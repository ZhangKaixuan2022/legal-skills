# 劳动争议证据体系Skill

## 功能概述

本Skill是劳动争议诉讼Agent的核心子Skill之一，专注于劳动争议案件的特殊证据规则与举证责任分配。基于劳动争议"举证责任倒置"的特殊规则，系统化进行举证责任分析、证据收集清单生成、证据效力评估、特殊保护检测、规章制度合法性审查、劳动者证据完整性检查和用人单位证据缺口评估。

### 核心功能

1. **举证责任分配**：依据《劳动争议调解仲裁法》第6条及司法解释，自动区分劳动者与用人单位的举证责任
2. **证据收集清单**：按12类劳动争议类型生成定制化证据清单
3. **证据效力评估**：三级证据效力评估体系（一级/二级/三级）
4. **特殊保护检测**：自动检测"三期"女职工、工伤职工、医疗期职工、临近退休职工四类特殊保护情形
5. **规章制度审查**：三要件审查（民主程序、内容合法、公示告知）
6. **公司端证据缺口评估**：围绕事实依据、制度依据、程序依据、送达依据和工资工时材料检查用人单位风险敞口

### 在劳动争议流程中的位置

```
初步法律分析 → 劳动关系认定与经济补偿计算 → 【本Skill】 → 委托合同管理
                                                  ↓
                                         输出：举证责任分配表
                                         输出：证据收集清单
                                         输出：特殊保护检测报告
                                         输出：规章制度合法性审查意见
```

## 核心模块

### 1. 举证责任分配分析

#### 基本原则

**一般规则**：谁主张，谁举证（《劳动争议调解仲裁法》第6条前半句）

**特殊规则**：举证责任倒置（《劳动争议调解仲裁法》第6条后半句、最高院司法解释（一）第44条）

#### 用人单位法定举证义务（举证责任倒置）

| 举证事项 | 法律依据 | 证据形式 | 保存期限 |
|---------|---------|---------|---------|
| **考勤记录** | 《工资支付暂行规定》第6条 | 考勤表、打卡记录、签到表 | 至少2年 |
| **工资发放记录** | 《工资支付暂行规定》第6条 | 工资条、银行转账记录、工资表 | 至少2年 |
| **劳动合同文本** | 《劳动合同法》第50条 | 书面劳动合同 | 至少2年备查 |
| **规章制度制定程序** | 《劳动合同法》第4条 | 职工代表大会/全体职工讨论记录、工会协商记录 | 长期 |
| **规章制度公示告知** | 《劳动合同法》第4条 | 培训签到、员工手册签收、公告栏照片、邮件送达记录 | 长期 |
| **解除/终止通知** | 《劳动合同法》第43条、第50条 | 解除通知书、终止通知书、送达回证 | 长期 |
| **社保缴纳记录** | 《社会保险法》第58-60条 | 社保缴费凭证、减员表 | 长期 |
| **加班审批记录** | 司法解释（一）第42条 | 加班申请单、审批记录 | 至少2年 |

#### 劳动者基本举证责任

| 举证事项 | 证明目的 | 常见证据 |
|---------|---------|---------|
| **劳动关系存在** | 证明与用人单位存在劳动关系 | 劳动合同、工资支付凭证、工作证、招聘记录、考勤记录、同事证言 |
| **权利受侵害事实** | 证明权益受到侵害 | 解除通知书、欠薪证明、工伤认定材料、医疗记录 |
| **损失金额** | 证明索赔金额的合理性 | 工资流水、医疗费用票据、伤残鉴定结论 |
| **入职时间** | 计算工作年限 | 入职登记表、首月工资发放记录、社保起始缴纳记录 |
| **离职时间** | 确定劳动关系终止时点 | 离职交接单、最后工作日考勤记录 |

#### 特殊举证规则

**解除合法性举证责任**（最高院司法解释（一）第44条）：
- 用人单位作出开除、除名、辞退、解除劳动合同等决定引发的争议
- **由用人单位负举证责任**证明解除的合法性
- 需证明：事实依据 + 制度依据 + 程序依据

**加班事实举证责任**（司法解释（一）第42条）：
- 劳动者需证明存在加班事实（初步证据）
- 用人单位掌握加班证据不提供的，承担不利后果

**用人单位掌握证据不提供后果**（第6条）：
- 与争议事项有关的证据属于用人单位掌握管理的，用人单位应当提供
- 用人单位不提供的，应当承担不利后果

#### 劳动者代理证据完整性检查

代理劳动者时，先检查以下材料是否齐全；缺失时输出缺口，不得直接下确定结论：

| 事项 | 优先证据 | 缺失时的替代证据 |
|---|---|---|
| 劳动关系 | 劳动合同、社保记录、工资流水 | 工作证、考勤、工作群、邮件、工牌、同事证言 |
| 入离职时间 | 入职登记、离职证明、解除通知 | 首末月工资、社保起止、聊天记录 |
| 工资基数 | 前12个月工资流水、工资条 | 个税记录、银行摘要、奖金/提成记录 |
| 加班事实 | 考勤、加班审批、排班表 | 工作成果时间戳、聊天安排、门禁记录 |
| 解除事实 | 解除/终止通知、送达记录 | 聊天通知、录音、停用账号记录 |

#### 用人单位争议/合规证据清单

代理公司或处理公司端合规时，必须检查以下材料：

| 证据类别 | 具体材料 | 证明目的 |
|---|---|---|
| 基本事实 | 劳动者身份信息、岗位、入职时间、合同期限、工资结构、社保记录 | 确定劳动关系和计算基础 |
| 特殊状态 | 三期、工伤、医疗期、残疾、临近退休、近期投诉举报记录 | 识别解除/调岗/处罚高风险 |
| 合同文件 | 劳动合同、续签记录、补充协议、Offer | 证明权利义务和岗位/工资/工时约定 |
| 规章制度 | 员工手册、薪酬制度、考勤制度、绩效制度、奖惩制度 | 证明管理依据 |
| 制度生效 | 民主程序材料、工会/职代会记录、公示告知、培训签到、签收记录 | 证明规章制度可作为处理依据 |
| 工资工时 | 工资表、工资条、银行流水、考勤、排班、加班审批、特殊工时审批 | 证明工资支付和工时合规 |
| 处理程序 | 调查记录、违纪事实、员工申辩、工会通知、解除/处罚/调岗通知、送达回证 | 证明事实依据、制度依据、程序依据 |
| 一致性 | 同类员工处理记录、历史处分记录 | 证明处理尺度一致，降低选择性执法风险 |

输出公司端证据缺口时，按“事实依据 / 制度依据 / 程序依据 / 送达依据 / 计算依据”分类列示，并标注风险等级。

### 2. 按12类争议类型生成证据收集清单

根据《劳动争议调解仲裁法》第2条及司法实践，劳动争议分为12类主要纠纷类型：

| 争议类型 | 核心证据要点 | W角色策略 | E角色策略 |
|---------|-------------|----------|----------|
| 确认劳动关系 | 劳动合同、工资流水、社保记录、工作证 | 强调三性从属 | 论证劳务关系 |
| 追索劳动报酬 | 工资条、银行流水、考勤记录、加班审批 | 举证责任倒置 | 已付工资凭证 |
| 经济补偿金 | 解除通知、工资流水、入职登记、离职交接 | 最大化年限基数 | 挑战计算依据 |
| 违法解除赔偿 | 解除通知、规章制度、违纪证据、工会通知 | 程序/实体违法 | 三依据论证 |
| 双倍工资 | 入职时间证明、工资记录、未签合同证据 | 时效逐月计算 | 劳动者拒签 |
| 工伤保险待遇 | 工伤认定书、鉴定结论、医疗票据、工资流水 | 最大化待遇 | 程序合规 |
| 加班费 | 加班审批、考勤记录、工资条、工时条款 | 初步证据即可 | 不存在加班 |
| 年休假工资 | 社保记录、入职登记、考勤记录、休假安排 | 用人单位举证 | 已安排休假 |
| 社会保险 | 社保记录、劳动合同、工资流水、损失证明 | 损失赔偿 | 已缴纳 |
| 竞业限制 | 竞业协议、工资流水、补偿金支付、离职证明 | 主张补偿 | 主张违约 |
| 培训服务期 | 培训协议、费用凭证、培训记录、差旅凭证 | 费用真实性 | 服务期约定 |
| 职业病/工伤认定前 | 诊断证明、工伤申请、医疗记录、工资流水 | 停工留薪期 | 程序配合 |

#### 主要争议类型证据要点

**确认劳动关系纠纷**：
- 核心证据：劳动合同、工资支付凭证、社保缴纳记录
- 辅助证据：工作证、考勤记录、入职登记表、工作邮件、同事证言
- W策略：强调三性从属（人身/经济/组织）
- E策略：论证为劳务关系或合作关系

**追索劳动报酬纠纷**：
- 核心证据：劳动合同（工资条款）、工资条、银行工资流水、考勤记录
- 辅助证据：加班审批单、绩效考核记录、欠薪确认单
- W策略：充分利用举证责任倒置，要求单位提供工资发放记录
- E策略：提供已足额支付工资的证据

**违法解除赔偿纠纷**：
- 核心证据：解除通知书、规章制度、违纪证据、工会通知记录
- 辅助证据：送达回证、劳动合同、工资流水
- W策略：重点攻击解除理由不成立或程序违法
- E策略：三依据论证（事实依据+制度依据+程序依据）

**工伤保险待遇纠纷**：
- 核心证据：工伤认定决定书、劳动能力鉴定结论、医疗费用票据
- 辅助证据：病历、工资流水、停工留薪期确认
- W策略：最大化待遇主张
- E策略：程序合规、待遇核算准确

**经济补偿金纠纷**：
- 核心证据：解除通知书、工资流水（前12个月）、入职登记表、离职交接单
- 辅助证据：年终奖发放记录、社保缴纳记录、劳动合同
- W策略：最大化月工资基数和工作年限，质疑解除合法性
- E策略：证明解除合法、挑战工资基数计算

**双倍工资纠纷**：
- 核心证据：入职时间证明、劳动合同签订时间、工资发放记录
- 辅助证据：社保起始缴纳记录、入职登记表、工牌/工作证
- W策略：证明用人单位故意不签合同，时效逐月计算
- E策略：证明劳动者拒绝签订或已签订但劳动者未交回

**加班费纠纷**：
- 核心证据：考勤记录、加班审批单、工资条（加班费项目）
- 辅助证据：工作邮件时间戳、打卡记录、排班表
- W策略：初步证据证明加班事实，用人单位需提供考勤记录
- E策略：证明不存在加班或已足额支付加班费

**竞业限制纠纷**：
- 核心证据：竞业限制协议、经济补偿支付记录、离职证明
- 辅助证据：新单位劳动合同、社保转移记录、工商登记信息
- W策略（主张补偿）：证明履行义务、单位未按时支付补偿
- E策略（主张违约）：证明劳动者违反竞业限制义务

### 3. 证据效力三级评估

| 证据等级 | 证明力 | 典型证据 | 特点 |
|---------|-------|---------|------|
| **一级证据** | 最强 | 劳动合同、仲裁裁决书、社保证明、公证书、工伤认定书、银行流水 | 直接证明，来源权威，无需补强 |
| **二级证据** | 较强 | 工资条、考勤表、工作证、入职登记表、工作邮件、社保减员表 | 间接证明，需配合使用 |
| **三级证据** | 需补强 | 微信聊天、电话录音、证人证言、照片、视频、网页截图 | 辅助证明，必须补强 |

### 5. 规章制度合法性审查三要件

| 要件 | 审查内容 | 合格标准 | 常见瑕疵 |
|-----|---------|---------|---------|
| **民主程序** | 职工代表大会讨论、与工会协商 | 有会议记录、签到表、协商记录 | 未召开、仅管理层决定 |
| **内容合法** | 不违反强制性规定、保障基本权利 | 过罚相当、条款明确 | 末位淘汰、放弃社保 |
| **公示告知** | 培训签到、员工手册签收、合同附件 | 能证明劳动者知悉 | 未公示、方式不当 |

### 6. 证据保全与固定策略

| 证据类型 | 保全方式 | 操作要点 | 法律效力 |
|---------|---------|---------|---------|
| **电子数据** | 公证保全 | 原始设备、完整操作过程 | 最强，直接采信 |
| **微信记录** | 公证截图+区块链存证 | 保留原始聊天、头像、账号信息 | 强，需辅助验证 |
| **录音录像** | 公证转录+原件保存 | 保持完整、禁止剪辑 | 需辅助证据 |
| **书证复印** | 核对原件后公证 | 标注"与原件核对一致" | 中等，需质证 |
| **证人证言** | 书面证词+身份证明 | 证人应知悉案件事实 | 弱，需出庭 |

**证据固定优先级**：
1. 公证保全（效力最强）
2. 区块链存证（时间戳可验证）
3. 司法鉴定机构存证
4. 律师见证
5. 自行保存（效力最弱）

### 7. 举证策略W/E角色差异

**W（劳动者代理）举证策略**：
| 策略要点 | 具体措施 |
|---------|---------|
| 充分利用倒置 | 明确用人单位法定举证事项，主动申请其提供 |
| 降低举证标准 | 提供初步证据即可，如加班费仅需证明加班事实存在 |
| 申请调查令 | 用人单位拒不提供证据时，申请仲裁机构调取 |
| 证据保全 | 对易灭失电子证据及时申请保全 |

**E（用人单位代理）举证策略**：
| 策略要点 | 具体措施 |
|---------|---------|
| 证据充分完整 | 主动提供考勤、工资、规章制度等证据 |
| 程序合规证明 | 重点证明规章制度"三要件"合法 |
| 解除依据论证 | 事实依据+制度依据+程序依据三位一体 |
| 反驳劳动者主张 | 质疑对方证据真实性、关联性、合法性 |

## 注意事项

### 证据收集注意事项

1. **合法性原则**：取证方式必须合法，不得采用非法手段获取证据
2. **及时性原则**：对易灭失证据及时采取保全措施
3. **完整性原则**：收集证据应全面，避免选择性收集
4. **原件优先**：尽量收集原件，复印件需核对无误
5. **原始载体**：电子数据注意保存原始载体（手机、电脑等）

### 举证责任倒置的适用

1. **准确识别倒置情形**：不是所有劳动争议都适用举证责任倒置
2. **劳动者仍有初步举证责任**：倒置不等于劳动者无需举证
3. **用人单位不提供证据的后果**：承担不利后果，但不等于直接败诉
4. **结合其他证据综合判断**：即使用人单位不提供证据，也需结合其他证据认定事实

### 特殊保护的适用边界

1. **保护不是绝对**：严重违纪等情形仍可解除
2. **注意地方差异**：各地对特殊保护的具体执行可能有差异
3. **时效问题**：特殊保护不影响仲裁时效的计算
4. **证据要求**：主张特殊保护需提供相应证据（如怀孕证明、工伤认定等）

### 规章制度审查的实务要点

1. **区分制定时间**：2008年前后的规章制度适用不同标准
2. **注意行业特殊规定**：部分行业有特殊的规章制度要求
3. **个案分析**：即使规章制度合法，具体适用也需个案分析
4. **补救可能性**：存在瑕疵的规章制度可能通过补救措施完善
