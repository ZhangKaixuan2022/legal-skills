# 证据效力三级评估框架

## 评估维度

对每份证据从四个维度进行评估：
1. **关联性**：与待证事实的关联程度
2. **合法性**：证据来源和形式是否合法
3. **真实性**：证据内容是否真实可靠
4. **证明力**：证据对待证事实的证明强度

---

## 一级证据（直接证据，证明力最强）

| 证据类型 | 证明力说明 | 适用场景 |
|---------|-----------|---------|
| 书面劳动合同 | 直接证明劳动关系及权利义务 | 确认劳动关系、追索报酬 |
| 仲裁裁决书/法院判决书 | 生效法律文书，具有既判力 | 已裁决/判决事项 |
| 社保缴纳证明 | 官方记录，证明力强 | 确认劳动关系、社保纠纷 |
| 公证书 | 经公证的证据具有推定真实效力 | 电子数据、证人证言 |
| 工伤认定决定书 | 行政确认行为，具有公信力 | 工伤保险待遇 |
| 劳动能力鉴定结论 | 专业机构出具 | 伤残等级相关 |
| 银行工资流水（盖章） | 金融机构出具，客观性强 | 工资标准、欠薪 |

### 一级证据特点
- 直接证明待证事实
- 来源权威、形式规范
- 一般不需要补强即可采信
- 对方质证难度大

---

## 二级证据（间接证据，证明力较强）

| 证据类型 | 证明力说明 | 补强建议 |
|---------|-----------|---------|
| 工资条/工资表 | 证明工资构成，但易被质疑 | 配合银行流水使用 |
| 考勤表（有签名） | 证明出勤情况 | 需有劳动者确认 |
| 用人单位盖章文件 | 代表单位意志 | 注意公章真实性 |
| 工作证/工牌 | 证明身份隶属关系 | 配合其他证据 |
| 入职登记表 | 证明入职事实 | 需有单位盖章或签字 |
| 培训记录 | 证明接受单位管理 | 配合签到表使用 |
| 工作邮件（公司邮箱） | 证明工作内容 | 保存原始载体 |
| 社保减员表 | 证明离职事实 | 官方记录 |
| 解除/终止通知书 | 证明解除意思表示 | 需有送达证据 |

### 二级证据特点
- 间接证明待证事实
- 一般需要配合其他证据使用
- 可能需要补强才能采信
- 对方可能提出质疑

---

## 三级证据（辅助证据，需补强）

| 证据类型 | 证明力说明 | 补强方式 |
|---------|-----------|---------|
| 微信/QQ聊天记录 | 易被篡改，真实性存疑 | 公证保全、原始载体 |
| 电话录音 | 合法性边界需注意 | 内容清晰、未剪辑 |
| 短信记录 | 可伪造性较高 | 运营商记录佐证 |
| 证人证言 | 主观性强，可能有利害关系 | 出庭作证、多证印证 |
| 照片 | 可编辑，证明力弱 | 原始存储设备、时间戳 |
| 视频录像 | 需证明来源合法 | 原始存储设备 |
| 网页截图 | 易被修改 | 公证保全 |
| 电子邮件（个人邮箱） | 发件人身份需确认 | 配合其他证据 |

### 三级证据特点
- 辅助证明待证事实
- 单独使用难以认定事实
- 必须补强才能采信
- 对方容易提出质疑

## 证据收集注意事项

1. **合法性原则**：取证方式必须合法，不得采用非法手段获取证据
2. **及时性原则**：对易灭失证据及时采取保全措施
3. **完整性原则**：收集证据应全面，避免选择性收集
4. **原件优先**：尽量收集原件，复印件需核对无误
5. **原始载体**：电子数据注意保存原始载体（手机、电脑等）

## 电子证据保全方式

| 保全方式 | 适用情形 | 证明力 |
|---------|---------|-------|
| 公证保全 | 重要电子数据 | 最强 |
| 区块链存证 | 电子合同、交易记录 | 较强 |
| 时间戳认证 | 电子文件 | 较强 |
| 原始载体保存 | 聊天记录、邮件 | 中等 |
| 截图保存 | 临时保存 | 较弱 |
