# 四类特殊保护检测详细标准

## 一、三期女职工保护检测

### 检测触发条件
案件事实中涉及女性劳动者 + 解除/终止争议

### 检测项

| 检测项目 | 检测内容 | 法律依据 | 风险提示 |
|---------|---------|---------|---------|
| 是否处于孕期 | 根据怀孕时间、预产期判断 | 《劳动合同法》第42条第4项 | 孕期不得解除 |
| 是否处于产期 | 根据分娩时间判断 | 《劳动合同法》第42条第4项 | 产假期间不得解除 |
| 是否处于哺乳期 | 根据婴儿出生时间计算（1年内） | 《劳动合同法》第42条第4项 | 哺乳期内不得解除 |
| 解除行为是否违法 | 如在三期内解除，原则上违法 | 《劳动合同法》第42条 | 除非劳动者严重违纪 |
| 产假工资是否保障 | 检查产假期间工资发放情况 | 《女职工劳动保护特别规定》第8条 | 产假工资照发 |
| 哺乳时间是否保障 | 检查每日1小时哺乳时间 | 《女职工劳动保护特别规定》第9条 | 不得扣工资 |

### 保护规则
- 孕期、产期、哺乳期内，用人单位不得依据《劳动合同法》第40条（非过失性解除）、第41条（经济性裁员）解除劳动合同
- 劳动合同期满，应当续延至相应情形消失时终止
- 违法解除的，劳动者可要求继续履行或2N赔偿

### 输出模板

```
三期女职工保护检测报告

一、检测基本信息
   - 客户姓名：
   - 性别：女
   - 检测日期：

二、三期状态检测
   | 检测项目 | 检测结果 | 检测依据 | 法律状态 |
   |---------|---------|---------|---------|
   | 孕期 | 是/否 | 怀孕时间： | 受保护/不受保护 |
   | 产期 | 是/否 | 分娩时间： | 受保护/不受保护 |
   | 哺乳期 | 是/否 | 婴儿出生： | 受保护/不受保护 |

三、解除/终止合法性分析
   - 解除时间：
   - 三期状态：
   - 解除依据：
   - 合法性判断：合法/违法
   - 法律后果：...

四、保护建议
   1. ...
   2. ...

五、法律依据
   - 《劳动合同法》第42条、第45条
   - 《女职工劳动保护特别规定》第5-9条
```

## 二、工伤职工保护检测

### 检测触发条件
案件事实中涉及工伤认定或工伤期间解除

### 检测项

| 检测项目 | 检测内容 | 法律依据 | 风险提示 |
|---------|---------|---------|---------|
| 工伤认定是否完成 | 是否有工伤认定决定书 | 《工伤保险条例》第14-20条 | 未认定先申请认定 |
| 伤残等级 | 劳动能力鉴定结论等级 | 《工伤保险条例》第22条 | 1-10级不同待遇 |
| 停工留薪期是否到期 | 根据诊断证明和鉴定结论 | 《工伤保险条例》第33条 | 停工留薪期内不得解除 |
| 解除行为是否违法 | 停工留薪期内解除原则上违法 | 《劳动合同法》第42条第2项 | 除非劳动者严重违纪 |
| 工伤保险待遇是否落实 | 医疗费、伤残补助金等 | 《工伤保险条例》第30-37条 | 用人单位未参保需承担 |

### 保护规则
- 患职业病或者因工负伤并被确认丧失或者部分丧失劳动能力的，用人单位不得依据第40条、第41条解除
- 停工留薪期内，原工资福利待遇不变
- 1-4级伤残：保留劳动关系，退出工作岗位
- 5-6级伤残：保留劳动关系，安排适当工作
- 7-10级伤残：劳动合同期满终止或劳动者提出可解除

### 输出模板

```
工伤职工保护检测报告

一、检测基本信息
   - 客户姓名：
   - 工伤时间：
   - 检测日期：

二、工伤认定状态
   - 工伤认定决定书：有/无
   - 认定结果：认定为工伤/不认定为工伤/认定中
   - 认定时间：

三、劳动能力鉴定
   - 鉴定结论：有/无
   - 伤残等级：X级
   - 鉴定时间：

四、停工留薪期检测
   - 停工留薪期起止：
   - 是否到期：是/否
   - 解除时间：
   - 解除时停工留薪期状态：

五、解除/终止合法性分析
   - 合法性判断：合法/违法
   - 法律后果：...

六、法律依据
   - 《劳动合同法》第42条
   - 《工伤保险条例》第14-37条
```

## 三、医疗期内职工保护检测

### 检测触发条件
案件事实中涉及劳动者患病/非因工负伤 + 解除争议

### 医疗期长度计算

| 实际工作年限 | 本单位工作年限 | 医疗期 |
|-------------|---------------|-------|
| 10年以下 | 5年以下 | 3个月 |
| 10年以下 | 5年以上 | 6个月 |
| 10年以上 | 5年以下 | 6个月 |
| 10年以上 | 5-10年 | 9个月 |
| 10年以上 | 10-15年 | 12个月 |
| 10年以上 | 15-20年 | 18个月 |
| 10年以上 | 20年以上 | 24个月 |

### 保护规则
- 医疗期内，用人单位不得依据第40条（非过失性解除）、第41条（经济性裁员）解除
- 劳动合同期满，应当续延至医疗期满时终止
- 医疗期满解除需满足：不能从事原工作 + 不能从事另行安排的工作 + 提前30日通知/N+1

## 四、距法定退休年龄不足5年职工保护检测

### 检测触发条件
案件事实中涉及劳动者年龄信息 + 解除争议

### 检测项

| 检测项目 | 检测内容 | 法律依据 |
|---------|---------|---------|
| 劳动者当前年龄 | 根据身份证信息 | - |
| 距法定退休年龄 | 男60岁/女干部55岁/女工人50岁 | 《劳动合同法》第42条第5项 |
| 解除依据 | 是否依据第40条或第41条 | 《劳动合同法》第42条第5项 |

### 保护规则
- 在本单位连续工作满15年，且距法定退休年龄不足5年的，用人单位不得依据第40条（非过失性解除）、第41条（经济性裁员）解除劳动合同
- 可以依据第39条（过失性解除）解除
- 劳动合同期满可以终止（但需注意地方特殊规定）
