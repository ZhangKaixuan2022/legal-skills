# 审理报告排版规范

> **来源**：审理报告Skill - 审理报告专属排版规范
> **用途**：法院内部审理报告的表格化排版标准
> **说明**：审理报告排版与判决书基本一致，使用仿宋三号/行距28磅。审理报告为内部文书，无法院名称、无审判人员签名等外部文书要素。

## 四、编号层级规范

审理报告使用以下层级编号体系：

| 层级 | 格式 | 使用场景 |
|-----|------|---------|
| 一级 | 一、二、三、四……（中文数字加顿号） | 七大区块标题 |
| 二级 | （一）（二）（三）……（中文数字加括号） | 区块内分项（如证据分析、事实认定） |
| 三级 | 1. 2. 3.……（阿拉伯数字加句号） | 要点列举 |
| 四级 | （1）（2）（3）…… | 子要点列举 |

**重要规则**：
- 七大区块标题使用"一、""二、"……"七、"格式
- 二级标题使用"（一）""（二）"格式
- 争议焦点编号可使用"争议焦点一""争议焦点二"等文字标注
- 附件标题使用"附件："前缀

## 五、标题格式规范

### 5.1 案件名称行

案件名称分两行排列，居中：

```
关于原告XXX、XXX与被告XXX、XXX及第三人XXX XXX纠纷一案

                    审 理 报 告
```

### 5.2 格式说明

| 元素 | 格式 |
|-----|------|
| 案件名称第一行 | "关于原告XX、XX与被告XX、XX及第三人XX XX纠纷一案"，仿宋三号，居中，列出全部当事人 |
| "审 理 报 告" | 宋体二号加粗，居中，字间有空格，与第一行空一行 |
| 案号 | 仿宋三号，右对齐，位于"审 理 报 告"下方 |

---

## 六、法条引用格式

| 元素 | 格式 |
|-----|------|
| 法律名称 | 用书名号《》包裹完整名称 |
| 条文引用 | "第×条"或"第×条第×款"，多个条文用顿号分隔 |
| 引用示例 | 依照《中华人民共和国民法典》第五百零九条、第五百七十七条、第五百八十五条规定 |

**禁止**：在法条编号后加括号注释，如"第五百零九条（全面履行原则）"，只写条文编号。

## 七、与民事判决书排版的主要差异

| 对比项 | 审理报告 | 民事判决书 |
|-------|---------|-----------|
| 法院名称 | 无（内部文书） | 有（方正小标宋二号居中） |
| 文书名称 | "审理报告"（宋体二号加粗居中） | "民事判决书"（宋体二号加粗居中） |
| 标题格式 | 案件名称分两行："关于原告XX、XX与被告XX纠纷一案"/"审 理 报 告" | 法院名称+"民事判决书" |
| 尾部信息 | 无（无审判人员签名、无上诉告知） | 有（审判人员签名、上诉告知、院印、书记员） |
| 一级标题 | 仿宋三号不加粗 | 仿宋三号（正文样式） |
| 二级标题 | 仿宋三号不加粗 | 仿宋三号（正文样式） |
| 语体风格 | 分析论证语体 | 裁判宣告语体 |
| 独有区块 | 争议焦点评价、需说明问题、处理意见（含法律适用分析）、附件：类案检索 | 判项、诉讼费分担、上诉告知 |
| 正文字体 | 仿宋三号 | 仿宋三号 |
| 行距 | 28磅固定值 | 28磅固定值 |
| 页边距 | 上2.8/下2.6/左2.6/右2.5 | 上2.8/下2.6/左2.6/右2.5 |
| 页码 | 无 | 无 |
