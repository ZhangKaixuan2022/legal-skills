# 庭审流程告知书

> 适用场景：第四次会见（庭前辅导会见）使用，向被告人说明庭审程序和注意事项
> 使用方式：会见时向被告人讲解，并留存副本供其参考
> 保密等级：【会见材料】——可向被告人提供

## 一、庭审基本流程

### 1.1 庭审阶段概览

```
┌─────────────────────────────────────────────────────────────┐
│                        开 庭 准 备                           │
│  (核实当事人身份、告知权利、询问回避申请)                      │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        法 庭 调 查                           │
│  (起诉书宣读→被告人陈述→讯问发问→质证→举证)                   │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        法 庭 辩 论                           │
│  (公诉人发言→辩护人发言→被告人自行辩护→多轮辩论)              │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      被告人最后陈述                          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     休庭或当庭宣判                           │
└─────────────────────────────────────────────────────────────┘
```

## 二、各阶段详细说明

### 2.1 开庭准备阶段

#### 2.1.1 书记员宣布法庭纪律
- 请起立迎接合议庭入庭
- 认真听取法庭纪律要求

#### 2.1.2 核实身份
审判长会核实你的身份信息，请如实回答：
- 姓名、出生年月日
- 民族、文化程度
- 户籍地、住址
- 是否收到起诉书副本
- 收到起诉书的时间

#### 2.1.3 告知权利
审判长会告知你在庭审中享有的权利：
- 申请回避的权利
- 进行辩护的权利
- 申请证人出庭的权利
- 最后陈述的权利
- 核对庭审笔录的权利

#### 2.1.4 回避申请
审判长会询问你是否申请回避。
**回复建议**：[根据案件情况，告知被告人如何回答]

---

### 2.2 法庭调查阶段

#### 2.2.1 宣读起诉书
公诉人会宣读起诉书。请认真听取，不要打断。

#### 2.2.2 被告人陈述
审判长会问你对起诉书指控的犯罪事实有无异议。
**回复建议**：
- 如果有异议：说"对起诉书指控的部分事实有异议"，然后简要说明异议内容
- 如果没有异议：说"没有异议"
- **不要在这个环节长篇大论**，详细意见在后续环节表达

#### 2.2.3 公诉人讯问
公诉人会对你进行讯问。
**注意事项**：
- 认真听问题，只回答问题本身
- 不清楚的问题可以请公诉人解释
- 记不清的如实说"记不清"，不要猜测
- 不要主动陈述与问题无关的内容
- 保持冷静，不要情绪激动

#### 2.2.4 辩护人发问
我会对你进行发问。
**回复要点**：[根据案件情况，告知被告人发问要点]

#### 2.2.5 质证环节
公诉人会出示证据。我会发表质证意见，你也有权发表意见。
**注意事项**：
- 主要由辩护人发表质证意见
- 如果你有补充，可以举手向审判长申请发言
- 如果证据内容与事实不符，简要指出即可

#### 2.2.6 举证环节
辩护人可能会向法庭出示证据。
**配合事项**：
- 如果需要你说明证据来源或内容，请如实说明
- 按照我们事先商量的要点陈述

---

### 2.3 法庭辩论阶段

#### 2.3.1 公诉人发言
公诉人会发表公诉意见。请认真听取，不要打断。

#### 2.3.2 辩护人发言
我会发表辩护意见。请认真听取。

#### 2.3.3 被告人自行辩护
审判长会问你是否有补充意见。
**建议**：[根据案件情况，告知被告人是否需要自行辩护及要点]

#### 2.3.4 多轮辩论
可能会有第二轮、第三轮辩论。
**注意事项**：
- 以辩护人发言为主
- 如需表态，简明扼要

---

### 2.4 被告人最后陈述

这是你在法庭上最后一次发言机会，由你独立完成，辩护人不能代替。

**陈述要点建议**：
1. [要点一]
2. [要点二]
3. [要点三]

**时间控制**：建议控制在3-5分钟内

**态度建议**：真诚、恳切，适度表达悔意（如认罪案件）

---

## 三、庭审注意事项

### 3.1 着装仪表
- 穿戴整洁得体
- 庭审期间可能会穿号服，这是程序要求，不影响定罪量刑

### 3.2 言行举止
- 称呼审判长为"审判长"，公诉人为"公诉人"
- 发言前举手示意，得到许可后再发言
- 站立回答问题，回答完毕说"回答完毕"
- 保持情绪稳定，不要与公诉人争吵
- 不要打断他人发言

### 3.3 与辩护人沟通
- 如需与我沟通，可以举手向审判长说明
- 不要在庭上私下交头接耳
- 休庭期间可以进行简短沟通

### 3.4 突发情况处理

| 情况 | 处理方式 |
|------|----------|
| 记不清事实细节 | 如实说"记不清"，不要猜测或编造 |
| 听不懂问题 | 请审判长或公诉人重复或解释 |
| 身体不适 | 举手向审判长说明情况 |
| 情绪激动 | 深呼吸，请求休息片刻 |
| 公诉人提出新证据 | 保持冷静，由辩护人应对 |
| 被法官打断 | 服从法庭指挥，不要顶撞 |

---

## 四、禁止事项

- **禁止**：辱骂、威胁公诉人或法官
- **禁止**：扰乱法庭秩序
- **禁止**：未经许可擅自发言
- **禁止**：与旁听人员交流
- **禁止**：故意说谎（可能构成妨害作证）
- **禁止**：庭上翻供前不与辩护人沟通

---

## 五、认罪认罚案件特别说明

【如适用】

如果你已经签署了认罪认罚具结书，庭审中：
1. 审判长会核实你是否自愿认罪认罚
2. 公诉人会说明量刑建议
3. 辩护人会确认是否同意量刑建议
4. 你需要当庭确认认罪态度

**回复建议**：
- 关于自愿性：说"我是自愿认罪认罚的"
- 关于量刑建议：说"我同意公诉机关的量刑建议"或"我希望法庭从轻判处"

## 六、庭后事项说明

### 6.1 庭审笔录
- 庭审结束后，法警会让你核对庭审笔录
- 认真阅读，如有错误可以要求更正
- 确认无误后签名

### 6.2 庭后会见
- 庭审结束后我会尽快安排会见
- 会见时复盘庭审情况

### 6.3 判决等待
- 普通程序案件审限为3-6个月
- 判决结果出来后我会第一时间告知你
