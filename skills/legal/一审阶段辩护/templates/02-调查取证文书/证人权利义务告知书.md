# 证人权利义务告知书

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 证人权利义务告知书 |
| 使用阶段 | 一审阶段辩护人调查证人前 |
| 使用目的 | 向证人告知其依法享有的权利和应当履行的义务 |
| 法律依据 | 《刑事诉讼法》第六十二条至第六十四条 |

---

## 文书格式

```
                    证人权利义务告知书

被指控______罪一案

【案号】____________第______号

证人：____________

    ____________律师事务所____________律师，系____________被指控____________罪一案被告人____________的辩护人。根据《中华人民共和国刑事诉讼法》的相关规定，现就辩护人向您调查取证的有关事项，告知您依法享有的权利和应当履行的义务如下：

一、您依法享有的权利

    1. **自愿接受调查的权利**：您有权自愿决定是否接受辩护人的调查。辩护人向您调查取证，应当经您同意。

    2. **如实陈述的权利**：您有权就您所知道的案件情况如实陈述。

    3. **核对笔录的权利**：调查结束后，您有权阅读或请辩护人向您宣读调查笔录。如有遗漏或差错，您有权提出补充或更正。

    4. **获得保护的权利**：根据《中华人民共和国刑事诉讼法》第六十四条之规定，对于危害国家安全犯罪、恐怖活动犯罪、黑社会性质的组织犯罪、毒品犯罪等案件，证人、鉴定人、被害人因在诉讼中作证，本人或者其近亲属的人身安全面临危险的，人民法院、人民检察院和公安机关应当采取保护措施。

    5. **获得补助的权利**：证人因履行作证义务而支出的交通、住宿、就餐等费用，应当给予补助。

    6. **使用本民族语言文字的权利**：您有权使用本民族语言文字进行陈述。

二、您应当履行的义务

    1. **如实作证的义务**：根据《中华人民共和国刑事诉讼法》第六十二条之规定，凡是知道案件情况的人，都有作证的义务。您应当如实向辩护人陈述您所知道的与案件有关的事实，不得隐瞒、歪曲事实。

    2. **保密的义务**：对于调查过程中知悉的案件情况，您应当予以保密，不得向无关人员透露。

三、法律责任

    根据《中华人民共和国刑法》第三百零五条之规定，在刑事诉讼中，证人对与案件有重要关系的情节，故意作虚假证明，意图陷害他人或者隐匿罪证的，处三年以下有期徒刑或者拘役；情节严重的，处三年以上七年以下有期徒刑。

    本告知书一式两份，一份交证人，一份存卷。

    证人签名确认：本人已阅读/听取宣读上述内容，知悉并理解本人依法享有的权利和应当履行的义务。

    证人签名：____________        日期：____年____月____日

    辩护人签名：____________      日期：____年____月____日
```

---

## 使用说明

1. **使用时机**：在向证人调查取证前，应当先向证人出示并宣读本告知书。

2. **告知要点**：
   - 证人有自愿接受调查的权利
   - 证人有如实陈述的义务
   - 作虚假证明的法律后果

3. **注意事项**：
   - 告知书应当由证人签名确认
   - 证人不识字的，应当向其宣读并由其按指印
   - 一份交证人，一份存卷

4. **执业风险提示**：
   - 不得威胁、引诱证人作证
   - 不得指使证人作伪证
   - 调查过程应当客观、中立
