# 书面辩护意见模板

> 💡 **实务参考**：本文书撰写时可检索 `reference/刑事审判参考案例索引.md` 中相关模块案例，选择裁判要旨契合的案例作为书面辩护论据支撑。特别是法律适用部分的"类案参考"，可援引《刑事审判参考》高价值案例增强说服力。

> 适用场景：休庭后3日内提交的补充书面辩护意见
> 法律依据：《刑事诉讼法》第二百条、《人民法院办理刑事案件庭前会议规程》

---

## 一、文书格式规范

### 1.1 排版参数
| 项目 | 规范 |
|------|------|
| 纸张 | A4纵向 |
| 页边距 | 上2.54cm 下2.54cm 左3.18cm 右3.18cm |
| 标题 | 黑体 三号 加粗 居中 |
| 一级标题 | 黑体 小三号 |
| 二级标题 | 宋体 四号 加粗 |
| 正文 | 宋体 四号 |
| 行距 | 固定值27磅 |
| 页码 | 底部居中 |

### 1.2 首部段标准
```
书面辩护意见

案号：（20XX）X刑初XXX号

尊敬的审判长、审判员：

    广东广和（长春）律师事务所接受被告人[姓名]（或其近亲属[姓名]）的委托，
指派[律师姓名]律师担任被告人[姓名]涉嫌[罪名]一案一审阶段的辩护人。

    辩护人于[庭审日期]参加了本案的公开审理，现根据庭审查明的事实
和证据，依据相关法律规定，就本案发表补充辩护意见如下：
```

## 二、正文结构（五段论）

### 2.1 案件基本情况概述

**【写作要点】**
- 简述起诉书指控的基本事实
- 列明辩护人的基本辩护观点
- 明确本意见书的主要论证方向

**【参考模板】**
```
一、案件基本情况

    [XXXX年XX月XX日]，[XX]人民检察院以[案号]号起诉书指控被告人
[姓名]犯[罪名]罪，向贵院提起公诉。

    起诉书指控：[概述起诉书认定的主要犯罪事实]

    辩护人认为：[概述辩护观点——无罪/轻罪/从轻/缓刑]

    以下辩护人将从[XX]个方面进行详细论述：
```

### 2.2 事实认定部分

**【写作要点】**
- 针对庭审查明的事实进行梳理
- 与起诉书指控的事实进行对比
- 指出控方事实认定的问题

**【参考模板】**
```
二、关于本案事实认定的意见

（一）庭审查明的事实

    通过庭审调查，辩护人认为以下事实已经查明：
    1. [已查明事实一]
    2. [已查明事实二]
    3. [已查明事实三]

（二）起诉书认定的事实存在以下问题

    1. [事实问题一]：[具体说明]
    2. [事实问题二]：[具体说明]
    3. [事实问题三]：[具体说明]

（三）庭审中新发现的有利事实

    1. [新发现事实一]
    2. [新发现事实二]
```

### 2.3 证据分析部分

**【写作要点】**
- 对庭审质证情况进行总结
- 分析控方证据的问题（三性）
- 强调辩方证据的证明力

**【参考模板】**
```
三、关于本案证据的补充意见

（一）控方证据存在的问题

    1. 关于[证据名称]
       - 合法性问题：[具体问题]
       - 真实性问题：[具体问题]
       - 关联性问题：[具体问题]

    2. 关于[证据名称]
       [同上分析]

（二）辩方证据的证明力

    辩护人在庭审中提交的证据[证据名称]，能够证明：
    1. [证明事项一]
    2. [证明事项二]

（三）证据不足的事项

    根据"疑罪从无"原则，以下事项因证据不足，不应认定：
    1. [事项一]
    2. [事项二]
```

### 2.4 法律适用部分

**【写作要点】**
- 分析罪名定性问题
- 论述量刑情节
- 引用法律依据

**【参考模板】**
```
四、关于本案法律适用的意见

（一）关于罪名定性

    1. [定性意见]
    2. 法律依据：[具体法条]
    3. 类案参考：[案例名称/案号]

（二）关于量刑情节

    1. 法定从轻/减轻情节
       (1) [情节一]：[具体说明]
       (2) [情节二]：[具体说明]

    2. 酌定从轻情节
       (1) [情节一]：[具体说明]
       (2) [情节二]：[具体说明]

    3. 量刑建议
       [具体建议]
```

### 2.5 结论部分

**【参考模板】**
```
五、辩护意见总结

    综上所述，辩护人认为：

    1. [核心辩护观点一]
    2. [核心辩护观点二]
    3. [核心辩护观点三]

    恳请合议庭依法采纳辩护人意见，[具体请求——宣告无罪/
从轻处罚/适用缓刑/改变罪名等]。

    以上意见，敬请审查采纳！

                                    辩护人：广东广和（长春）律师事务所
                                            潘睿律师
                                    日  期：[年 月 日]

附件：
1. [附件名称一]
2. [附件名称二]
```

## 三、与庭审辩护意见的区别

| 对比项 | 庭审辩护意见 | 书面辩护意见 |
|--------|--------------|--------------|
| 提交时间 | 开庭前或当庭宣读 | 休庭后3日内 |
| 内容侧重 | 预设的辩护方案 | 针对庭审情况调整 |
| 表述风格 | 适合口头表达 | 书面论证，可更详细 |
| 证据引用 | 概括引用 | 可详细引用庭审笔录 |
| 篇幅 | 一般控制在30分钟内 | 不受限制 |

## 四、提交注意事项

### 4.1 时限要求
- 休庭后3日内提交
- 如需延期，应提前向合议庭说明

### 4.2 提交方式
- 书面提交正本1份，副本按对方当事人人数提交
- 可同时提交电子版

### 4.3 送达确认
- 要求法院出具《接收材料清单》
- 保留提交记录备查

---

## 五、常见问题处理

### 5.1 庭审中发现新的辩护要点
- 在书面意见中详细论述
- 如需补充证据，另行提交调取证据申请

### 5.2 公诉人庭审中提出新观点
- 在书面意见中予以回应
- 逐点反驳，不遗漏

### 5.3 被告人庭后提出新情况
- 核实后决定是否补充
- 如属重要情节，应申请恢复法庭调查

---

**使用说明**：
1. 本模板适用于休庭后提交的补充书面辩护意见
2. 应结合庭审实际情况进行调整
3. 与当庭宣读的辩护意见形成互补
