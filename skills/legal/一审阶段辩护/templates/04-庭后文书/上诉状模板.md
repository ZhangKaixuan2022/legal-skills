# 刑事上诉状模板

> 适用场景：被告人不服一审判决，在法定期限内提起上诉
> 法律依据：《刑事诉讼法》第二百二十七条、第二百三十条
> 重要提示：上诉状应以被告人名义提出，由被告人签名，律师可代为书写

---

## 一、文书格式规范

### 1.1 排版参数
| 项目 | 规范 |
|------|------|
| 纸张 | A4纵向 |
| 页边距 | 上2.54cm 下2.54cm 左3.18cm 右3.18cm |
| 标题 | 黑体 三号 加粗 居中 |
| 正文 | 宋体 四号 |
| 行距 | 固定值27磅 |
| 页码 | 底部居中 |

### 1.2 上诉期限
- 不服判决：自收到判决书之日起**10日内**
- 不服裁定：自收到裁定书之日起**5日内**

## 二、上诉状标准格式

```
                        刑事上诉状

上诉人（原审被告人）：[姓名]，[性别]，[出生年月日]，[民族]，
[文化程度]，[籍贯]，[住址]，[身份证号]。[现羁押于/取保候审]。

辩护人：【律师姓名】，【律师事务所名称】律师。
        联系电话：【联系电话】
        执业证号：[证号]

上诉人因[罪名]一案，不服[XX]人民法院（20XX）X刑初XXX号刑事判决书，
现依法提起上诉。

上诉请求：

    [请求事项——撤销原判/改判无罪/改判罪名/从轻改判等]

上诉理由：

一、[上诉理由一：事实认定错误]

    [具体论述原判认定事实的错误之处]

二、[上诉理由二：证据不足]

    [具体论述原判证据方面的问题]

三、[上诉理由三：适用法律错误]

    [具体论述原判适用法律的错误之处]

四、[上诉理由四：量刑不当]

    [具体论述原判量刑过重的理由]

五、[上诉理由五：程序违法]（如有）

    [具体论述一审程序违法之处]

    综上所述，上诉人认为原审判决[认定事实错误/证据不足/
适用法律错误/量刑畸重/程序违法]，恳请二审法院依法[改判/
发回重审]。

    此致

[XX]市中级人民法院（或高级人民法院）

                                    上诉人：[签名/捺印]
                                    日  期：[年 月 日]

附：
1. 本上诉状副本X份
2. 一审判决书复印件1份
3. [其他附件]
```

## 三、上诉请求表述规范

### 3.1 无罪上诉
```
上诉请求：
    请求撤销[XX]人民法院（20XX）X刑初XXX号刑事判决书，
依法改判上诉人无罪。
```

### 3.2 改判罪名上诉
```
上诉请求：
    请求撤销[XX]人民法院（20XX）X刑初XXX号刑事判决书，
依法将上诉人的罪名由[原罪名]改判为[新罪名]。
```

### 3.3 从轻上诉
```
上诉请求：
    请求撤销[XX]人民法院（20XX）X刑初XXX号刑事判决书第X项，
依法对上诉人从轻判处。
```

### 3.4 缓刑上诉
```
上诉请求：
    请求撤销[XX]人民法院（20XX）X刑初XXX号刑事判决书，
依法对上诉人改判适用缓刑。
```

### 3.5 发回重审上诉
```
上诉请求：
    请求撤销[XX]人民法院（20XX）X刑初XXX号刑事判决书，
依法将本案发回原审人民法院重新审判。
```

## 四、上诉理由撰写指引

### 4.1 事实认定错误

**【常见类型】**
1. 原判认定的犯罪事实与客观事实不符
2. 原判遗漏了对上诉人有利的重要事实
3. 原判对关键事实的认定缺乏证据支持
4. 原判混淆了不同行为的性质

**【论述要点】**
- 指出原判认定的具体事实
- 提出正确的事实认定
- 说明证据支持
- 论述事实认定错误对定罪量刑的影响

### 4.2 证据不足

**【常见类型】**
1. 定罪证据未达到"确实、充分"标准
2. 关键证据存在合法性问题
3. 证据之间存在矛盾
4. 孤证定案

**【论述要点】**
- 列举控方证据的问题
- 分析证据三性缺陷
- 论述合理怀疑
- 强调疑罪从无原则

### 4.3 适用法律错误

**【常见类型】**
1. 罪名定性错误
2. 法条适用错误
3. 司法解释适用错误
4. 刑事责任年龄/能力认定错误

**【论述要点】**
- 引用正确的法律规定
- 对比原判适用的法条
- 分析法律适用的区别
- 论述正确定性

### 4.4 量刑不当

**【常见类型】**
1. 未认定法定从轻/减轻情节
2. 未充分考虑酌定从轻情节
3. 量刑明显不均衡
4. 数罪并罚错误

**【论述要点】**
- 列举应当认定的量刑情节
- 引用量刑指导意见
- 提供类案量刑参考
- 提出合理的量刑建议

### 4.5 程序违法

**【常见类型】**
1. 管辖错误
2. 审判组织不合法
3. 剥夺/限制辩护权
4. 证据收集程序违法
5. 其他影响公正审判的程序违法

**【论述要点】**
- 指出具体违法事项
- 引用程序法规定
- 论述对审判公正的影响
- 说明应当发回重审

## 五、刑辩百问要点嵌入

### Q84要点：上诉时机判断
- 无罪案件：应当上诉
- 有罪案件：与委托人充分沟通后决定
- 考虑因素：改判可能性、上诉成本、羁押期限

### Q85要点：上诉状撰写注意事项
- 以被告人名义提出，律师代书
- 不能出现明显的律师专业语言
- 表述应通俗易懂
- 被告人应当签名或捺印

### Q87要点：上诉审辩护策略
- 重点突出关键问题
- 避免面面俱到
- 注意与一审辩护意见的衔接
- 关注二审审理方式（书面/开庭）

### Q88要点：上诉与上诉期内抗诉的关系
- 上诉不加刑原则的适用
- 抗诉后上诉不加刑原则的例外
- 策略选择建议

## 六、上诉状提交程序

### 6.1 提交方式
1. **通过原审法院提交**（推荐）
   - 向原审法院递交上诉状
   - 原审法院转送上级法院

2. **直接向二审法院提交**
   - 二审法院转送原审法院
   - 原审法院报送案卷

### 6.2 份数要求
- 正本1份
- 副本按对方当事人人数（通常1份给检察院）

### 6.3 送达确认
- 要求出具收据或在副本上加盖收文章
- 记录提交时间

---

## 七、特别提醒

### 7.1 上诉期限计算
- 自收到判决书/裁定书的**次日**起算
- 期间届满的最后一日为节假日的，以节假日后的第一日为期满日
- 邮寄上诉的，以交邮日期为准

### 7.2 在押被告人上诉
- 可以通过看守所递交上诉状
- 以看守所收到上诉状的日期为上诉日期
- 口头上诉的，看守所应当制作笔录

### 7.3 超期上诉的处理
- 超过上诉期限的，上诉无效
- 如有正当理由，可以申请恢复上诉期限
- 申请应当在障碍消除后5日内提出

---

**使用说明**：
1. 本模板由辩护律师代被告人起草
2. 上诉状内容应与被告人充分沟通后确定
3. 上诉状必须由被告人本人签名或捺印
4. 注意核实上诉期限，确保在法定期限内提交
