# 答辩补充意见模板

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 第二轮/第三轮辩论意见 |
| 使用阶段 | 一审阶段法庭辩论第二轮及以后 |
| 使用目的 | 针对控方辩论意见进行答辩和补充 |
| 法律依据 | 《刑事诉讼法》第一百九十八条 |

---

## 文书格式

```
被指控______罪一案
第____轮辩论意见

【案号】____________第______号

审判长、审判员（人民陪审员）：

    针对公诉人刚才发表的意见，辩护人补充发表以下辩论意见：

一、对公诉人意见的回应

    【针锋相对】

    （一）关于____________问题

    公诉人认为：____________
    
    辩护人回应：____________

    （二）关于____________问题

    公诉人认为：____________
    
    辩护人回应：____________

    （三）关于____________问题

    公诉人认为：____________
    
    辩护人回应：____________

二、补充辩护意见

    【引经据典】

    （一）法律依据补充
    
    根据《____________》第____条之规定：____________
    
    本案中，____________

    （二）案例援引
    
    在____________案（案号：____________）中，法院认为：____________
    
    本案情况与该案类似，应当参照处理。

    （三）学理观点
    
    ____________

三、再次强调的核心观点

    【言简意赅】

    1. ____________
    2. ____________
    3. ____________

四、总结提升

    【总结提升】

    综上所述，辩护人坚持认为：
    
    ____________

    恳请合议庭依法采纳辩护人的意见。

                                    辩护人：____________律师事务所
                                                      ____________律师
                                              ____年____月____日
```

## 使用说明

1. **Q82第二轮、第三轮辩论要做到**：
   - **针锋相对**：直接回应控方观点
   - **引经据典**：援引法律依据和案例
   - **言简意赅**：精炼表达、突出重点
   - **总结提升**：升华辩护观点

2. **回应控方意见的技巧**：
   - 准确概括控方观点
   - 找准控方观点的薄弱环节
   - 用事实和法律进行反驳
   - 避免重复第一轮意见

3. **Q83辩护人观点分歧处理**：
   - 同一被告人多名辩护人应事先协调
   - 不同被告人辩护人可各自独立辩护
   - 在法庭上不宜与其他辩护人辩论

4. **Q78被法官打断应对**：
   - 服从法庭指挥
   - 择机继续陈述核心观点
   - 书面补充完整意见

5. **注意事项**：
   - 第二轮辩论重在回应，避免简单重复
   - 抓住要点，不宜过长
   - 保持冷静，据理力争
