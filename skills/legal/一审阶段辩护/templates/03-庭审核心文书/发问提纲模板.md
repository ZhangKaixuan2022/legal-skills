# 发问提纲模板

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 一审庭审发问提纲 |
| 使用阶段 | 一审阶段庭审准备时 |
| 使用目的 | 准备法庭调查阶段对被告人、同案犯、证人、鉴定人的发问 |
| 法律依据 | 《刑事诉讼法》第一百九十一条、第一百九十二条 |

## 使用说明

1. **发问原则**：
   - 围绕案件争议焦点
   - 问题明确具体
   - 避免诱导性问题
   - 有利于查明事实

2. **发问技巧**：
   - 开放性问题与封闭性问题结合
   - 先易后难，循序渐进
   - 关键问题要追问
   - 注意观察被发问人反应

3. **Q71/Q72/Q74要点**：
   - 对公诉人不当讯问可提出异议
   - 审判长不当发问保持冷静处理
   - 举证辩论阶段可申请补充发问

4. **庭前准备**：
   - 模拟发问场景
   - 预判各种回答
   - 准备应对方案
