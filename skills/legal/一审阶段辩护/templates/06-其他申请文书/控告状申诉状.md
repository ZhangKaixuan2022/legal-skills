# 控告状/申诉状模板

> 适用场景：针对办案机关违法行为提出控告或申诉
> 法律依据：《刑事诉讼法》第一百一十七条、《人民检察院刑事诉讼规则》

---

## 第一部分：控告状

### 一、文书格式规范

#### 1.1 排版参数
| 项目 | 规范 |
|------|------|
| 纸张 | A4纵向 |
| 页边距 | 上2.54cm 下2.54cm 左3.18cm 右3.18cm |
| 标题 | 黑体 三号 加粗 居中 |
| 正文 | 宋体 四号 |
| 行距 | 固定值27磅 |

---

### 二、控告状正文格式

```
                        控 告 状

控告人（辩护人）：潘睿，广东广和（长春）律师事务所律师。
                  联系电话：18686488305
                  执业证号：[证号]

被控告人：[被控告的办案人员或机关名称]

控告事项：

    控告[被控告人]在办理被告人[姓名][罪名]一案过程中的违法行为。

控告请求：

    1. 依法纠正[被控告人]的违法行为；
    2. 对[被控告人]的违法行为依法处理；
    3. [其他具体请求]。

事实与理由：

    控告人系被告人[姓名]涉嫌[罪名]一案（案号：（20XX）X刑初XXX号）
的辩护人。在本案办理过程中，控告人发现[被控告人]存在以下违法行为：

一、[违法行为一]

    [具体描述违法行为的时间、地点、经过、后果]

    该行为违反了《刑事诉讼法》第XX条/《XX规定》第XX条的规定。

二、[违法行为二]

    [具体描述违法行为的时间、地点、经过、后果]

    该行为违反了《刑事诉讼法》第XX条/《XX规定》第XX条的规定。

三、[违法行为三]

    [具体描述违法行为的时间、地点、经过、后果]

    该行为违反了《刑事诉讼法》第XX条/《XX规定》第XX条的规定。

    上述违法行为严重侵害了被告人的合法权益，影响了司法公正。
根据《刑事诉讼法》第一百一十七条的规定，当事人和辩护人、
诉讼代理人、利害关系人对于司法机关及其工作人员有侵犯其
诉讼权利和人身权利等违法行为的，有权向该机关申诉或者控告。

    特此控告，恳请依法处理。

    此致

[控告受理机关名称]

                                    控告人：潘睿律师
                                    广东广和（长春）律师事务所（盖章）
                                    日  期：[年 月 日]

附件：
1. 委托书、授权委托书、律师事务所函各1份
2. 律师执业证复印件1份
3. [证据材料清单]
```

### 三、常见控告事由

#### 3.1 侦查阶段违法行为

| 违法行为 | 法律依据 | 控告要点 |
|----------|----------|----------|
| 刑讯逼供 | 《刑诉法》第52条 | 时间、地点、方式、伤情 |
| 非法拘禁 | 《刑诉法》第85条 | 超期羁押、违法羁押 |
| 非法搜查 | 《刑诉法》第138条 | 无证搜查、违法扣押 |
| 阻碍律师会见 | 《刑诉法》第39条 | 拒绝安排、无故拖延 |
| 诱供、骗供 | 《刑诉法》第52条 | 诱供方式、骗供内容 |

#### 3.2 审查起诉阶段违法行为

| 违法行为 | 法律依据 | 控告要点 |
|----------|----------|----------|
| 阻碍阅卷 | 《刑诉法》第40条 | 拒绝提供、选择性提供 |
| 超期羁押 | 《刑诉法》第172条 | 羁押期限计算 |
| 不当起诉 | 《刑诉法》第176条 | 证据不足起诉 |

#### 3.3 审判阶段违法行为

| 违法行为 | 法律依据 | 控告要点 |
|----------|----------|----------|
| 限制辩护权 | 《刑诉法》第14条 | 打断发言、不让质证 |
| 程序违法 | 《刑诉法》相关规定 | 具体违法情形 |
| 不当言行 | 《法官职业道德准则》 | 威胁、训斥辩护人 |

## 第二部分：申诉状

### 一、申诉状正文格式

```
                        申 诉 状

申诉人（辩护人）：潘睿，广东广和（长春）律师事务所律师。
                  联系电话：18686488305
                  执业证号：[证号]

申诉事项：

    对[机关名称]作出的[决定/行为]提出申诉。

申诉请求：

    1. 撤销[机关名称]作出的[决定名称]；
    2. 依法重新作出决定；
    3. [其他具体请求]。

事实与理由：

    申诉人系被告人[姓名]涉嫌[罪名]一案（案号：（20XX）X刑初XXX号）
的辩护人。

一、[原决定/行为]的基本情况

    [描述原决定/行为的内容]

二、申诉理由

    （一）[理由一]
    
    [详细论述]

    （二）[理由二]
    
    [详细论述]

    （三）[理由三]
    
    [详细论述]

三、法律依据

    根据《刑事诉讼法》第XX条/《XX规定》第XX条的规定，
[引用法律依据说明原决定错误]。

    综上所述，[机关名称]作出的[决定/行为][认定事实错误/
适用法律错误/程序违法]，恳请依法予以纠正。

    此致

[申诉受理机关名称]

                                    申诉人：潘睿律师
                                    广东广和（长春）律师事务所（盖章）
                                    日  期：[年 月 日]

附件：
1. 原决定文书复印件
2. [证据材料清单]
```

### 二、常见申诉事项

#### 2.1 对强制措施决定的申诉

```
申诉事项：

    对[XX]人民法院作出的不予变更强制措施决定提出申诉。

申诉理由：

    申诉人于[日期]向贵院提交了《取保候审申请书》，贵院于[日期]
作出不予变更强制措施的决定。申诉人认为该决定错误，理由如下：

    [详细论述取保候审条件成就、不予变更决定错误的理由]
```

#### 2.2 对驳回申请的申诉

```
申诉事项：

    对[XX]人民法院驳回辩护人[调查取证/证人出庭/回避]申请
的决定提出申诉。

申诉理由：

    [详细论述申请应当被批准的理由、驳回决定错误的依据]
```

### 三、控告/申诉渠道

#### 3.1 控告渠道

| 被控告对象 | 控告受理机关 |
|------------|--------------|
| 公安机关及其工作人员 | 同级或上级公安机关、同级人民检察院 |
| 人民检察院及其工作人员 | 上级人民检察院 |
| 人民法院及其工作人员 | 上级人民法院、同级人民检察院 |

#### 3.2 申诉渠道

| 申诉事项 | 申诉受理机关 |
|----------|--------------|
| 对逮捕决定不服 | 同级或上级人民检察院 |
| 对不起诉决定不服 | 上级人民检察院 |
| 对法院裁定不服 | 作出裁定的人民法院或上级人民法院 |

---

### 四、注意事项

#### 4.1 控告/申诉时效
- 应当及时提出，避免因时间过长影响调查
- 一般在违法行为发生后尽快提出

#### 4.2 证据收集
- 控告/申诉前应当收集保全相关证据
- 可以包括：书面材料、录音录像、证人证言等

#### 4.3 多渠道并行
- 可以同时向多个机关提出控告/申诉
- 注意协调不同渠道的进展

#### 4.4 后续跟进
- 提出控告/申诉后及时跟进处理进展
- 对处理结果不服的，可以向上级机关继续申诉

---

**使用说明**：
1. 控告状用于针对办案人员的违法行为
2. 申诉状用于对不当决定提出异议
3. 注意区分控告与申诉的适用情形
4. 提供充分的证据支持控告/申诉主张
