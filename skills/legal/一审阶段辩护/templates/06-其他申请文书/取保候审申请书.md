# 取保候审申请书

> **适用场景**：一审阶段向**人民法院**申请对被告人变更强制措施为取保候审
> **法律依据**：《刑事诉讼法》第六十七条、第九十七条、第九十八条

---

## 一审阶段特殊说明

### 1. 申请对象确认

> **重要提示**：一审阶段取保候审的申请对象为**人民法院**，区别于：
> - 侦查阶段：向公安机关申请
> - 审查起诉阶段：向人民检察院申请

### 2. 一审阶段两种取保情形

| 情形 | 适用时机 | 特殊考量 |
|-----|---------|---------|
| **庭前取保** | 案件移送法院后、开庭前 | 案件尚未进入实体审理，羁押必要性可能变化 |
| **延审期间取保** | 延期审理期间 | 审理期限延长，继续羁押可能不当 |

### 3. 一审阶段特有申请事由

除《刑事诉讼法》第六十七条规定的一般事由外，一审阶段可援引以下特殊事由：

| 特殊事由 | 说明 | 法律依据 |
|---------|------|---------|
| 案件进入审判阶段后发现新的从轻/减轻情节 | 阅卷后发现对被告人有利的新情况 | 《刑事诉讼法》第六十七条第（二）项 |
| 延期审理期间羁押必要性变化 | 因延期审理导致羁押期限延长 | 《刑事诉讼法》第九十八条 |
| 被告人身体状况变化 | 羁押期间健康恶化 | 《刑事诉讼法》第六十七条第（三）项 |
| 已达成刑事和解/被害人谅解 | 社会危险性降低 | 《刑事诉讼法》第六十七条第（二）项 |
| 羁押期限即将届满 | 一审审限内案件尚未审结 | 《刑事诉讼法》第六十七条第（四）项 |
| 认罪认罚案件量刑建议较轻 | 可能判处缓刑或轻刑 | 《刑事诉讼法》第六十七条第（一）、（二）项 |

现羁押于[看守所名称]。

案由：[罪名]

案号：（20XX）X刑初XXX号

    现被告人[姓名]被羁押于[看守所名称]，申请人认为被告人符合

    [选择适用的情形，详细论述]

    （一）可能判处管制、拘役或者独立适用附加刑的
    
    [如适用，详细论述]

    [如适用，详细论述]

    （三）患有严重疾病、生活不能自理，怀孕或者正在哺乳自己婴儿的
    妇女，采取取保候审不致发生社会危险性的
    
    [如适用，详细论述]

    （四）羁押期限届满，案件尚未办结，需要采取取保候审的
    
    [如适用，详细论述]

二、被告人取保候审不致发生社会危险性

    （一）被告人无前科劣迹
    
    [详细论述]

    （二）被告人有固定住所和稳定的社会关系
    
    [详细论述]

    （三）被告人认罪态度良好
    
    [详细论述]

    [详细论述]

三、取保候审具有现实必要性

    [详细论述，如家庭困难、老人需要赡养、子女需要抚养、企业经营
需要等特殊情况]

四、保证方式

    被告人愿意采取以下保证方式：
    
    □ 提供保证人：[保证人姓名]，[与被告人关系]，[住址]，

## 四、常见申请理由详述

### 4.1 一审阶段特有理由

```
    本案被告人涉嫌[罪名]，根据《刑法》第XX条的规定，法定刑为
[刑期范围]。综合考虑被告人的犯罪情节、认罪态度及量刑情节，
辩护人认为被告人可能被判处[管制/拘役/有期徒刑X年以下]。
根据《刑事诉讼法》第六十七条第（一）项/第（二）项的规定，
符合取保候审的条件。
```

### 4.2 案件进入审判阶段后发现新的从轻/减轻情节

```
    本案移送贵院审理后，辩护人通过阅卷和会见，发现以下
对被告人有利的新情况：

    一、[新发现的从轻情节，如：被害人过错、自首情节未被
认定、坦白情节等]

    二、上述情节的发现，使得被告人的量刑预期发生变化，
继续羁押的必要性相应降低。

    根据《刑事诉讼法》第六十七条第（二）项的规定，可能判处
有期徒刑以上刑罚，采取取保候审不致发生社会危险性的，可以
取保候审。
```

### 4.3 延期审理期间羁押必要性变化

```
    本案因[延期审理原因]，贵院已决定延期审理。被告人自
[日期]被羁押至今已[X]个月，原定开庭时间为[日期]，因延期
审理导致羁押期限相应延长。

    根据《刑事诉讼法》第九十八条的规定，犯罪嫌疑人、被告人
被羁押的案件，不能在法定期限内办结的，对被羁押的犯罪嫌疑人、
被告人应当予以释放或者变更强制措施。

    被告人在羁押期间表现良好，无违规行为，且案件延期审理
并非被告人原因所致。在延期审理期间，继续羁押被告人已无必要，
恳请贵院依法变更强制措施为取保候审。
```

### 4.4 无社会危险性

```
    （一）被告人在案发前无任何违法犯罪记录，系初犯、偶犯。

    （二）被告人有固定住所，居住在[地址]，有稳定的家庭关系，
其父母/配偶/子女均在本地，具备有效的监管条件。

    （三）被告人归案后如实供述，认罪态度良好，无翻供、串供行为。

    （四）本案主要证据已经收集固定，被告人不具备毁灭、伪造证据
或干扰证人作证的条件。

    （五）被害人已谅解/双方已达成和解协议，被告人无继续危害
社会的可能。
```

### 4.5 身体健康原因

```
    被告人患有[疾病名称]，根据[医院名称]出具的诊断证明，
被告人病情[描述病情]，需要[治疗方式]，看守所的医疗条件
难以满足被告人的治疗需要。

    根据《刑事诉讼法》第六十七条第（三）项的规定，患有严重疾病
的犯罪嫌疑人、被告人，采取取保候审不致发生社会危险性的，
可以取保候审。
```

### 4.6 羁押必要性审查

```
    被告人自[日期]被刑事拘留/逮捕至今，已被羁押[X]个月。
在此期间，被告人一直认罪悔罪，积极配合调查，无任何违规行为。

    辩护人认为，继续羁押被告人已无必要性，恳请贵院依法变更
强制措施为取保候审。
```

## 六、注意事项

### 6.1 一审阶段申请时机
- 案件移送法院后，根据案件情况择机申请
- 羁押期限即将届满时
- 出现新的有利情节时（如达成和解、取得谅解）
- **延期审理决定作出后**（延审期间取保的最佳时机）
- **庭前会议后**（程序性问题解决、争点明确后）

### 6.2 申请方式
- 书面申请，提交正本1份
- 附相关证明材料
- 建议同时口头说明情况

### 6.3 后续跟进
- 申请后3日内跟进处理结果
- 如被驳回，分析原因，必要时补充材料再次申请
- 取保候审获批后，及时办理相关手续

---

**使用说明**：
1. 根据案件具体情况选择适用的申请理由
2. 确保附件材料齐全、真实
3. 申请书应简明扼要，重点突出
