# 适用普通程序建议书

> 适用场景：简易程序/速裁程序案件需要转为普通程序审理时，向人民法院提出建议
> 法律依据：《刑事诉讼法》第二百二十条、第二百二十五条

---

## 二、建议书正文格式

```
                  适用普通程序审理建议书

建议人：潘睿，广东广和（长春）律师事务所律师。
        联系电话：18686488305
        执业证号：[证号]

被告人：[姓名]，[性别]，[出生年月日]，[民族]，[住址]。

案由：[罪名]

案号：（20XX）X刑初XXX号

建议事项：

    建议将本案由[简易程序/速裁程序]转为普通程序审理。

建议理由：

    广东广和（长春）律师事务所接受被告人[姓名]（或其近亲属[姓名]）的委托，
指派建议人担任被告人[姓名]涉嫌[罪名]一案一审阶段的辩护人。

    本案目前适用[简易程序/速裁程序]审理，但辩护人认为本案不符合
适用[简易程序/速裁程序]的条件，依法应当转为普通程序审理。
具体理由如下：

一、[建议理由一：被告人对适用简易程序/速裁程序有异议]

    根据《刑事诉讼法》第二百一十五条的规定，适用简易程序审理案件，
应当征得被告人的同意。被告人[姓名]现对适用简易程序审理本案有异议，
表示不同意适用简易程序。

    [或：根据《刑事诉讼法》第二百二十二条的规定，适用速裁程序审理
案件，应当符合"被告人认罪认罚"的条件。被告人[姓名]现表示撤回
认罪认罚的意思表示，不再认罪认罚。]

    根据上述法律规定，本案应当转为普通程序审理。

二、[建议理由二：被告人对指控的犯罪事实有异议]

    本案适用[简易程序/速裁程序]的前提是被告人对指控的犯罪事实
没有异议。但被告人[姓名]对起诉书指控的以下犯罪事实有异议：

    1. [异议内容一]
    2. [异议内容二]
    3. [异议内容三]

    上述异议涉及案件的关键事实认定，需要通过完整的法庭调查
予以查明，简易程序/速裁程序无法满足审理需要。

三、[建议理由三：案件存在疑难复杂情形]

    本案存在以下疑难复杂情形，不适宜适用简易程序/速裁程序审理：

    1. [情形一：涉及复杂的法律适用问题]
    2. [情形二：证据存在重大争议]
    3. [情形三：共同犯罪中各被告人意见不一致]
    4. [情形四：其他疑难复杂情形]

四、[建议理由四：辩护人需要充分行使辩护权]

    简易程序/速裁程序的审理时间较短，辩护人难以充分行使辩护权。
本案涉及以下需要重点辩护的问题：

    1. [辩护要点一]
    2. [辩护要点二]
    3. [辩护要点三]

    为保障被告人获得充分有效的辩护，本案应当转为普通程序审理。

五、[建议理由五：其他不宜适用简易程序/速裁程序的情形]

    [根据案件具体情况补充其他理由]

    综上所述，本案不符合适用[简易程序/速裁程序]的条件，依法应当
转为普通程序审理。根据《刑事诉讼法》第二百二十条/第二百二十五条
的规定，恳请贵院将本案转为普通程序审理。

    此致

[XX]人民法院

                                    建议人：潘睿律师
                                    广东广和（长春）律师事务所（盖章）
                                    日  期：[年 月 日]

附件：
1. 委托书、授权委托书、律师事务所函各1份
2. 律师执业证复印件1份
3. [其他证明材料]
```

## 三、法定转为普通程序的情形

### 3.1 简易程序转普通程序

根据《刑事诉讼法》第二百二十条，有下列情形之一的，应当转为普通程序审理：

| 法定情形 | 说明 |
|----------|------|
| 被告人的行为不构成犯罪 | 可能宣告无罪 |
| 被告人应当判处三年以上有期徒刑 | 可能判处较重刑罚 |
| 被告人当庭对起诉指控的犯罪事实予以否认 | 事实存在争议 |
| 事实不清或证据不足 | 需要进一步调查 |
| 其他不宜适用简易程序的情形 | 兜底条款 |

### 3.2 速裁程序转普通程序

根据《刑事诉讼法》第二百二十五条，有下列情形之一的，应当转为普通程序审理：

| 法定情形 | 说明 |
|----------|------|
| 被告人违背意愿认罪认罚 | 认罪认罚自愿性存疑 |
| 被告人否认指控的犯罪事实 | 事实存在争议 |
| 其他不宜适用速裁程序的情形 | 兜底条款 |

## 四、常见建议理由详述

### 4.1 被告人撤回认罪认罚

```
    被告人[姓名]在审查起诉阶段签署了《认罪认罚具结书》，
但被告人现表示其当时对案件事实和法律后果认识不足，
系在不了解真实情况下签署具结书。

    现被告人明确表示撤回认罪认罚的意思表示，对起诉书
指控的[部分/全部]犯罪事实予以否认。

    根据《刑事诉讼法》第二百二十五条的规定，被告人否认
指控的犯罪事实的，应当转为普通程序审理。
```

### 4.2 定性存在争议

```
    本案公诉机关以[罪名A]提起公诉，但辩护人认为被告人的
行为不构成[罪名A]，应当认定为[罪名B/无罪]。

    该定性问题涉及复杂的法律适用，需要在法庭辩论中
充分论证，简易程序难以保障充分的辩论时间。
```

### 4.3 证据存在重大争议

```
    本案存在以下证据争议：
    
    1. [证据名称]的合法性存疑，辩护人已申请非法证据排除；
    2. [证据名称]与其他证据存在矛盾，需要通过法庭调查查明；
    3. 辩护人申请[证人姓名]出庭作证，对关键事实进行核实。
    
    上述证据问题需要通过完整的举证、质证程序予以查明，
简易程序无法满足审理需要。
```

### 4.4 共同犯罪意见不一致

```
    本案系共同犯罪案件，各被告人对案件事实的陈述存在差异：
    
    1. 关于[事实点]，被告人A称[内容]，被告人B称[内容]；
    2. 关于[事实点]，各被告人对各自的作用认定存在争议。
    
    需要通过普通程序的法庭调查，对各被告人进行讯问、
对质，以查明案件事实。
```

## 五、注意事项

### 5.1 提出时机
- 收到起诉书后尽快审查是否适宜简易程序/速裁程序
- 如有异议，在开庭前及时提出
- 开庭后发现不宜适用的，也可以当庭提出

### 5.2 与被告人沟通
- 转为普通程序可能延长审理时间
- 需要告知被告人利弊，取得被告人同意
- 如被告人坚持认罪认罚，应尊重被告人意见

### 5.3 后果考量
- 转为普通程序后，认罪认罚从宽可能受影响
- 审理时间延长
- 诉讼成本增加

### 5.4 备选方案
- 如建议不被采纳，准备好简易程序/速裁程序下的辩护方案
- 在庭审中继续提出相关意见

## 六、当庭申请转程序

如在庭审中发现需要转为普通程序：

```
审判长，辩护人认为本案不宜继续适用[简易程序/速裁程序]审理，
申请转为普通程序。

理由是：[简要说明理由]

根据《刑事诉讼法》第二百二十条/第二百二十五条的规定，
恳请合议庭将本案转为普通程序审理。
```

---

**使用说明**：
1. 本建议书应在开庭前提交
2. 需要与被告人充分沟通，取得被告人同意
3. 建议理由应当具体、明确、有法律依据
4. 如建议未被采纳，应做好相应程序下的辩护准备
