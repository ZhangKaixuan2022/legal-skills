# 延期审理申请书

> **适用场景**：因法定事由需要延期审理时，向人民法院提出申请
> **法律依据**：《刑事诉讼法》第二百零六条、《最高人民法院关于适用〈中华人民共和国刑事诉讼法〉的解释》第二百七十五条

---

## 使用说明

### 1. 程序定位

延期审理申请是一审程序中的程序性申请，可在以下时机提出：

| 提出时机 | 提出方式 | 处理程序 |
|---------|---------|---------|
| **庭前会议中** | 口头或书面 | 由审判长当场决定或会后研究 |
| **开庭前** | 书面申请 | 法院审查后决定 |
| **庭审中** | 当庭口头申请 | 合议庭当庭或休庭后决定 |

### 2. 与庭前会议的关系

> **重要提示**：延期审理申请可在庭前会议中提出，也可单独书面提交。
> - 在庭前会议中提出的，纳入庭前会议讨论事项，由审判长主持讨论后决定
> - 单独书面提交的，由法院书面审查后作出决定
> - 庭审中发现新情况需要延期的，应当庭申请

### 3. 延期事由分类

根据《刑事诉讼法》第二百零六条，延期事由分为以下类型：

| 类型 | 具体事由 | 法律依据 |
|-----|---------|---------|
| **证据事由** | 需要通知新的证人到庭 | 第206条第（一）项 |
| | 需要调取新的物证 | 第206条第（二）项 |
| | 需要重新鉴定或者勘验 | 第206条第（二）项 |
| **诉讼程序事由** | 检察人员提出补充侦查建议 | 第206条第（三）项 |
| | 由于申请回避而不能进行审判 | 第206条第（四）项 |
| **当事人事由** | 被告人患病无法出庭 | 司法解释 |
| | 辩护人有正当理由无法出庭 | 司法解释 |
| **其他事由** | 辩护律师庭审冲突 | 实务惯例 |
| | 需要通知有专门知识的人出庭 | 司法解释 |
| | 其他不可抗力 | 司法解释 |

### 4. 延期期限与羁押关系

| 延期情形 | 延期期限 | 与羁押的关系 |
|---------|---------|------------|
| 一般延期 | 由法院决定 | 羁押期限相应延长 |
| 建议补充侦查 | 1个月 | 羁押期限顺延 |
| 被告人患病 | 视病情而定 | 可申请变更强制措施 |

> **注意**：延期审理期间，被告人仍处于羁押状态的，如羁押必要性发生变化，应当及时申请变更强制措施。详见 → [取保候审申请书](./取保候审申请书.md)

## 二、申请书正文格式

```
                      延期审理申请书

申请人：【律师姓名】，【律师事务所名称】律师。
        联系电话：【联系电话】
        执业证号：[证号]

被告人：[姓名]，[性别]，[出生年月日]，[民族]，[住址]。

案由：[罪名]

案号：（20XX）X刑初XXX号

申请事项：

    申请延期审理本案。

申请理由：

    【律师事务所名称】接受被告人[姓名]（或其近亲属[姓名]）的委托，
指派申请人担任被告人[姓名]涉嫌[罪名]一案一审阶段的辩护人。

    本案定于[年 月 日 时]在贵院开庭审理，但因[具体原因]，
特申请延期审理，具体理由如下：

    [选择适用的理由，详细论述]

一、[申请理由一：需要通知新的证人到庭]

    辩护人在准备辩护过程中，发现[证人姓名]对本案关键事实知情，
其证言对查明案件事实具有重要意义。根据《刑事诉讼法》第二百零六条
第（一）项的规定，需要通知新的证人到庭的，可以延期审理。

    辩护人已向贵院提交《通知证人出庭申请书》，为确保[证人姓名]
能够到庭作证，特申请延期审理。

二、[申请理由二：需要调取新的物证]

    辩护人发现本案存在对被告人有利的重要物证，即[物证名称/内容]，
该物证现存放于[地点/机关]。根据《刑事诉讼法》第二百零六条
第（二）项的规定，需要调取新的物证的，可以延期审理。

    辩护人已向贵院提交《调查取证申请书》，为确保该物证能够在
庭审中出示，特申请延期审理。

三、[申请理由三：需要重新鉴定]

    辩护人认为本案[鉴定事项]的鉴定意见存在[问题]，需要重新鉴定。
根据《刑事诉讼法》第二百零六条第（二）项的规定，需要重新鉴定或者
勘验的，可以延期审理。

    辩护人已向贵院提交《重新鉴定申请书》，特申请延期审理。

四、[申请理由四：辩护人有正当理由无法出庭]

    辩护人因[具体原因，如：身体原因/出差/有其他案件开庭冲突等]，
无法于定于[年 月 日]参加本案开庭审理。

    申请人作为被告人的辩护人，出庭辩护是保障被告人合法权益的
重要途径，辩护人缺席将影响被告人辩护权的行使。

    恳请贵院依法准许延期审理。

五、[申请理由五：被告人身体状况]

    被告人因[疾病/身体原因]，目前身体状况不适宜出庭受审。
根据[医院/看守所]出具的证明，被告人[具体情况]。

    为保障被告人的诉讼权利和人身权利，特申请延期审理。

    综上所述，申请人认为本案符合延期审理的法定情形，恳请贵院
依法准许延期审理申请。

    此致

[XX]人民法院

                                    申请人：【律师姓名】律师
                                    【律师事务所名称】（盖章）
                                    日  期：[年 月 日]

附件：
1. [证明材料清单]
```

## 三、法定延期审理情形

根据《刑事诉讼法》第二百零六条，有下列情形之一，影响审判进行的，可以延期审理：

### 3.1 需要通知新的证人到庭、调取新的物证、重新鉴定或者勘验的

**适用情形**：
- 发现新的证人需要出庭作证
- 需要调取新的物证、书证
- 对鉴定意见有异议，需要重新鉴定
- 需要进行现场勘验

**注意事项**：
- 应当同时提交相应的申请书（通知证人出庭申请书、调取证据申请书等）
- 说明新证据/新证人对案件事实认定的必要性

### 3.2 检察人员发现提起公诉的案件需要补充侦查、提出建议的

**说明**：此为检察机关申请，辩护人一般不适用

### 3.3 由于申请回避而不能进行审判的

**适用情形**：
- 辩护人提出回避申请，合议庭需要研究决定
- 回避申请涉及复杂情况，需要时间处理

---

## 四、其他可以延期审理的情形

根据相关司法解释，以下情形也可以延期审理：

### 4.1 当事人、辩护人、诉讼代理人申请

- 需要准备新的证据
- 发现新的事实需要调查
- 其他需要延期的合理理由

### 4.2 诉讼参与人无法出庭

- 被告人患病无法出庭
- 辩护人有正当理由无法出庭
- 关键证人无法按时到庭

### 4.3 其他不可抗力

- 自然灾害
- 公共卫生事件
- 其他不可预见的情况

---

## 五、申请注意事项

### 5.1 申请时机
- 发现延期事由后及时提出
- 尽量在开庭前提出
- 如庭审中发现需要延期，可当庭申请

### 5.2 申请方式
- 原则上应当书面申请
- 紧急情况可口头申请，但应补交书面申请
- 需要提供相应的证明材料

### 5.3 证明材料

| 申请理由 | 需要提供的证明材料 |
|----------|-------------------|
| 通知新证人 | 证人信息、证言内容说明、必要性说明 |
| 调取新物证 | 物证线索、存放地点、调取申请书 |
| 重新鉴定 | 原鉴定意见问题说明、重新鉴定申请书 |
| 辩护人无法出庭 | 病假证明/行程安排/其他案件开庭通知等 |
| 被告人身体原因 | 医院/看守所出具的证明 |

### 5.4 延期次数和时间
- 原则上延期审理次数不宜过多
- 延期时间由法院决定
- 应当配合法院安排新的开庭时间

---

## 六、庭审中申请延期

如在庭审过程中发现需要延期审理的情形：

### 6.1 申请方式
```
审判长，辩护人申请延期审理。

理由是：[当庭说明理由]

恳请合议庭准许延期审理申请。
```

### 6.2 常见情形
- 公诉人当庭出示新证据，辩护人需要时间准备质证意见
- 庭审中发现需要申请新证人出庭
- 庭审中发现需要补充调取证据
- 被告人当庭翻供，需要核实情况

---

**使用说明**：
1. 根据案件具体情况选择适用的申请理由
2. 申请理由应当具体、明确
3. 提供相应的证明材料
4. 延期审理申请不一定会被准许，应有备选方案
