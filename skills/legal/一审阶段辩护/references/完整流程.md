## 一、功能概述

### 1.1 阶段定位
> 本Skill亦可由刑事辩护总调度Skill路由进入：skills/legal/刑事辩护总调度/SKILL.md

一审阶段是刑事诉讼从"检察院公诉"到"法院判决"的核心程序阶段，辩护工作从审查起诉阶段的"影响指控"转向"影响裁判"。本阶段以法庭审理为中心，辩护律师全面行使法定辩护权利。

### 1.2 核心目标

| 目标层级 | 具体内容 | 实现路径 |
|---------|---------|---------|
| 无罪辩护 | 事实不清、证据不足，或不构成犯罪 | 质证+辩论+证据排除 |
| 轻罪辩护 | 改变罪名认定，降低法定刑幅度 | 法律适用辩论 |
| 量刑辩护 | 争取从轻、减轻或免除处罚 | 量刑情节举证 |
| 缓刑辩护 | 符合条件时争取缓刑适用 | 社会调查+认罪认罚 |

### 1.3 辩护权利升级

| 权利类别 | 审查起诉阶段 | 一审阶段新增/升级 |
|---------|------------|-----------------|
| 阅卷权 | 全部案卷材料 | 补充侦查卷+新证据 |
| 调查取证权 | 向检察院申请 | 向法院申请调取 |
| **举证权** | 有限 | **完整举证权** |
| **质证权** | 书面意见 | **当庭质证权** |
| **辩论权** | 无 | **法庭辩论权** |
| 申请排除非法证据 | 可申请 | 庭审中当场启动 |

## 二、双模式启动

### 2.1 串联模式（推荐）

当已完成审查起诉阶段辩护工作时，读取前阶段输出：审查起诉阶段辩护方案、证据审查报告、案件法律分析报告、量刑预判报告、阶段工作总结。

**关键对比审查**：起诉书与起诉意见书的差异（指控事实变化、罪名变化、量刑建议变化）。

### 2.2 独立模式

首次介入一审阶段时，需完成信息采集：收集起诉书、案卷材料；首次会见被告人了解案情；确定辩护方向和策略。

## 三、十大基本原则（精要版）

| 序号 | 原则名称 | 核心要义 |
|-----|---------|---------|
| 1 | 当事人利益原则 | 一切以被告人合法权益最大化为目标 |
| 2 | 依法辩护原则 | 在法律框架内穷尽辩护手段 |
| 3 | 保密义务原则 | 对案件信息严格保密 |
| 4 | 独立辩护原则 | 独立发表辩护意见，不受干预 |
| 5 | 忠诚义务原则 | 不得损害委托人利益 |
| 6 | 有效辩护原则 | 实质性参与、积极辩护 |
| 7 | 尊重意愿原则 | 重大决策尊重被告人意愿 |
| 8 | 诚信原则 | 不作虚假陈述、不伪造证据 |
| 9 | 勤勉原则 | 认真负责、尽职尽责 |
| 10 | 规范执业原则 | 遵守执业纪律和行业规范 |

## 五、案件接收与管辖审查

### 5.1 委托手续办理

案件移送法院后，辩护人应在**3日内**向法院提交：授权委托书、律师事务所函、律师执业证复印件

### 5.2 管辖权审查

**审查要点**：级别管辖、地域管辖、指定管辖

📋 管辖权异议申请书 → templates/01-庭前程序文书/管辖权异议申请书.md

### 5.3 起诉书与起诉意见书比对

| 比对项目 | 审查要点 |
|---------|---------|
| 指控事实 | 是否有增减、是否有变化 |
| 罪名认定 | 罪名是否变更、是否增减 |
| 量刑建议 | 检察院是否提出、幅度如何 |
| 证据情况 | 是否有新证据、是否退回补充侦查 |

## 六、补充阅卷与证据审查

### 6.1 审判阶段阅卷特点

1. **范围把控**：重点关注补充侦查卷、新增证据
2. **比对审查**：起诉书与起诉意见书的差异
3. **法律研究**：收集相关法律法规、司法解释
4. **判例检索**：搜索类似案例的裁判结果

### 6.2 证据完整性检验

| 证据类型 | 检验要点 |
|---------|---------|
| 物证 | 提取、保管、鉴定程序合法性 |
| 书证 | 原件/复印件、来源、真实性 |
| 证人证言 | 取证程序、证人资格、内容一致性 |
| 被告人供述 | 自愿性、合法性、稳定性 |
| 鉴定意见 | 资质、程序、依据、结论 |
| 视听资料 | 来源、完整性、关联性 |
| 电子数据 | 提取程序、完整性、真实性 |

## 十三、法庭工作四大核心环节

### 13.1 发问提纲设计

| 发问对象 | 发问目的 | 设计要点 |
|---------|---------|---------|
| 被告人 | 澄清事实、有利陈述 | 引导性适度、避免诱导 |
| 同案犯 | 区分责任、查明事实 | 注意利益冲突 |
| 证人 | 质疑证言、发现矛盾 | 封闭式问题为主 |
| 鉴定人 | 质疑鉴定意见 | 专业性问题 |

### 13.2 质证意见三性框架

**合法性**：取证程序、证据形式
**真实性**：内容真实、来源可靠
**关联性**：与待证事实的关联程度

### 13.3 举证提纲

无罪证据、从轻证据、程序违法证据 → 提交时限：开庭前5日

### 13.4 辩论三种路径

| 辩护类型 | 核心论点 |
|---------|---------|
| 无罪辩护 | 事实不清、证据不足或不构成犯罪 |
| 罪轻辩护 | 轻罪、从轻/减轻情节 |
| 骑墙式辩护 | 无罪为主+量刑为辅 |

## 十八、当事人和解

### 和解的法律后果

一审阶段达成和解，法院必须定罪判刑，但可以依法对被告人从宽处罚。

### 庭审中和解事实的运用

- 举证阶段：提交和解协议书、谅解书作为量刑证据
- 辩论阶段：强调和解系被告人真诚悔罪、被害人自愿谅解
- 量刑建议：结合和解情节争取从宽量刑
