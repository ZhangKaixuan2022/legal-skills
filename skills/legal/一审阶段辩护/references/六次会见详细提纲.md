# 一审阶段六次会见详细提纲

---

## 会见总体安排

一审阶段设计六次关键会见节点，每次会见有明确的核心任务和输出要求。

---

## 第一次会见：收到起诉书后立即

**时机**：收到起诉书副本后第一时间

**核心目的**：核实起诉事实、了解诉讼意愿

**会见任务清单**：
```
□ 向被告人送达起诉书副本
□ 逐条告知起诉书指控的事实
□ 核实被告人对指控事实的态度（认可/部分认可/不认可）
□ 了解被告人对证据的初步意见
□ 询问是否有新情况需要说明
□ 了解被告人的诉讼意愿（是否认罪认罚）
□ 告知一审程序和被告人权利
□ 制作详细会见笔录
```

**重点询问**：
- 起诉书指控的事实与你之前的供述是否一致？
- 有哪些事实认定你认为不准确？
- 是否有新的辩解或需要补充的情况？

**输出文书**：会见笔录

---

## 第二次会见：阅卷后

**时机**：律师完成全部案卷阅卷后

**核心目的**：核实证据、形成初步辩护意见

**会见任务清单**：
```
□ 向被告人通报阅卷情况
□ 逐项核实关键证据材料
□ 听取被告人对证据的意见
□ 识别证据中的矛盾和疑点
□ 与被告人核实案件事实细节
□ 沟通初步辩护意见和思路
□ 记录被告人提出的辩护线索
□ 制作会见笔录
```

**重点核实**：
- 你的供述笔录内容是否属实？是否存在被迫或诱导？
- 证人证言中关于你的描述是否准确？
- 物证、书证与你记忆中的情况是否一致？
- 鉴定意见你是否有异议？

**输出文书**：会见笔录、证据核实记录

---

## 第三次会见：庭前会议前

**时机**：法院决定召开庭前会议后、庭前会议召开前

**核心目的**：沟通庭前会议策略

**会见任务清单**：
```
□ 告知被告人庭前会议的时间和议题
□ 沟通是否申请证人出庭
□ 沟通是否申请非法证据排除
□ 确认是否有其他程序性申请
□ 沟通辩护策略是否调整
□ 了解被告人是否有新的想法
□ 制作会见笔录
```

**重点沟通**：
- 是否需要申请证人出庭？哪些证人？
- 是否有非法取证情形需要申请排除？
- 庭前会议上律师拟提出哪些问题？

**输出文书**：会见笔录

---

## 第四次会见：开庭前（庭前辅导）

**时机**：开庭前1-3天

**核心目的**：庭前辅导、庭审程序讲解

**会见任务清单**：
```
□ 告知开庭时间、地点、合议庭组成
□ 详细讲解庭审流程和各环节注意事项
□ 预演可能的讯问和发问
□ 指导法庭陈述的要点和技巧
□ 指导被告人如何回应公诉人讯问
□ 指导最后陈述的准备
□ 确认辩护方案，取得被告人同意
□ 缓解被告人庭审紧张情绪
□ 制作庭审流程告知书
```

**庭审流程告知要点**：
1. 法庭调查阶段：公诉人宣读起诉书 → 被告人陈述 → 公诉人讯问 → 辩护人发问 → 举证质证
2. 法庭辩论阶段：公诉人发表公诉意见 → 辩护人发表辩护意见 → 双方辩论
3. 最后陈述：被告人最后陈述
4. 注意事项：尊重法庭、如实陈述、不抢话、不争吵

**输出文书**：会见笔录、庭审流程告知书

---

## 第五次会见：庭审后

**时机**：庭审休庭后，判决前

**核心目的**：庭审复盘、补充辩护意见

**会见任务清单**：
```
□ 与被告人一起复盘庭审情况
□ 了解被告人对庭审的感受
□ 总结庭审中的有利和不利因素
□ 沟通是否需要补充辩护意见
□ 告知书面辩护意见的提交计划
□ 预判可能的判决结果
□ 制作会见笔录
```

**复盘要点**：
- 庭审中哪些环节有利？哪些不利？
- 法官对哪些问题特别关注？
- 公诉人的意见有哪些新内容？
- 是否需要补充举证或辩护意见？

**输出文书**：会见笔录

---

## 第六次会见：判决后

**时机**：收到判决书后

**核心目的**：判决分析、上诉沟通

**会见任务清单**：
```
□ 向被告人送达判决书
□ 逐条解读判决书内容
□ 分析判决对辩护意见的采纳情况
□ 评估判决的合理性
□ 分析上诉的利弊
□ 了解被告人是否决定上诉
□ 如决定上诉，告知上诉程序和期限
□ 如不上诉，告知执行相关事项
□ 制作会见笔录
```

**上诉分析要点**：
- 判决在事实认定、证据采信、法律适用、量刑方面是否有错误
- 上诉的可能结果（改判、维持、发回重审）
- 上诉的利弊权衡
- 上诉期限（10日）

**输出文书**：会见笔录
