# 保密合规检查清单

> 适用场景：一审阶段辩护工作全程保密合规管理
> 适用对象：辩护律师、律师助理、实习律师
> 保密等级：律师内部使用

---

## 一、保密等级划分

### 1.1 三级保密架构

| 等级 | 名称 | 适用范围 | 可见对象 |
|-----|------|---------|---------|
| 第一级 | 内部底稿 | 律师工作底稿、案件分析、量刑预判 | 仅承办律师及团队 |
| 第二级 | 会见材料 | 会见笔录、庭审告知书、确认表 | 律师、被告人、委托人 |
| 第三级 | 反馈函 | 对外正式函件、辩护意见书 | 可向办案机关提交 |

### 1.2 各类文书保密等级对照

| 文书类型 | 保密等级 | 备注 |
|---------|---------|------|
| 量刑预判报告 | 内部底稿 | 绝对保密，不对外 |
| 辩护方案 | 内部底稿 | 绝对保密，不对外 |
| 案件分析报告 | 内部底稿 | 绝对保密，不对外 |
| 会见笔录 | 会见材料 | 可向委托人提供 |
| 庭审流程告知书 | 会见材料 | 可向被告人提供 |
| 辩护意见书 | 反馈函 | 可向法院提交 |
| 法律文书 | 反馈函 | 可向办案机关提交 |

## 三、量刑预判报告保密规范

### 3.1 保密原则

| 原则 | 具体要求 |
|-----|---------|
| 绝对保密 | 量刑预判报告不得向任何外部人员提供 |
| 内部限制 | 仅限承办律师及直接参与案件的团队成员查阅 |
| 分级展示 | 向委托人说明时仅口头沟通要点，不提供书面材料 |
| 存档保密 | 归档后按保密材料管理 |

### 3.2 保密操作流程

```
┌─────────────────────────────────────────────────────────────┐
│                    量刑预判报告保密流程                        │
├─────────────────────────────────────────────────────────────┤
│                                                               │
│  1. 制作阶段                                                  │
│     └── 仅由承办律师制作，标注【内部底稿】                      │
│                                                               │
│  2. 审核阶段                                                  │
│     └── 团队内部讨论，不向外传阅                               │
│                                                               │
│  3. 使用阶段                                                  │
│     └── 向委托人口头说明要点，不提供书面                       │
│     └── 向被告人仅告知辩护策略，不透露具体预判                  │
│                                                               │
│  4. 存档阶段                                                  │
│     └── 标注保密等级，按保密材料归档                           │
│                                                               │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 向委托人沟通的脱敏处理

| 原始内容 | 脱敏后表述 |
|---------|-----------|
| 预判刑期3-5年 | 根据案件情况，可能判处有期徒刑 |
| 无罪概率较低 | 案件存在一定难度，我们会尽力辩护 |
| 缓刑可能性大 | 具备争取缓刑的有利条件 |
| 死刑风险高 | 案件性质严重，量刑可能较重 |

## 六、保密事件应急处理

### 6.1 泄密事件等级

| 等级 | 定义 | 示例 |
|-----|------|------|
| 一级（严重） | 涉及国家秘密或造成严重后果 | 泄露涉密案件信息 |
| 二级（较重） | 泄露案件核心信息 | 量刑预判报告外泄 |
| 三级（一般） | 泄露一般案件信息 | 委托人信息外泄 |
| 四级（轻微） | 轻微违规 | 在公共场所讨论案件 |

### 6.2 应急处理流程

```
发现泄密 → 立即报告律所负责人 → 评估泄密等级
    ↓
┌───────────────────────────────────────┐
│ 一级/二级：                            │
│ • 立即采取补救措施                      │
│ • 向委托人说明情况                      │
│ • 必要时向司法机关报告                  │
│ • 配合相关调查                         │
│ • 保留证据                            │
└───────────────────────────────────────┘
    ↓
┌───────────────────────────────────────┐
│ 三级/四级：                            │
│ • 采取补救措施                         │
│ • 内部警示教育                         │
│ • 完善保密制度                         │
└───────────────────────────────────────┘
```

## 七、保密合规自查表

### 7.1 日常自查（每日）

- [ ] 离开工位时电脑已锁屏
- [ ] 案件材料已妥善存放
- [ ] 未在公共场所讨论案件
- [ ] 未通过非加密渠道传输敏感信息

### 7.2 阶段自查（每周）

- [ ] 案卷材料存放位置合规
- [ ] 电子文件备份和加密
- [ ] 会见笔录整理归档
- [ ] 对外沟通记录合规

### 7.3 结案自查（结案时）

- [ ] 所有材料已归档
- [ ] 废弃材料已销毁
- [ ] 电子文件已清理
- [ ] 委托人确认无遗留问题
