## 五、《刑法》第306条详解

### 5.1 条文内容

> 在刑事诉讼中，辩护人、诉讼代理人毁灭、伪造证据，帮助当事人毁灭、伪造证据，威胁、引诱证人违背事实改变证言或者作伪证的，处三年以下有期徒刑或者拘役；情节严重的，处三年以上七年以下有期徒刑。

### 5.2 构成要件分析

| 要件 | 内容 |
|-----|------|
| 主体 | 辩护人、诉讼代理人 |
| 主观 | 故意 |
| 客观行为 | 毁灭伪造证据/帮助毁灭伪造/威胁引诱证人 |
| 发生阶段 | 刑事诉讼中 |

### 5.3 风险防范要点

| 风险行为 | 安全做法 |
|---------|---------|
| 向证人了解情况 | 两人以上、制作笔录、告知权利义务 |
| 核实被告人供述 | 如实记录、不引导不暗示 |
| 发现有利证据 | 申请司法机关调取、保全证据 |
| 会见被告人 | 不透露他人供述、不教唆翻供 |

## 六、庭审风险防范指引

### 6.1 发言风险控制

| 场景 | 风险行为 | 安全做法 |
|-----|---------|---------|
| 发问阶段 | 诱导被告人翻供 | 客观发问，不暗示答案 |
| 质证阶段 | 人身攻击公诉人 | 针对证据发表意见 |
| 辩论阶段 | 攻击司法机关 | 基于事实和法律辩护 |
| 最后陈述 | 代替被告人陈述 | 提示被告人自行陈述 |

### 6.2 与法官沟通风险控制

| 风险行为 | 安全做法 |
|---------|---------|
| 私下请托 | 正式场合书面沟通 |
| 利益输送暗示 | 仅就案件事实和法律问题沟通 |
| 施加不当压力 | 尊重司法独立 |

### 6.3 与公诉人关系风险控制

| 风险行为 | 安全做法 |
|---------|---------|
| 庭下交易 | 依法沟通辩护意见 |
| 人身攻击 | 保持职业尊重 |
| 恶意对抗 | 理性辩护 |

## 七、风险自查清单

### 7.1 会见前自查

- [ ] 是否携带违禁物品
- [ ] 委托手续是否齐全
- [ ] 是否存在利益冲突
- [ ] 会见内容是否合规

### 7.2 阅卷后自查

- [ ] 案卷材料是否妥善保管
- [ ] 是否向不当人员泄露
- [ ] 复制材料是否合规
- [ ] 是否在网络上传播

### 7.3 取证前自查

- [ ] 是否向被害方调查
- [ ] 是否有两人以上
- [ ] 是否制作规范笔录
- [ ] 是否告知证人权利义务

### 7.4 庭审前自查

- [ ] 辩护策略是否与被告人沟通一致
- [ ] 是否存在利益冲突
- [ ] 证据材料是否合法取得
- [ ] 发言内容是否合规

### 7.5 庭审中自查

- [ ] 是否服从法庭指挥
- [ ] 发言是否文明规范
- [ ] 是否涉及不当言论
- [ ] 是否遵守法庭纪律
