## 操作指引159：犯罪嫌疑人、被告人逃匿、死亡案件违法所得的没收程序

**操作指引159：律师可以接受当事人的委托，担任犯罪嫌疑人、被告人逃匿、死亡案件违法所得案件的代理人，参加法院的诉讼活动。**

【解析】这是新《刑事诉讼法》规定的新程序，在操作层面上，律师应当把握以下几个问题：

### 1. 哪些案件适用"违法所得没收程序"？

根据新《刑事诉讼法》的规定，有两种情况：

**情况一**：贪污贿赂犯罪、恐怖活动犯罪等重大犯罪案件，犯罪嫌疑人、被告人逃匿，在通缉一年后不能到案的；

**情况二**：贪污贿赂犯罪、恐怖活动犯罪等重大犯罪案件，犯罪嫌疑人、被告人死亡的。

这里规定的案件类型必须是"贪污贿赂犯罪、恐怖活动犯罪等重大犯罪"，对一般的犯罪不适用。

根据《最高院刑诉法解释》，**"重大犯罪案件"**是指：
1. 犯罪嫌疑人、被告人可能被判处无期徒刑以上刑罚；
2. 案件在本省、自治区、直辖市或者全国范围内有较大影响的；
3. 其他重大犯罪案件。

一般而言，没收程序所适用的案件是指那些涉案金额巨大，涉及的违法所得金额巨大，有必要提起单独的没收违法所得诉讼程序的案件。

### 2. "违法所得没收程序"如何启动？

与刑事诉讼起诉自然人的程序一样，先是由侦查机关向审查起诉机关提出《没收违法所得意见书》，由检察院向法院提出《没收违法所得申请》，人民法院受案后，由犯罪嫌疑人、被告人居住地的中级人民法院组成合议庭，应当先予公告，公告期为6个月，公告期满，合议庭应当进行审理。

**程序启动流程**：
```
侦查机关《没收违法所得意见书》
    ↓
检察院《没收违法所得申请》
    ↓
人民法院受案（中级人民法院）
    ↓
公告（公告期6个月）
    ↓
公告期满后审理
```

在此期间，犯罪嫌疑人、被告人的近亲属和其他利害关系人可以申请诉讼，也可以委托律师作为诉讼代理人参加诉讼。

**裁判结果**：
- 经过审理，人民法院对案件事实清楚，证据确实充分，申请没收的财产确属违法所得及其他涉案财产的，除依法返还被害人的以外，应当裁定没收；
- 不属于的，应当裁定驳回申请。

### 3. 律师在"违法所得没收程序"中有何作用？

**作用一**：律师可以接受犯罪嫌疑人、被告人近亲属或者利害关系人的委托，担任其诉讼代理人，参与违法所得没收案件的审理，维护当事人的合法权益。

**作用二**：律师可以就涉案财产是否属于案件违法所得进行调查和研究，提出意见。

根据《最高院刑诉法司法解释》，**违法所得及其他涉案财产**是指"实施犯罪行为所取得的财物及其孳息，以及被告人非法持有的违禁品、供犯罪所用的本人财物"。

因此，律师要运用自己的专业知识，查阅证据材料，对标的物进行审查，维护当事人的合法权益。

## 操作指引160：依法不负刑事责任的精神病人的强制医疗程序

**操作指引160：律师可以接受强制医疗案当事人的委托，担任诉讼代理人，参加依法不负刑事责任的精神病人的强制医疗程序。**

【解析】这是新《刑事诉讼法》规定的特别程序之一。

### 法律框架

在我国《刑法》中，精神病人：
- **完全丧失辨认与控制能力的**：不承担刑事责任；
- **在行为时尚未完全丧失辨认与控制能力**：应当承担刑事责任，但可以从轻或减轻处罚。

对于不承担刑事责任的精神病人，新《刑事诉讼法》确立了"强制医疗程序"，经法定程序，将精神病人送强制医疗机构强制医疗。

律师代理此类案，有几个问题需要把握。

### 1. 强制医疗案件的基本诉讼程序？

类似自然人犯罪的刑事诉讼程序，也是按照侦查、审查起诉、一审和二审的程序进行。

**适用对象**：实施暴力行为，危害公共安全或者严重危害公民人身安全，社会危害性已达到犯罪程度，但经法定程序鉴定其为不负刑事责任的精神病人。

**程序流程**：
```
侦查阶段
公安机关《强制医疗意见书》
    ↓
审查起诉阶段
人民检察院审查 → 《强制医疗的申请》
    ↓
审判阶段
人民法院受理 → 组成合议庭审理 → 决定强制医疗
    ↓
强制医疗期间
被强制医疗人的近亲属有权申请解除强制医疗
```

### 2. 谁可以委托律师参与强制医疗程序？

可以委托律师的主体：
1. 被申请人或被告人；
2. 被申请人或被告人的法定代理人；
3. 被害人；
4. 被害人死亡的，其近亲属可以委托；
5. （法律援助指派）

**需要注意的是，双方都可以委托律师参与强制医疗程序。**

### 3. 律师在强制医疗程序中可以做些什么工作？

律师在代理此类案件中的代理目标，因人而异，因案而异。

**"因人而异"**：是指当事人的要求，还要看是哪一方的当事人，不同方的当事人要求是不一样的。

**"因案而异"**：是指重罪还是轻罪。

#### 从被申请人一方的利益来看，有以下几种情况：

**情况一：重罪案件 → 一般追求强制医疗目标**

比如故意杀人罪，被告人可能被判处死刑的，如果被告人被鉴定为不负刑事责任的精神病人，可以不被判处极刑，而到强制医疗机构去强制医疗，还有希望治愈成正常人。

律师如果接受委托，则要尽力去收集被申请人的病历资料，申请司法机关对被申请人进行精神病鉴定。在法院决定强制医疗后，可以要求对被申请人定期诊疗，解除强制医疗措施。

**情况二：轻罪案件 → 一般不希望强制医疗**

如果是轻罪案件，量刑本来就较轻，就算有精神病，家人也可以选择相关医院进行治疗，因此，家属是不希望被告人被强制医疗的。

律师就要从被申请人没有精神病的方向提供专业服务。

**情况三：不构成犯罪情形 → 反对进入强制医疗程序**

如果被申请人本人的行为危害性根本没有达到犯罪的程度，或者其有正当防卫、紧急避险等法定不负刑事责任的事由，那么根本就不需要进入强制医疗程序。

律师应该着重就被申请人是否构成犯罪的角度进行辩护。

#### 从被害人这一方看，也有以下两种情况：

**情况一：严重加害 → 反对认定精神病**

被害人被严重加害，需要对被告人处以严厉的刑罚甚至极刑的，一般要反对被告人被认定为精神病人，反对其进入强制医疗程序。

因为如果没有精神病，被告人可能被判处极刑，而有精神病，可能住院治疗一段时间治好就回归社会了，被害人情感上不能接受。

**情况二：轻罪加害 → 希望强制医疗**

被告人罪行不严重，如果有精神病，那么就要被强制医疗，有些被害人是希望被告人被强制医疗的。

因为刑罚有期限，而被强制医疗的话，以病情康复为标准，期限可能要比刑罚长得多。

### 4. 律师代理此类案件必备的三项能力

笔者认为，在代理此类案件过程中，律师一定要具备三项能力：

1. **法医学专业知识**：较为丰富的法医学专业知识，特别是精神病学方面的专业知识；
2. **程序规定掌握**：熟练掌握强制医疗程序规定，在各个环节都能抓住机会，提出该提出的意见；
3. **职业道德**：有良好的人品，一定要坚持实事求是，坚守职业道德，用专业知识说话，用材料和鉴定意见说话。
