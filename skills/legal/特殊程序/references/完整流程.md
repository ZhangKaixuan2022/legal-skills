### 1.2 触发条件与路由入口

#### 触发条件一：违法所得没收程序

```
用户提及以下情形之一：
- 犯罪嫌疑人/被告人逃匿、外逃
- 犯罪嫌疑人/被告人死亡
- 贪污贿赂/恐怖活动等重大犯罪案件的财产处置
- 收到《没收违法所得申请》
→ 进入模块二
```

#### 触发条件二：强制医疗程序

```
用户提及以下情形之一：
- 精神病人犯罪/涉嫌犯罪
- 收到《强制医疗申请》
- 被强制医疗人申请解除
- 精神病鉴定相关问题
→ 进入模块三
```

#### 触发条件三：再审/申诉程序

```
用户提及以下情形之一：
- 二审维持原判后申诉
- 生效判决确有错误
- 发现新证据
- 申请再审/申诉
→ 进入模块四
```

## 模块二：违法所得没收程序

> 📖 完整操作指引 → reference/特殊程序操作指引.md#操作指引159

### 2.1 适用案件范围

#### 案件类型要求

**必须是"贪污贿赂犯罪、恐怖活动犯罪等重大犯罪"**，一般犯罪不适用。

#### 适用情形（两种）

| 情形 | 具体条件 |
|-----|---------|
| 情形一 | 犯罪嫌疑人、被告人**逃匿**，在通缉**一年后**不能到案 |
| 情形二 | 犯罪嫌疑人、被告人**死亡** |

#### "重大犯罪案件"定义（最高院解释）

根据《最高人民法院关于适用〈中华人民共和国刑事诉讼法〉的解释》：

| 序号 | 情形 | 说明 |
|-----|------|------|
| 1 | 可能被判处无期徒刑以上刑罚 | 刑期标准 |
| 2 | 在省级或全国范围内有较大影响 | 影响力标准 |
| 3 | 其他重大犯罪案件 | 兜底条款 |

**实务判断**：涉案金额巨大，违法所得金额巨大，有必要提起单独的没收违法所得诉讼程序的案件。

### 2.2 程序启动流程

```
┌─────────────────────────────────────────────────────────┐
│                   违法所得没收程序流程                    │
├─────────────────────────────────────────────────────────┤
│  侦查阶段                                                │
│  侦查机关 → 《没收违法所得意见书》                        │
│      ↓                                                   │
│  审查起诉阶段                                            │
│  检察院 → 《没收违法所得申请》                           │
│      ↓                                                   │
│  受案阶段                                                │
│  人民法院受案（犯罪嫌疑人/被告人居住地中级人民法院）       │
│      ↓                                                   │
│  公告阶段                                                │
│  法院公告（公告期6个月）                                 │
│  ※ 近亲属/利害关系人可申请参加诉讼                       │
│  ※ 可委托律师作为诉讼代理人                              │
│      ↓                                                   │
│  审理阶段                                                │
│  公告期满 → 合议庭审理                                   │
│      ↓                                                   │
│  裁定阶段                                                │
│  裁定没收 / 裁定驳回申请                                 │
└─────────────────────────────────────────────────────────┘
```

### 2.3 参与人员与委托资格

**可以委托律师的主体**：
1. 犯罪嫌疑人、被告人的**近亲属**
2. **其他利害关系人**（如财产共有人、债权人等）

**律师身份**：诉讼代理人（非辩护人）

**委托时机**：公告期间（6个月内）

### 2.4 裁决标准

| 裁决类型 | 适用条件 |
|---------|---------|
| **裁定没收** | 案件事实清楚，证据确实充分，申请没收的财产确属违法所得及其他涉案财产 |
| **裁定驳回** | 申请没收的财产不属于违法所得及其他涉案财产 |

**例外情形**：依法应当返还被害人的财产，不在没收范围内。

### 2.5 违法所得界定

根据《最高人民法院关于适用〈中华人民共和国刑事诉讼法〉的解释》：

**违法所得及其他涉案财产**包括：
1. 实施犯罪行为所取得的财物
2. 上述财物的**孳息**
3. 被告人非法持有的**违禁品**
4. **供犯罪所用的本人财物**

### 2.6 律师的作用（3项）

| 序号 | 作用 | 具体内容 |
|-----|------|---------|
| 1 | 代理参与诉讼 | 接受近亲属或利害关系人委托，担任诉讼代理人，参与审理 |
| 2 | 财产审查 | 就涉案财产是否属于违法所得进行调查和研究 |
| 3 | 提出意见 | 运用专业知识，查阅证据材料，对标的物进行审查，提出代理意见 |

### 2.7 代理工作要点

**接案阶段**：
```
□ 确认委托人身份（近亲属/利害关系人）
□ 了解案件基本情况
□ 签署委托协议
□ 办理诉讼代理手续
```

**阅卷阶段**：
```
□ 查阅《没收违法所得申请》
□ 审查申请没收的财产清单
□ 分析财产来源与性质
□ 识别合法财产与违法所得
```

**审理阶段**：
```
□ 出庭参加诉讼
□ 质证申请没收的财产证据
□ 提出代理意见
□ 强调合法财产权益
```

## 模块三：精神病人强制医疗程序

> 📖 完整操作指引 → reference/特殊程序操作指引.md#操作指引160

### 3.1 法律框架

#### 刑法关于精神病人刑事责任的规定

| 情形 | 刑事责任 | 处置方式 |
|-----|---------|---------|
| **完全丧失**辨认与控制能力 | 不承担刑事责任 | 强制医疗程序 |
| **尚未完全丧失**辨认与控制能力 | 承担刑事责任 | 可从轻或减轻处罚 |

#### 强制医疗程序的适用对象

实施暴力行为，**同时满足**以下条件：
1. 危害公共安全**或**严重危害公民人身安全
2. 社会危害性已达到**犯罪程度**
3. 经法定程序鉴定为**不负刑事责任的精神病人**

### 3.2 基本诉讼程序

```
┌─────────────────────────────────────────────────────────┐
│                   强制医疗程序流程                        │
├─────────────────────────────────────────────────────────┤
│  侦查阶段                                                │
│  公安机关《强制医疗意见书》                               │
│      ↓                                                   │
│  审查起诉阶段                                            │
│  人民检察院审查                                          │
│  → 确系不负刑事责任的精神病人应当给予强制医疗             │
│  → 《强制医疗的申请》                                    │
│      ↓                                                   │
│  审理阶段                                                │
│  人民法院受理 → 组成合议庭审理                           │
│  → 认定为不负刑事责任的精神病人                          │
│  → 决定强制医疗                                          │
│      ↓                                                   │
│  执行阶段                                                │
│  送强制医疗机构强制医疗                                  │
│      ↓                                                   │
│  解除阶段                                                │
│  近亲属有权申请解除强制医疗                              │
└─────────────────────────────────────────────────────────┘
```

### 3.3 委托资格（5类人员）

| 序号 | 委托主体 | 说明 |
|-----|---------|------|
| 1 | 被申请人或被告人 | 本人委托 |
| 2 | 被申请人/被告人的法定代理人 | 通常为近亲属 |
| 3 | 被害人 | 可以委托律师参与 |
| 4 | 被害人死亡的，其近亲属 | 代为委托 |
| 5 | 法律援助机构指派 | 法律援助案件 |

**重要提示**：被申请人一方和被害人一方**都可以**委托律师参与强制医疗程序。

### 3.4 双视角代理策略

#### 从被申请人一方代理

| 案件类型 | 代理目标 | 代理策略 |
|---------|---------|---------|
| **重罪案件** | 追求强制医疗 | 收集病历资料，申请精神病鉴定，争取强制医疗决定 |
| **轻罪案件** | 反对强制医疗 | 论证被申请人无精神病或不需要强制医疗 |
| **不构成犯罪** | 反对进入程序 | 论证行为不构成犯罪或有法定免责事由 |

**重罪案件代理要点**：
```
1. 尽力收集被申请人的病历资料
2. 申请司法机关对被申请人进行精神病鉴定
3. 在法院决定强制医疗后，要求定期诊疗
4. 条件成熟时申请解除强制医疗措施
```

**轻罪案件代理要点**：
```
1. 从被申请人没有精神病的方向提供服务
2. 申请重新鉴定或补充鉴定
3. 质疑鉴定程序或鉴定结论
4. 家属可选择自行送医治疗
```

**不构成犯罪情形代理要点**：
```
1. 论证行为危害性未达到犯罪程度
2. 论证存在正当防卫、紧急避险等免责事由
3. 主张不应进入强制医疗程序
4. 从是否构成犯罪角度进行辩护
```

#### 从被害人一方代理

| 加害程度 | 代理目标 | 代理策略 |
|---------|---------|---------|
| **严重加害** | 反对精神病认定 | 质疑鉴定，主张应承担刑事责任 |
| **轻罪加害** | 希望强制医疗 | 支持鉴定结论，主张强制医疗 |

**严重加害案件代理要点**：
```
原因：如果没有精神病，被告人可能被判处极刑；
      有精神病，可能治愈后回归社会
      
策略：
1. 反对被告人被认定为精神病人
2. 反对其进入强制医疗程序
3. 申请重新鉴定
4. 质疑鉴定程序合法性
```

**轻罪加害案件代理要点**：
```
原因：刑罚有期限，强制医疗以病情康复为标准
      期限可能比刑罚长得多
      
策略：
1. 支持精神病鉴定结论
2. 主张应当强制医疗
3. 强调社会危险性
```

### 3.5 律师必备能力（3项）

| 序号 | 能力要求 | 具体内容 |
|-----|---------|---------|
| 1 | 法医学知识 | 较为丰富的法医学专业知识，特别是**精神病学**方面 |
| 2 | 程序掌握 | 熟练掌握强制医疗程序规定，在各环节抓住机会提出意见 |
| 3 | 职业道德 | 坚持实事求是，用专业知识和鉴定意见说话 |

### 3.6 代理工作要点

**接案阶段**：
```
□ 确认委托人身份（被申请人方/被害人方）
□ 明确代理目标（支持/反对强制医疗）
□ 评估案件类型（重罪/轻罪）
□ 签署委托协议
```

**证据收集阶段**：
```
□ 收集被申请人的病历资料
□ 了解精神病鉴定情况
□ 审查鉴定程序合法性
□ 必要时申请重新鉴定
```

**审理阶段**：
```
□ 出庭参加诉讼
□ 对鉴定意见进行质证
□ 发表代理意见
□ 提出具体诉求
```

## 模块四：再审/申诉基础框架

> **注意**：参考书覆盖不足，本模块基于法条构建基础框架

### 4.1 法律依据

**《中华人民共和国刑事诉讼法》审判监督程序**：
- 第252条：当事人及其法定代理人、近亲属的申诉权
- 第253条：申诉应当提交的材料
- 第254条：人民法院、人民检察院提起再审的条件
- 第255条：再审案件的审理程序
- 第256条：再审不加刑原则
- 第257条：再审期间的强制措施
- 第258条：再审裁判

### 4.2 再审启动条件（5种法定事由）

根据《刑事诉讼法》第253条，当事人及其法定代理人、近亲属的申诉符合下列情形之一的，人民法院应当重新审判：

| 序号 | 法定事由 | 辩护要点 |
|-----|---------|---------|
| 1 | 有新的证据证明原判决、裁定认定的事实确有错误，可能影响定罪量刑 | 寻找新证据 |
| 2 | 据以定罪量刑的证据不确实、不充分、依法应当予以排除，或者证明案件事实的主要证据之间存在矛盾 | 证据审查 |
| 3 | 原判决、裁定适用法律确有错误 | 法律适用分析 |
| 4 | 违反法律规定的诉讼程序，可能影响公正审判 | 程序违法审查 |
| 5 | 审判人员在审理该案件的时候，有贪污受贿，徇私舞弊，枉法裁判行为 | 司法腐败线索 |

### 4.3 申诉受理程序

#### 申诉材料要求

```
1. 申诉状（载明当事人基本情况、申诉请求、申诉事实与理由）
2. 原一、二审判决书、裁定书等法律文书复印件
3. 申诉所依据的证据材料
4. 委托律师的，提交授权委托书
```

#### 申诉受理机关

| 受理机关 | 说明 |
|---------|------|
| 原审人民法院 | 可向作出生效裁判的法院申诉 |
| 上一级人民法院 | 可向上级法院申诉 |
| 人民检察院 | 可向检察院申诉，检察院可提出抗诉 |

#### 申诉处理流程

```
申诉人提交申诉材料
    ↓
法院/检察院审查
    ↓
    ├── 不符合再审条件 → 驳回申诉
    │
    └── 符合再审条件 → 决定再审
                           ↓
                      组成合议庭审理
                           ↓
                      作出再审裁判
```

### 4.4 辩护操作要点

#### 寻找原审未发现的关键证据

```
1. 审查原审证据是否完整
2. 调查是否存在未收集的有利证据
3. 联系证人是否有新的证言
4. 审查是否存在被隐匿的证据
5. 关注同案犯归案后的供述
```

#### 法律适用错误审查

```
1. 审查罪名认定是否准确
2. 审查量刑情节是否遗漏
3. 审查法律条款适用是否正确
4. 对比类案判决寻找差异
5. 关注司法解释或法律变化
```

#### 程序违法审查

```
1. 审查侦查程序是否合法
2. 审查证据收集程序是否合法
3. 审查审判程序是否合法
4. 审查是否存在非法证据
5. 审查辩护权是否得到保障
```

### 4.5 与二审Skill的衔接

**衔接场景**：上诉失利后的后续路径

```
二审维持原判
    ↓
分析是否符合再审条件
    ↓
    ├── 有新证据 → 准备再审申请
    │
    ├── 法律适用错误 → 准备再审申请
    │
    ├── 程序违法 → 准备再审申请
    │
    └── 无再审事由 → 告知申诉现实
```

**与当事人沟通要点**：
```
1. 如实告知申诉成功率极低
2. 不得承诺再审结果
3. 分析是否存在法定再审事由
4. 说明申诉途径和程序
5. 评估是否值得继续申诉
```

### 4.6 再审不加刑原则

根据《刑事诉讼法》第256条：

> 人民法院按照审判监督程序重新审判的案件，应当另行组成合议庭进行。如果原判决认定事实没有错误，量刑适当的，应当裁定驳回申诉或者抗诉；如果原判决认定事实没有错误，但适用法律有错误或者量刑不当的，应当撤销原判决，依法改判；如果原判决事实不清或者证据不足的，可以裁定撤销原判决，发回原审人民法院重新审理。

**重要原则**：
- 人民法院审理被告人或者其法定代理人、近亲属提起的申诉案件，不得加重被告人的刑罚
- 检察院抗诉的除外

## 模块五：执业风险防范

### 5.1 违法所得没收程序执业风险

| 风险点 | 风险描述 | 防范措施 |
|-------|---------|---------|
| 委托资格审查不严 | 委托人非近亲属或利害关系人 | 严格审查委托人身份 |
| 财产权属争议 | 对财产性质判断失误 | 充分调查财产来源 |
| 公告期限把握 | 错过参与诉讼时机 | 及时关注公告信息 |
| 利益冲突 | 同时代理多方利害关系人 | 做好利益冲突审查 |

### 5.2 强制医疗程序执业风险

| 风险点 | 风险描述 | 防范措施 |
|-------|---------|---------|
| 鉴定意见理解偏差 | 对精神病学知识不足 | 学习相关专业知识 |
| 代理目标定位错误 | 未准确把握当事人诉求 | 充分沟通确认目标 |
| 程序节点遗漏 | 错过关键程序时机 | 熟练掌握程序规定 |
| 职业道德风险 | 协助伪造精神病证据 | 坚守职业道德底线 |

### 5.3 再审/申诉程序执业风险

| 风险点 | 风险描述 | 防范措施 |
|-------|---------|---------|
| 虚假承诺 | 对再审结果作出承诺 | 如实告知成功率 |
| 证据造假 | 协助伪造新证据 | 严格审查证据来源 |
| 缠诉行为 | 无理由反复申诉 | 理性评估申诉条件 |
| 干扰司法 | 不当方式影响司法 | 依法行使申诉权利 |

### 5.4 通用执业伦理

#### 诚信原则
- 不得向当事人作出虚假承诺
- 如实告知案件前景和风险
- 不得夸大自身能力或关系

#### 忠诚原则
- 忠于当事人的合法利益
- 不得损害当事人利益
- 尊重当事人意愿

#### 保密原则
- 对执业过程中知悉的信息严格保密
- 不得向第三方泄露案件信息
- 档案资料妥善保管

#### 利益冲突审查
- 不得同时代理利益对立的当事人
- 在同一案件中不得代理双方
- 发现利益冲突应当及时披露

#### 法律援助
- 符合条件的案件应当提供法律援助
- 强制医疗案件被申请人未委托代理人的，应当通知法律援助机构指派

## 附录四：错误处理

### 常见错误场景及处理

| 错误场景 | 处理方式 |
|---------|---------|
| 案件类型不符合没收程序条件 | 告知不适用，引导至常规刑事程序 |
| 委托人无委托资格 | 告知委托资格要求，协助寻找合格委托人 |
| 公告期已过 | 告知已无法参与诉讼，分析救济途径 |
| 精神病鉴定存疑 | 申请重新鉴定或补充鉴定 |
| 不符合再审条件 | 如实告知，不鼓励无理由申诉 |
| 申诉材料不完整 | 指导补充完善申诉材料 |

### 风险提示场景

| 场景 | 风险提示 |
|-----|---------|
| 没收程序代理 | 注意财产权属调查的完整性 |
| 强制医疗代理 | 准确把握代理目标，因案制宜 |
| 再审/申诉 | 如实告知成功率极低，不得承诺结果 |
| 鉴定质证 | 需要专业知识支撑，必要时聘请专家 |
| 双方代理 | 严禁同时代理利益对立的双方 |
