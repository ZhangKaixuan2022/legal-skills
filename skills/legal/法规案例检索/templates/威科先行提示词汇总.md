# 威科先行检索提示词汇总

> 来源：法规案例检索Skill主文件  
> 用途：在正式检索前，自动生成结构化搜索提示词，供用户直接复制粘贴到威科先行平台进行高效检索

---

## 提示词生成依据

从前序步骤读取以下信息作为提示词生成依据：
- **案由**：从法律服务建议书/Excel获取
- **请求权路径**：各可行路径及构成要件
- **争议焦点**：共识要件与争议要件
- **法律关系类型**：合同关系、侵权关系等
- **审理法院**：确定地域检索范围
- **核心事实特征**：关键证据类型、交易方式等

---

## 一、法规检索提示词

**检索目的：** 精准定位请求权基础对应的法律条文

按请求权路径生成法条关键词组合：

**生成规则：**
- 核心法律名称 + 条款编号/关键词
- 使用AND连接必要条件，OR连接可选条件

**输出格式：**

```
【法规检索提示词 - 路径A：[请求权名称]】
用于搜索：该请求权的法律依据

"民法典" AND ("第五百七十七条" OR "违约责任" OR "继续履行")
```

```
【法规检索提示词 - 路径B：[请求权名称]】
用于搜索：备选路径的法律依据

"民法典" AND ("第一千一百六十五条" OR "侵权责任" OR "损害赔偿")
```

**常用法规检索模板：**

| 案由类型 | 推荐提示词模板 |
|---------|--------------|
| 合同纠纷 | `"民法典" AND "合同编" AND ("[具体条款]" OR "[争议关键词]")` |
| 劳动争议 | `("劳动法" OR "劳动合同法") AND "[具体条款/关键词]"` |
| 侵权纠纷 | `"民法典" AND "侵权责任编" AND "[侵权类型]"` |
| 公司纠纷 | `"公司法" AND ("[具体条款]" OR "[争议类型]")` |

## 二、类案检索提示词（核心）

**检索目的：** 从多维度搜索相似案例，提高命中率

按以下五个维度生成检索提示词，**按推荐优先级排序**：

### 维度1：案由维度（优先级：★★★★★）

**检索目的：** 快速定位同类案件，覆盖面最广

**生成规则：** 直接使用规范案由名称

```
【案由检索】
用于搜索：同案由的全部案件

案由：买卖合同纠纷
```

```
【扩展案由检索】
用于搜索：相关案由的案件（扩大搜索范围时使用）

案由：买卖合同纠纷 OR 产品责任纠纷
```

### 维度2：争议焦点维度（优先级：★★★★★）

**检索目的：** 精准定位争点相同的案例，参考价值最高

**生成规则：** 
- 每个争议焦点生成一组关键词
- 核心争议词 + 相关法律概念

```
【争议焦点检索 - 焦点1：[焦点名称]】
用于搜索：与"[焦点名称]"相关的裁判规则

"逾期交付" AND "违约金" AND ("合同解除" OR "继续履行")
```

```
【争议焦点检索 - 焦点2：[焦点名称]】
用于搜索：与"[焦点名称]"相关的裁判规则

"质量瑕疵" AND ("减价" OR "修理" OR "更换") AND "举证责任"
```

### 维度3：法律关系维度（优先级：★★★★☆）

**检索目的：** 搜索同类法律关系的处理方式

**生成规则：**
- 法律关系类型 + 权利义务关键词 + 法律效果

```
【法律关系检索】
用于搜索：同类法律关系的案件

"买卖合同" AND "质量瑕疵" AND ("减价" OR "退货")
```

```
【法律关系检索 - 变体】
用于搜索：类似法律关系的案件

"承揽合同" AND "工作成果" AND "质量不符"
```

### 维度4：关键事实维度（优先级：★★★☆☆）

**检索目的：** 搜索事实特征相似的案例，用于证据采信参考

**生成规则：**
- 证据类型 + 交易方式 + 特殊情形

```
【事实特征检索 - 证据类型】
用于搜索：涉及相同证据类型的案件认定规则

"微信聊天记录" AND ("口头约定" OR "合意") AND "借贷"
```

```
【事实特征检索 - 交易方式】
用于搜索：相同交易模式的案件

"分期付款" AND "所有权保留" AND "取回权"
```

```
【事实特征检索 - 特殊情形】
用于搜索：涉及特殊情形的案件

"疫情" AND "不可抗力" AND "合同履行"
```

### 维度5：裁判规则维度（优先级：★★★☆☆）

**检索目的：** 直接搜索特定裁判观点，用于法律适用参考

**生成规则：**
- "本院认为" + 法律概念 + 裁判结论关键词

```
【裁判规则检索 - 举证责任】
用于搜索：关于举证责任分配的裁判观点

"本院认为" AND "举证责任" AND "借贷关系"
```

```
【裁判规则检索 - 责任认定】
用于搜索：关于责任认定标准的裁判观点

"本院认为" AND "违约金过高" AND ("调整" OR "酌减")
```

```
【裁判规则检索 - 事实认定】
用于搜索：关于事实认定标准的裁判观点

"本院认为" AND "高度盖然性" AND "证据采信"
```

## 三、高级检索建议

**在威科先行平台使用时，建议配合以下筛选条件：**

### 1. 法院层级筛选建议

```
【推荐筛选顺序】
① 最高人民法院（指导性案例、公报案例优先）
② 高级人民法院（省级裁判规则参考）
③ 中级人民法院（同区域裁判倾向）
④ 基层人民法院（案件量大，需筛选优质案例）
```

### 2. 时间范围建议

```
【时间筛选策略】
- 首选：近3年（2023-2025）— 法律适用最新，裁判规则最具参考性
- 扩展：近5年（2021-2025）— 《民法典》施行后的案例
- 特殊：《民法典》施行前案例需注意法律变化

注意：涉及《民法典》的案例，2021年1月1日后的案例更具参考价值
```

### 3. 地域范围建议

```
【地域筛选策略 - 根据审理法院确定】
① 优先：[审理法院所在地] — 如：广东省深圳市
② 次选：[审理法院所在省份] — 如：广东省
③ 扩展：全国范围（当本地案例不足时）

如审理法院为：深圳市福田区人民法院
推荐筛选顺序：福田区法院 → 深圳中院 → 广东高院 → 全国
```

### 4. 案件类型建议

```
【案件类型筛选】
- 二审案件：法律适用论证更充分，优先参考
- 再审案件：涉及疑难问题，裁判规则更具指导性
- 一审案件：案件量大，需筛选高质量案例
```

## 使用说明

1. **一键复制**：每组提示词用代码块包裹，点击即可复制
2. **优先级指引**：★越多优先使用，命中率越高
3. **组合使用**：建议先用高优先级提示词检索，再用低优先级补充
4. **迭代优化**：根据检索结果调整关键词组合
