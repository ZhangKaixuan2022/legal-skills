# 立案管理Skill

## 功能概述

本Skill定位在民事一审诉讼体系的第5环节（承接"调查取证与证据管理"，衔接"案件讨论与提纲"），系统化指引立案全流程，确保立案材料齐备、管辖正确、程序合规，输出立案材料清单和跟踪管理记录。

### 核心功能

1. **立案材料核查**：根据案由自动匹配差异化材料清单，逐项核对前序文档
2. **管辖权预检**：根据案由和案件事实判断管辖规则，提示特殊案件管辖
3. **立案方式选择**：对比网上立案与线下立案优劣，提供操作指引
4. **立案后跟踪**：诉讼费计算、送达地址确认、举证期限管理、保全联动
5. **信息回写**：将案号、承办法官等信息同步当前事项系统记录

## 触发方式

### 被动触发

用户发送客户编号 + 提及以下关键词时自动触发：
- 【客户编号】+ "立案"
- 【客户编号】+ "去立案"
- 【客户编号】+ "立案准备"
- 【客户编号】+ "立案材料"
- 【客户编号】+ "网上立案"
- 【客户编号】+ "立案核查"

### 主动询问

当律师向AI提供起诉状、证据材料路径时：
1. 检测到用户提供了诉讼文书路径
2. 读取该路径下的起诉状、证据目录等材料
3. 读取当前事项系统记录中的案件信息
4. 比对已有材料与立案所需材料清单
5. 若存在缺失 → 主动提示："检测到以下材料缺失：【缺失清单】，是否开启立案准备流程？"
6. 用户确认后启动完整流程

### 步骤3：管辖权预检

#### 3.1 管辖规则判断

根据案由和案件事实，自动判断应适用的管辖规则：

**一般地域管辖（被告住所地）**：
- 适用：大多数民事案件
- 判断：被告户籍所在地或经常居住地
- 提示："本案应由被告住所地【具体法院】管辖"

**合同纠纷特殊管辖**：
- 合同履行地管辖
- 约定管辖检查（需审查合同中的管辖条款）
- 提示："检测到合同约定管辖条款，约定由【法院】管辖，请审查该约定是否有效"

**侵权纠纷管辖**：
- 侵权行为地（包括侵权行为实施地和侵权结果发生地）
- 被告住所地
- 提示："本案可由侵权行为地【地点】或被告住所地【地点】法院管辖"

**专属管辖**：
| 案件类型 | 管辖法院 | 法律依据 |
|---------|---------|---------|
| 不动产纠纷 | 不动产所在地法院 | 《民事诉讼法》第34条 |
| 港口作业纠纷 | 港口所在地法院 | 《民事诉讼法》第34条 |
| 继承遗产纠纷 | 被继承人死亡时住所地或主要遗产所在地法院 | 《民事诉讼法》第34条 |

#### 3.2 特殊案件管辖提示

| 案件类型 | 管辖规则 | 注意事项 |
|---------|---------|---------|
| 专利纠纷案件 | 知识产权法院/最高人民法院指定的中级人民法院 | 需确认当地是否有知识产权法庭 |
| 海事海商案件 | 海事法院专属管辖 | 全国仅10家海事法院 |
| 公益诉讼案件 | 中级人民法院管辖 | 需符合公益诉讼主体资格 |
| 证券虚假陈述赔偿 | 省、自治区、直辖市人民政府所在的市、计划单列市和经济特区中级人民法院 | 有专门管辖规定 |
| 破产案件 | 债务人住所地法院 | 可能涉及集中管辖 |

#### 3.3 管辖风险预警

**存在以下情形时提示风险**：
1. **约定管辖无效风险**：约定不明确、违反级别管辖或专属管辖
2. **管辖争议**：多个法院均有管辖权，对方可能提出管辖异议
3. **特殊主体**：被告为军队、铁路等特殊主体

**应对方案建议**：
- 约定管辖无效 → 按法定管辖重新确定
- 管辖争议 → 准备管辖异议答辩材料
- 特殊主体 → 查询专门管辖规定

#### 3.4 管辖权预检报告

**输出格式**：
```
管辖权预检报告

一、案件基本信息
案由：【案由】
当事人：【原告】诉【被告】

二、管辖分析
1. 被告住所地：【地址】→ 【法院名称】有管辖权
2. 特殊连接点：【如合同履行地/侵权行为地等】→ 【法院名称】有管辖权
3. 约定管辖：【如有，分析效力】

三、管辖建议
建议管辖法院：【法院名称】
理由：【简要说明】

四、风险提示
【如有管辖风险，在此提示】

五、应对方案
【针对风险的应对建议】
```

### 步骤4：立案方式选择

#### 4.1 立案方式对比

| 对比维度 | 网上立案 | 线下立案 |
|---------|---------|---------|
| 便捷性 | 随时随地提交，无需排队 | 需前往法院，可能排队 |
| 材料要求 | 需扫描件，对清晰度要求高 | 可携带原件现场核对 |
| 审核时间 | 通常3-7个工作日 | 当场或当日 |
| 补正机会 | 可在线补正，时间充裕 | 需再次前往法院 |
| 适合场景 | 材料齐全、时间充裕 | 材料复杂、需现场咨询 |
| 注意事项 | 注意扫描件格式和大小限制 | 携带原件备查 |

#### 4.2 网上立案操作指引

**常见平台**：
- 人民法院在线服务（微信小程序）
- 各地法院诉讼服务网
- 律师服务平台

**材料扫描要求**：
| 材料类型 | 格式要求 | 注意事项 |
|---------|---------|---------|
| 起诉状 | PDF/JPG | 需签字/盖章页扫描清晰 |
| 身份证明 | JPG | 正反面均需扫描 |
| 证据材料 | PDF | 按顺序合并为一个文件或分卷上传 |
| 授权手续 | PDF | 授权委托书、所函、律师证 |

**网上立案提示**：
> "网上立案材料扫描要求：
> 1. 所有材料需清晰可辨，建议300dpi以上分辨率
> 2. 单张图片建议不超过10MB
> 3. PDF文件建议不超过100MB
> 4. 起诉状需当事人签字/盖章后扫描
> 5. 证据材料建议按证据清单顺序编号"

#### 4.3 线下立案操作指引

**需携带原件清单**：
- 原告身份证明原件（核对用）
- 授权委托书原件
- 律师证原件（核对用）
- 证据材料原件（如法院要求核对）

**法院窗口注意事项**：
1. 提前确认立案庭工作时间
2. 准备被告工商档案信息复印件（如被告为公司）
3. 离婚案件须携带户口簿、结婚证原件及复印件
4. 填写送达地址确认书时务必认真，填写完毕拍照留存

**线下立案提示**：
> "线下立案注意事项：
> 1. 提前电话确认立案庭位置和办公时间
> 2. 准备足够份数的起诉状（被告人数+1-2份）
> 3. 被告为公司时，携带工商档案信息复印件
> 4. 送达地址确认书填写后拍照留存
> 5. 立案后索要案件受理通知书"

#### 5.1 诉讼费计算

| 标的额区间 | 计算公式 | 速算公式 |
|-----------|---------|---------|
| 不超过1万元 | 50元 | 50元 |
| 1万-10万元 | 标的额×2.5%-200元 | 标的额×2.5%-200 |
| 10万-20万元 | 标的额×2%+300元 | 标的额×2%+300 |
| 20万-50万元 | 标的额×1.5%+1300元 | 标的额×1.5%+1300 |
| 50万-100万元 | 标的额×1%+3800元 | 标的额×1%+3800 |
| 100万-200万元 | 标的额×0.9%+4800元 | 标的额×0.9%+4800 |

## 第六部分：注意事项

### 6.1 立案材料注意事项

1. **身份信息核对**：起诉状中原被告名称、身份证号必须与证明材料完全一致
2. **起诉状份数**：按被告人数+1-2份准备
3. **证据份数**：按法院要求准备，可能需要给每个被告单独准备
4. **送达地址确认书**：认真填写，填写完毕拍照留存
5. **特殊材料**：离婚案件带户口簿、结婚证；公司被告带工商档案

### 6.2 管辖权注意事项

1. **再次核实**：立案前再次核实管辖问题，不能仅根据级别或地域管辖判断
2. **特殊案件**：专利、海事、公益诉讼、证券赔偿有特殊规定
3. **约定管辖**：审查合同中的管辖条款效力
4. **管辖异议**：预判对方可能提出的管辖异议，提前准备应对

### 6.3 诉讼费注意事项

1. **计算准确**：严格按照《诉讼费用交纳办法》计算
2. **缴费期限**：收到缴费通知7日内缴费，逾期按撤诉处理
3. **退费处理**：调解结案、撤诉等情形可申请退费
4. **保全费**：财产保全需另行交纳保全费

### 6.4 举证期限注意事项

1. **及时记录**：收到举证通知书后立即记录截止日期
2. **提前准备**：举证期限届满前3天完成证据提交
3. **逾期风险**：逾期举证可能面临证据不被采纳、训诫、罚款
4. **延期申请**：确有困难可在期限届满前申请延期

### 6.5 网上立案注意事项

1. **扫描质量**：确保扫描件清晰可辨
2. **文件格式**：按平台要求准备PDF或图片格式
3. **文件大小**：注意单文件和总文件大小限制
4. **补正时限**：收到补正通知后及时补正
