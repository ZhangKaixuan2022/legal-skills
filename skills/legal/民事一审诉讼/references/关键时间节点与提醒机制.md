# 关键时间节点与提醒机制

## 一、民事诉讼关键节点确认

| 阶段 | 确认内容 | 确认方式 |
|-----|---------|---------|
| 法律分析 | 请求权路径选择（最优方案/备选方案） | 选项A/B/C |
| 委托合同 | 收费方式、代理权限、代理阶段 | 逐项确认 |
| 起诉状 | 当事人信息完整性、诉讼请求准确性 | 表格核对 |
| 争议焦点 | AI归纳的争议焦点是否准确 | 展示-确认/修改 |
| 庭审提纲 | 陈述要点、法官预案、法律依据 | 逐项确认 |
| 代理词 | 核心观点、法条引用、案例支撑 | 重点确认 |

## 二、破产程序关键时间节点

| 事件 | 期限 | 法律依据 | 提醒策略 |
|------|------|---------|---------|
| 法院受理决定 | 收到申请后15日 | 破产法第10条 | 提交后自动计算提醒 |
| 通知已知债权人 | 受理后10日 | 破产法第14条 | R1受理后立即提醒 |
| 债权申报期限 | 30-90日 | 破产法第45条 | 设置申报截止日提醒 |
| 合同处理决定 | 受理后2个月 | 破产法第18条 | R1接管后计算提醒 |
| 重整计划提交 | 6个月（可延3个月） | 破产法第79条 | 分段提醒 |
| 第一次债权人会议 | 申报期满后15日 | 破产法第62条 | 申报截止后提醒 |
| 追加分配 | 程序终结后2年内 | 破产法第123条 | 终结后设置提醒 |

## 三、民事诉讼法定期限

| 事项 | 期限 | 法律依据 | 备注 |
|------|------|---------|------|
| 一审上诉期限 | 判决送达后15日内 | 民诉法第171条 | 不服判决 |
| 裁定上诉期限 | 裁定送达后10日内 | 民诉法第171条 | 不服裁定 |
| 申请再审期限 | 判决/裁定生效后6个月内 | 民诉法第212条 | 特殊情形除外 |
| 举证期限 | 不少于15日 | 民诉法解释第99条 | 法院指定 |
| 答辩期 | 收到起诉状副本15日内 | 民诉法第128条 | 被告答辩 |
| 申请财产保全 | 诉前30日内起诉 | 民诉法第104条 | 诉前保全 |
| 执行申请期限 | 2年内 | 民诉法第246条 | 判决生效后 |

## 四、提醒时间策略

### 4.1 破产程序分段提醒

**重整计划提交（总计6个月+3个月延期）**：
- 第3个月：进度检查提醒
- 第5个月：临近预警
- 第6个月：最后期限警告
- 申请延期后：重新计算

**合同处理决定（2个月）**：
- 第1个月：开始处理提醒
- 第45天：半程提醒
- 第55天：期限预警

### 4.2 民事诉讼关键提醒

**上诉期限（15日）**：
- 判决送达当日：告知上诉期限
- 第7天：半程提醒
- 第12天：最后预警

**举证期限**：
- 确定举证期限后立即记录
- 届满前7日：提醒准备
- 届满前3日：最后预警

## 五、提醒方式

### 自动提醒触发
```
系统检测到以下节点即将到期：
┌─────────────────────────────────────────┐
│ 【紧急】上诉期限将于3日后届满            │
│ 案件：张先生 vs XX公司买卖合同纠纷       │
│ 判决送达日：2026年3月15日               │
│ 上诉截止日：2026年3月30日               │
│                                         │
│ 建议操作：                              │
│ □ 与当事人确认是否上诉                  │
│ □ 准备上诉状材料                        │
│ □ 计算二审诉讼费                        │
└─────────────────────────────────────────┘
```

### 期限预警样式
```
⚠️ 【期限预警】
案件：【客户姓名】破产清算案
事项：合同处理决定期限
剩余时间：5天
法律依据：《企业破产法》第18条

> 注意：逾期未通知对方当事人视为解除合同

[查看待处理合同清单] [延期说明] [已处理]
```

## 六、Excel字段与时间节点关联

| 字段名 | 用途 | 触发时机 |
|-------|------|---------|
| 首次咨询日期 | 计算案件时效 | 建档时写入 |
| 委托日期 | 计算服务周期 | 签约时写入 |
| 立案日期 | 计算举证期限起点 | 立案后写入 |
| 举证截止日期 | 触发举证期限提醒 | 立案后写入 |
| 开庭日期 | 触发庭前准备提醒 | 法院通知后写入 |
| 判决送达日期 | 计算上诉期限 | 收到判决后写入 |
| 上诉截止日期 | 触发上诉决策提醒 | 判决送达后自动计算 |
| 受理日期（破产） | 计算各项破产期限 | 法院裁定后写入 |
| 债权申报截止日 | 触发申报提醒 | 公告发布后写入 |
| 第一次债权人会议日期 | 触发会议准备提醒 | 确定后写入 |
