# Skill调用序列详解

## 一、子Skill完整清单（25个）

### 民事诉讼Skill（15个）

| 序号 | Skill名称 | 核心功能 |
|-----|----------|---------|
| 1 | **法律咨询助手** | 咨询入口路由，区分微信/线下咨询，新客户建档 |
| 2 | **案件沟通记录** | 对话式笔录格式，含附表（关键信息摘要+证据初步清单） |
| 3 | **初步法律分析** | 六来源体系+请求权基础分析，输出法律服务建议书 |
| 4 | **委托合同管理** | 利益冲突检查+阶段收费+诉讼风险告知 |
| 5 | **法规案例检索** | 三层检索法，输出法官版+律师版双文档 |
| 6 | **调查取证与证据管理** | 证据缺口分析、取证方案、调查令申请、财产保全 |
| 7 | **立案管理** | 立案材料核查、管辖权预检、诉讼费计算、立案后跟踪 |
| 8 | **诉讼可视化** | 6种Mermaid图表，输出法官版+律师版双报告 |
| 9 | **案件讨论与提纲** | 四维度AI自问自答分析、三方模拟法庭、生成案件提纲 |
| 10 | **诉讼文书起草** | 起诉状/答辩状/代理词/质证意见/证据目录/保全申请书 |
| 11 | **庭前准备** | 庭审提纲+攻防预案+举证策略+代理词+诉讼方案 |
| 12 | **庭审与庭后工作** | 庭审纪要、复盘分析、庭后代理词、法官沟通（支持多次开庭） |
| 13 | **调解与和解** | 调解可行性分析、调解方案、撤诉管理（任何阶段可触发） |
| 14 | **诉讼分析工具** | 判决书九层拆解+上诉/再审决策支持 |
| 15 | **结案归档** | 案卷归档、结案报告、客户移交、费用结算、客户回访 |

## 二、民事诉讼标准流程（原告方）

```
1. 法律咨询助手
   触发条件：无客户编号的新客户
   输出：客户编号（KH-年份-序号）、复盘台账记录
   衔接：自动进入案件沟通记录
    ↓
2. 案件沟通记录
   前置检查：Excel中有客户记录
   输出：案件沟通记录（含附表：关键信息摘要+证据初步清单）
    ↓
3. 初步法律分析
   前置检查：案件沟通记录存在
   输出：法律服务建议书（含请求权路径、争点清单、证据评估）
    ↓
4. 委托合同管理
   前置检查：法律服务建议书存在
   输出：委托代理合同（含利益冲突声明+阶段收费）、诉讼风险告知书、诉讼案件跟踪记录表
    ↓
5. 法规案例检索
   前置检查：法律服务建议书存在（获取请求权路径）、委托代理合同存在
   输出：法律法规及案例汇编（法官版）+ 法规案例检索报告（律师版）
    ↓
6. 调查取证与证据管理
   前置检查：法律服务建议书存在（获取请求权要件清单）、案件沟通记录存在
   输出：取证方案、补充材料清单、调查令申请书（如需要）、财产保全申请书（如需要）
   提示：可与诉讼文书起草并行或先后进行
    ↓
7. 立案管理
   前置检查：起诉状、证据目录、委托代理合同存在
   输出：立案材料准备清单、管辖权预检报告、诉讼费计算书、案件信息登记表
   提示：获取案号后主动提醒执行案件讨论与提纲
    ↓
8. 案件讨论与提纲
   前置检查：案号已获取，前序材料齐备
   输出：前序材料检查表、四维度分析报告、模拟法庭纪要、模拟法庭复盘报告、案件提纲
   特点：AI自主完成，律师只审阅最终输出
    ↓
9. 诉讼可视化
   前置检查：法律服务建议书存在、法规案例检索报告存在
   输出：诉讼可视化报告（法官版）+ 诉讼可视化分析报告（律师版）+ 6种图表
    ↓
10. 诉讼文书起草（起诉状）
   前置检查：法律服务建议书、法规案例汇编、诉讼可视化报告存在
   输出：民事起诉状 + 证据目录（可选：代理词、质证意见、保全申请书）
    ↓
11. 庭前准备
   前置检查：起诉状、案件提纲、法规案例检索报告、诉讼可视化分析报告、质证意见（如有）存在
   输出：庭前准备材料（律师版，含庭审提纲+攻防预案矩阵+举证策略+开庭陈述提纲+法庭辩论提纲+代理词+诉讼法律服务方案）
    ↓
12. 【等待开庭】
    ↓
13. 庭审与庭后工作
   触发条件：用户确认"已开庭"
   输出：庭审纪要、庭审复盘报告、庭后代理词、补充材料清单、法官沟通记录
   特点：支持多次开庭，自动编号和关联前次记录
    ↓
14. 【判决后】诉讼分析工具
   前置检查：判决书存在
   输出：诉讼分析报告（含判决书九层拆解、上诉可行性评估、再审事由检索、败诉原因分析、补救策略建议）
    ↓
15. 结案归档
   触发条件：判决生效或调解完成
   输出：案卷归档目录、案件结案报告、判决结果对比表、案卷移交清单、费用结算提醒清单、客户回访记录
   终态：Excel案件状态更新为"已结案"
```

## 三、民事诉讼被告方流程

```
1-3步：与原告方相同（法律咨询 → 案件沟通 → 初步法律分析）
    ↓
4. 委托合同管理（同上）
    ↓
5. 法规案例检索（同上）
    ↓
6. 诉讼文书起草（答辩状）
   前置检查：对方起诉状、法律服务建议书、法规案例汇编存在
   输出：民事答辩状 + 证据目录 + 质证意见（可选：代理词、保全申请书）
    ↓
7. 庭前准备
   前置检查：答辩状、法律服务建议书、法规案例检索报告、诉讼可视化分析报告、质证意见存在
   输出：庭前准备材料（律师版）
    ↓
后续步骤同原告方...
```

## 四、横向Skill：调解与和解

```
调解与和解
   触发条件：任何阶段，用户提及"调解"/"和解"/"撤诉"
   输出：调解方案确认函、和解协议审查清单、撤诉申请书
   衔接：调解成功→结案归档；调解失败→返回原流程
```

## 八、流程衔接提示表

| 当前阶段完成 | 下一步提示 | 可触发Skill |
|-------------|-----------|------------|
| 诉讼文书起草完成 | 是否需要进行调查取证与证据管理？ | 调查取证与证据管理 |
| 调查取证完成 | 是否准备立案？建议进行立案管理 | 立案管理 |
| 立案完成（获取案号） | 建议执行案件讨论与提纲，进行庭前深度分析 | 案件讨论与提纲 |
| 案件讨论完成 | 是否制作诉讼可视化？ | 诉讼可视化 |
| 诉讼可视化完成 | 是否起草起诉状/答辩状？ | 诉讼文书起草 |
| 诉讼文书起草完成 | 是否进行庭前准备？ | 庭前准备 |
| 庭前准备完成 | 提示等待开庭，开庭后启动庭审与庭后工作 | 庭审与庭后工作 |
| 庭审完成 | 根据情况提示调解与和解或等待判决 | 调解与和解 |
| 判决生效/调解完成 | 是否启动结案归档流程？ | 结案归档 |
| 任何阶段 | 对方提出调解意向或需要评估调解可行性 | 调解与和解（横向） |

## 十、破产衍生诉讼衔接

破产程序中可能产生多种民事诉讼，这些诉讼可复用现有民事一审Skill体系：

| 诉讼类型 | 复用Skill | 文档存放路径 |
|---------|----------|-------------|
| 债权确认之诉 | 诉讼文书起草、庭前准备等 | `07-破产衍生诉讼/债权确认之诉/` |
| 撤销权之诉 | 诉讼文书起草、庭前准备等 | `07-破产衍生诉讼/撤销权之诉/` |
| 取回权之诉 | 诉讼文书起草、庭前准备等 | `07-破产衍生诉讼/取回权之诉/` |
| 追收债权之诉 | 诉讼文书起草、庭前准备等 | `07-破产衍生诉讼/追收债权之诉/` |
| 追缴出资之诉 | 诉讼文书起草、庭前准备等 | `07-破产衍生诉讼/追缴出资之诉/` |
