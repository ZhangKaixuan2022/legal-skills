# 死刑案件执业风险清单

> **适用范围**：死刑案件全流程（一审/二审/死刑复核）
> **使用说明**：辩护人在办理死刑案件时应对照本清单进行风险防范

---

## 一、死刑案件特有风险

### 1.1 被害人家属情绪激烈风险

| 风险等级 | 高风险 |
|---------|--------|
| 风险表现 | 被害人家属对辩护律师的敌意、法庭上的情绪爆发、法庭外的冲突 |
| 触发场景 | 法庭辩护、赔偿协商、出入法院 |

**防范措施**：

```
□ 法庭辩护时注意措辞，被害人过错点到为止
□ 详细的辩护意见以书面形式提交法庭
□ 避免与被害人家属正面冲突
□ 私下协商时选择公共场所或司法机关
□ 必要时请求法警保护
□ 出入法院时注意观察周围环境
```

### 1.2 人身安全风险

| 风险等级 | 高风险 |
|---------|--------|
| 风险表现 | 遭受人身攻击、恐吓威胁 |
| 触发场景 | 出入法院、私下协商赔偿、案件高关注度时 |

**防范措施**：

```
□ 尽量在公共场所活动
□ 避免单独与情绪激动的被害人家属接触
□ 出入法院注意安全
□ 必要时申请司法保护
□ 保留威胁恐吓的证据
□ 向律协和司法行政部门报告
```

### 1.3 舆论压力风险

| 风险等级 | 中高风险 |
|---------|---------|
| 风险表现 | 社会舆论攻击、"为坏人辩护"的道德指责、网络暴力 |
| 触发场景 | 社会关注度高的死刑案件 |

**防范措施**：

```
□ 坚持依法辩护，不受舆论干扰
□ 不回应非理性的攻击
□ 不主动接受媒体采访（除非必要且经律所同意）
□ 必要时通过律协发表专业声明
□ 保留网络暴力证据，必要时依法维权
□ 寻求律协和同行支持
```

### 1.4 证据处理风险

| 风险等级 | 高风险 |
|---------|--------|
| 风险表现 | 证据处理不当影响案件结果，可能影响被告人生命权利 |
| 触发场景 | 阅卷、调查取证、证据分析 |

**防范措施**：

```
□ 极度审慎对待每一份证据
□ 多次复核、交叉验证
□ 重大问题集体讨论
□ 保留详细的工作底稿
□ 证据材料妥善保管
□ 不得毁损、伪造、隐匿证据
```

## 二、各阶段风险清单

### 2.1 一审阶段风险

| 风险类型 | 风险描述 | 防范措施 |
|---------|---------|---------|
| 阅卷风险 | 复制传播案卷、泄露案件信息 | 严格保管、禁止外传 |
| 会见风险 | 传递违禁物品/信息 | 规范会见、如实记录 |
| 取证风险 | 引诱威胁证人 | 规范取证、全程留痕 |
| 庭审言论风险 | 发表不当言论、刺激被害人家属 | 点到为止、依据事实 |
| 策略风险 | 未经同意改变辩护策略 | 充分沟通、尊重意愿 |

### 2.2 二审阶段风险

| 风险类型 | 风险描述 | 防范措施 |
|---------|---------|---------|
| 时间风险 | 错过上诉期、延误提交材料 | 严格管理时间节点 |
| 衔接风险 | 一审材料交接不完整 | 核对材料清单 |
| 和解风险 | 和解协商失败或引发冲突 | 谨慎操作、法院协调 |
| 沟通风险 | 与法官沟通不当 | 把握节奏、有新内容再沟通 |

### 2.3 死刑复核阶段风险

| 风险类型 | 风险描述 | 防范措施 |
|---------|---------|---------|
| 介入风险 | 未及时介入复核程序 | 主动跟踪、尽早提交 |
| 会见风险 | 复核期间会见受阻 | 依法主张、向法院反映 |
| 意见风险 | 辩护意见未被重视 | 详尽充分、多渠道反映 |

## 三、红线清单（绝对禁止）

### 3.1 证据相关红线

```
✗ 毁灭、伪造证据
✗ 帮助当事人毁灭、伪造证据
✗ 威胁、引诱证人违背事实改变证言或作伪证
✗ 故意隐匿对当事人不利的证据
✗ 泄露案件证据材料
```

### 3.2 会见相关红线

```
✗ 为被告人传递违禁物品
✗ 为被告人传递与案件有关的信息
✗ 帮助被告人与外界串通
✗ 接受被告人委托携带现金或贵重物品
✗ 违规拍摄、录音、录像
```

### 3.3 执业相关红线

```
✗ 向当事人承诺案件结果
✗ 向办案人员行贿
✗ 私自收费
✗ 接受对方当事人财物或其他利益
✗ 利用案件信息谋取私利
```

### 3.4 言行相关红线

```
✗ 在法庭上辱骂、诽谤他人
✗ 扰乱法庭秩序
✗ 发表诋毁司法机关的言论
✗ 在案件审理期间发表有损公正审判的言论
✗ 诋毁被害人人格
```

## 四、风险预警机制

### 4.1 高风险预警（立即处理）

| 预警情形 | 应对措施 |
|---------|---------|
| 被害人家属情绪极度激动、有攻击倾向 | 请求司法保护、避免直接接触 |
| 遭受人身威胁 | 报警、向律协和司法局报告 |
| 案件社会关注度极高、舆论压力巨大 | 谨慎发言、请示律所、必要时聘请公关顾问 |
| 发现案件存在重大冤假错案风险 | 详细记录、书面报告、保留证据 |

### 4.2 中风险预警（密切关注）

| 预警情形 | 应对措施 |
|---------|---------|
| 被害人家属拒绝和解、态度强硬 | 调整方案、寻求法官协调 |
| 辩护效果不佳、法院不重视辩护意见 | 及时调整策略、书面补充、多渠道反映 |
| 媒体关注案件、记者采访 | 不擅自接受采访、请示律所 |

### 4.3 一般风险预警（持续监控）

| 预警情形 | 应对措施 |
|---------|---------|
| 沟通不畅 | 变换沟通方式、寻求协助 |
| 被告人情绪不稳定 | 心理疏导、耐心沟通 |
| 家属期望值过高 | 如实告知风险、管理预期 |

## 五、应急处置预案

### 5.1 遭遇人身攻击

```
步骤一：确保自身安全
├── 迅速撤离现场
├── 寻求周围人帮助
└── 必要时报警

步骤二：保留证据
├── 拍照或录像
├── 保留伤情证据
└── 记录在场证人

步骤三：报告
├── 向律所报告
├── 向律协报告
├── 向司法行政部门报告
└── 必要时向媒体说明情况

步骤四：依法维权
├── 申请司法保护
├── 追究侵权责任
└── 寻求法律援助
```

### 5.2 遭遇舆论攻击

```
步骤一：评估影响
├── 评估舆论规模
├── 评估对案件的影响
└── 评估对个人的影响

步骤二：谨慎应对
├── 不回应非理性攻击
├── 保留网络暴力证据
└── 请示律所意见

步骤三：必要时回应
├── 通过律协发表声明
├── 强调依法辩护的正当性
└── 呼吁尊重法治

步骤四：寻求支持
├── 律协支持
├── 同行支持
├── 心理疏导
└── 必要时法律维权
```

### 5.3 和解协商破裂

```
步骤一：分析原因
├── 分析破裂原因
├── 评估挽回可能性
└── 调整协商策略

步骤二：寻求协调
├── 请求法官协调
├── 寻求第三方调解
└── 调整赔偿方案

步骤三：调整辩护策略
├── 强调其他从轻情节
├── 加强事实和证据辩护
└── 书面补充辩护意见

步骤四：告知当事人
├── 如实告知和解破裂
├── 说明可能的后果
└── 确认后续辩护策略
```

## 六、执业风险自查表

### 6.1 接案阶段自查

```
□ 是否充分评估自身能力
□ 是否如实告知当事人案件风险
□ 是否未对案件结果作出承诺
□ 委托手续是否完备
□ 收费是否规范
```

### 6.2 办案阶段自查

```
□ 阅卷是否详细、是否保管好案卷材料
□ 会见是否规范、是否如实记录
□ 调查取证是否合法
□ 是否与当事人充分沟通辩护策略
□ 是否尊重当事人意愿
□ 辩护意见是否有事实和法律依据
```

### 6.3 庭审阶段自查

```
□ 是否遵守法庭纪律
□ 言论是否适当、是否避免刺激被害人家属
□ 详细辩护意见是否书面提交
□ 是否注意人身安全
```

### 6.4 结案阶段自查

```
□ 是否妥善保管案件材料
□ 是否及时告知当事人案件结果
□ 是否依法告知救济途径
□ 是否做好案件总结
```
