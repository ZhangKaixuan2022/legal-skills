# 死刑复核程序详细说明

## 一、法律依据

**《刑事诉讼法》第246-251条**

| 条款 | 主要内容 |
|-----|---------|
| 第246条 | 死刑由最高人民法院核准 |
| 第247条 | 中级人民法院判处死刑的，高级人民法院复核同意后报最高人民法院核准 |
| 第248条 | 复核死刑案件，应当讯问被告人，辩护律师提出要求的，应当听取辩护律师的意见 |
| 第249条 | 最高人民法院复核死刑案件，应当作出核准或者不核准死刑的裁定 |
| 第250条 | 最高人民法院复核死刑案件，对于不核准死刑的，可以发回重新审判或者予以改判 |
| 第251条 | 死刑缓期执行的复核 |

---

## 二、死刑复核的基本程序

### （一）最高人民法院核准死刑（死刑立即执行）

**程序流程**：

```
中级人民法院一审判处死刑
    ↓
被告人上诉或自动进入二审
    ↓
高级人民法院二审维持死刑
    ↓
高级人民法院复核同意
    ↓
报最高人民法院核准
    ↓
最高人民法院审查
    ↓
作出核准/不核准裁定
```

### （二）高级人民法院复核死缓

**适用情形**：死刑缓期执行案件

**程序流程**：中级人民法院判处死缓 → 二审 → 高级人民法院核准死缓

---

## 三、律师介入方式

### （一）介入时机

| 阶段 | 律师介入方式 |
|-----|------------|
| 二审结束后 | 继续委托或新委托 |
| 案件报送复核后 | 向复核法院提交辩护手续 |
| 复核审查期间 | 提交书面辩护意见 |

### （二）提交书面辩护意见

**《刑事诉讼法》第248条**：
> 复核死刑案件，应当讯问被告人，**辩护律师提出要求的，应当听取辩护律师的意见**。

**操作要求**：

```
1. 提交时机
   - 案件报送复核后尽快提交
   - 不等待通知，主动提交

2. 提交方式
   - 书面形式（必须）
   - 邮寄或当面递交
   - 保留送达回执

3. 提交对象
   - 最高人民法院（死刑立即执行）
   - 高级人民法院（死缓）
```

### （三）会见权保障

**法律依据**：辩护律师在死刑复核期间有权会见被告人

**操作要点**：

```
1. 会见目的
   - 了解被告人最新情况
   - 核实案件事实
   - 听取被告人意见
   - 告知复核进展

2. 会见障碍及应对
   - 部分看守所可能设置障碍
   - 依法主张会见权
   - 必要时向复核法院反映
```

## 四、辩护意见要点（针对复核阶段的特殊性）

### （一）辩护意见结构

```
第一部分：程序说明
├── 辩护人身份
├── 案件来源
└── 辩护意见提交依据（第248条）

第二部分：案件事实和证据问题
├── 事实认定错误
├── 证据问题
├── 证明标准未达到
└── 存在合理怀疑

第三部分：法律适用问题
├── 罪名认定错误
├── 量刑情节遗漏
├── 法律适用不当
└── 死刑适用不当

第四部分：量刑情节
├── 被害人过错
├── 案件起因（民间纠纷）
├── 悔罪表现
├── 赔偿谅解
└── 其他从轻情节

第五部分：辩护意见
├── 不核准死刑
├── 或发回重审
├── 或改判死缓
└── 给予被告人重新做人的机会
```

### （二）复核阶段特殊辩护策略

**聚焦核心问题**：
- 是否必须适用死刑立即执行
- 是否存在"可杀可不杀"的情形
- 是否应当留有余地

**强调政策因素**：
- "少杀、慎杀"的刑事政策
- 死刑案件证明标准的最高要求
- 疑罪从轻、留有余地的原则

---

## 五、复核可能结果

### （一）三种裁定结果

| 结果 | 法律后果 | 辩护应对 |
|-----|---------|---------|
| 核准死刑 | 交付执行 | 告知家属、做好善后 |
| 不核准，发回重审 | 回到原审程序 | 重新准备辩护 |
| 改判 | 直接改变判决结果 | 分析改判内容 |

### （二）不核准的可能情形

- 事实不清、证据不足
- 法律适用错误
- 量刑不当
- 程序严重违法
- 出现新证据

---

## 六、《刑事诉讼法》关键条款

### 死刑复核相关

| 条款 | 内容概要 |
|-----|---------|
| 第234条 | 死刑案件二审应当开庭 |
| 第246条 | 死刑由最高人民法院核准 |
| 第248条 | 复核应讯问被告人、听取律师意见 |
| 第249-250条 | 核准或不核准的裁定及处理 |

### 《刑法》关键条款

| 条款 | 内容概要 |
|-----|---------|
| 第48条 | 死刑只适用于罪行极其严重的犯罪分子 |
| 第49条 | 不适用死刑：未满18周岁、审判时怀孕的妇女 |
| 第50条 | 死缓执行：无故意犯罪可减刑 |
