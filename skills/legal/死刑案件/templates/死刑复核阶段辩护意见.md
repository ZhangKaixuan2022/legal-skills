# 死刑复核阶段辩护意见模板

> **适用阶段**：死刑复核程序
> **提交对象**：最高人民法院（死刑立即执行）/ 高级人民法院（死缓）
> **格式规范**：参照《刑事辩护法律文书标准化文本》

---

## 文书格式规范

### 标题格式

| 行数 | 内容 | 格式要求 |
|-----|------|---------|
| 第一行 | XX被判XX罪一案 | 黑体三号加粗居中 |
| 第二行 | 死刑复核阶段辩护意见 | 黑体三号加粗居中 |

### 正文格式

| 项目 | 规范要求 |
|-----|---------|
| 正文字体 | 宋体四号 |
| 行距 | 固定值27磅 |
| 上下边距 | 25.4mm（2.54cm） |
| 左右边距 | 31.8mm（3.18cm） |
| 一级标题 | 黑体四号加粗，首行缩进2字符 |
| 二级标题 | 宋体四号加粗，首行缩进2字符 |
| 三级标题 | 楷体四号加粗，首行缩进2字符 |
| 序号 | 依次使用"一、""（一）""1.""（1）" |

## 模板正文

```
                    XX被判XX罪一案
                  死刑复核阶段辩护意见

最高人民法院：

尊敬的承办法官：

    受XX及其近亲属的委托，【律师事务所名称】指派我担任XX被判
XX罪一案死刑复核阶段的辩护人。接受指派后，辩护人认真研究
了本案一审、二审判决书及全部案卷材料，多次依法会见了XX，
详细听取其陈述和辩解。现依据《中华人民共和国刑事诉讼法》
第248条之规定，向贵院提交以下辩护意见，恳请贵院在死刑复
核时予以充分考虑。

    辩护人总体认为，本案【在事实认定/证据采信/法律适用/
量刑等方面存在问题】，【具体概括核心观点】，不应核准死刑
立即执行，恳请贵院依法【不核准死刑/发回重审/改判死缓】。
具体辩护意见如下：

    一、【关于案件事实和证据问题】

    （一）【一审、二审认定的事实存在问题】

    1. 【具体事实问题1】

    【详细论述】

    2. 【具体事实问题2】

    【详细论述】

    （二）【证据方面存在的问题】

    1. 【证据问题1：如证据不足、证据矛盾等】

    【详细论述】

    2. 【证据问题2】

    【详细论述】

    （三）【证明标准问题】

    死刑案件的证明标准要求达到"确实充分"的最高标准，
本案存在以下疑点未能排除合理怀疑：

    1. 【疑点1】

    2. 【疑点2】

    二、【关于法律适用问题】

    （一）【罪名认定问题（如有）】

    【详细论述】

    （二）【法律条款适用问题（如有）】

    【详细论述】

    三、【关于量刑情节】

    （一）【被害人过错】

    本案系【情感纠纷/邻里纠纷/其他】引发，被害人对案件
的发生存在一定过错。【详细论述被害人过错情况】

    （二）【悔罪表现】

    XX到案后认罪态度好，真诚悔罪，【具体悔罪表现】。

    （三）【赔偿情况】

    XX及其家属在【极其困难的情况下】，积极筹措资金，对
被害人家属进行了赔偿，赔偿金额为XX元。【赔偿具体情况】

    （四）【谅解情况】

    经过多方努力，XX及其家属与被害人家属达成和解，取得
了被害人家属的谅解。被害人家属出具的谅解书明确表示【具体
谅解内容】，请求对XX从轻处罚。

    根据《最高人民法院关于贯彻宽严相济刑事政策的若干意
见》，因婚姻家庭等民间纠纷激化引发的犯罪，取得被害人谅解
的，应当从宽处理。本案符合上述情形，不应适用死刑立即执行。

    （五）【其他从轻情节】

    1. 【自首/立功情节（如有）】

    2. 【初犯/偶犯情节（如有）】

    3. 【其他酌定从轻情节】

    四、【关于死刑适用政策】

    （一）【"少杀、慎杀"的刑事政策】

    我国长期坚持"少杀、慎杀"的死刑政策，对于"可杀可
不杀"的案件不杀。本案【具体论述为何属于可杀可不杀的情
形】。

    （二）【死刑案件证明标准的最高要求】

    死刑是剥夺生命的最严厉刑罚，必须达到"事实清楚，证
据确实、充分"的最高证明标准。本案【存在的疑点/证据问
题】尚未完全排除，不应适用死刑立即执行。

    五、【辩护意见】

    综上所述，辩护人认为：

    1. 本案【事实认定/证据采信/法律适用】存在问题；

    2. XX具有【悔罪表现/积极赔偿/取得谅解等】从轻情节；

    3. 本案系【民间纠纷/情感纠纷】引发，被害人存在过错；

    4. 依据"少杀、慎杀"的刑事政策，本案不应适用死刑立
即执行。

    恳请贵院在死刑复核时充分考虑上述辩护意见，依法作出
【不核准死刑/发回重审/改判死缓】的裁定，给予XX重新做人
的机会。

    以上意见，敬请重视，望予采纳！

                                辩护人：【律师事务所名称】
                                        【律师姓名】律师
                                     X×年X×月X×日

附：
1. 授权委托书；
2. 律师事务所公函；
3. 律师执业证复印件；
4. 【谅解书（如有）】；
5. 【和解协议（如有）】；
6. 【赔偿凭证（如有）】；
7. 【悔过书（如有）】；
8. 【其他证据材料】。
```

## 填写说明

### 标题填写

- **XX被判XX罪一案**：填写被告人姓名和判决罪名
- 示例：张三被判故意杀人罪一案

### 首部填写

- **委托来源**：说明委托人及委托情况
- **工作说明**：简要说明辩护人开展的工作
- **法律依据**：引用《刑事诉讼法》第248条

### 正文填写要点

| 部分 | 填写要点 |
|-----|---------|
| 事实证据 | 详细列举事实认定和证据方面的问题 |
| 法律适用 | 如有法律适用错误，详细论述 |
| 量刑情节 | 重点突出从轻情节，特别是谅解情况 |
| 政策论述 | 引用死刑政策，强调慎重适用 |
| 辩护意见 | 明确提出不核准死刑的请求 |

### 附件清单

**必备附件**：
- 授权委托书
- 律师事务所公函
- 律师执业证复印件

**选择性附件**：
- 谅解书
- 和解协议
- 赔偿凭证
- 悔过书
- 其他有利证据

## 注意事项

### 提交时机

- 案件报送复核后**尽快**提交
- 不必等待法院通知，**主动**提交

### 提交方式

- **书面形式**（必须）
- 邮寄（保留邮寄凭证）或当面递交
- 保留送达回执

### 提交对象

| 案件类型 | 提交对象 |
|---------|---------|
| 死刑立即执行 | 最高人民法院 |
| 死刑缓期执行 | 高级人民法院 |

### 内容要求

- **详细充分**：这是最后的辩护机会，必须详尽
- **有针对性**：针对判决中的具体问题
- **有说服力**：事实依据+法律依据+政策依据
- **恳切诚恳**：语气恳切，请求慎重

---

## 参考法条

### 《刑事诉讼法》第248条

> 最高人民法院复核死刑案件，应当讯问被告人，**辩护律师提出要求的，应当听取辩护律师的意见**。
>
> 在复核死刑案件过程中，最高人民检察院可以向最高人民法院提出意见。最高人民法院应当将死刑复核结果通报最高人民检察院。
