## Q56：审查起诉阶段辩护意见什么时候递交检察院

### 问题全文

审查起诉阶段辩护意见什么时候递交检察院？

### 完整答案

**第一，如果是简单案件或适用速裁程序、简易程序、认罪认罚从宽程序的案件**，律师应当及时阅卷，及时会见，及时撰写书面辩护意见，及时递交检察院。

这里的及时，一般是指**一周以内**，**速裁程序则应在四天以内完成**。

这类案件辩护意见的重点是量刑情节的挖掘、阐述和分析。如自首、立功、从犯、未成年人、在校学生、赔偿、坦白、被害人过错、初犯偶犯、认罪悔罪、和解、获得谅解等，并对照最高人民法院的量刑规范化司法解释，对具体量刑期限进行计算，最终得出一个具体的量刑期限，并且与承办检察官当面进行磋商，争取意见被采纳。

**第二，如果是犯罪嫌疑人作无罪辩解，控辩双方争议较大的案件**，律师需要较宽裕的时间详细研究案卷材料，撰写辩护意见的，律师应当先口头向承办检察官提出主要的辩护意见，向检察官提出此案应慎重审查的意见，关注案件的审限及是否退查等审查进展，综合采取申请调查取证，申请非法证据排除，申请对重要证人核实，提请专家对疑难法律问题进行指导等辩护措施，最终提出详尽的辩护意见。

这里要注意，对案件作事实不清、证据不足辩的，应当在**两次退查结束后**提出，以防止律师辩护意见成为司法机关的补充侦查提纲。

**第三，如果系共同犯罪案件**，律师应当把握全案审查的节奏，根据当事人的具体情况，结合上述两点，适时提出辩护意见。

**总之**，在审查起诉阶段，律师要针对法院、检察院对案件的看法，结合案件的具体情况，提出符合时宜的对嫌疑人有利的辩护意见。

### 核心要点提炼

| 案件类型 | 提交时限 | 重点内容 |
|---------|---------|---------|
| 简单案件/认罪认罚 | 一周以内 | 量刑情节挖掘 |
| **速裁程序** | **4天以内** | 量刑情节分析 |
| 简易程序 | 一周以内 | 量刑情节分析 |
| 无罪辩解案件 | 两次退查后 | 详尽的无罪辩护意见 |
| 共同犯罪案件 | 把握节奏 | 结合具体情况 |

### 适用指引

**速裁程序案件辩护意见工作流程**：

```
Day 1：接受委托、阅卷
       ↓
Day 2：会见当事人、核实案情
       ↓
Day 3：撰写辩护意见、计算量刑
       ↓
Day 4：提交辩护意见、与检察官磋商
```

## Q61：辩护律师如何在认罪认罚从宽制度下，最大限度地维护当事人的合法权益

### 问题全文

辩护律师如何在认罪认罚从宽制度下，最大限度地维护当事人的合法权益？

### 完整答案

认罪认罚制度适用一般是在审查起诉阶段。

**首先**，律师要仔细审查案卷，对证据进行细致分析，确定当事人是否实施了犯罪行为，如果实施了犯罪行为，是否构成被指控的犯罪。只有在案证据能够证明当事人构成被指控的犯罪前提下，律师才可以劝其认罪认罚，并为其争取从宽处理。

如果证据不足以证明当事人构成犯罪，律师是不应该劝其认罪认罚的，而且律师要就证据上存在的问题及时与检察院进行沟通。

**除非**当事人明确表明即使在案证据不足，其也愿意认罪认罚，只要能够判处较轻的刑罚。在这种情况下，律师可以向检察院提出来，案件证据是不足的，但是当事人愿意认罪，希望能判处较轻的刑罚，否则律师将作无罪辩护。

**在当事人认罪认罚之前**，律师要将认罪认罚的相关法律规定以及可能出现的结果向当事人解释清楚，让当事人作出选择。只有当事人选择认罪认罚，律师才能协助当事人进行落实，从各个方面，尽最大努力，去争取当事人利益的最大化。

### 核心要点提炼

**律师工作五步法**：

| 步骤 | 工作内容 | 要点 |
|-----|---------|------|
| **第一步** | 充分案卷审查 | 仔细审查案卷，细致分析证据 |
| **第二步** | 准确判断定性 | 确定是否实施犯罪、是否构成指控罪名 |
| **第三步** | 审慎提出建议 | 只有证据充分才能劝认罪认罚 |
| **第四步** | 充分告知后果 | 解释清楚法律规定和可能结果 |
| **第五步** | 尊重当事人选择 | 当事人选择后才协助落实 |

### 关键原则

```
┌─────────────────────────────────────────────────────────────┐
│                    认罪认罚辩护核心原则                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ① 充分案卷审查和证据分析是前提                              │
│                                                             │
│  ② 证据充分 → 可以劝认罪认罚                                │
│     证据不足 → 不应该劝认罪认罚                              │
│                                                             │
│  ③ 证据不足但当事人坚持认罪 → 向检察院说明情况               │
│                                                             │
│  ④ 认罪认罚前必须充分告知法律后果                            │
│                                                             │
│  ⑤ 只有当事人选择认罪认罚，律师才协助落实                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 证据不足情形的特殊处理

**当证据不足但当事人仍愿意认罪时**：

```
律师向检察院表态：
"本案证据不足，但当事人愿意认罪，
希望能判处较轻的刑罚，
否则律师将作无罪辩护。"

→ 这是一种特殊的协商策略
→ 既表明了证据问题
→ 又尊重了当事人意愿
→ 也为从轻处理创造了空间
```

## 实务操作指引

### 简易/速裁程序辩护意见撰写要点

**一、辩护意见结构**

```
1. 首部：委托情况、工作说明
2. 案件事实：简要、准确
3. 量刑情节：
   - 法定情节（自首、坦白、从犯、立功等）
   - 酌定情节（认罪悔罪、退赃、赔偿、谅解等）
4. 量刑计算：对照量刑指导意见
5. 具体建议：明确的量刑期限
6. 结语：请求采纳
```

**二、量刑情节挖掘清单**

| 类型 | 情节 | 从宽幅度参考 |
|-----|------|-------------|
| 法定 | 自首 | 减少基准刑40%以下 |
| 法定 | 坦白 | 减少基准刑20%以下 |
| 法定 | 从犯 | 减少基准刑20%-50% |
| 法定 | 立功 | 减少基准刑20%以下 |
| 法定 | 未成年人 | 减少基准刑10%-50% |
| 酌定 | 当庭自愿认罪 | 减少基准刑10%以下 |
| 酌定 | 退赃退赔 | 减少基准刑30%以下 |
| 酌定 | 赔偿+谅解 | 减少基准刑40%以下 |
| 酌定 | 初犯偶犯 | 酌情从轻 |

**三、与检察官磋商要点**

```
磋商准备：
□ 量刑计算表（有理有据）
□ 量刑情节证据材料
□ 类案判例检索结果
□ 具体量刑建议区间

磋商策略：
□ 先提出略低于预期的方案
□ 准备备选方案
□ 强调认罪认罚的态度
□ 说明从轻的社会效果
```

## 附：Q60相关内容摘要（与公诉人沟通）

### 审查起诉阶段与公诉人有效沟通

**第四点（与认罪认罚相关）**：重视认罪认罚从宽制度。以庭审为中心审判改革全方位落实后，繁简分流，**80%的案件可能都是在检察院走认罪认罚的程序**，因此律师与公诉人的良性沟通越来越重要，所以要保证沟通的常态化、节奏感和有效性，以充分维护当事人的合法权益。
