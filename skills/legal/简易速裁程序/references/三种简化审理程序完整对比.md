# 三种简化审理程序完整对比

## 一、基本对比表

| 对比维度 | 简易程序 | 速裁程序 | 普通程序简化审理 |
|---------|---------|---------|-----------------|
| **适用范围** | 基层法院管辖案件 | 可能判处一年以下刑罚案件 | 认罪认罚案件 |
| **证据要求** | 事实清楚、证据充分 | 事实清楚、证据充分 | 事实清楚、证据充分 |
| **被告人态度** | 认罪且无异议 | 自愿认罪+法律无争议 | 认罪认罚 |
| **审结期限** | 20日 | 10日（可延长至15日） | 普通审限 |
| **庭审特点** | 简化示证、可单独审判 | 不进行法庭调查和辩论 | 简化但保留完整环节 |
| **合议庭** | 可独任审判 | 可独任审判 | 原则上合议庭 |
| **二审规则** | 可开庭 | 可不开庭审理 | 可开庭 |
| **量刑建议** | 可提出 | 检察机关明确提出 | 检察机关明确提出 |

## 二、简易程序详解

### 适用条件（《刑事诉讼法》第214条）

基层人民法院管辖的案件，符合下列条件的，可以适用简易程序审判：

```
（一）案件事实清楚、证据充分的；
（二）被告人承认自己所犯罪行，对指控的犯罪事实没有异议的；
（三）被告人对适用简易程序没有异议的。
```

### 不应当适用简易程序的情形（指引127）

| 序号 | 不宜适用情形 | 异议理由 |
|-----|-------------|---------|
| 1 | 被告人是盲、聋、哑人 | 需要特殊诉讼保障 |
| 2 | 被告人是尚未完全丧失辨认或控制能力的精神病人 | 认罪自愿性存疑 |
| 3 | 有重大社会影响的 | 需要公开充分审理 |
| 4 | 共同犯罪案件中部分被告人不认罪或有异议的 | 案件完整性 |
| 5 | 辩护人作无罪辩护的 | 与简易程序前提矛盾 |
| 6 | 被告人认罪但经审查可能不构成犯罪的 | 防止错案 |
| 7 | 不宜适用简易程序审理的其他情形 | 兜底条款 |

### 转普通程序条件（指引128）

在法庭审理过程中，发现以下情形的，应当建议法庭中止审理，转为普通程序：

```
（一）被告人的行为可能不构成犯罪的；
（二）被告人可能不负刑事责任的；
（三）被告人当庭对起诉指控的犯罪事实予以否认的；
（四）案件事实不清、证据不足的；
（五）不应当或者不宜适用简易程序的其他情形。
```

### 庭审流程简化要点（指引129）

| 庭审环节 | 普通程序 | 简易程序 |
|---------|---------|---------|
| 起诉书宣读 | 完整宣读 | 可简化 |
| 证据出示 | 逐一出示质证 | 简明出示 |
| 发问环节 | 完整发问 | 律师酌情发问 |
| 质证意见 | 逐项质证 | 简明发表 |
| 辩护意见 | 完整辩护 | 简明辩护 |
| 辩论环节 | 控辩多轮 | 经许可互相辩论 |

## 三、速裁程序详解

### 适用条件（《刑事诉讼法》第222条）

```
适用条件（全部满足）：
□ 事实清楚
□ 证据确实、充分
□ 被告人自愿认罪
□ 当事人对适用法律没有争议
□ 可能判处一年以下有期徒刑、拘役、管制或单处罚金
```

**典型适用罪名**：危险驾驶、交通肇事、盗窃、诈骗、抢夺、伤害、寻衅滋事等情节较轻案件。

### 速裁程序七大特点

| 序号 | 特点 | 对辩护工作的影响 |
|-----|------|----------------|
| 1 | 被告人认罪 | 辩护重点在量刑 |
| 2 | 事实清楚、证据充分 | 无罪辩护空间极小 |
| 3 | 可能处刑在一年以内 | 刑期争取空间有限 |
| 4 | 诉讼用时短、进程快 | 辩护工作须快速响应 |
| 5 | 检察机关量刑建议明确 | 量刑协商成为重点 |
| 6 | 从轻幅度大 | 认罪认罚优势明显 |
| 7 | 往往当庭判决 | 庭前工作至关重要 |

### 速裁程序与简易程序关键区别

| 对比维度 | 简易程序 | 速裁程序 |
|---------|---------|---------|
| 刑期条件 | 基层法院管辖范围 | 一年以下有期徒刑 |
| 庭审调查 | 简化但保留 | **不进行法庭调查** |
| 庭审辩论 | 简化但保留 | **不进行法庭辩论** |
| 审结期限 | 20日 | **10日**（可延长至15日） |
| 二审规则 | 可开庭 | **可不开庭审理** |
| 辩护意见 | 庭上发表 | **必须庭前提交** |

### 速裁程序庭审流程

```
庭审特点：不进行法庭调查和法庭辩论

庭审流程（简化）：
1. 开庭核实身份
2. 公诉人宣读起诉书（简化）
3. 被告人对指控意见
4. 确认认罪认罚自愿性
5. 辩护人发表意见（简短）
6. 被告人最后陈述
7. 当庭宣判
```

## 四、普通程序简化审理

### 适用范围

- 认罪认罚案件
- 需要保留完整庭审环节但可以简化

### 与普通程序的区别

| 环节 | 普通程序 | 普通程序简化审理 |
|-----|---------|-----------------|
| 法庭调查 | 完整进行 | 简化进行 |
| 举证质证 | 逐一进行 | 简化进行 |
| 法庭辩论 | 完整进行 | 简化进行 |
| 合议庭 | 必须 | 原则上必须 |
| 审限 | 普通审限 | 普通审限 |

---

## 五、程序选择策略

### 辩护律师评估要点

```
选择简易程序的条件：
□ 案件事实清楚
□ 证据确实充分
□ 被告人真实认罪
□ 对适用简易程序没有异议
□ 律师确认确实构成犯罪

选择速裁程序的条件：
□ 上述条件全部满足
□ 可能判处一年以下
□ 当事人对法律适用无争议

建议转普通程序的情形：
□ 证据存在重大疑问
□ 可能不构成犯罪
□ 被告人翻供
□ 存在无罪辩护空间
```
