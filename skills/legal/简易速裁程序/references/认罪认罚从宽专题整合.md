# 认罪认罚从宽制度专题整合

> **说明**：本文件汇总侦查阶段、审查起诉阶段、审判阶段的认罪认罚工作内容，提供完整专题视角
> **参考依据**：操作指引63-65、《刑事诉讼法》相关条款、《关于在部分地区开展刑事案件认罪认罚从宽制度试点工作的办法》

## 一、认罪认罚从宽制度概述

### 1.1 制度定义

认罪认罚从宽制度是指犯罪嫌疑人、被告人自愿如实供述自己的罪行，承认指控的犯罪事实，愿意接受处罚的，可以依法从宽处理的制度。

### 1.2 制度发展历程

| 时间 | 事件 | 内容 |
|-----|------|------|
| 2010年2月 | 最高法《关于贯彻宽严相济刑事政策的若干意见》 | 制度雏形 |
| 2014年10月 | 十八届四中全会 | 首次提出完善认罪认罚从宽制度 |
| 2016年6月 | 《关于推进以审判为中心的刑事诉讼制度改革的意见》 | 完善速裁程序和认罪认罚从宽制度 |
| 2016年7月 | 《关于认罪认罚从宽制度改革试点方案》 | 改革试点方案 |
| 2016年9月 | 全国人大常委会授权决定 | 18个城市开展试点 |
| 2016年11月 | 两高三部《试点工作办法》 | 正式开展试点 |
| 2018年10月 | 刑事诉讼法第三次修改 | 认罪认罚从宽制度入法 |

### 1.3 适用范围

**可以适用的案件**：
- 除以下情形外，所有刑事案件均可适用

**排除适用的情形**：
1. 被告人是尚未完全丧失辨认或控制自己行为能力的精神病人
2. 未成年被告人的法定代理人、辩护人对认罪认罚有异议
3. 被告人违反取保候审、监视居住规定
4. 被告人不构成犯罪
5. 其他不宜适用的情形

### 1.4 从宽处理内容

| 情形 | 处理方式 |
|-----|---------|
| 犯罪情节显著轻微，不认为是犯罪 | 直接撤销案件、不起诉或宣告无罪 |
| 犯罪情节轻微，无须判处刑罚或免除刑罚 | 依法不起诉或宣告免予刑事处罚 |
| 同意检察院量刑建议或审判阶段认罪认罚签署具结书 | 可以免予刑事处罚、缓刑，或依法从轻、减轻处罚 |
| 自愿如实供述+重大立功或涉及国家重大利益 | 层报最高检撤案、不起诉 |

## 二、侦查阶段的认罪认罚

> **参考**：侦查阶段Skill → skills/legal/侦查阶段辩护/SKILL.md

### 2.1 告知义务

**公安机关告知义务**：

```
时间：第一次讯问或采取强制措施之日起

内容：
□ 向犯罪嫌疑人出示《认罪认罚从宽制度告知书》
□ 告知适用认罪认罚从宽制度产生的法律后果
□ 告知适用认罪认罚从宽制度产生的法律责任
□ 听取犯罪嫌疑人、辩护人或值班律师的意见
  - 对于事实认定的意见
  - 对于适用法律的意见
  - 对于适用程序的意见
```

### 2.2 记录固定

**公安机关记录要求**：

```
□ 将认罪认罚情况记录在案并附卷
□ 《起诉意见书》中应写明被告人认罪认罚情况
□ 移送审查起诉时一并移送相关材料
```

### 2.3 看守所工作

**看守所配合工作**：

```
犯罪嫌疑人在看守所表示愿意认罪认罚的：
□ 看守所工作人员应当在三日内书面告知办案单位
□ 或者辩护人、值班律师在三日内书面告知办案单位
```

### 2.4 审查逮捕中的认罪认罚

**检察院审查逮捕时**：

```
如发现案件符合适用认罪认罚从宽制度的条件：
□ 应当启动适用认罪认罚程序
□ 告知犯罪嫌疑人相关权利义务
□ 听取辩护人意见
```

### 2.5 侦查阶段律师工作要点

```
□ 会见时告知认罪认罚从宽制度
□ 了解案件证据情况
□ 评估认罪认罚的利弊
□ 不得在证据不充分时劝导认罪
□ 为当事人作出知情选择提供帮助
```

## 三、审查起诉阶段的认罪认罚

> **参考**：审查起诉阶段Skill → skills/legal/审查起诉阶段辩护/SKILL.md

### 3.1 检察院告知义务

**告知内容**：

```
□ 适用认罪认罚从宽制度产生的法律后果
□ 适用认罪认罚从宽制度产生的法律责任
□ 听取辩护人或值班律师的意见：
  - 事实认定
  - 适用法律
  - 适用程序
  - 量刑建议
```

### 3.2 认罪认罚具结书签署

**签署条件**：

```
全部满足：
□ 犯罪嫌疑人自愿认罪
□ 对犯罪事实没有异议
□ 对适用法律没有异议
□ 对量刑建议没有异议
□ 对适用程序没有异议
```

**签署程序**：

```
必须要求：
□ 辩护人或值班律师在场
□ 向犯罪嫌疑人充分告知权利义务
□ 确认认罪认罚出于自愿
□ 犯罪嫌疑人本人签字
□ 辩护人或值班律师签字见证
```

### 3.3 量刑建议协商

**协商前提**：

```
情形一：
□ 犯罪事实清楚、证据确实充分
□ 被告人不认罪，但承认犯罪事实

情形二：
□ 主要犯罪事实清楚、基本证据确实充分
□ 被告人不认罪也不承认犯罪事实
□ 但辩护人要求启动认罪认罚协商程序
```

**协商主体**：人民检察院、被告人及其辩护人或值班律师

**协商内容**：
1. 案件的适用程序
2. 量刑幅度
3. 涉案财物处理

### 3.4 值班律师在场见证

**值班律师职责**（无委托辩护人时）：

```
□ 为犯罪嫌疑人提供法律咨询
□ 程序选择建议
□ 申请变更强制措施
□ 对案件处理提出意见
□ 在认罪认罚具结书签署时在场见证
```

### 3.5 审查起诉阶段律师工作要点（操作指引63-65）

**操作指引63**：

```
适用条件判断：
□ 犯罪嫌疑人愿意如实供述犯罪事实
□ 对指控没有异议
□ 认罪认罚有利于维护当事人最大合法权益
→ 律师应当建议适用认罪认罚从宽程序

不能适用情形：
□ 尚未完全丧失辨认或控制能力的精神病人
□ 未成年人的法定代理人、辩护人有异议
□ 可能不构成犯罪的案件
```

**操作指引64**：

```
会见时告知内容：
□ 认真、全面听取案情陈述
□ 充分了解当事人要求
□ 介绍相关法律规定
□ 告知认罪认罚从宽程序的规定内容
□ 制作笔录
```

**操作指引65（四种情况处理）**：

| 情形 | 律师建议 |
|-----|---------|
| 事实清楚、证据充分，被告人认罪 | 建议适用认罪认罚，加强量刑辩护 |
| 被告人认罪但可能不构成犯罪 | 向被告人详尽咨询，提出无罪意见，不建议认罪认罚 |
| 被告人不认罪且确实无法定罪 | 要求实事求是辩解，提出无罪意见，不建议认罪认罚 |
| 足以定罪但被告人不认罪 | 详细讲解法律，建议认罪；如坚持可协助无罪辩护并告知后果 |

## 四、审判阶段的认罪认罚

> **参考**：一审阶段Skill → skills/legal/一审阶段辩护/SKILL.md

### 4.1 审判阶段启动认罪认罚

**法律依据**：

虽然认罪认罚从宽制度没有明确审判阶段启动程序，但从"审判阶段认罪认罚，签署具结书，可以免予刑事处罚、判处缓刑，或者依法从轻、减轻处罚"的规定来看，该阶段亦可启动认罪认罚程序。

**二审启动**：

一审程序中被告人未认罪认罚，第二审程序中被告人认罪认罚的，可以适用认罪认罚从宽制度。

### 4.2 庭审简化

**适用认罪认罚案件的庭审简化**：

| 环节 | 简化方式 |
|-----|---------|
| 讯问 | 简化讯问 |
| 示证 | 公诉人仅就证据名称及证明事项作出说明，不再详细宣读或出示 |
| 法庭审理 | 除被告人最后陈述外，其余环节可以省略 |
| 速裁程序 | 明确规定不进行法庭调查、法庭辩论 |

### 4.3 量刑建议采纳规则

**一般规则**：

```
人民法院一般应当采纳人民检察院指控的罪名和量刑建议

例外情形（不采纳）：
□ 被告人的行为不构成犯罪
□ 被告人不应当追究刑事责任
□ 被告人违背意愿认罪认罚
□ 被告人否认指控的犯罪事实
□ 起诉指控的罪名与审理认定的不一致
□ 量刑建议明显不当
□ 其他可能影响公正审判的情形
```

### 4.4 法院确认程序

**法院应当确认的内容**：

```
□ 被告人是否自愿认罪认罚
□ 是否知悉认罪认罚的法律后果
□ 是否签署认罪认罚具结书
□ 具结书内容是否真实
□ 辩护人或值班律师是否在场见证
```

### 4.5 审判阶段律师工作要点

```
□ 确保被告人知情、自愿
□ 见证签署具结书（如在审判阶段签署）
□ 审查量刑建议合理性
□ 必要时协商调整量刑建议
□ 协助被告人作最后陈述
□ 关注认罪认罚自愿性的庭审确认
```

## 五、认罪认罚的撤回及法律后果

### 5.1 可以撤回的情形

```
被告人有权撤回认罪认罚：
□ 被告人违背意愿认罪认罚
□ 被告人否认指控的犯罪事实
□ 起诉指控的犯罪事实与审理认定的不一致
□ 其他不应当适用的情形
```

### 5.2 撤回的法律后果

```
撤回后：
□ 案件转为普通程序审理
□ 不影响案件继续审理
□ 但会影响从宽幅度
□ 法院将根据实际情况重新量刑
□ 可能面临比原量刑建议更重的处罚
```

### 5.3 上诉与认罪认罚

**上诉后的处理**：

```
单纯以量刑过重为由上诉：
□ 二审一般不开庭审理
□ 主要审查认罪认罚的自愿性、真实性、合法性
□ 上诉可能影响认罪认罚从宽幅度

对事实、证据有异议的上诉：
□ 应明确提出异议理由
□ 二审应当审查相关争议
```

## 六、对办案期限的影响

### 6.1 侦查阶段

**速裁程序案件**：公安机关一般应当在对犯罪嫌疑人执行逮捕之日或采取取保候审、监视居住之日起**一个月内**移送审查起诉。

### 6.2 审查起诉阶段

| 案件类型 | 审查期限 |
|---------|---------|
| 速裁程序（一年以下） | 一般**10日内**决定是否起诉 |
| 速裁程序（一年以上） | 可延长至**15日** |
| 简易程序（二年以上） | 一般**20日内**决定是否起诉 |

### 6.3 审判阶段

| 程序类型 | 审结期限 |
|---------|---------|
| 速裁程序 | 一般**10日内**审结 |
| 速裁程序（一年以上） | 可延长至**15日** |
| 简易程序 | **20日内**审结 |

## 七、律师全程工作要点汇总

### 7.1 侦查阶段

```
□ 会见时告知认罪认罚从宽制度
□ 了解案件基本情况
□ 评估认罪认罚的利弊
□ 不得在证据不充分时劝导认罪
□ 协助当事人作出知情选择
```

### 7.2 审查起诉阶段

```
□ 仔细审查案卷，细致分析证据
□ 确定是否构成被指控的犯罪
□ 只有证据充分才能劝认罪认罚
□ 充分告知法律后果
□ 尊重当事人选择
□ 协助量刑协商
□ 在场见证具结书签署
□ 及时提交辩护意见（速裁4天内）
```

### 7.3 审判阶段

```
□ 确保被告人知情、自愿
□ 审查量刑建议合理性
□ 必要时协商调整
□ 关注庭审确认程序
□ 协助最后陈述
□ 庭后跟进判决结果
```
