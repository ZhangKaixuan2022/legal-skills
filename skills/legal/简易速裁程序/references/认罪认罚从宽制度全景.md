# 认罪认罚从宽制度全景

## 一、三阶段认罪认罚工作线

```
侦查阶段                 审查起诉阶段               审判阶段
    │                        │                        │
    ▼                        ▼                        ▼
┌─────────────┐      ┌─────────────────┐      ┌─────────────────┐
│ 告知义务     │ ──▶ │ 具结书签署       │ ──▶ │ 庭审简化        │
│ 记录固定     │      │ 量刑建议协商     │      │ 量刑建议采纳    │
│ 启动程序     │      │ 值班律师在场     │      │ 确认自愿性      │
└─────────────┘      └─────────────────┘      └─────────────────┘
```

**实务背景**："以庭审为中心审判改革全方位落实后，繁简分流，**80%的案件**可能都是在检察院走认罪认罚的程序。"

---

## 二、认罪认罚案件从宽处理内容

| 情形 | 处理方式 |
|-----|---------|
| 犯罪情节显著轻微，不认为是犯罪 | 直接撤销案件、不起诉或宣告无罪 |
| 犯罪情节轻微，无须判处刑罚或免除刑罚 | 依法不起诉或宣告免予刑事处罚 |
| 同意检察院量刑建议或审判阶段认罪认罚签署具结书 | 可以免予刑事处罚、缓刑，或依法从轻、减轻处罚 |
| 自愿如实供述+重大立功或涉及国家重大利益 | 层报最高检撤案、不起诉 |

## 三、排除适用认罪认罚从宽的情形

```
以下情形不适用认罪认罚从宽制度：
1. 被告人是尚未完全丧失辨认或控制能力的精神病人
2. 未成年被告人的法定代理人、辩护人对认罪认罚有异议
3. 被告人违反取保候审、监视居住规定
4. 被告人不构成犯罪
5. 其他不宜适用的情形
```

## 四、律师核心职责（Q61完整嵌入）

**Q61：辩护律师如何在认罪认罚从宽制度下，最大限度地维护当事人的合法权益？**

### 第一步：充分案卷审查和证据分析是前提

```
□ 仔细审查案卷
□ 对证据进行细致分析
□ 确定当事人是否实施了犯罪行为
□ 如果实施了犯罪行为，是否构成被指控的犯罪
```

### 第二步：告知法律规定和可能结果

```
□ 在案证据能够证明当事人构成被指控犯罪的前提下
□ 律师才可以劝其认罪认罚
□ 为其争取从宽处理
```

### 第三步：证据不足情形的处理

```
□ 如证据不足以证明当事人构成犯罪
□ 律师不应该劝其认罪认罚
□ 应就证据问题及时与检察院沟通
```

### 第四步：特殊情形处理

```
当事人明确表明即使证据不足也愿意认罪认罚，只要能判处较轻刑罚：
□ 律师可以向检察院提出：案件证据不足，但当事人愿意认罪
□ 希望能判处较轻的刑罚
□ 否则律师将作无罪辩护
```

### 第五步：只有当事人选择认罪认罚，律师才协助落实

```
□ 在当事人认罪认罚之前
□ 律师要将认罪认罚的相关法律规定及可能结果向当事人解释清楚
□ 让当事人作出选择
□ 只有当事人选择认罪认罚，律师才能协助当事人落实
□ 从各个方面，尽最大努力，去争取当事人利益的最大化
```

## 五、认罪认罚具结书签署规范

### 签署条件

- 犯罪嫌疑人自愿认罪
- 对犯罪事实、适用法律、量刑建议、适用程序没有异议
- **辩护人或值班律师在场**

### 律师在场职责

1. 向被告人详尽告知其享有的诉讼权利
2. 告知认罪认罚可能导致的法律后果
3. 仔细询问和观察被告人认罪认罚是否出于自愿
4. 向人民检察院提出辩护意见

---

## 六、认罪认罚的撤回及法律后果

### 撤回情形

- 被告人违背意愿认罪认罚的
- 被告人否认指控的犯罪事实的
- 起诉指控的犯罪事实与审理认定的不一致
- 其他不应当适用的情形

### 法律后果

- 撤回认罪认罚不影响案件继续审理
- 但会影响从宽幅度
- 法院将根据实际情况重新量刑

---

## 七、律师告知义务清单

```
必须告知当事人的内容：
□ 认罪认罚从宽制度的具体内容
□ 从宽幅度（可能获得的量刑优惠）
□ 具结书的法律效力
□ 翻供的法律后果
□ 放弃的诉讼权利
□ 量刑建议的具体内容
□ 律师对案件的分析意见
□ 认罪认罚后的程序（可能适用简易或速裁程序）
```
