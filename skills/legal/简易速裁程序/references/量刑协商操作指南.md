# 量刑协商操作指南

## 一、量刑协商概述

### 协商主体
- 人民检察院
- 被告人及其辩护人或值班律师

### 协商内容
- 案件的适用程序
- 量刑幅度
- 涉案财物处理

### 协商前提
- 主要犯罪事实清楚
- 基本证据确实充分

---

## 二、律师与公诉人的量刑协商

### 协商时机

**审查起诉阶段**：最佳协商时机

### 协商策略

```
步骤一：全面梳理量刑情节
□ 法定从轻情节（自首、坦白、立功、从犯等）
□ 酌定从轻情节（初犯、偶犯、赔偿、谅解等）
□ 按量刑情节体系逐项核对

步骤二：对照量刑规范化司法解释计算
□ 确定基准刑
□ 计算各情节调整比例
□ 得出建议量刑区间

步骤三：提出具体量刑建议
□ 明确建议的刑期
□ 是否建议适用缓刑
□ 罚金数额建议

步骤四：与公诉人当面磋商
□ 提交书面量刑建议
□ 当面阐述理由
□ 听取公诉人意见

步骤五：争取意见被采纳
□ 针对分歧继续沟通
□ 适当调整建议
□ 达成一致意见
```

---

## 三、量刑建议的审查与异议

### 审查要点

```
□ 量刑建议是否在法定刑幅度内
□ 量刑情节是否遗漏
□ 从宽幅度是否合理
□ 是否明显不当
□ 是否与类案相符
```

### 异议权利

- 被告人、辩护人可以对量刑建议提出异议
- 法院应当告知检察院，检察院可以调整
- 检察院不调整或调整后仍明显不当的，法院应依法判决

---

## 四、新情节出现时的量刑调整

### 【案例5-89】量刑协商典型案例

**案情**：故意毁坏公私财物案
- 双方原系夫妻，离婚后因抚养费发生口角
- 男方将女方手机摔碎（价值8000元）
- 检察院建议适用简易程序
- 公诉人初步量刑建议：6个月有期徒刑

**辩护过程**：

```
第一步：侦查阶段介入，尝试和解（未成功）

第二步：案件移送法院后，建议男方将8000元赔偿款交到法院

第三步：将新的酌情量刑情节告知公诉人

第四步：公诉人调整量刑建议至3个月以内

第五步：辩护律师提出免刑意见，公诉人不反对

第六步：与法官沟通量刑建议情况

第七步：法官建议促成和解谅解

第八步：发动家属做女方工作，获得谅解书

结果：成功争取到免刑
```

### 案例启示

1. 新情节出现要及时告知公诉人
2. 可以请公诉人考虑调整量刑建议
3. 辩护律师与公诉人可以提出不同意见
4. 开庭前与法官沟通量刑问题
5. 和解谅解对量刑有重大影响

---

## 五、开庭前与法官沟通量刑意见

### 沟通时机

**庭前**：必须在开庭前完成沟通

### 沟通内容

```
□ 辩护意见要点
□ 量刑协商情况
□ 新的量刑情节
□ 和解谅解进展
□ 对量刑建议的看法
```

### 注意事项

```
□ 简易/速裁程序案件很可能当庭宣判
□ 必须在庭前完成沟通
□ 书面辩护意见提前提交
□ 沟通以有实质内容为前提
```

## 六、速裁程序辩护意见提交时限

### Q56要点（关键时间要求）

```
速裁程序辩护意见提交时限：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  4天以内完成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

工作内容：
1. 及时阅卷、及时会见
2. 及时撰写书面辩护意见
3. 及时递交检察院/法院

辩护意见重点：
- 量刑情节的挖掘、阐述和分析
- 对照最高人民法院量刑规范化司法解释
- 计算具体量刑期限
- 与承办人当面磋商
```

## 七、量刑辩护重点清单

```
法定从轻情节：
□ 自首
□ 坦白
□ 立功
□ 从犯、胁从犯
□ 未遂、中止
□ 未成年人
□ 限定刑事责任能力人

酌定从轻情节：
□ 初犯、偶犯
□ 赔偿损失
□ 退赃退赔
□ 取得谅解
□ 和解
□ 认罪悔罪
□ 被害人过错
□ 在校学生
□ 家庭特殊情况
```

## 八、缓刑争取策略

### 缓刑适用条件

```
1. 被判处拘役、三年以下有期徒刑
2. 犯罪情节较轻
3. 有悔罪表现
4. 没有再犯罪的危险
5. 宣告缓刑对所居住社区没有重大不良影响
```

### 争取缓刑的辩护要点

```
□ 强调犯罪情节较轻
□ 展示悔罪表现（认罪认罚、赔偿、谅解）
□ 论证没有再犯罪危险（初犯、有稳定工作、家庭）
□ 说明对社区无不良影响
□ 提供社区矫正条件
```
