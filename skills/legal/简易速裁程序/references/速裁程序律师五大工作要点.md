# 速裁程序律师五大工作要点

> 参考指引131解析

## 要点一：及时介入、迅速反应

### 办案期限极短

```
速裁程序办案期限：
- 侦查阶段：逮捕后1个月内移送审查起诉
- 审查起诉：一般10日内决定是否起诉
- 审判阶段：10日内审结
```

### 律师必须

```
□ 第一时间接受委托
□ 迅速完成阅卷
□ 快速会见当事人
□ 及时形成辩护意见
□ 每个环节争分夺秒
```

---

## 要点二：诉前与庭前发表意见

### 审查起诉阶段

```
□ 了解案件是否适用速裁程序
□ 向检察机关提出辩护意见
□ 参与量刑建议协商
□ 核实认罪认罚的自愿性
□ 见证具结书签署
```

### 审判阶段

```
□ 庭前提交书面辩护意见
□ 与法官沟通量刑问题
□ 确保法院判决时充分考虑辩护意见
□ 辩护意见必须庭前到位
```

---

## 要点三：挖掘辩点、协商量刑

### 量刑辩护重点

```
法定情节：
□ 自首、坦白情节
□ 立功情节
□ 从犯、胁从犯
□ 未成年人

酌定情节：
□ 在校学生
□ 赔偿、退赃
□ 认罪悔罪
□ 被害人过错
□ 初犯偶犯
□ 和解、获得谅解
□ 家庭特殊情况
```

### 协商策略

```
1. 全面梳理量刑情节
2. 对照量刑规范化司法解释
3. 计算具体量刑期限
4. 与公诉人当面磋商
5. 争取最有利的量刑建议
```

---

## 要点四：提前提交辩护意见

### Q56要点（关键时间要求）

```
速裁程序辩护意见提交时限：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  4天以内完成
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 工作内容

```
1. 及时阅卷、及时会见
2. 及时撰写书面辩护意见
3. 及时递交检察院/法院
```

### 辩护意见重点

```
- 量刑情节的挖掘、阐述和分析
- 对照最高人民法院量刑规范化司法解释
- 计算具体量刑期限
- 与承办人当面磋商
```

### 速裁程序特殊性

```
由于速裁程序不进行法庭调查和法庭辩论：
- 庭审中辩护发言时间极短
- 辩护意见主要依靠书面提交
- 必须确保庭前意见充分表达
```

---

## 要点五：争取工作时间

### 特殊案件处理（如危险驾驶案）

```
当诉讼时限特别紧时：
□ 如有必要，及时提出取保候审申请
□ 目的：赢得工作时间
□ 争取最有利的结果
```

### 争取时间的方法

```
1. 申请取保候审
   - 延长辩护工作时间
   - 便于当事人配合

2. 申请延期审理
   - 有新证据需要核实
   - 需要补充调查

3. 充分利用有限时间
   - 优化工作流程
   - 并行开展多项工作
```

---

## 综合工作要求

### "程序简化≠工作简化"原则

**核心理念**：程序的简化不代表律师的工作可以简化。

```
即使案件采用速裁程序审理，律师也应当：
□ 尽心竭力、尽职尽责
□ 重视案件公正性的保障
□ 保证被告人诉讼权利
□ 将绝大部分工作做在开庭前
□ 开庭前提出辩护意见
□ 与承办人进行充分沟通
```

### 时间节点管理

| 节点 | 期限 | 工作内容 |
|-----|------|---------|
| 接受委托 | 第1天 | 办理委托手续 |
| 阅卷会见 | 第1-2天 | 完成阅卷和会见 |
| 形成意见 | 第2-3天 | 撰写辩护意见 |
| 提交沟通 | 第3-4天 | 提交意见、与承办人沟通 |
| 庭审 | 第5-10天 | 出庭辩护 |
