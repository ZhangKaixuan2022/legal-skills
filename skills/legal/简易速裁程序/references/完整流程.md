> **Skill定位**：一审Skill的简易/速裁分支模式，专门处理简化审理程序的差异部分

- 庭前准备、会见等通用模块 → 路由至一审Skill对应模块
- 仅处理简易/速裁程序的**差异部分**（庭审简化、量刑协商、时间压缩等）

### 1.2 三种简化审理方式对比

📋 完整对比表参见 [references/三种简化审理程序完整对比.md](三种简化审理程序完整对比.md)

| 对比维度 | 简易程序 | 速裁程序 |
|---------|---------|---------|
| 适用范围 | 基层法院管辖案件 | 可能判处一年以下 |
| 审结期限 | 20日 | 10日（可延长至15日） |
| 庭审特点 | 简化示证 | 不进行法庭调查和辩论 |

## 模块二：简易程序辩护

> **法律依据**：《刑事诉讼法》第214-221条

### 2.1 适用条件

- 案件事实清楚、证据充分
- 被告人承认自己所犯罪行，对指控无异议
- 被告人对适用简易程序没有异议

**核心认知**："简易程序究其实质，是以牺牲被告人的部分诉讼权利来换取诉讼效率的提高。"

### 2.2 不应当适用简易程序的情形

| 序号 | 不宜适用情形 |
|-----|-------------|
| 1 | 被告人是盲、聋、哑人 |
| 2 | 被告人是精神病人 |
| 3 | 有重大社会影响的 |
| 4 | 共同犯罪中部分被告人不认罪 |
| 5 | 辩护人作无罪辩护的 |
| 6 | 被告人认罪但可能不构成犯罪 |

📋 按需读取：适用普通程序建议书 → templates/适用普通程序建议书.md

### 2.3 转普通程序条件

- 被告人的行为可能不构成犯罪
- 被告人当庭否认指控
- 案件事实不清、证据不足

### 2.4 "程序简化≠工作简化"原则

**核心理念**：程序的简化不代表律师的工作可以简化。

```
即使案件采用简易程序审理，律师也应当：
□ 将绝大部分工作做在开庭前
□ 开庭前提出辩护意见
□ 与合议庭进行充分沟通
```

**时间要求**：简易程序审理案件，法院应在受理后**20日以内审结**，较大可能当庭宣判。

## 模块三：速裁程序辩护

> **法律依据**：《刑事诉讼法》第222-227条

### 3.1 适用条件

- 事实清楚、证据确实充分
- 被告人自愿认罪
- 当事人对适用法律没有争议
- 可能判处一年以下有期徒刑

**典型适用罪名**：危险驾驶、交通肇事、盗窃、诈骗、伤害等情节较轻案件。

### 3.2 律师五大工作要点

📋 完整详解参见 [references/速裁程序律师五大工作要点.md](速裁程序律师五大工作要点.md)

| 要点 | 核心内容 |
|-----|---------|
| 及时介入、迅速反应 | 第一时间接受委托、快速完成阅卷会见 |
| 诉前与庭前发表意见 | 审查起诉阶段提辩护意见、庭前书面提交 |
| 挖掘辩点、协商量刑 | 全面梳理量刑情节、与公诉人当面磋商 |
| 提前提交辩护意见 | **4天内完成** |
| 争取工作时间 | 必要时申请取保候审 |

### 3.3 速裁程序庭审流程

```
庭审特点：不进行法庭调查和法庭辩论

流程：开庭→起诉书→被告人意见→确认自愿性→辩护人意见→最后陈述→当庭宣判
```

## 模块四：认罪认罚从宽制度

📋 完整制度全景参见 [references/认罪认罚从宽制度全景.md](认罪认罚从宽制度全景.md)

### 4.1 律师核心职责（Q61要点）

1. **充分案卷审查和证据分析是前提**
2. 在案证据能够证明构成犯罪，才可以劝认罪认罚
3. 证据不足，不应该劝其认罪认罚
4. **只有当事人选择认罪认罚，律师才协助落实**

### 4.2 认罪认罚具结书签署规范

**签署条件**：自愿认罪+无异议+**辩护人或值班律师在场**

**律师在场职责**：告知诉讼权利、告知法律后果、审查是否出于自愿

### 4.3 不宜适用的情形

- 可能不构成犯罪
- 事实不清证据不足
- 对定性有重大异议
- 量刑建议明显过重

## 模块五：量刑协商与建议

📋 完整操作指南参见 [references/量刑协商操作指南.md](量刑协商操作指南.md)

### 5.1 协商策略

1. 全面梳理量刑情节
2. 对照量刑规范化司法解释计算
3. 提出具体量刑建议
4. 与公诉人当面磋商
5. 争取意见被采纳

### 5.2 量刑建议的审查与异议

**审查要点**：是否在法定刑幅度内、情节是否遗漏、从宽幅度是否合理

**异议权利**：被告人、辩护人可以对量刑建议提出异议，检察院可以调整

### 5.3 开庭前与法官沟通

**注意事项**：简易/速裁程序案件很可能当庭宣判，必须在庭前完成沟通

## 模块六：被告人权利保障

### 6.1 同意适用简易/速裁程序的权利

- 有权同意或不同意适用
- 同意后可以反悔，转为普通程序

### 6.2 律师告知义务

```
必须告知被告人的内容：
□ 简易/速裁程序的特点
□ 放弃的诉讼权利
□ 可能的法律后果
□ 认罪认罚从宽的幅度
□ 量刑建议的内容
□ 程序转换的权利
```

### 6.3 防止虚假认罪的审查义务

审查要点：自愿性、是否受威胁引诱、与证据是否一致、是否理解法律后果
