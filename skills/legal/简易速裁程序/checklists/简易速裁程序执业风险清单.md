# 简易/速裁程序执业风险清单

> **适用对象**：办理简易程序、速裁程序、认罪认罚案件的辩护律师
> **使用方式**：案件办理过程中逐项核对，确保不遗漏关键环节

---

## 一、时间风险

### 1.1 速裁程序时间节点

| 节点 | 期限 | 风险等级 | 核对 |
|-----|------|---------|------|
| 辩护意见提交 | **4天内** | ⚠️ 极高 | □ |
| 审查起诉决定 | 10日内 | ⚠️ 高 | □ |
| 审结期限 | 10日内 | ⚠️ 高 | □ |
| 当庭宣判 | 庭审当日 | ⚠️ 极高 | □ |

### 1.2 简易程序时间节点

| 节点 | 期限 | 风险等级 | 核对 |
|-----|------|---------|------|
| 辩护意见提交 | 一周内 | ⚠️ 高 | □ |
| 审结期限 | 20日内 | ⚠️ 高 | □ |
| 可能当庭宣判 | 庭审当日 | ⚠️ 高 | □ |

### 1.3 时间风险防控措施

```
□ 接案当日即开始阅卷
□ 接案次日完成首次会见
□ 制定详细工作时间表
□ 设置多个时间提醒节点
□ 提前与办案机关确认开庭时间
□ 庭前完成所有准备工作
```

## 二、程序适用风险

### 2.1 不应适用简易程序情形核对

**发现以下情形应提出异议**：

| 序号 | 情形 | 核对 | 处理措施 |
|-----|------|------|---------|
| 1 | 被告人是盲、聋、哑人 | □ | 建议转普通程序 |
| 2 | 被告人是精神病人（未完全丧失辨认控制能力） | □ | 建议转普通程序 |
| 3 | 有重大社会影响的案件 | □ | 建议转普通程序 |
| 4 | 共同犯罪中部分被告人不认罪或有异议 | □ | 建议转普通程序 |
| 5 | 辩护人作无罪辩护 | □ | 建议转普通程序 |
| 6 | 被告人认罪但可能不构成犯罪 | □ | 建议转普通程序 |
| 7 | 其他不宜适用情形 | □ | 建议转普通程序 |

### 2.2 庭审中应申请转程序情形

| 序号 | 情形 | 核对 | 处理措施 |
|-----|------|------|---------|
| 1 | 被告人行为可能不构成犯罪 | □ | 当庭申请转程序 |
| 2 | 被告人可能不负刑事责任 | □ | 当庭申请转程序 |
| 3 | 被告人当庭否认起诉事实 | □ | 当庭申请转程序 |
| 4 | 案件事实不清、证据不足 | □ | 当庭申请转程序 |
| 5 | 其他不宜适用情形 | □ | 当庭申请转程序 |

## 三、认罪认罚风险

### 3.1 虚假认罪风险核对

```
审查要点：
□ 被告人认罪是否出于自愿
□ 是否受到威胁、引诱、欺骗
□ 认罪内容与案件证据是否一致
□ 被告人是否理解认罪的法律后果
□ 是否存在替人顶罪情况
□ 是否存在为保护他人而认罪情况
```

### 3.2 认罪认罚告知风险

```
必须告知并确认：
□ 认罪认罚的法律规定
□ 认罪认罚可能的从宽幅度
□ 认罪认罚放弃的诉讼权利
□ 简易/速裁程序的庭审特点
□ 当庭宣判的可能性
□ 上诉权利及上诉后果
```

### 3.3 具结书签署风险

```
签署前确认：
□ 辩护人或值班律师在场
□ 被告人对犯罪事实无异议
□ 被告人对适用法律无异议
□ 被告人对量刑建议无异议
□ 被告人对适用程序无异议
□ 被告人签字系本人自愿

签署后记录：
□ 保留签署过程记录
□ 记录被告人当时状态
□ 记录告知内容
```

## 四、量刑协商风险

### 4.1 量刑协商准备核对

```
协商前准备：
□ 全面梳理量刑情节
□ 对照量刑指导意见计算
□ 收集量刑情节证据材料
□ 检索类案判例
□ 准备具体量刑建议区间
□ 准备备选方案
```

### 4.2 量刑建议审查核对

```
审查量刑建议：
□ 是否在法定刑幅度内
□ 是否遗漏量刑情节
□ 从宽幅度是否合理
□ 是否明显不当
□ 与类案量刑是否相当
```

### 4.3 量刑协商记录

```
应当记录：
□ 协商时间、地点
□ 参与人员
□ 协商内容
□ 双方意见
□ 最终达成的量刑建议
□ 被告人确认意见
```

---

## 五、工作质量风险

### 5.1 "程序简化≠工作简化"核对

```
即使程序简化，也必须完成：
□ 完整阅卷
□ 证据审查
□ 与当事人充分沟通
□ 形成书面辩护意见
□ 与办案机关沟通
□ 庭前准备
□ 庭后跟进
```

### 5.2 会见工作核对

```
简易/速裁程序会见要点：
□ 及时会见（接案后立即）
□ 告知程序特点
□ 核实案件事实
□ 确认认罪态度
□ 告知量刑协商情况
□ 庭前辅导
□ 制作会见笔录
```

### 5.3 辩护意见核对

```
辩护意见必须包含：
□ 案件基本情况
□ 量刑情节分析
□ 量刑计算依据
□ 具体量刑建议
□ 法律依据
□ 证据材料附件
```

---

## 六、沟通风险

### 6.1 与当事人沟通核对

```
必须沟通确认的内容：
□ 对指控事实的态度
□ 是否同意认罪认罚
□ 是否同意适用简易/速裁程序
□ 对量刑建议的意见
□ 对辩护意见的确认
□ 程序转换权利的告知
```

### 6.2 与办案机关沟通核对

```
与检察机关沟通：
□ 确认适用程序
□ 量刑协商
□ 提交辩护意见
□ 确认开庭安排

与审判机关沟通：
□ 确认开庭时间
□ 庭前沟通量刑意见
□ 庭后跟进判决结果
```

## 七、文书风险

### 7.1 文书提交核对

| 文书类型 | 提交对象 | 提交时限 | 核对 |
|---------|---------|---------|------|
| 辩护意见 | 检察院/法院 | 速裁4天/简易一周 | □ |
| 量刑协商意见书 | 检察院 | 审查起诉阶段 | □ |
| 适用普通程序建议书 | 法院 | 发现问题时立即 | □ |
| 会见笔录 | 存档 | 每次会见后 | □ |

### 7.2 文书格式核对

```
文书格式要求：
□ 标题：黑体三号加粗居中
□ 正文：宋体四号
□ 行距：固定值27磅
□ 页边距：上下2.54cm，左右3.18cm
□ 占位符：使用"X×"格式
```

## 八、庭审风险

### 8.1 简易程序庭审核对

```
庭审前准备：
□ 确认开庭时间地点
□ 携带全部材料
□ 辩护意见已提交
□ 与被告人已沟通庭审注意事项

庭审中注意：
□ 确认被告人收到起诉书
□ 确认被告人对指控无异议
□ 简明发表质证意见
□ 简明发表辩护意见
□ 指导被告人最后陈述
```

### 8.2 速裁程序庭审核对

```
庭审前准备：
□ 辩护意见已提交法院
□ 量刑问题已与法官沟通
□ 被告人知悉庭审流程

庭审中注意：
□ 不进行法庭调查和辩论
□ 确认认罪认罚自愿性
□ 简要发表辩护意见
□ 指导被告人最后陈述
□ 准备接受当庭宣判
```

### 8.3 庭审应急处理

| 突发情况 | 应急处理 | 核对 |
|---------|---------|------|
| 被告人当庭翻供 | 申请休庭沟通 | □ |
| 发现新的无罪证据 | 申请转普通程序 | □ |
| 量刑建议明显不当 | 当庭提出异议 | □ |
| 公诉人出示新证据 | 申请休庭审查 | □ |

## 九、执业禁止清单

### 9.1 绝对禁止行为

```
⛔ 在证据不充分时劝导当事人认罪
⛔ 强迫当事人认罪认罚
⛔ 向当事人承诺案件结果
⛔ 隐瞒认罪认罚的不利后果
⛔ 未经当事人同意变更辩护策略
⛔ 泄露案件信息
⛔ 伪造证据
⛔ 指使当事人作虚假陈述
```

### 9.2 避免行为

```
⚠️ 因程序简化而简化工作
⚠️ 因时间紧迫而遗漏关键环节
⚠️ 未与当事人充分沟通即提交辩护意见
⚠️ 未审查量刑建议合理性即同意
⚠️ 未告知当事人程序转换权利
⚠️ 未在场见证具结书签署
```

## 十、风险应对预案

### 10.1 期限错过应对

```
如果错过辩护意见提交期限：
□ 立即补充提交
□ 书面说明延误原因
□ 请求法院/检察院予以考虑
□ 庭审中充分发表意见
□ 庭后补充书面辩护意见
```

### 10.2 程序转换应对

```
如果需要从简易/速裁转普通程序：
□ 及时书面提出申请
□ 说明转换理由
□ 提供相关证据材料
□ 与当事人沟通程序变化
□ 调整辩护策略
```

### 10.3 认罪认罚撤回应对

```
如果当事人需要撤回认罪认罚：
□ 与当事人充分沟通原因
□ 告知撤回的法律后果
□ 评估撤回的利弊
□ 如确需撤回，协助提出申请
□ 准备转为普通程序后的辩护方案
```
