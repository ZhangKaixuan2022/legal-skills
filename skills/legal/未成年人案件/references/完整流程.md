## 模块二：基本原则

### 2.1 "教育、感化、挽救"方针

对未成年人案件，《刑事诉讼法》确立了"教育、感化、挽救"的方针，确立了"教育为主、惩罚为辅"的原则。

| 方针 | 内涵 | 辩护体现 |
|-----|------|---------|
| 教育 | 通过诉讼程序教育未成年人认识错误 | 促使认罪悔罪、参与法治教育 |
| 感化 | 通过人文关怀感化未成年人 | 争取社区帮教、家庭监护 |
| 挽救 | 给予改过自新的机会 | 争取不逮捕、不起诉、轻刑 |

### 2.2 "教育为主、惩罚为辅"原则

| 原则 | 司法适用 | 辩护策略 |
|-----|---------|---------|
| 教育为主 | 优先考虑非羁押措施 | 强调监护帮教条件 |
| 惩罚为辅 | 慎用逮捕、慎重起诉 | 突出悔罪表现、从轻情节 |

### 2.3 十大基本原则强化适用

在未成年人案件中，以下原则需要特别强调：

| 序号 | 原则 | 未成年人案件强化适用 |
|-----|------|-------------------|
| 1 | 当事人利益原则 | 特别保护未成年人利益，优先考虑其成长 |
| 2 | 保密义务原则 | 严格保密，不得泄露身份信息 |
| 3 | 依法辩护原则 | 充分利用未成年人特别程序 |
| 4 | 尊重意愿原则 | 兼顾未成年人和法定代理人意愿 |
| 5 | 有效辩护原则 | 积极争取最有利结果 |

### 2.4 未成年人人权六项保护

> **操作指引152核心要点**

| 序号 | 权利内容 | 法律依据 | 辩护职责 |
|-----|---------|---------|---------|
| 1 | 姓名不公开 | 《刑事诉讼法》相关规定 | 提醒司法机关保密 |
| 2 | 与成年人分开关押 | 看守所管理规定 | 核实羁押条件 |
| 3 | 讯问时法定代理人在场 | 第281条 | 确保合法讯问 |
| 4 | 不公开开庭 | 第285条 | 申请不公开审理 |
| 5 | 犯罪记录封存 | 第286条 | 全程申请封存 |
| 6 | 减轻处罚 | 《刑法》第17条 | 强调法定从轻 |

## 模块三：社会调查

### 3.1 社会调查概述

> **操作指引153核心要点**

《刑事诉讼法》规定，司法机关办理未成年人犯罪案件，根据情况**可以**对未成年人犯罪嫌疑人、被告人的成长经历、犯罪原因、监护教育等情况进行调查。

**注意**：法律用的是"可以"而非"应当"，司法机关有选择性地进行社会调查。

### 3.2 辩护律师三项工作

| 工作类型 | 具体内容 | 操作要点 |
|---------|---------|---------|
| **申请调查** | 申请司法机关对当事人进行社会调查 | 主动提交申请书 |
| **自行调查** | 律师自行调查成长经历、犯罪原因 | 走访家庭、学校、社区 |
| **提出修正** | 对司法机关调查报告提出修正补充意见 | 审查报告内容，补充有利信息 |

### 3.3 社会调查内容清单

#### 成长经历

| 调查项目 | 调查内容 |
|---------|---------|
| 家庭背景 | 父母职业、婚姻状况、经济条件、教育方式 |
| 教育经历 | 就读学校、学习成绩、在校表现、师生关系 |
| 社会交往 | 朋友圈、日常活动、兴趣爱好 |
| 心理状况 | 性格特点、心理健康状况 |

#### 犯罪原因

| 因素类型 | 分析要点 |
|---------|---------|
| 直接诱因 | 导致犯罪的直接原因 |
| 家庭因素 | 家庭教育缺失、亲子关系等 |
| 社会因素 | 不良交往、社会环境等 |
| 心理因素 | 冲动、认知偏差等 |

#### 监护教育条件

| 评估项目 | 评估内容 |
|---------|---------|
| 监护能力 | 法定代理人是否具备监护能力 |
| 家庭条件 | 居住条件、经济条件 |
| 帮教资源 | 社区、学校帮教条件 |
| 改造条件 | 是否具备教育改造条件 |

### 3.4 申请时机与对象

| 诉讼阶段 | 申请对象 | 建议时机 |
|---------|---------|---------|
| 侦查阶段 | 公安机关 | 首次会见后尽快 |
| 审查起诉阶段 | 人民检察院 | 案件移送后3日内 |
| 审判阶段 | 人民法院 | 收到起诉书后5日内 |

## 模块四：审查批捕阶段辩护（不逮捕申请）

### 4.1 核心目标

> **操作指引154核心要点**

促成检察院作出**不批准逮捕决定**，使未成年犯罪嫌疑人变更为取保候审。

### 4.2 法律依据

《最高人民检察院刑事诉讼规则》规定：

> "人民检察院办理未成年犯罪嫌疑人审查逮捕案件，应当根据未成年犯罪嫌疑人涉嫌犯罪的事实、主观恶性、有无监护与社会帮教条件等，综合衡量其社会危险性，**严格限制适用逮捕措施**"

### 4.3 不批准逮捕的法定情形

#### 情形一：罪行较轻型（应当不批准逮捕）

| 条件 | 内容 |
|-----|------|
| 罪行程度 | 罪行较轻 |
| 监护条件 | 具备有效监护条件或社会帮教措施 |
| 社会危险性 | 没有社会危险性或社会危险性较小 |
| 诉讼影响 | 不逮捕不致妨害诉讼正常进行 |

#### 情形二：罪行较重但有悔罪表现型（可以不批准逮捕）

对于罪行比较严重，但主观恶性不大，有悔罪表现，具备有效监护条件或社会帮教措施，具有下列情形之一的：

| 序号 | 法定情形 |
|-----|---------|
| （一） | 初次犯罪、过失犯罪的 |
| （二） | 犯罪预备、中止、未遂的 |
| （三） | 有自首或者立功表现的 |
| （四） | 犯罪后如实交代罪行，真诚悔罪，积极退赃，尽力减少和赔偿损失，被害人谅解的 |
| （五） | 不属于共同犯罪的主犯或者集团犯罪中的首要分子的 |
| （六） | 属于已满十四周岁不满十六周岁的未成年人或者系在校学生的 |

### 4.4 三方面申请材料（核心）

#### 材料一：监护条件或社会帮教措施证明

| 当事人类型 | 证明出具单位 | 证明内容 |
|-----------|------------|---------|
| **本地人** | 居委会/村委会 | 家庭监护条件、社区帮教措施 |
| **外地人** | 所在企业 | 单位监管条件、帮教方案 |
| **在校学生** | 学校 | 在校表现、学校帮教方案 |

#### 材料二：认罪悔罪证明

| 材料名称 | 内容要求 | 制作要点 |
|---------|---------|---------|
| 会见笔录 | 记录认罪悔罪态度 | 律师向嫌疑人讲解法律，促使认罪 |
| 悔过书 | 亲笔书写 | 深刻认识错误和危害 |
| 法律讲解记录 | 律师教育引导过程 | 体现教育感化工作 |

#### 材料三：损失挽回证明

| 材料名称 | 内容 |
|---------|------|
| 赔礼道歉函 | 家属代为向被害方赔礼道歉 |
| 赔偿协议 | 赔偿金额、支付方式 |
| 和解协议 | 双方达成和解的协议 |
| 谅解书 | 被害方出具的谅解文书 |

### 4.5 成功要点

一般而言，达到上述三个条件，检察院不予逮捕，改变为取保候审的强制措施**还是很有希望的**。

## 模块五：附条件不起诉

### 5.1 制度概述

> **操作指引155核心要点**

"附条件不起诉"是《刑事诉讼法》规定的未成年人特别程序。

**重大意义**：被决定不起诉，相当于**没有犯罪记录**，对未成年人的人权保障极其有利，也极有利于未成年人的成长。

### 5.2 法定适用条件

| 条件类型 | 具体要求 |
|---------|---------|
| **年龄条件** | 未成年人（犯罪时不满18周岁） |
| **罪名条件** | 刑法第四章（侵犯公民人身权利、民主权利罪）、第五章（侵犯财产罪）、第六章（妨害社会管理秩序罪） |
| **刑期条件** | 可能判处**一年有期徒刑以下**刑罚 |
| **起诉条件** | 符合起诉条件 |
| **悔罪条件** | 有悔罪表现 |

### 5.3 考验期规定

| 项目 | 内容 |
|-----|------|
| 考验期限 | **6个月至1年** |
| 考验期起算 | 附条件不起诉决定作出之日 |
| 考验期满结果 | 检察院作出不起诉决定 = 没有犯罪记录 |

### 5.4 撤销情形

在考验期内有下列情形的，撤销附条件不起诉，提起公诉：

| 序号 | 撤销情形 |
|-----|---------|
| 1 | 实施新的犯罪或者发现决定附条件不起诉以前还有其他犯罪需要追诉的 |
| 2 | 违反治安管理规定或者考察机关有关附条件不起诉的监督管理规定，情节严重的 |

### 5.5 申请材料清单

| 序号 | 材料名称 | 内容要求 |
|-----|---------|---------|
| 1 | **附条件不起诉申请书** | 基本情况、犯罪事实、悔罪表现、法律适用 |
| 2 | **会见笔录** | 证实认罪悔罪 |
| 3 | **社会帮教方案** | 证明可以帮教、可以改过自新 |
| 4 | 监护人承诺书 | 承诺监督管教 |
| 5 | 谅解书（如有） | 被害方谅解 |

### 5.6 后续监督义务

在检察院作出附条件不起诉后，辩护律师还要：

1. **日常监督**：监督未成年人当事人服从规定、遵纪守法
2. **定期联系**：定期了解当事人情况
3. **风险提示**：告知违规后果
4. **确保结果**：确保最终获得不起诉决定

## 模块六：犯罪记录封存

### 6.1 制度概述

> **操作指引156核心要点**

《刑事诉讼法》规定："犯罪的时候不满十八周岁，被判处五年有期徒刑以下刑罚的，应当对相关犯罪记录予以封存。"

**封存效果**：封存后，"**不得向任何单位和个人提供**"。这对当事人来说，是一种最大的保护。

### 6.2 封存条件

| 条件类型 | 具体要求 |
|---------|---------|
| **年龄条件** | 犯罪的时候不满十八周岁 |
| **刑罚条件** | 被判处五年有期徒刑以下刑罚 |

**注意**：不起诉决定、附条件不起诉决定也应当封存相关记录。

### 6.3 辩护律师全流程监督职责

辩护律师首先要判断当事人是否属于应当封存的对象。如果属于，在各阶段应当：

| 诉讼阶段 | 律师职责 | 具体操作 |
|---------|---------|---------|
| **侦查阶段** | 提醒保密 | 提醒侦查机关加强保密意识 |
| **审查起诉阶段** | 防止外泄 | 提醒监管单位保密；不起诉的立即封存 |
| **审判阶段** | 申请不公开 | 提醒法院不公开审理，宣判时不组织旁听 |
| **宣判后** | 申请封存 | 提请法庭对犯罪记录予以封存 |

### 6.4 核心要求

**律师首先要有犯罪记录封存的意识**，在整个诉讼过程中不断地提醒和提示相关办案机关注意对案件的保密，及时进行犯罪记录的封存。

## 模块七：庭审阶段特殊规范

### 7.1 不公开审理

#### 法律依据

《刑事诉讼法》第285条规定，审判的时候被告人不满十八周岁的案件，**不公开审理**。

#### 辩护律师工作

| 工作事项 | 具体内容 |
|---------|---------|
| 申请确认 | 主动申请或确认法院不公开审理 |
| 旁听管理 | 提醒法院限制旁听人员 |
| 宣判方式 | 宣判时不得组织旁听，不得向外泄露 |
| 材料管理 | 庭审材料不得外传 |

### 7.2 法定代理人/合适成年人到场

#### 到场要求

| 诉讼环节 | 到场人员 | 法律依据 |
|---------|---------|---------|
| 讯问 | 法定代理人到场 | 第281条 |
| 庭审 | 法定代理人到场 | 第281条 |
| 宣判 | 法定代理人到场 | 相关规定 |

#### 合适成年人制度

当法定代理人无法到场时，可以通知：
- 其他成年近亲属
- 所在学校、单位、居住地基层组织的代表
- 其他合适成年人

#### 辩护律师注意事项

1. 核实到场人身份（法定代理人或合适成年人）
2. 在会见笔录中记录到场情况
3. 讯问时法定代理人未到场，可作为程序性辩护意见

### 7.3 与一审Skill庭审模块的衔接

## 模块八：执业风险防范

### 8.1 保密义务（最高风险）

> **【案例8-1警示】**

一起未成年人强奸案，律师在不公开开庭以后，将无罪辩护的辩护意见在网络上公布，其中包括未成年被告人的身份特征以及被害人的身份特征，造成同案未成年被告人信息对外泄露，也造成被害人的隐私公开。**律师协会对该律师的执业行为进行了调查**。

### 8.2 红线禁止清单

| 序号 | 禁止行为 | 风险等级 |
|-----|---------|---------|
| 1 | 公开未成年人姓名、身份特征 | ⛔ 极高 |
| 2 | 泄露未成年被害人信息 | ⛔ 极高 |
| 3 | 公布不公开审理案件的辩护意见 | ⛔ 极高 |
| 4 | 接受媒体采访透露案件信息 | ⛔ 高 |
| 5 | 在社交媒体发布案件相关内容 | ⛔ 高 |

### 8.3 合规要求

| 合规事项 | 具体要求 |
|---------|---------|
| 文书保管 | 未成年人案件文书单独保管，标注"保密" |
| 信息处理 | 脱敏处理后方可用于教学、研讨 |
| 媒体应对 | 拒绝接受采访，或仅提供程序性说明 |
| 网络发布 | 禁止发布任何可识别未成年人身份的信息 |

## 附录四：错误处理

### 常见错误场景及处理

| 错误场景 | 处理方式 |
|---------|---------|
| 年龄判断错误 | 重新核实犯罪时年龄，以出生日期为准 |
| 遗漏不公开审理申请 | 立即补充申请，说明当事人为未成年人 |
| 遗漏封存申请 | 各阶段均可补充申请 |
| 信息泄露 | 立即删除，书面道歉，报告律所 |
| 附条件不起诉条件不符 | 转向普通不起诉或量刑辩护 |

### 风险预警

| 阶段 | 风险预警内容 |
|-----|-------------|
| 接案时 | 确认当事人犯罪时年龄 |
| 会见时 | 通知法定代理人，注意沟通方式 |
| 阅卷时 | 案卷单独保管，标注保密 |
| 取证时 | 涉及未成年人需申请许可 |
| 庭审时 | 确认不公开审理 |
| 结案时 | 申请犯罪记录封存 |

### 违规处理预案

| 违规情形 | 补救措施 |
|---------|---------|
| 不慎泄露信息 | 立即删除，书面道歉，报告律所 |
| 媒体曝光 | 配合律所危机处理，不再发表评论 |
| 被投诉 | 如实陈述，积极配合调查 |
