# 未成年人案件执业风险清单

> 来源：《刑事辩护实务操作技能与执业风险防范（第三版）》
> 适用对象：办理未成年人刑事案件的辩护律师

---

## 一、保密义务风险（最高风险）

### 1.1 红线禁止事项

| 序号 | 禁止行为 | 风险等级 | 法律后果 |
|-----|---------|---------|---------|
| 1 | 公开未成年人姓名、身份特征 | ⛔ 极高 | 行业处分、民事赔偿 |
| 2 | 泄露未成年被害人信息 | ⛔ 极高 | 行业处分、民事赔偿 |
| 3 | 公布不公开审理案件的辩护意见 | ⛔ 极高 | 行业处分 |
| 4 | 接受媒体采访透露案件信息 | ⛔ 高 | 行业处分 |
| 5 | 在社交媒体发布案件相关内容 | ⛔ 高 | 行业处分 |

### 1.2 警示案例

> **【案例8-1】**一起未成年人强奸案，由于受媒体高度关注，其中一名未成年被告人的律师在不公开开庭以后，将无罪辩护的辩护意见在网络上公布，其中包括未成年被告人的身份特征以及被害人的身份特征，造成同案未成年被告人信息对外泄露，也造成被害人的隐私公开。鉴于此，**律师协会对该律师的执业行为进行了调查**。

### 1.3 合规要求

| 合规事项 | 具体要求 |
|---------|---------|
| 文书保管 | 未成年人案件文书单独保管，标注"保密" |
| 信息处理 | 脱敏处理后方可用于教学、研讨 |
| 媒体应对 | 拒绝接受采访，或仅提供程序性说明 |
| 网络发布 | 禁止发布任何可识别未成年人身份的信息 |

## 二、不公开审理程序风险

### 2.1 程序性要求

| 检查项 | 合规要求 | 违规风险 |
|-------|---------|---------|
| 申请不公开审理 | 主动申请或确认法院不公开审理 | 程序瑕疵 |
| 旁听人员管理 | 提醒法院限制旁听人员 | 信息泄露 |
| 宣判方式 | 不得组织旁听、不得公开报道 | 隐私泄露 |
| 材料管理 | 庭审材料不得外传 | 保密违规 |

### 2.2 风险场景与应对

| 风险场景 | 正确应对 |
|---------|---------|
| 媒体要求采访 | 婉拒，说明涉及未成年人不便透露 |
| 家属要求公开 | 解释不公开审理的意义，争取理解 |
| 同行询问案情 | 仅可讨论法律问题，不透露当事人信息 |

---

## 三、法定代理人到场风险

### 3.1 到场要求检查清单

| 诉讼环节 | 到场要求 | 未到场处理 |
|---------|---------|-----------|
| 讯问 | 法定代理人应到场 | 申请合适成年人到场 |
| 询问（证人） | 法定代理人应到场 | 申请合适成年人到场 |
| 庭审 | 法定代理人应到场 | 申请合适成年人到场 |
| 宣判 | 法定代理人应到场 | 申请合适成年人到场 |

### 3.2 辩护律师注意事项

| 注意事项 | 具体要求 |
|---------|---------|
| 核实到场人身份 | 确认是否为法定代理人或合适成年人 |
| 记录到场情况 | 在会见笔录中记录法定代理人到场情况 |
| 发现违规 | 讯问时法定代理人未到场，可作为程序性辩护意见 |

---

## 四、犯罪记录封存风险

### 4.1 全流程提醒义务

| 诉讼阶段 | 辩护律师职责 | 风险点 |
|---------|------------|-------|
| 侦查阶段 | 提醒侦查机关加强保密意识 | 遗漏提醒导致信息泄露 |
| 审查起诉阶段 | 提醒检察院保密，不起诉时申请封存 | 未及时申请封存 |
| 审判阶段 | 提醒法院不公开审理、宣判时不组织旁听 | 公开审理导致信息泄露 |
| 宣判后 | 提请法庭对犯罪记录予以封存 | 遗漏封存申请 |

### 4.2 封存条件确认

| 确认项 | 内容 |
|-------|------|
| 年龄条件 | 犯罪时不满18周岁 |
| 刑罚条件 | 被判处五年以下刑罚 |
| 封存范围 | 相关犯罪记录 |
| 封存效果 | 不得向任何单位和个人提供 |

---

## 五、会见工作风险

### 5.1 会见特殊要求

| 要求 | 内容 | 违规后果 |
|-----|------|---------|
| 法定代理人告知 | 会见前通知法定代理人 | 程序瑕疵 |
| 会见方式 | 注意沟通方式，考虑未成年人心理特点 | 沟通效果差 |
| 笔录制作 | 使用未成年人能理解的语言 | 理解偏差 |

### 5.2 禁止行为

| 禁止行为 | 风险等级 |
|---------|---------|
| 单独与未成年异性当事人会见时关闭门窗 | ⛔ 高 |
| 使用恐吓、威胁语言 | ⛔ 高 |
| 传递与案件无关的物品 | ⛔ 高 |
| 未经许可拍照、录音 | ⛔ 中 |

---

## 六、调查取证风险

### 6.1 特殊注意事项

| 取证对象 | 注意事项 |
|---------|---------|
| 未成年证人 | 应通知其法定代理人到场 |
| 未成年被害人 | 需经检察院或法院许可 |
| 学校 | 注意保护未成年人隐私 |
| 社区 | 调查时不透露具体案情 |

### 6.2 社会调查风险

| 风险点 | 防范措施 |
|-------|---------|
| 调查时泄露案情 | 仅说明调查目的，不透露具体犯罪事实 |
| 调查材料保管 | 单独归档，标注保密 |
| 调查报告使用 | 仅用于辩护目的 |

---

## 七、文书工作风险

### 7.1 文书保密要求

| 文书类型 | 保密要求 |
|---------|---------|
| 辩护意见 | 不得公开发布 |
| 会见笔录 | 单独保管 |
| 社会调查报告 | 仅限辩护使用 |
| 申请书 | 不透露给无关第三方 |

### 7.2 文书内容规范

| 检查项 | 要求 |
|-------|------|
| 当事人称谓 | 使用化名或简称 |
| 身份信息 | 不完整披露 |
| 案件细节 | 避免过度详细描述 |

---

## 八、与办案机关沟通风险

### 8.1 沟通注意事项

| 沟通对象 | 注意事项 |
|---------|---------|
| 侦查人员 | 提醒未成年人特殊保护要求 |
| 检察官 | 强调"教育为主、惩罚为辅"原则 |
| 法官 | 申请不公开审理、犯罪记录封存 |

### 8.2 禁止行为

| 禁止行为 | 风险等级 |
|---------|---------|
| 承诺案件结果 | ⛔ 高 |
| 暗示"走关系" | ⛔ 极高 |
| 贬低司法机关 | ⛔ 中 |

---

## 九、附条件不起诉后续风险

### 9.1 考验期管理

| 管理事项 | 要求 |
|---------|------|
| 跟踪联系 | 定期了解当事人情况 |
| 法治教育 | 提醒遵纪守法 |
| 风险提示 | 告知考验期内违规后果 |

### 9.2 风险场景

| 风险场景 | 应对措施 |
|---------|---------|
| 当事人违规 | 及时提醒，协助纠正 |
| 考验期内再犯 | 如实告知法律后果 |
| 考验期满 | 提醒检察院作出不起诉决定 |

## 十、执业风险自查表

### 接案阶段

- [ ] 确认当事人年龄（犯罪时是否不满18周岁）
- [ ] 通知法定代理人参与委托
- [ ] 签署保密承诺书

### 会见阶段

- [ ] 通知法定代理人会见时间
- [ ] 会见笔录记录法定代理人到场情况
- [ ] 使用适合未成年人的沟通方式

### 阅卷阶段

- [ ] 案卷材料单独保管
- [ ] 标注"未成年人案件-保密"

### 调查取证阶段

- [ ] 涉及未成年人取证申请许可
- [ ] 调查时保护隐私

### 审判阶段

- [ ] 申请/确认不公开审理
- [ ] 提醒法院限制旁听
- [ ] 提醒宣判时保密

### 结案阶段

- [ ] 申请犯罪记录封存
- [ ] 销毁不必要的案件材料副本
- [ ] 归档材料标注保密

## 十一、违规处理预案

| 违规情形 | 补救措施 |
|---------|---------|
| 不慎泄露信息 | 立即删除，书面道歉，报告律所 |
| 媒体曝光 | 配合律所危机处理，不再发表评论 |
| 被投诉 | 如实陈述，积极配合调查 |
