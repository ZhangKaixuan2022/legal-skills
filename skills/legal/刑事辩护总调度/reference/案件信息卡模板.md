# 刑事案件信息卡模板

> **用途**：标准化记录刑事案件核心信息，便于刑事辩护总调度进行专业路由和阶段判断
> **维护边界**：本模板是系统记录字段参考；实际创建、更新和归档由法律工作总控按事项工作区协议执行，阶段 Skill 只补充其职责范围内的信息

---

| **联系方式** | 本人或家属联系电话 | 138XXXXXXXX |

| **现住址** | 经常居住地 | 同上 |

## 二、案件状态标记

### 2.1 当前诉讼阶段（核心路由字段）

**填写规则**：从以下选项中选择一个

| 阶段值 | 说明 | 路由目标 |
|-------|------|---------|
| `案件承接` | 刚接受委托，尚未正式介入 | 案件承接与委托 |
| `侦查阶段` | 公安机关侦查期间 | 侦查阶段辩护 |
| `审查起诉` | 检察院审查起诉期间 | 审查起诉阶段辩护 |
| `一审阶段` | 法院一审审理期间 | 一审阶段辩护 |
| `二审阶段` | 上诉/抗诉，二审审理期间 | 二审阶段辩护 |
| `执行阶段` | 判决生效后执行期间 | 暂无专用Skill |
| `特殊程序` | 涉及违法所得没收/强制医疗/再审申诉 | 特殊程序 |

### 2.2 阶段时间节点

| 字段 | 填写说明 | 示例 |
|-----|---------|------|
| **当前阶段启动日期** | 进入当前阶段的日期 | 2024年X月X日 |
| **刑事拘留日期** | 如有 | 2024年X月X日 |
| **逮捕日期** | 如有 | 2024年X月X日 |
| **移送审查起诉日期** | 侦查终结日期 | 2024年X月X日 |
| **提起公诉日期** | 收到起诉书日期 | 2024年X月X日 |
| **一审判决日期** | 一审宣判日期 | 2024年X月X日 |
| **上诉日期** | 如有 | 2024年X月X日 |
| **二审裁判日期** | 二审宣判日期 | 2024年X月X日 |

### 2.3 关键时间节点预警

| 字段 | 计算规则 | 当前值 |

## 三、特殊标签

### 3.1 特殊主体标签

| 标签 | 选项 | 当前值 | 路由影响 |
|-----|------|-------|---------|
| **是否未成年人案件** | 是/否 | 否 | 是→附加未成年人案件 |
| **是否可能判处死刑** | 是/否 | 否 | 是→附加死刑案件 |
| **是否涉外案件** | 是/否 | 否 | 提示特殊程序注意事项 |
| **是否单位犯罪** | 是/否 | 否 | 区分单位与自然人辩护 |

### 3.2 程序性标签

| 标签 | 选项 | 当前值 | 路由影响 |
|-----|------|-------|---------|
| **是否认罪认罚** | 是/否 | 否 | 是→启用认罪认罚嵌入模块 |
| **适用审理程序** | 普通程序/简易程序/速裁程序 | 普通程序 | 简易/速裁→附加简易速裁程序 |
| **是否涉及特殊程序** | 无/违法所得没收/强制医疗/再审申诉 | 无 | 有→附加特殊程序 |

### 3.3 案件性质标签

| 标签 | 选项 | 当前值 | 说明 |
|-----|------|-------|------|
| **案件复杂程度** | 简单/一般/复杂/重大 | 一般 | 影响工作量评估 |
| **是否共同犯罪** | 是/否 | 否 | 是→注意同案犯辩护利益冲突 |
| **当事人在共同犯罪中的地位** | 主犯/从犯/胁从犯/不适用 | 不适用 | 影响量刑辩护策略 |
| **是否有被害人** | 是/否 | 是 | 是→考虑和解、谅解策略 |

## 六、填写规范说明

### 6.1 必填字段

以下字段为案件承接时必须填写：
- 当事人姓名
- 涉嫌罪名
- 委托人姓名及与当事人关系
- 承办律师及律所
- 委托日期
- 当前诉讼阶段

### 6.2 动态更新规则

| 更新场景 | 更新字段 |
|---------|---------|
| 阶段转换时 | 当前诉讼阶段、阶段启动日期、已完成阶段记录 |
| 收到新文书时 | 对应案件编号、罪名（如有变化）|
| 时间节点变化时 | 关键时间节点预警字段 |
| 标签条件触发时 | 特殊标签字段 |

### 6.3 信息补充来源

| 阶段 | 信息补充来源 | 负责Skill |
|-----|-------------|----------|
| 案件承接 | 委托人陈述、委托材料 | 案件承接与委托 |
| 侦查阶段 | 会见笔录、强制措施文书 | 侦查阶段辩护 |
| 审查起诉 | 起诉意见书、案卷材料 | 审查起诉阶段辩护 |
| 一审阶段 | 起诉书、判决书 | 一审阶段辩护 |
| 二审阶段 | 一审判决书、二审裁定书 | 二审阶段辩护 |
