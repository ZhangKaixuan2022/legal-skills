# 二审阶段会见工作详细提纲

---

## 会见总体安排

二审阶段，律师一般至少会见**三次以上**：

| 会见次数 | 时机 | 主要任务 |
|---------|------|---------|
| 第一次 | 接受委托后 | 通报委托、签署上诉状、听取上诉理由 |
| 第二次 | 阅卷后/开庭前 | 通报进程、沟通辩护意见、告知开庭情况 |
| 第三次 | 案件审结前 | 案件进展、二审可能结果、裁判后应对 |

---

## 第一次会见任务清单

**时机**：接受委托后第一时间

**核心目的**：通报委托情况、听取上诉理由、签署上诉状

**任务清单**：
```
□ 向上诉人通报已委托律师的情况
□ 确认上诉人同意家属委托
□ 签署上诉状（如在上诉期内）
□ 介绍上诉程序和二审审限
□ 听取上诉人的辩解和上诉理由
□ 了解上诉辩解的依据
□ 核实一审判决认定的事实
□ 记录上诉人提供的新证据线索
□ 制作会见笔录
```

**重点询问**：
- 你对一审判决哪些部分不服？
- 你认为一审判决在事实认定上有什么错误？
- 你有没有新的证据或线索可以提供？
- 一审判决的量刑你是否接受？

**输出文书**：会见笔录、上诉状（如适用）

---

## 第二次会见任务清单

### 开庭案件

**时机**：阅卷后、开庭前

**核心目的**：通报进程、沟通辩护意见、庭前准备

**任务清单**：
```
□ 通报案件进程（已决定开庭）
□ 告知检察院阅卷导致审限延长
□ 介绍二审开庭程序
□ 告知二审开庭时上诉人的权利与注意事项
□ 沟通、商议并确定辩护意见
□ 核实辩护重点相关的事实和证据
□ 制作会见笔录
```

**重点沟通**：
- 二审开庭的流程和注意事项
- 辩护意见的主要内容
- 上诉人在庭上需要强调的要点
- 是否有新的情况需要说明

### 不开庭案件

**时机**：阅卷后、知悉不开庭决定后

**核心目的**：通报进程、确定书面辩护意见

**任务清单**：
```
□ 通报案件进程（不开庭审理）
□ 告知其应在法官来提问时需强调的供述与辩解内容
□ 沟通、确定书面辩护意见
□ 告知不开庭不等于不改判，律师会继续努力
□ 制作会见笔录
```

**重点沟通**：
- 不开庭的原因和后续程序
- 书面辩护意见的要点
- 如果法官来讯问需要强调什么
- 保持信心，律师会通过书面意见充分辩护

**输出文书**：会见笔录

---

## 第三次会见任务清单

**时机**：案件审结前

**核心目的**：通报进展、分析结果、告知裁判后应对

**任务清单**：
```
□ 通报案件的进展情况
□ 分析二审可能的结果
□ 告知二审裁判后的应对工作
□ 重点说明二审宣判方式：
   - 二审一般不作公开宣判
   - 二审裁判文书交由一审法院代为送达
   - 一审法院收到后直接到看守所送达
   - 送达后即生效
   - 律师可能后于上诉人知道结果
□ 制作会见笔录
```

**重点说明**：
- 二审的三种可能结果（维持、改判、发回重审）
- 每种结果的法律后果
- 裁判后的救济途径（如有）
- 律师的后续工作安排

**输出文书**：会见笔录

---

## 会见笔录生成要求

**会见笔录应包含**：

```
1. 会见基本信息
   - 会见时间、地点
   - 会见律师
   - 上诉人姓名
   
2. 会见内容记录
   - 律师告知的内容
   - 上诉人陈述的内容
   - 上诉人签名确认
   
3. 重要事项备注
   - 上诉人提供的新线索
   - 上诉人对辩护意见的意见
   - 需要进一步核实的问题
```

---

## 二审宣判方式特别说明

**向当事人说明的要点**：

1. **二审一般不作公开宣判**
2. **裁判文书由一审法院代为送达**：二审裁判文书不是由二审法院直接送达，而是交由一审法院代为送达
3. **送达方式**：一审法院收到后，直接到看守所向上诉人送达
4. **生效时间**：裁判文书送达后即生效，不存在上诉期
5. **信息不对称**：由于送达方式的原因，律师可能后于上诉人知道裁判结果
6. **这是正常程序**：这是法定的送达程序，不是律师不尽责的表现

**提前告知的意义**：
- 避免当事人误解律师工作
- 让当事人有心理准备
- 当事人知道结果后可以主动联系律师
