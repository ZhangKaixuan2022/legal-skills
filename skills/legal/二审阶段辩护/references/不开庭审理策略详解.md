# 二审阶段不开庭审理策略详解

---

## 一、不开庭的实践现状

### 1.1 开庭率现实

- 二审以不开庭审理为主流（实践中）
- 多数法院刑事二审上诉案件开庭率不超过10%
- 死刑立即执行案件一律开庭

### 1.2 不开庭的条件

合议庭通过阅卷认为"初步审查认定的事实与第一审认定的没有变化的"，可以不开庭审理。

### 1.3 不开庭的程序保障

- 合议庭"应当讯问被告人，听取其他当事人、辩护人、诉讼代理人的意见"
- 合议庭成员每个人都必须经过阅卷，甚至形成书面意见

## 二、书面辩护意见的充分性要求

### 2.1 核心原则

**不开庭≠不改判**，书面辩护意见同样重要

### 2.2 书面辩护意见要求

**详细、充分**：
- 不能因为不开庭就简化辩护意见
- 要像准备开庭一样准备书面意见
- 每个辩护要点都要详细论证

**有针对性**：
- 针对一审判决的每一个问题逐条回应
- 提出明确的辩护请求
- 不泛泛而谈，聚焦核心问题

**有说服力**：
- 事实依据+证据支撑+法律适用
- 逻辑清晰，说理充分
- 援引相关判例和法律依据

### 2.3 辩护意见两大方向

**方向一：审查一审辩护意见为什么没有得到采信**
```
步骤1：吃透一审辩护意见，审查和了解一审辩护意见提出的依据
步骤2：审查一审法院为什么不采纳
步骤3：针对性强化
   - 事实认定问题 → 加强证据的调查、分析与论证
   - 法律认定问题 → 加强对法律适用原理的研究与分析
步骤4：强化辩护意见，抓住一审未认定存在的实体与程序错误，加强说理
```

**方向二：在原有辩护意见外寻找新的辩护意见**
```
步骤1：对原审侦查、起诉的所有证据材料进行全面审查
步骤2：不拘泥于侦查、检察及审判人员的意见，也不拘泥于原有辩护意见
步骤3：对新的辩护观点进行反思：
   - 是否有确凿的依据？
   - 当时为什么没有人提出？
   - 是否需要补充调查取证？
   - 提出后是否会得到法院的采信？
步骤4：将新的辩护意见与原辩护意见进行整合
步骤5：提出全面、充分、法理情兼具的辩护意见
```

## 三、与承办法官的沟通技巧

### 3.1 沟通方式

1. 提交充分而详细的辩护意见
2. 与承办法官电话沟通
3. 预约见面沟通（如有必要）
4. 充分阐述辩护意见，使法官重视

### 3.2 沟通内容

1. 确认辩护意见已收悉
2. 询问是否需要补充材料
3. 了解案件审理进度
4. 提出新的意见、申请（如有）

### 3.3 沟通技巧

```
1. 把握好节奏，以免引起法官反感
2. 打电话联系应以有新的内容为前提
3. 如确实希望了解案件进展，也可进行沟通
4. 沟通目的：保持信息畅通，充分表达诉求，维护当事人利益
```

### 3.4 沟通时机

**庭前沟通**：围绕如何开好庭（非法证据排除、申请证人出庭、申请调取新证据等）

**庭后沟通**：案件如何审判（新的意见、证据、申请，合议庭/审委会讨论情况）

## 四、不开庭案件的工作心态

### 4.1 正确认识

1. 不可以因为不开庭就失去信心
2. 仍要保质保量地完成辩护任务
3. 不开庭不等于不改判
4. 继续努力，争取最好的结果

### 4.2 工作标准

即使不开庭，也要做到：
- 辩护意见详尽完整
- 法律论证严密有据
- 沟通反馈及时到位
- 对当事人负责到底

---

## 五、初步辩护意见7项内容

向承办法官提交的初步辩护意见，一般包括：

| 序号 | 内容要点 | 说明 |
|-----|---------|------|
| 1 | 一审判决事实不清、证据不足 | 上诉辩护意见的重点，也是开庭的法定理由 |
| 2 | 被告人的辩解可以成立 | 论证被告人辩解的合理性 |
| 3 | 辩护意见可以成立 | 一审辩护意见的延续与强化 |
| 4 | 一审判决定性不准 | 罪名认定错误 |
| 5 | 一审判决量刑失当 | 量刑过重 |
| 6 | 有新的证据证明一审判决不能成立 | 附新证据材料 |
| 7 | 提出开庭审理本案的申请 | 争取开庭 |

## 六、争取开庭审理策略

### 6.1 二审开庭的重要性

- 实践规律："开庭改判可能性大，不开庭基本维持原判"
- 争取开庭是二审辩护的**第一个工作目标**

### 6.2 法定应开庭情形（刑诉法第234条）

1. 被告人、自诉人及其法定代理人对第一审认定的事实、证据提出异议，可能影响定罪量刑的上诉案件
2. 被告人被判处死刑的上诉案件
3. 人民检察院抗诉的案件
4. 其他应当开庭审理的案件

### 6.3 争取开庭的策略

**策略一：强调事实证据争议**
- 这是法定开庭理由
- 必须认真撰写，充分说理
- 收集和强调原审事实和证据中存在的问题
- 论证与定罪量刑的关联性

**策略二：提交新证据**
- 有新证据的，开庭可能性大增
- 新证据需要质证，法院倾向于开庭

**策略三：及时提交初步辩护意见**
- 在案件到达承办法官处半个月内提交
- 让合议庭及时了解辩护意见
- 明确提出开庭审理申请
