> **适用对象**：上诉人（原审被告人）、检察院抗诉案件的原审被告人

## 模块一：功能概述与双模式启动

### 1.1 Skill功能描述

> 本Skill亦可由刑事辩护总调度Skill路由进入：skills/legal/刑事辩护总调度/SKILL.md

| 功能领域 | 具体内容 |

| 委托接案 | 委托手续递交、案件移送跟踪、一审材料衔接 |
| 阅卷审查 | 二审阅卷7大重点、一审判决分析、新证据线索发现 |
| 会见策略 | 三次+会见任务分解、会见笔录生成 |

### 1.2 适用范围

**适用**：被告人上诉案件、检察院抗诉案件、上诉+抗诉案件

**不适用/需路由**：
- 死刑立即执行案件 → 死刑案件子Skill
- 发回重审案件 → 一审阶段Skill
- 再审/申诉案件 → 特殊程序Skill

### 1.3 双模式启动（摘要）

**模式A：全新案件启动** - 确认案件基本信息、委托情况，进入委托接案模块
**模式B：从一审Skill衔接** - 读取一审工作成果，跳过委托手续，直接进入材料衔接

📋 完整流程 → [references/双模式启动详细流程.md](双模式启动详细流程.md)

### 1.4 二审核心特点

| 对比维度 | 一审 | 二审 |
|---------|------|------|
| 审理对象 | 起诉书指控的犯罪事实 | 一审判决的认定 |
| 辩护目标 | 无罪/罪轻/量刑 | 改判/发回重审 |
| 开庭情况 | 原则上开庭 | 以不开庭为主（实践中） |
| 裁判效力 | 上诉期满生效 | 送达即生效 |

**二审辩护两大工作目标**：1. 争取开庭审理  2. 争取改判或发回重审

## 模块三：二审委托接案与信息接收（摘要）

### 3.1 委托手续递交

**方式一**：随上诉状一并递交（上诉期内接受委托）
**方式二**：案件到达二审法院后递交

**关键提醒**：随时关注案件移送情况，在案件落实承办人的第一时间内递交委托手续

### 3.2 一审信息衔接

**情形A：一审律师继续代理** - 重点关注庭审笔录、新材料
**情形B：新律师接手** - 从一审律师处获取材料或从二审法院阅卷

📋 完整流程 → [references/双模式启动详细流程.md](双模式启动详细流程.md)

### 3.3 关键时间节点

| 节点 | 期限 |
|-----|------|
| 上诉期 | 判决书送达后10日内 |
| 二审审限 | 2个月（可延长2个月） |
| 辩护意见提交 | 案件到达承办人后半个月内 |

## 模块四：二审阅卷审查

### 4.1 阅卷材料构成

- 第一部分：检方起诉时提供的证据材料卷
- 第二部分：一审审判阶段形成和收集的材料（含庭审笔录）

### 4.2 二审阅卷7大重点

| 序号 | 阅卷重点 | 审查目的 |
|-----|---------|---------|
| 1 | 上诉人上诉理由及其相关依据 | 明确辩护方向 |
| 2 | 是否有印证上诉人辩解的证据 | 支持辩护意见 |
| 3 | 一审定案证据是否客观 | 发现证据问题 |
| 4 | 一审定案证据之间是否存在矛盾 | 突破证据体系 |
| 5 | 一审判决证据体系是否排除合理怀疑 | 无罪/存疑辩护 |
| 6 | 是否有从轻情节未得到认定 | 量刑辩护 |
| 7 | 是否有有利于上诉人的新证据线索 | 调查取证方向 |

## 模块六：争取开庭与辩护意见

### 6.1 争取开庭审理策略

**实践规律**："开庭改判可能性大，不开庭基本维持原判"

**法定应开庭情形**：
1. 对事实、证据提出异议，可能影响定罪量刑的上诉案件
2. 被告人被判处死刑的上诉案件
3. 人民检察院抗诉的案件

**争取策略**：强调事实证据争议、提交新证据、及时提交初步辩护意见

📋 公开开庭审理申请书 → templates/公开开庭审理申请书.md

### 6.2 初步辩护意见7项内容

1. 一审判决事实不清、证据不足
2. 被告人的辩解可以成立
3. 辩护意见可以成立
4. 一审判决定性不准
5. 一审判决量刑失当
6. 有新的证据证明一审判决不能成立
7. 提出开庭审理本案的申请

### 6.3 辩护意见两大方向

**方向一**：审查一审辩护意见为什么没有得到采信 → 针对性强化
**方向二**：在原有辩护意见外寻找新的辩护意见

📋 二审辩护意见模板 → templates/二审辩护意见模板.md

## 模块七：开庭审理增量模块

> **触发条件**：二审法院决定开庭审理本案

### 7.1 二审开庭与一审的区别

| 对比维度 | 一审 | 二审 |
|---------|------|------|
| 审理范围 | 起诉书指控的事实 | 全面审查 |
| 辩护重点 | 证据+法律适用 | 上诉请求和理由，突出重点 |
| 质证重点 | 全部证据 | 争议证据+新证据 |
| 辩论重点 | 全案分析 | 紧抓一审判决错误 |

### 7.2 庭审准备要点

- **发问**：针对上诉理由相关事实，避免对无争议事实过多询问
- **质证**：重点质证一审定案关键证据，指出证据矛盾
- **举证**：提交新证据，强调一审遗漏的有利证据
- **辩论**：紧抓上诉请求与理由，避免讲无关事项

## 模块八：不开庭审理辩护策略

### 8.1 不开庭的实践现状

- 二审以不开庭审理为主流
- 多数法院刑事二审上诉案件开庭率不超过10%

### 8.2 书面辩护意见的充分性要求

**不开庭≠不改判**，书面辩护意见同样重要：
- 要像准备开庭一样准备书面意见
- 针对一审判决的每一个问题逐条回应
- 事实依据+证据支撑+法律适用，逻辑清晰

### 8.3 不开庭案件的工作心态

1. 不可以因为不开庭就失去信心
2. 仍要保质保量地完成辩护任务
3. 不开庭不等于不改判
4. 继续努力，争取最好的结果

## 模块九：庭后工作与裁判分析

### 9.1 二审阶段和解工作

二审阶段仍可进行和解，对死刑案件尤其重要，可能影响生死。

### 9.2 二审裁判结果类型

| 裁判类型 | 应对策略 |
|---------|---------|
| 维持原判 | 分析是否符合申诉/再审条件 |
| 改判 | 分析改判是否到位 |
| 发回重审 | 引导回一审程序 |

### 9.3 二审宣判方式说明

**重要提醒**（向当事人说明）：
- 二审一般不作公开宣判
- 裁判文书由一审法院代为送达
- 送达后即生效
- 律师可能后于上诉人知道结果
