# 二审阶段双模式启动详细流程

---

## 模式A：全新案件启动

**触发条件**：用户首次提及二审阶段案件，无前序Skill工作记录

### 启动流程

#### 第一步：确认案件基本信息

```
必采信息：
□ 上诉人/原审被告人姓名
□ 一审判决罪名及刑期
□ 上诉理由/抗诉理由
□ 是否已过上诉期
□ 一审法院名称
□ 一审判决日期
□ 案件编号
```

#### 第二步：确认委托情况

```
必采信息：
□ 委托人身份（上诉人本人/近亲属）
□ 是否为一审律师继续代理
□ 是否需要协助代书上诉状
□ 委托阶段范围
□ 联系方式
```

#### 第三步：进入委托接案模块

完成信息采集后，进入委托手续办理和材料接收流程。

---

## 模式B：从一审Skill衔接启动

**触发条件**：存在一审阶段Skill工作记录，案件进入二审

### 衔接流程

#### 第一步：读取一审Skill工作成果

```
读取文件清单：
□ 一审辩护意见及采纳情况
□ 一审判决书分析
□ 关键证据清单
□ 当事人沟通记录
□ 一审阶段工作总结
□ 案件法律分析报告
```

#### 第二步：确认衔接信息

```
确认事项：
□ 一审判决结果（罪名、刑期）
□ 是否上诉/抗诉
□ 上诉理由
□ 律师是否继续代理
□ 是否需要补充委托手续
```

#### 第三步：进入材料衔接

跳过委托手续办理，直接进入材料衔接和二审阅卷。

---

## 委托手续递交方式

### 方式一：随上诉状一并递交

**适用情形**：在上诉期内接受委托

**操作流程**：
1. 接受委托，签署委托协议
2. 协助或代书上诉状（如被告人要求）
3. 将委托书、律所公函、律师执业证复印件与上诉状一并递交一审法院
4. 一审法院移送案卷时将委托手续一并递交二审法院

**上诉状代书注意事项**：
- 上诉状不同于辩护意见，不能照搬律师的辩护意见
- 要按照被告人的口吻、学识、能力水平进行撰写
- 上诉状寄至法院要保留EMS回执单，并存档
- 一段时间后询问法院是否收到文件

### 方式二：案件到达二审法院后递交

**适用情形**：案件已移送二审法院

**操作流程**：
1. 了解案件二审受理时间及承办人
2. 向承办法官递交：授权委托书、律师事务所公函、律师执业证复印件
3. 提出阅卷要求

**关键提醒**：
- 无论哪种递交方式，都要**随时关注案件移送情况**
- 要在案件落实承办人的**第一时间内**递交或确认委托手续已收到
- 案例教训：有案件上诉不到1个月就维持原判，律师连手续都没递交

## 一审信息衔接逻辑

### 情形A：一审律师继续代理

**信息衔接要点**：

```
已有材料：
├── 一审起诉书、判决书
├── 一审案卷材料（已复制）
├── 一审辩护意见
├── 一审会见笔录
└── 一审工作底稿

需重点关注的新材料：
├── 庭审笔录（二审阅卷重点之一）
├── 一审审判阶段形成的新材料
└── 上诉人新的辩解和要求

工作重点：
1. 分析一审辩护意见为何未被采纳
2. 寻找新的辩护意见和证据线索
3. 针对一审判决提出二审辩护意见
```

### 情形B：新律师接手

**材料获取渠道**：

```
1. 从一审律师处获取材料
   - 向一审律所提交公函+委托书
   - 一审律师应予配合，复制全部材料
   - 注意：部分未公开证据需谨慎处理
   
2. 从二审法院阅卷
   - 案件到达后尽快阅卷
   - 注意案卷移送期间的时间空档
   
3. 材料清单核对
   - 检方起诉时提供的证据材料卷
   - 审判阶段形成和收集的材料
   - 庭审笔录
   - 一审辩护词
   - 上诉状
```

## 案件移送跟踪

**跟踪要点**：
1. 了解一审法院移送案卷的通常时间（各法院不同，一般1周至1个月）
2. 主动联系一审法院了解移送进度
3. 案件移送后，及时联系二审法院确认承办人
4. 不要假设移送需要固定时间而不主动核实

**案例警示**：
某案件上诉后，律师以为移送需要1个月，结果该案随一批案件集中送到中院，1个月内就维持原判结案，律师连手续都没递交。

---

## 关键时间节点管理

| 节点 | 期限 | 注意事项 |
|-----|------|---------|
| 上诉期 | 判决书送达后10日内 | 协助代书上诉状需在期限内完成 |
| 案件移送 | 无固定期限 | 主动跟踪，不要等待 |
| 二审审限 | 2个月（可延长2个月） | 疑难复杂案件可再延长 |
| 辩护意见提交 | 案件到达承办人后半个月内 | 争取开庭的关键时机 |
