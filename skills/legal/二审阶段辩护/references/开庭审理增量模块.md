# 二审阶段开庭审理增量模块

> **触发条件**：二审法院决定开庭审理本案

---

## 一、二审开庭与一审的区别

| 对比维度 | 一审 | 二审 |
|---------|------|------|
| 审理范围 | 起诉书指控的事实 | 全面审查，不限于上诉/抗诉范围 |
| 辩护重点 | 证据+法律适用 | 上诉请求和理由，突出重点 |
| 发问重点 | 案件事实全面核实 | 与上诉请求相关的事实 |
| 质证重点 | 全部证据 | 争议证据+新证据 |
| 辩论重点 | 全案分析 | 紧抓一审判决错误 |

---

## 二、庭审准备要点

### 2.1 发问提纲

**设计原则**：
1. 针对上诉理由相关的事实进行发问
2. 针对一审判决认定有误的事实进行发问
3. 避免对双方无争议的事实过多询问

**发问类型**：
- 澄清性发问：澄清一审认定有误的事实
- 补充性发问：补充一审遗漏的有利事实
- 核实性发问：核实新证据相关的事实

### 2.2 质证要点

**质证策略**：
1. 重点质证一审定案的关键证据
2. 质证一审未采信的有利证据
3. 对新证据进行充分质证
4. 指出证据之间的矛盾

**质证重点**：
- 一审判决采信但有瑕疵的证据
- 一审判决未采信但对被告人有利的证据
- 二审期间出现的新证据

### 2.3 举证要点

**举证内容**：
1. 提交在二审期间调取的新证据
2. 强调一审遗漏或未采信的有利证据
3. 证据与上诉请求的关联性

**举证方法**：
- 逐一说明证据的证明目的
- 强调证据与上诉理由的对应关系
- 证据之间形成体系

### 2.4 辩论要点

**辩论原则**：
1. 紧紧抓住上诉请求与理由
2. 针对一审判决中不一致的地方着重辩论
3. 不要对大家共同认可的事实进行过多阐述
4. 避免讲与上诉请求与理由无关的事项

---

## 三、出庭辩论技巧

**核心原则**：紧抓上诉请求和理由，突出重点，避免重复

### 3.1 辩论结构

**开场**：明确上诉请求
```
尊敬的审判长、审判员：
辩护人认为一审判决[存在事实认定错误/证据采信错误/法律适用错误/量刑不当]，
请求二审法院[改判/发回重审]。
```

**主体**：逐条回应一审判决的错误
- 事实认定错误 → 证据分析
- 证据采信错误 → 证据规则
- 法律适用错误 → 法律依据
- 量刑不当 → 量刑情节

**总结**：重申上诉请求
```
综上所述，一审判决[具体错误]，辩护人请求贵院依法[改判/发回重审]。
```

### 3.2 辩论技巧

- **聚焦核心**：围绕上诉请求展开，不偏离主题
- **对比论证**：将一审判决认定与证据对比，突出矛盾
- **引用规范**：准确援引法律条文和司法解释
- **逻辑严密**：论证过程环环相扣，结论顺理成章

## 四、二审质证复审工作

### 4.1 二审质证的核心定位

二审质证不同于一审的全面质证，其核心任务是**论证一审判决所采信证据存在的问题**，为改判或发回重审提供证据基础。

### 4.2 二审质证复审三大重点

| 质证方向 | 工作内容 | 目标 |
|---------|---------|------|
| 一审采信证据 | 审查程序瑕疵、真实性疑问、关联性不足 | 动摇定案根基 |
| 一审未采信有利证据 | 论证应被采信的理由、补强证据 | 争取有利认定 |
| 二审新证据 | 新证据的质证准备与质证应对 | 改变事实认定 |

### 4.3 证据矛盾的发现与论证

二审辩护律师应重点关注一审判决认定的事实与证据之间是否存在矛盾，包括：
- 证据间的逻辑矛盾
- 关键证据与客观事实的冲突
- 证据链条的断裂和缺环

### 4.4 二审质证工作清单

```
□ 逐项梳理一审判决采信的全部证据
□ 标注每份证据的质证空间（强/中/弱）
□ 制作证据矛盾对照表
□ 准备二审质证意见（开庭时使用）
□ 或将证据问题融入辩护意见（不开庭时）
```

## 五、认罪认罚案件二审开庭要点

### 上诉人反悔认罪认罚的情形

**审查要点**：
1. 审查认罪认罚的自愿性、合法性
2. 区分对事实的认可与对量刑的异议
3. 注意：单纯以量刑过重为由上诉，二审一般不开庭
4. 重点论证：认罪认罚程序存在瑕疵/量刑建议明显不当

**辩护策略**：
- 如上诉人确有冤情，应重点论证认罪认罚非自愿
- 如仅对量刑不服，应聚焦量刑建议是否明显不当
- 注意认罪认罚后上诉可能影响从宽幅度

## 六、庭审应急处理

### 6.1 公诉人提出新证据

**应对步骤**：
1. 申请休庭审查新证据
2. 需要时申请延期审理
3. 准备质证意见

### 6.2 上诉人当庭改变陈述

**应对步骤**：
1. 申请休庭与上诉人沟通
2. 了解改变陈述的原因
3. 调整辩护策略（如有必要）

### 6.3 合议庭打断辩护发言

**应对步骤**：
1. 服从法庭指挥
2. 择机继续陈述核心观点
3. 庭后提交书面补充意见
