## 文书范式

```
                X×被判X×罪（上诉、抗诉）一案
                      二审阶段辩护意见

X×人民法院

尊敬的承办法官、合议庭各位法官：

    受X×及其近亲属的委托，X×律师事务所指派我担任其上诉阶段的辩护人，为其辩护。接受指派后，辩护人认真分析了一审判决书，再次详细查阅了本案全部案卷材料，多次依法会见了X×，详细听取其辩解，现发表如下辩护意见，供合议庭审查。

    辩护人总体认为，……对一审判决提出异议……

    以上意见，敬请重视，望予照准！

        【律师姓名】律师
        ×年×月×日
```

## 辩护意见五段论结构

### 第一段：辩护人身份与工作说明
- 委托来源（上诉人及其近亲属）
- 指派单位（律师事务所）
- 辩护人所做工作（分析判决书、查阅案卷、会见当事人）

### 第二段：辩护人总体认为
- 对一审判决的总体评价
- 提出异议的核心观点
- 一句话概括辩护立场

### 第三段：观点概述
- 辩护请求（变更罪名/认定情节/从轻减轻/适用缓刑）
- 简要列明主要理由

### 第四段：具体理由详述
- 分点论述各项辩护意见
- 事实依据+证据支撑+法律适用
- 回应一审判决的认定

### 第五段：综上+结论请求
- 总结全文
- 明确提出辩护请求

## 二审辩护意见两大方向

### 方向一：审查一审辩护意见为何未被采信
1. **吃透一审辩护意见**：了解一审辩护意见提出的依据
2. **分析未采纳原因**：
   - 事实认定问题 → 加强证据的调查、分析与论证
   - 法律认定问题 → 加强对法律适用原理的研究与分析
3. **强化辩护意见**：抓住一审未认定存在的实体与程序错误，加强说理，明确依据

### 方向二：寻找新的辩护意见
1. **全面审查证据材料**：不拘泥于侦查、检察及审判人员的意见
2. **反思新观点**：
   - 是否有确凿依据？
   - 当时为何没人提出？
   - 是否需要补充调查取证？
3. **整合意见**：将新的辩护意见与原辩护意见进行整合

## 初步辩护意见七项内容

向承办法官提交的初步辩护意见，一般包括以下内容：

| 序号 | 内容要点 | 说明 |
|-----|---------|------|
| 1 | 一审判决事实不清、证据不足 | 上诉辩护意见的重点，也是开庭的法定理由 |
| 2 | 被告人的辩解可以成立 | 论证被告人辩解的合理性 |
| 3 | 辩护意见可以成立 | 一审辩护意见的延续与强化 |
| 4 | 一审判决定性不准 | 罪名认定错误 |
| 5 | 一审判决量刑失当 | 量刑过重 |
| 6 | 有新的证据证明一审判决不能成立 | 附新证据材料 |
| 7 | 提出开庭审理本案的申请 | 争取开庭 |

## 常见辩护类型与表述

### 类型一：无罪辩护
```
辩护人总体认为，一审判决认定X×构成X×罪，事实不清、证据不足，X×依法不构成X×罪。恳请贵院依法改判X×无罪。
```

### 类型二：罪轻辩护（变更罪名）
```
辩护人总体认为，一审判决认定X×构成X×罪（重罪）定性错误，X×的行为仅涉嫌X×罪（轻罪）。恳请贵院依法变更罪名，对其从轻处罚。
```

### 类型三：量刑辩护
```
辩护人总体认为，一审判决认定X×构成X×罪正确，但未认定X×具有……（从轻、减轻情节），导致量刑过重。恳请贵院依法认定上述情节，对X×从轻/减轻处罚。
```

### 类型四：程序辩护
```
辩护人总体认为，一审审判程序存在……违法情形，依法应当发回重审。
```

## 认罪认罚案件二审辩护要点

### 上诉人反悔认罪认罚的情形
1. **非自愿认罪认罚**：审查认罪认罚具结书签署过程
2. **量刑建议明显不当**：一审未依法调整
3. **事实认定有误**：认罪认罚不影响事实认定

### 辩护策略
- 重点审查认罪认罚的自愿性、合法性
- 区分对事实的认可与对量刑的异议
- 注意：单纯以量刑过重为由上诉，二审一般不开庭

## 提交时机与方式

### 提交时机
- **初步辩护意见**：案件到达承办法官处**半个月内**提交
- **正式辩护意见**：开庭前或不开庭案件的书面意见

### 提交方式
1. 直接送达承办法官
2. 通过法院收发室递交
3. 邮寄（保留EMS回执）

### 沟通配合
- 提交后与承办法官电话确认收悉
- 询问是否需要补充材料
- 了解案件审理进度
