# 二审阶段执业风险禁止清单

> 本清单针对二审阶段特有的执业风险，与一审阶段清单互补

---

## 一、委托接案风险

### 1.1 委托手续风险
- [ ] **禁止**：未核实委托人身份即接受委托
- [ ] **禁止**：委托手续不完整即开展工作
- [ ] **禁止**：未确认被告人本人同意即接受家属委托
- [ ] **禁止**：在上诉期届满后仍协助递交上诉状

### 1.2 案件移送跟踪风险
- [ ] **禁止**：不主动跟踪案件移送进度
- [ ] **禁止**：案件到达二审法院后不及时递交委托手续
- [ ] **禁止**：假设案件移送需要固定时间而不主动核实
- [ ] **警示**：案例教训——案件1个月内结案，律师连手续都未递交

---

## 二、与一审律师协作风险

### 2.1 材料交接风险
- [ ] **禁止**：未出具公函和委托书即要求一审律师复制材料
- [ ] **禁止**：获取一审材料后不核实完整性
- [ ] **禁止**：泄露从一审律师处获取的未公开证据
- [ ] **警示**：一审律师提供的材料可能不完整，需补充阅卷

### 2.2 案件信息风险
- [ ] **禁止**：不向一审律师了解案件背景和辩护思路
- [ ] **禁止**：完全否定一审辩护意见而不分析原因
- [ ] **禁止**：在未充分了解一审情况下仓促提出新观点

---

## 三、会见风险

### 3.1 会见程序风险
- [ ] **禁止**：首次会见不通报委托情况和介绍上诉程序
- [ ] **禁止**：不制作会见笔录或笔录不完整
- [ ] **禁止**：会见时诱导上诉人改变原有供述
- [ ] **禁止**：将案卷证据原件交给上诉人查看

### 3.2 会见内容风险
- [ ] **禁止**：传递与案件无关的人员信息
- [ ] **禁止**：帮助上诉人与同案犯串供
- [ ] **禁止**：向上诉人承诺案件结果
- [ ] **禁止**：代为转达威胁证人、被害人的信息

### 3.3 共同犯罪案件会见风险
- [ ] **禁止**：与同案其他辩护律师串通后统一口径再会见
- [ ] **禁止**：将同案犯的供述内容透露给上诉人
- [ ] **警示**：律师之间可就辩点沟通，但严禁串供、作伪证

---

## 四、阅卷风险

### 4.1 阅卷程序风险
- [ ] **禁止**：阅卷不全面、不细致
- [ ] **禁止**：仅复制检方证据而忽略审判阶段材料
- [ ] **禁止**：不复制庭审笔录（二审阅卷重点之一）
- [ ] **禁止**：不审查一审辩护意见及公诉答辩意见

### 4.2 阅卷内容风险
- [ ] **禁止**：阅卷后不制作详细阅卷笔录
- [ ] **禁止**：不关注一审未认定的从轻情节
- [ ] **禁止**：不寻找证据之间的矛盾点
- [ ] **禁止**：忽视有利于上诉人的新证据线索

---

## 五、调查取证风险

### 5.1 取证程序风险
- [ ] **禁止**：单独向被害人及其近亲属取证
- [ ] **禁止**：单独向证人取证（应两名律师同行）
- [ ] **禁止**：取证时威胁、引诱、欺骗证人
- [ ] **禁止**：伪造、变造证据或帮助当事人毁灭、伪造证据

### 5.2 取证内容风险
- [ ] **禁止**：取证后不及时向法院提交
- [ ] **禁止**：隐瞒对上诉人不利的证据线索
- [ ] **警示**：申请法院调取的证据类型（立功材料、同步录音录像等）

---

## 六、辩护意见风险

### 6.1 意见撰写风险
- [ ] **禁止**：辩护意见没有事实和法律依据
- [ ] **禁止**：辩护意见与上诉人意愿严重冲突且未沟通
- [ ] **禁止**：照搬一审辩护意见而不针对一审判决
- [ ] **禁止**：使用侮辱性、攻击性语言

### 6.2 意见提交风险
- [ ] **禁止**：不在规定时间内提交辩护意见
- [ ] **禁止**：提交后不确认承办法官是否收悉
- [ ] **禁止**：案件不开庭时放弃书面辩护

---

## 七、开庭审理风险

### 7.1 庭前风险
- [ ] **禁止**：开庭前不与上诉人沟通庭审要点
- [ ] **禁止**：不告知上诉人二审开庭的权利和注意事项
- [ ] **禁止**：不准备针对一审判决争议点的发问提纲

### 7.2 庭审风险
- [ ] **禁止**：庭审中偏离上诉请求和理由
- [ ] **禁止**：对一审已无争议的事实过多阐述
- [ ] **禁止**：当庭与同案辩护律师激烈对抗（可保留意见）
- [ ] **禁止**：未经法庭许可擅自离开法庭

---

## 八、与法官沟通风险

### 8.1 沟通方式风险
- [ ] **禁止**：频繁无实质内容地联系承办法官
- [ ] **禁止**：向法官送礼、请吃或进行利益输送
- [ ] **禁止**：通过不正当途径打探合议庭讨论情况
- [ ] **禁止**：在社交媒体上公开案件细节向法官施压

### 8.2 沟通内容风险
- [ ] **禁止**：向法官承诺当事人会认罪而实际并未沟通
- [ ] **禁止**：代替当事人作出实体权利处分的承诺
- [ ] **警示**：沟通目的在于保持信息畅通、充分表达诉求

---

## 九、裁判结果风险

### 9.1 宣判跟踪风险
- [ ] **禁止**：不主动关注案件宣判情况
- [ ] **禁止**：当事人先知道结果而律师不知道
- [ ] **禁止**：不向当事人说明二审委托宣判的送达方式
- [ ] **警示**：二审裁判文书送达后即生效，律师可能后于当事人知悉

### 9.2 结果沟通风险
- [ ] **禁止**：对不利裁判结果使用非法律用语评价
- [ ] **禁止**：诋毁司法机关形象
- [ ] **禁止**：承诺申诉或再审一定能成功
- [ ] **警示**：发回重审案件，引导当事人重新委托一审辩护

---

## 十、舆论与媒体风险

### 10.1 信息发布风险
- [ ] **禁止**：未经当事人同意公开案件信息
- [ ] **禁止**：在裁判前通过媒体炒作案件
- [ ] **禁止**：泄露不公开审理案件的信息
- [ ] **禁止**：发布可能影响司法公正的言论

### 10.2 舆论应对风险
- [ ] **禁止**：接受采访时透露案件保密信息
- [ ] **禁止**：通过舆论向法院施压
- [ ] **禁止**：在社交媒体发表不当言论

---

## 风险等级说明

| 等级 | 说明 | 后果 |
|-----|------|------|
| **禁止** | 绝对不可触碰的红线 | 可能导致刑事责任、行政处分、行业处分 |
| **警示** | 需要特别注意的风险点 | 可能导致执业不规范、投诉或纠纷 |

---

## 二审阶段特有风险提示

1. **时间紧迫风险**：二审审限2个月（可延长），需在案件到达承办人后半个月内提交辩护意见
2. **案件移送空档风险**：案件移送期间无法阅卷，需提前从一审律师处获取材料
3. **不开庭≠不改判**：即使不开庭，仍需认真完成辩护工作
4. **委托宣判风险**：二审裁判文书通过一审法院送达，律师可能后于当事人知悉结果
5. **发回重审衔接风险**：发回重审后案件回到一审程序，需引导当事人重新委托
