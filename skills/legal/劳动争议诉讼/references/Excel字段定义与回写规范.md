# 劳动争议复盘台账字段映射与系统记录同步规范

> **全局强制规则**：本文档仅作为历史复盘台账字段映射和系统记录同步参考。当前执行以法律工作总控为准：先更新当前事项系统记录；复盘台账仅在用户确认或总控规则要求时同步。

## 基础字段（主流程L0→L1创建）

| 字段名 | 说明 | 初始值 |
|-------|------|-------|
| 客户编号 | 格式：客户名称_联系方式 | 用户输入 |
| 客户姓名/单位名称 | | 用户输入 |
| 联系方式 | | 用户输入 |
| 首次咨询日期 | | 当前日期 |
| 案件状态 | 各Skill更新 | 已咨询 |
| 案件类型 | "劳动争议" | 劳动争议 |
| 劳动争议角色 | "劳动者代理(W)"/"用人单位代理(E)" | 确认后填写 |
| 承办律师 | | 潘睿 |
| 案件文件夹路径 | | 创建后填写 |
| 备注 | | |
