# 劳动争议诉讼Agent

## 功能概述

整合17个子Skill（14个复用 + 3个新建），实现劳动争议案件阶段判断和流程编排。本 Skill 根据总控确认的当前事项和既有记录判断案件状态（L0-L15/LM系列），按序调用专业 Skill，并在关键节点请求用户确认。

公司端日常合规、员工手册/规章制度/劳动合同更新、工资工时咨询、内部劳动政策起草，优先路由至「用人单位劳动合规」；已经发生仲裁、诉讼或需要答辩/裁决后衔接的，仍由本 Skill 编排。

**子Skill清单**：
1. **法律咨询助手** - 咨询入口路由，区分微信/线下咨询，新客户建档
2. **案件沟通记录** - 对话式笔录格式，含附表（关键信息摘要+证据初步清单）
3. **初步法律分析** - 六来源体系+请求权基础分析，输出法律服务建议书
4. **委托合同管理** - 利益冲突检查+阶段收费+诉讼风险告知
5. **法规案例检索** - 三层检索法，输出法官版+律师版双文档
6. **调查取证与证据管理** - 证据缺口分析、取证方案、调查令申请、财产保全
7. **诉讼可视化** - 6种Mermaid图表，输出法官版+律师版双报告
8. **案件讨论与提纲** - 四维度AI自问自答分析、三方模拟法庭、生成案件提纲
9. **庭前准备** - 庭审提纲+攻防预案+举证策略+代理词+诉讼方案
10. **庭审与庭后工作** - 庭审纪要、复盘分析、庭后代理词、法官沟通（支持多次开庭）
11. **诉讼文书起草** - 仲裁申请书/答辩状/代理词/质证意见/证据目录
12. **调解与和解** - 调解可行性分析、调解方案、撤诉管理（任何阶段可触发）
13. **诉讼分析工具** - 判决书九层拆解+上诉/再审决策支持
14. **结案归档** - 案卷归档、结案报告、客户移交、费用结算、客户回访
15. **劳动争议仲裁程序管理**（新建）- 仲裁前评估、仲裁策略、时效计算、一裁终局判断
16. **劳动关系认定与经济补偿计算**（新建）- 劳动关系三性认定、N/N+1/2N计算、加班费计算
17. **劳动争议证据体系**（新建）- 举证责任分配、证据收集清单、特殊保护检测

## 核心机制

### 1. 客户编号规则与咨询入口路由

**客户编号格式**：客户名称_联系方式
**示例**：张先生_138xxxx8888、李女士_微信、XX公司_法务部

**咨询入口路由逻辑**：
```
用户输入客户信息
    ↓
检测客户编号
    ├─ 有客户编号 → 读取当前事项系统记录获取当前状态 → 根据状态进入后续流程
    └─ 无客户编号（新客户）→ 路由到【法律咨询助手】Skill
       → 事实齐备后生成客户编号 → 按确认写入复盘台账 → 衔接至案件沟通记录
```

### 2. 角色系统（W/E双角色）

| 角色 | 代码 | 说明 | 典型场景 |
|-----|------|------|---------|
| 劳动者代理 | W | 代理员工维权 | 追索报酬、违法解除赔偿、工伤待遇、加班费 |
| 用人单位代理 | E | 代理企业应诉/合规 | 答辩仲裁申请、调岗合法性论证、规章制度合规审查 |

角色通过系统记录或复盘台账「劳动争议角色」字段确定，首次录入时交互确认。角色影响文书模板选择、策略方向、举证重点分配。

**W角色核心策略**：最大化权益主张、举证责任向用人单位倾斜、强调用人单位违法行为
**E角色核心策略**：风险敞口控制、合规性论证、降低赔偿金额或争取无责解除

### 2A. 任务类型分流（强制）

确认角色后，必须继续判断任务类型：

| 任务类型 | 判断标准 | 路由 |
|---|---|---|
| W劳动者代理 | 劳动者咨询、准备仲裁/诉讼、索赔、维权 | 本 Skill + 劳动争议证据体系，先做劳动者证据完整性检查 |
| E用人单位争议代理 | 已收到仲裁申请、劳动监察、律师函、员工明确主张权利，或已经进入争议处理 | 本 Skill + 劳动争议仲裁程序管理 + 劳动争议证据体系 |
| E用人单位日常合规 | 员工手册、规章制度、劳动合同更新、工资工时问答、内部政策起草、争议前解除/调岗/降薪风险预判 | 路由至「用人单位劳动合规」 |

角色或任务类型不明确时，先询问用户，不默认选择。涉及公司端解除、调岗、降薪、处罚、待岗、停职、绩效处理时，即使用户未明确说“合规”，也要先做公司端基本事实和证据收集。

E角色已发生争议时，应检查是否存在同一企业客户的劳动合规档案：
- 路径：`<LEGAL_WORKSPACE>/_系统记录/劳动合规/<企业客户>/`
- 优先读取：`公司基本信息.md`、`公司材料清单.md`、`制度版本台账.md`、`变更记录.md`、`生效流程记录.md`。
- 按争议类型定向读取相关制度版本、制度冲突检查、工资工时计算记录和专项事项记录。
- 读取结果写入当前诉讼事项系统记录时，必须标注“来源于公司劳动合规档案”，并进行争议事项复查。
- 不得默认读取无关员工、无关专项事项或公司全量资料。

### 3. 状态检测机制（L系列状态链）

本 Skill 只判断专业阶段，不维护独立事项记录：
1. 读取法律工作总控确认的当前事项和系统记录。
2. 必要时结合 复盘台账、业务文件区输出物和子 Skill 完成记录。
3. 根据文档存在情况判断当前阶段。

**状态定义简表**：

| 阶段 | 状态编码 | 状态名称 | 关键判断依据 |
|-----|---------|---------|-------------|
| 前期 | L0-L4 | 新客户→已委托 | 系统记录+01委托阶段文档 |
| 仲裁 | L5-L9 | 已检索→仲裁裁决 | 02仲裁阶段文档+裁决书 |
| L9分支 | L9a-L9e | 五种裁决后走向 | 一裁终局判断+起诉/撤销状态 |
| 一审 | L10-L12 | 一审准备→判决 | 03一审阶段文档 |
| 二审 | L13-L14 | 二审准备→判决 | 04二审阶段文档 |
| 终结 | L15 | 结案归档 | 05结案归档+结案报告 |
| 横向 | LM | 调解与和解 | 任何阶段可触发 |

## 工作流程

### 主流程

```
用户输入客户编号
    ↓
检测案件类型（系统记录或复盘台账「案件类型」字段 = "劳动争议"）
    ↓
检测角色（系统记录或复盘台账「劳动争议角色」字段）
    ↓
检测任务类型
    ├─ W（劳动者代理）→ 劳动者证据完整性检查 + 原劳动争议流程
    ├─ E（已发生争议）→ 公司端事实/证据/风险敞口检查 + 原劳动争议流程
    └─ E（日常合规/制度更新/工资工时/政策起草）→ 用人单位劳动合规
```

#### W角色（劳动者代理）标准流程

```
L0→L1：法律咨询助手 → 案件沟通记录 → 复盘台账记录+事项文件夹创建
L1→L2：案件沟通记录
L2→L5：初步法律分析 → 劳动关系认定与经济补偿计算 + 劳动争议证据体系
L5→L3：委托合同管理
L3→L4：法规案例检索
L4→L6：劳动争议仲裁程序管理（仲裁前评估+策略报告）
L6→L7：提交仲裁申请书 + 证据
L7→L8：调查取证 → 诉讼可视化 → 案件讨论 → 庭前准备
L8→L9：仲裁庭审（庭审与庭后工作）→ 裁决分析（诉讼分析工具）
L9→分支路由：
  L9a 终局+接受 → L15 结案归档
  L9b 终局+劳动者不服 → L10 一审起诉
  L9c 终局+单位申请撤销 → L10 撤销之诉
  L9d 非终局+不服 → L10 一审
  L9e 生效+执行 → L15 结案归档
L10→L11→L12：一审（复用诉讼文书起草+庭前准备+庭审）
L12→分支：接受→L15 / 不服→L13
L13→L14：二审
L14→L15：结案归档
LM：任何阶段可触发调解与和解
```

#### E角色差异要点

- L6策略方向：防守型，控制风险敞口
- L7文书：仲裁答辩书（而非申请书）
- 举证重点：规章制度合法性、劳动者过错
- 调解策略：降低赔偿金额或争取无责
- 前置检查：劳动者特殊状态、近期投诉举报、员工手册/规章制度生效、劳动合同、工资工时、考勤、解除/处罚程序和送达记录
- 日常合规事项：不进入L0-L15诉讼状态链，转「用人单位劳动合规」处理并由总控记录事项
- 公司档案衔接：如存在该企业的劳动合规长期档案，读取相关制度版本、变更记录和生效流程，作为公司端抗辩和风险敞口评估的来源之一

- **中止**：不可抗力或正当理由 → 中止事由消除后继续计算

## 新建Skill核心功能概要

### 劳动争议仲裁程序管理Skill

**核心功能模块**：
1. **仲裁前置强制检测**：判断案件是否必须经过仲裁前置程序
2. **仲裁时效计算器**：1年时效+中断/中止/持续侵害特殊规则
3. **一裁终局自动判断**：金额限制型（月最低工资×12）+ 标准执行型（工时/休假/社保）
4. **特殊保护检测**：三期女职工、工伤职工、医疗期职工、职业病职工
5. **仲裁策略报告**：W角色最大化请求 / E角色风险敞口控制

**输出文档**：仲裁时效评估、特殊保护检测报告、仲裁策略报告

### 劳动关系认定与经济补偿计算Skill

**核心功能模块**：
1. **劳动关系三性从属分析**：
   - 人身从属性：工作时间、地点、纪律、指挥监督、请假制度
   - 经济从属性：报酬来源、稳定性、工具设备、经营风险
   - 组织从属性：组织体系、业务关联、身份标识、团队协作
2. **12类劳动争议自动分类**：确认劳动关系、追索报酬、经济补偿、违法解除、双倍工资、工伤待遇等
3. **工时制度定性**：标准工时制、综合工时制、不定时工作制（审批要件检查）
4. **经济补偿计算引擎**：
   - N/N+1/2N判断逻辑（7类N情形、N+1情形、2N情形）
   - 月工资基数确定（前12个月平均，含基本工资/绩效/奖金/津贴/加班费/提成）
   - 三层封顶规则（社平3倍封顶、12年年限封顶、总额封顶）
5. **加班费三档计算**：工作日150%、休息日200%、法定节假日300%
6. **双倍工资计算**：入职第2月起至第12月，最多11个月

**输出文档**：劳动关系认定分析、经济补偿测算表、加班费计算表、双倍工资计算表

### 劳动争议证据体系Skill

**核心功能模块**：
1. **举证责任分配**：
   - 一般规则：谁主张，谁举证
   - 特殊规则：用人单位法定举证义务（考勤/工资/合同/规章制度/解除通知/社保/加班审批）
   - 解除合法性由用人单位举证、加班事实劳动者初步举证+单位不提供承担不利后果
2. **12类争议证据收集清单**：按争议类型定制化证据清单模板
3. **证据效力三级评估**：
   - 一级证据（直接证据）：劳动合同、裁决书/判决书、社保证明、公证书、工伤认定决定书、银行流水
   - 二级证据（间接证据）：工资条、考勤表、工作证、入职登记表、工作邮件
   - 三级证据（辅助证据）：微信/QQ聊天、电话录音、证人证言、照片、视频
4. **四类特殊保护检测**：三期女职工、工伤职工、医疗期职工、距退休不足5年职工
5. **规章制度合法性审查三要件**：民主程序、内容合法、公示告知

**输出文档**：举证责任分配表、证据收集清单、特殊保护检测报告、规章制度合法性审查意见

## 四类特殊保护群体

劳动法对以下四类群体提供特殊保护，用人单位不得依据第40条（非过失性解除）、第41条（经济性裁员）解除劳动合同：

| 保护类型 | 适用条件 | 保护内容 | 法律依据 |
|---------|---------|---------|---------|
| 三期女职工 | 孕期、产期、哺乳期 | 不得解除，合同期满应续延 | 劳动合同法第42条第4项 |
| 工伤职工 | 丧失或部分丧失劳动能力 | 1-4级保留劳动关系，5-6级安排工作 | 劳动合同法第42条第2项 |
| 医疗期职工 | 患病/非因工负伤医疗期内 | 不得解除，期满后需满足条件方可解除 | 劳动合同法第42条第3项 |
| 距退休不足5年 | 本单位连续满15年+距退休不足5年 | 不得依第40/41条解除 | 劳动合同法第42条第5项 |

## 仲裁前置与例外

**仲裁前置原则**：劳动争议必须先仲裁后诉讼（《劳动争议调解仲裁法》第5条）

**四级递进处理程序**：
```
协商（自愿）→ 调解（企业/行业/社会）→ 仲裁（必经）→ 诉讼（不服仲裁）
```

**例外情形**（可直接诉讼）：
1. 劳动者凭工资欠条直接起诉（按普通民事纠纷）
2. 解除/终止协议不履行（司法解释一第35条）
3. 非劳动争议的民事纠纷（借款、担保等）

## 一裁终局判断规则

根据《劳动争议调解仲裁法》第47条：

**金额限制型终局**（不超过当地月最低工资×12）：
- 追索劳动报酬
- 工伤医疗费
- 经济补偿
- 赔偿金

**标准执行型终局**：
- 工时争议
- 休假争议
- 社保争议

**诉讼权差异**：
| 裁决类型 | 劳动者 | 用人单位 |
|---------|-------|---------|
| 一裁终局 | 15日内起诉（基层法院） | 30日内申请撤销（中级法院） |
| 非终局 | 15日内起诉（基层法院） | 15日内起诉（基层法院） |

## 经济补偿计算核心规则

**N/N+1/2N适用场景**：
- **N（经济补偿）**：7类法定情形（第38条依法解除、协商一致、非过失性解除、经济性裁员、合同到期不续签、破产、解散）
- **N+1（经济补偿+代通知金）**：依据第40条解除但未提前30日书面通知
- **2N（违法解除赔偿金）**：无合法理由解除、程序违法解除、违法终止

**三层封顶规则**：
1. 月工资封顶：超过当地社平3倍的按3倍计算
2. 年限封顶：触发月工资封顶的，年限最高12年
3. 总额封顶：封顶月工资 × 封顶年限

## 注意事项

1. **客户隐私保护**：所有案件信息严格保密，记忆删除后不可恢复
2. **文档备份**：关键文档生成后建议用户本地备份
3. **时效提醒**：关键时间节点主动提醒用户
4. **Skill版本**：调用的Skill以最新版本为准
5. **人工最终审核**：所有提交仲裁委/法庭的文书需律师最终审核签字
6. **仲裁前置原则**：劳动争议必须先仲裁，未经仲裁直接起诉将被驳回
7. **一裁终局注意**：终局裁决对用人单位限制更大（30日撤销期vs劳动者15日起诉期）
8. **特殊保护优先**：涉及"三期"女职工、工伤职工等特殊群体时，优先适用保护性规定
9. **举证责任倾斜**：充分利用劳动争议举证责任倒置规则，减轻劳动者举证负担
10. **调解优先**：劳动争议调解成功率较高，任何阶段都应重视调解机会
11. **工时制度影响**：综合/不定时工时需审批文件，未审批按标准工时计算加班费
12. **双倍工资时效**：逐月计算，超过1年的月份丧失请求权
