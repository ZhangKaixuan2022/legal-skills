# 劳动争议诉讼17个Skill调用序列详解

## 前置分流

进入本17个 Skill 诉讼序列前，先确认角色和任务类型：

```text
确认代理角色（W劳动者 / E用人单位）
    ↓
确认任务类型
    ├─ W劳动者代理 → 进入本诉讼序列，先做劳动者证据完整性检查
    ├─ E用人单位已发生争议 → 进入本诉讼序列，先做公司端事实/证据/风险敞口检查
    └─ E用人单位日常合规、制度更新、工资工时问答、政策起草、争议前风险预判 → 用人单位劳动合规
```

「用人单位劳动合规」是本诉讼序列外的公司端非诉入口，不改变本17个 Skill 的诉讼阶段链。

## 标准流程（W角色 - 劳动者代理）

```
1. 法律咨询助手（法律咨询助手Skill）- 新客户入口
   触发条件：无客户编号的新客户
   输出：客户编号（KH-年份-序号）、复盘台账记录
   衔接：自动进入案件沟通记录
    ↓
2. 案件沟通记录（案件沟通记录Skill）
   前置检查：Excel中有客户记录
   输出：案件沟通记录（含附表：关键信息摘要+证据初步清单）
    ↓
3. 初步法律分析（初步法律分析Skill）
   前置检查：案件沟通记录存在
   输出：法律服务建议书（含请求权路径、争点清单、证据评估）
    ↓
4. 劳动关系认定与经济补偿计算（新建Skill）
   前置检查：案件沟通记录存在、法律服务建议书存在
   输出：劳动关系认定分析、经济补偿测算表、加班费计算表
   衔接：与劳动争议证据体系并行或先后进行
    ↓
5. 劳动争议证据体系（新建Skill）
   前置检查：案件沟通记录存在、法律服务建议书存在
   输出：举证责任分配表、证据收集清单、特殊保护检测报告
    ↓
6. 委托合同管理（委托合同管理Skill）
   前置检查：法律服务建议书存在
   输出：委托代理合同（含利益冲突声明+阶段收费）、诉讼风险告知书、诉讼案件跟踪记录表
    ↓
7. 法规案例检索（法规案例检索Skill）
   前置检查：法律服务建议书存在（获取请求权路径）、委托代理合同存在
   输出：法律法规及案例汇编（法官版）+ 法规案例检索报告（律师版）
    ↓
8. 劳动争议仲裁程序管理（新建Skill）- 仲裁前评估
   前置检查：法律服务建议书存在、经济补偿测算表存在、举证责任分配表存在
   输出：仲裁时效评估、仲裁策略报告
    ↓
9. 调查取证与证据管理（调查取证与证据管理Skill）
   前置检查：法律服务建议书存在（获取请求权要件清单）、证据收集清单存在
   输出：取证方案、补充材料清单、调查令申请书（如需要）
   提示：可与仲裁文书起草并行或先后进行
    ↓
10. 诉讼文书起草（诉讼文书起草Skill - 仲裁申请书）
    前置检查：法律服务建议书、法规案例汇编、经济补偿测算表存在
    输出：劳动仲裁申请书 + 证据目录 + 仲裁请求清单
     ↓
11. 案件讨论与提纲（案件讨论与提纲Skill）
    前置检查：仲裁申请书已起草，前序材料齐备
    输出：前序材料检查表、四维度分析报告、模拟法庭纪要、模拟法庭复盘报告、案件提纲
    特点：AI自主完成，律师只审阅最终输出
     ↓
12. 诉讼可视化（诉讼可视化Skill）
    前置检查：法律服务建议书存在、法规案例检索报告存在
    输出：诉讼可视化报告（法官版）+ 诉讼可视化分析报告（律师版）+ 6种图表
     ↓
13. 庭前准备（庭前准备Skill）
    前置检查：仲裁申请书、案件提纲、法规案例检索报告、诉讼可视化分析报告、质证意见（如有）存在
    输出：仲裁庭审准备材料（律师版，含庭审提纲+攻防预案矩阵+举证策略+开庭陈述提纲+仲裁庭辩论提纲+仲裁员提问预案+代理词+仲裁法律服务方案）
     ↓
14. 【等待开庭】
     ↓
15. 庭审与庭后工作（庭审与庭后工作Skill）
    触发条件：用户确认"已开庭"
    输出：庭审纪要、庭审复盘报告、庭后代理词、补充材料清单、仲裁员沟通记录
    特点：支持多次开庭，自动编号和关联前次记录
     ↓
16. 【裁决后】诉讼分析工具（诉讼分析工具Skill）
    前置检查：仲裁裁决书存在
    输出：诉讼分析报告（含裁决书拆解、一审起诉可行性评估、执行策略建议）
    分支：接受裁决→结案归档；不服裁决→一审/二审
     ↓
17. 结案归档（结案归档Skill）
    触发条件：裁决生效或调解完成
    输出：案卷归档目录、案件结案报告、裁决结果对比表、案卷移交清单、费用结算提醒清单、客户回访记录
    终态：Excel案件状态更新为"已结案"
```

## 横向Skill：调解与和解

```
调解与和解（调解与和解Skill）
   触发条件：任何阶段，用户提及"调解"/"和解"/"撤诉"
   输出：调解方案确认函、和解协议审查清单、撤诉申请书
   衔接：调解成功→结案归档；调解失败→返回原流程
```

## E角色与W角色的关键差异

| 阶段 | W角色（劳动者代理） | E角色（用人单位代理） |
|-----|-------------------|---------------------|
| L6策略方向 | 进攻型，最大化权益 | 防守型，控制风险敞口 |
| L7文书 | 仲裁申请书 | 仲裁答辩书 |
| 举证重点 | 用人单位违法行为 | 规章制度合法性、劳动者过错 |
| 调解策略 | 争取最大赔偿 | 降低赔偿金额或争取无责 |
| 核心论证 | 违法解除/未签合同/欠薪 | 合法解除/已签合同/已付薪 |

## 流程衔接提示

| 当前阶段完成 | 下一步提示 | 可触发Skill |
|-------------|-----------|------------|
| 初步法律分析完成 | 是否进行劳动关系认定与经济补偿计算？ | 劳动关系认定与经济补偿计算 |
| 经济补偿计算完成 | 是否进行劳动争议证据体系梳理？ | 劳动争议证据体系 |
| 证据体系完成 | 是否签订委托合同？ | 委托合同管理 |
| 委托合同签订完成 | 是否进行法规案例检索？ | 法规案例检索 |
| 法规检索完成 | 是否进行仲裁前评估？ | 劳动争议仲裁程序管理 |
| 仲裁评估完成 | 是否起草仲裁申请书/答辩书？ | 诉讼文书起草 |
| 仲裁文书起草完成 | 是否进行案件讨论与提纲？ | 案件讨论与提纲 |
| 案件讨论完成 | 是否制作诉讼可视化？ | 诉讼可视化 |
| 诉讼可视化完成 | 是否进行庭前准备？ | 庭前准备 |
| 庭前准备完成 | 提示等待开庭，开庭后启动庭审与庭后工作 | 庭审与庭后工作 |
| 仲裁庭审完成 | 根据裁决结果进入分支路由或结案归档 | 诉讼分析工具/结案归档 |
| 任何阶段 | 对方提出调解意向或需要评估调解可行性 | 调解与和解（横向） |
