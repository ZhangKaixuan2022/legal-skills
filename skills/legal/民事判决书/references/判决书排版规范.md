# 判决书排版规范

> **来源**：民事判决书Skill - 判决书专属排版规范；最高人民法院《人民法院民事裁判文书制作规范》
> **用途**：一审普通程序民事判决书的表格化排版标准
> **说明**：判决书排版与诉讼文书不同，使用仿宋三号/行距28磅，而非宋体小四/1.5倍行距

---

## 四、编号层级规范

判决书中不使用复杂的编号层级体系，主要使用以下格式：

| 使用场景 | 格式 |
|---------|------|
| 判项 | 一、二、三……（中文数字加顿号） |
| 诉讼请求分项 | 1. 2. 3.……（阿拉伯数字加句号） |
| 事实认定要点 | 1. 2. 3.……（阿拉伯数字加句号） |
| 法条引用 | 直接引用法条序号，不加层级编号 |

**重要规则**：判项编号使用"一、""二、"格式，不使用阿拉伯数字。

## 五、尾部格式规范

### 5.1 签名区布局

尾部签名区右对齐，各要素纵向排列：

```
                                  审  判  长  ×××
                                  审  判  员  ×××
                                  审  判  员  ×××

                                      ××××年××月××日
                                        （院印）

                                      书  记  员  ×××
```

### 5.2 签名区格式说明

| 元素 | 格式 |
|-----|------|
| 审判长/审判员/人民陪审员 | 职务名称与姓名之间空两格，右对齐 |
| 裁判日期 | "年月日"三字齐全，右对齐 |
| （院印） | 加括号，右对齐，位于日期下方 |
| 书记员 | "书记员"三字与姓名之间空两格，右对齐 |

## 六、法条引用格式

| 元素 | 格式 |
|-----|------|
| 法律名称 | 用书名号《》包裹完整名称 |
| 条文引用 | "第×条"或"第×条第×款"，多个条文用顿号分隔 |
| 引用示例 | 依照《中华人民共和国民法典》第五百零九条、第五百七十七条、第五百八十五条规定 |

**禁止**：在法条编号后加括号注释，如"第五百零九条（全面履行原则）"，只写条文编号。

---

## 七、与诉讼文书排版的主要差异

| 对比项 | 判决书 | 诉讼文书（起诉状/答辩状等） |
|-------|-------|------------------------|
| 正文字体 | 仿宋 | 宋体 |
| 正文字号 | 三号（16pt） | 小四（12pt） |
| 行距 | 28磅固定值 | 1.5倍行距 |
| 标题字体 | 方正小标宋（法院名）/宋体加粗（文书名） | 华文中宋/方正小标宋 |
| 法条引用字体 | 仿宋三号（与正文一致） | 楷体小四 |
| 页边距 | 上2.8/下2.6/左2.6/右2.5 | 上下2.54/左右3.18 |
| 页码 | 无 | 有（居中） |

## 八、印刷标准

### 8.1 版心与排版

| 元素 | 格式要求 |
|-----|---------|
| 版心尺寸 | 156mm × 225mm |
| 每面行数 | 一般排22行 |
| 每行字数 | 一般排28个字 |
| 印刷方式 | 双面印刷 |
| 单页页码 | 居右 |
| 双页页码 | 居左 |
| 印品质量 | 字迹清楚、均匀 |

### 8.2 标题与案号

| 元素 | 格式要求 |
|-----|---------|
| 标题位置 | 版心下空两行，居中排布 |
| 法院名称字体 | 二号小标宋体字 |
| 文书名称字体 | 二号小标宋体字 |
| 排列方式 | 法院名称与文书名称分两行排列 |
| 案号 | 之后空二个汉字空格至行末端 |
| 案号字体 | 三号仿宋体字 |

### 8.3 落款排版

| 元素 | 格式要求 |
|-----|---------|
| 落款位置 | 与正文同处一面，不用"此页无正文" |
| 空白处理 | 空白不足容下印章时，适当调整行距、字距 |
| 职务字间距 | 审判长、审判员每个字之间空二个汉字空格 |
| 职务与姓名间距 | 审判长、审判员与姓名之间空三个汉字空格 |
| 姓名后间距 | 姓名之后空二个汉字空格至行末端 |

落款间距示例：

```
                                  审　　判　　长　　　×××
                                  审　　判　　员　　　×××
                                  审　　判　　员　　　×××
```

### 8.4 院印规范

| 元素 | 格式要求 |
|-----|---------|
| 加盖位置 | 日期居中位置 |
| 上边界 | 上不压审判员 |
| 下边界 | 下不压书记员 |
| 骑年压月 | 下弧骑年压月在成文时间上 |
| 国徽边缘 | 底边缘及上下弧以不覆盖文字为限 |
| 印章质量 | 不应歪斜、模糊 |

### 8.5 误写处理

| 情形 | 处理方式 |
|-----|---------|
| 未送达的文书 | 重新制作 |
| 已送达的文书 | 以裁定补正 |
| 禁止做法 | 避免使用校对章 |

## 十一、标点符号规范

| 使用场景 | 标点 | 示例 |
|---------|------|------|
| "被告辩称""本院认为"等之后 | 逗号 | 本院认为，…… |
| "向本院提出诉讼请求""认定如下""判决如下"等之后 | 冒号 | 判决如下： |
| 判项序号后 | 顿号 | 一、…… |
| 其他标点 | 按GB/T15834-2011执行 | — |

标点符号用法示例：

```
本院认为，……

判决如下：
一、被告×××于本判决生效之日起十日内支付原告×××货款×××元；
二、驳回原告×××的其他诉讼请求。
```
