# 管理人工作Skill

## 功能概述

本Skill定位在破产程序体系的第2环节（BP-M2），承接破产法律分析报告（BP-M1），覆盖管理人自法院指定后至第一次债权人会议前的全部初期工作职责。

### 核心功能

| 功能模块 | 内容概述 |
|---------|---------|
| 印章与账户管理 | 指引刻制管理人印章、开立管理人银行账户 |
| 财产接管 | 全面接管债务人财产、印章、证照、账簿、档案等 |
| 内部事务管理 | 决定日常开支、营业继续/停止决策 |
| 合同处理 | 依据《企业破产法》第18条处理未履行完毕合同 |
| 聘用工作人员 | 经法院许可聘用必要工作人员 |
| 管理人团队管理 | 团队分工、工作进度跟踪、执业责任保险 |
| 报酬方案 | 制定并向法院提交管理人报酬方案 |

**财产接管四步流程**：
1. **通知交接**：向债务人法定代表人、实际控制人、财务负责人发送接管通知
2. **现场交接**：与债务人有关人员办理交接手续，逐项清点、共同签字确认
3. **处理保全**：了解已存在的保全措施和执行措施，依法申请解除保全、中止执行
4. **强制接管**：如债务人拒不配合，申请法院强制接管，全程录像或拍照

📋 详细操作手册参见 [references/管理人工作详细操作手册.md](管理人工作详细操作手册.md)

## 核心工作流程（9步）

### 工作步骤速览

| 步骤 | 工作内容 | 法律依据 | 输出文档 |
|-----|---------|---------|---------|
| 1 | 读取前序文档与信息整合 | - | - |
| 2 | 印章与账户管理 | 破产法第25条 | 印章刻制及备案记录 |
| 3 | 财产接管 | 破产法第25条 | 接管清单 |
| 4 | 内部事务管理 | 破产法第26条 | 营业继续/停止报告 |
| 5 | 合同处理 | 破产法第18条 | 合同处理决定书 |
| 6 | 聘用工作人员 | 破产法规定 | 聘用人员报告 |
| 7 | 管理人团队管理 | 各地管理人规范 | 管理人团队分工方案 |
| 8 | 报酬方案 | 破产法第28条 | 管理人报酬方案 |
| 9 | 文档生成与系统记录同步 | - | 各类文档 |

**步骤2：印章与账户管理要点**：
- 凭法院受理裁定书、指定管理人决定书向公安机关申请刻制管理人印章
- 印章样式："【债务人名称】管理人"，印章封样备案
- 管理人银行账户专户专用，不得与管理人自有资金混同

**步骤3：财产接管范围**：
实物财产、货币资金、知识产权、无形资产、印章证照、财务账簿、档案文件、电子数据、其他权益

**步骤4：营业决策要点**：
- 决策时点：第一次债权人会议前
- 评估继续营业的必要性和可行性
- 分析继续营业的成本与收益
- 报请人民法院许可

**步骤5：合同处理要点**：
- 决定期限：受理后2个月内决定解除或继续履行
- 催告答复期限：对方催告后30日内答复
- 逾期视为解除合同
- 继续履行时对方可要求提供担保

📋 各步骤详细操作参见 [references/管理人工作详细操作手册.md](管理人工作详细操作手册.md)

## 关键时间节点

| 节点 | 期限 | 法律依据 | 提醒策略 |
|-----|------|---------|---------|
| 合同处理决定 | 受理后2个月 | 破产法第18条 | 期限前15天、7天、3天提醒 |
| 对方催告答复 | 催告后30日 | 破产法第18条 | 收到催告后立即提醒 |
| 营业决定许可 | 第一次债权人会议前 | 破产法第26条 | 会议筹备阶段提醒 |

**逾期后果**：合同处理逾期未通知视为解除合同

**合同处理决策考量因素**：
- **继续履行考量**：合同履行对债务人财产增值有利、对方同意继续履行、管理人能够提供担保
- **解除合同考量**：继续履行成本过高、对方不愿履行、不符合债权人整体利益

## 交互规范

### 智能追问机制

**追问原则**：
- 每次只追问一个关键问题
- 追问语言简洁专业
- 说明追问的原因
- 等待用户回复后再继续

**常见追问情形**：

- 专业严谨，使用法律术语

## 注意事项

### 印章与账户管理注意事项

1. **印章刻制**：必须在公安机关指定单位刻制，不得私刻
2. **印章保管**：建立印章使用登记制度，专人保管
3. **账户专户**：管理人账户必须专户专用，不得与自有资金混同
4. **资金安全**：大额支出需经管理人会议讨论决定

### 财产接管注意事项

1. **全面接管**：确保接管范围完整，不得遗漏
2. **现场记录**：接管过程全程记录，必要时录像或公证
3. **交接手续**：交接清单必须双方签字确认
4. **财产保管**：妥善保管接管财产，防止毁损灭失
5. **强制接管**：债务人拒不配合时及时申请法院强制接管

### 合同处理注意事项

1. **期限把控**：严格遵守2个月决定期限和30日催告答复期限
2. **决策依据**：以债权人整体利益最大化为决策原则
3. **担保提供**：继续履行时对方要求担保的，应及时提供
4. **解除后果**：解除后对方损害赔偿请求权作为破产债权
5. **书面通知**：所有决定必须以书面形式通知对方

### 营业决策注意事项

1. **法院许可**：第一次债权人会议前的营业决定必须经法院许可
2. **必要性论证**：充分论证继续营业的必要性和可行性
3. **资金保障**：继续营业必须有可靠的资金保障方案
4. **风险评估**：充分评估继续营业的风险

### 管理人团队管理注意事项

1. **分工明确**：团队分工应明确具体，责任到人
2. **进度跟踪**：建立工作进度跟踪机制
3. **会议制度**：定期召开工作例会，及时沟通
4. **执业保险**：确保执业责任保险有效覆盖

### 报酬方案注意事项

1. **合理预估**：财产价值预估应客观合理
2. **比例适用**：严格按照司法解释规定的比例计算
3. **分期收取**：建议分期收取，与工作量挂钩
4. **调整机制**：工作量显著变化时及时申请调整

## 关键法条

### 《中华人民共和国企业破产法》

**第18条**（合同处理）：
> 人民法院受理破产申请后，管理人对破产申请受理前成立而债务人和对方当事人均未履行完毕的合同有权决定解除或者继续履行，并通知对方当事人。管理人自破产申请受理之日起二个月内未通知对方当事人，或者自收到对方当事人催告之日起三十日内未答复的，视为解除合同。
>
> 管理人决定继续履行合同的，对方当事人应当履行；但是，对方当事人有权要求管理人提供担保。管理人不提供担保的，视为解除合同。

**第25条**（管理人职责）：
> 管理人履行下列职责：
> （一）接管债务人的财产、印章和账簿、文书等资料；
> （二）调查债务人财产状况，制作财产状况报告；
> （三）决定债务人的内部管理事务；
> （四）决定债务人的日常开支和其他必要开支；
> （五）在第一次债权人会议召开之前，决定继续或者停止债务人的营业；
> ...

**第26条**：在第一次债权人会议召开之前，管理人决定继续或者停止债务人的营业，应当经人民法院许可。

**第28条**：管理人报酬由人民法院确定。

## Wiki 知识库查阅规则

当处理的案件涉及具体罪名或法律领域时，必须查阅 Wiki 知识库：

1. **检查知识库**：检查 `~/wiki/wiki/` 下是否有对应知识页（刑事：`books/criminal/{罪名}.md`，民事：`books/civil/{领域}/{具体问题}.md`）
2. **读取知识卡片**：若有对应页面，读取页面顶部 `<!-- CARD_START -->` 到 `<!-- CARD_END -->` 之间的知识卡片（约800字浓缩摘要）
3. **按需深入**：卡片信息足够则直接引用；需要深入了解具体辩护策略、案例细节、司法解释原文时，读取全文对应板块
4. **融入输出**：将 Wiki 知识自然融入当前工作输出，引用时标注来源页面路径
5. **案例知识页**：同时检查 `~/wiki/wiki/cases/` 下是否有相关判例知识页可供参考
