# 管理人工作详细操作手册

## 步骤1：读取前序文档与信息整合

### 1.1 Excel客户档案读取

**数据来源**：当前事项系统记录；复盘台账按总控确认后同步

**读取字段**：
| 字段 | 用途 |
|-----|------|
| 客户编号 | 定位客户 |
| 客户称呼 | 文档中显示 |
| 破产案号 | 文书引用 |
| 受理法院 | 文书引用 |
| 受理日期 | 计算合同处理期限（2个月） |
| 破产法律分析报告路径 | 读取前置分析结论 |

### 1.2 前置文档检查

**必查文档**：

| 文档名称 | 检查要点 | 缺失处理 |
|---------|---------|---------|
| 法院指定管理人决定书 | 确认管理人身份、指定日期 | 提示用户补充提供 |
| 破产法律分析报告 | 债务人基本情况、财产线索、已知合同 | 如未生成，先调用破产法律分析Skill |

### 1.3 操作指令

1. **读取Excel**：根据客户编号读取复盘台账，获取客户称呼、破产案号、受理法院、受理日期、破产法律分析报告路径

2. **读取破产法律分析报告**：
   - 提取债务人基本信息（名称、类型、注册资本、法定代表人）
   - 提取已知财产线索
   - 提取已知未履行完毕合同清单
   - 提取已知债权债务概况

3. **检查前置文档**：
   - 确认法院指定管理人决定书是否已提供
   - 如未提供，提示用户补充

4. **信息整合提示**：
   > "已读取【客户称呼】的破产案件档案。破产案号：【案号】，受理法院：【法院】，受理日期：【日期】。开始管理人接管工作。"

## 步骤2：印章与账户管理

### 2.1 刻制管理人印章

**法律依据**：《企业破产法》第25条、各地管理人工作规范

**办理流程**：
1. 凭法院受理裁定书、指定管理人决定书
2. 向公安机关申请刻制管理人印章
3. 印章样式："【债务人名称】管理人"
4. 印章封样备案

**输出文档**：印章刻制及备案记录（内部存档）

### 2.2 开立管理人银行账户

**办理流程**：
1. 凭法院指定管理人决定书、管理人印章
2. 选择开户银行（建议在受理法院所在地或债务人主要账户行开立）
3. 账户名称："【债务人名称】管理人"
4. 所有破产资金收支通过该账户

**账户管理规范**：
- 专户专用，不得与管理人自有资金混同
- 建立收支台账
- 定期向法院报告资金状况

## 步骤3：财产接管

### 3.1 接管范围（全面接管）

依据《企业破产法》第25条，管理人应当接管债务人的全部财产和事务：

| 序号 | 接管类别 | 具体内容 |
|-----|---------|---------|
| 1 | 实物财产 | 动产（设备、存货、车辆等）、不动产（房屋、土地等） |
| 2 | 货币资金 | 银行存款、现金、其他货币资金 |
| 3 | 知识产权 | 专利、商标、著作权、商业秘密等 |
| 4 | 无形资产 | 特许经营权、域名、商誉等 |
| 5 | 印章证照 | 公章、财务章、合同章、营业执照、资质证书等 |
| 6 | 财务账簿 | 会计凭证、账簿、报表、审计报告等 |
| 7 | 档案文件 | 合同档案、人事档案、诉讼档案等 |
| 8 | 电子数据 | 电脑数据、信息系统、电子账簿等 |
| 9 | 其他权益 | 对外投资、应收债权、合同权益等 |

### 3.2 接管程序

**第一步：通知交接**
- 向债务人法定代表人、实际控制人、财务负责人发送接管通知
- 要求准备交接材料
- 确定交接时间、地点

**第二步：现场交接**
- 与债务人有关人员办理交接手续
- 逐项清点、核对
- 共同签字确认交接清单

**第三步：处理保全/执行措施**
- 了解已存在的保全措施和执行措施
- 依法申请解除保全、中止执行
- 协调查封机关移交财产

**第四步：强制接管（如需要）**
- 如债务人拒不配合，申请法院强制接管
- 强制接管时全程录像或拍照（参考厦门指引）
- 邀请公证机关见证

📋 接管清单模板参见 `templates/接管清单.md`

## 步骤4：内部事务管理

### 4.1 接管后内部事务管理权

**管理人权限**：
- 接管后由管理人管理债务人的内部事务
- 管理人决定债务人的日常开支和其他必要开支
- 代表债务人参加诉讼、仲裁或其他法律程序

### 4.2 营业继续/停止决策

**决策时点**：第一次债权人会议前

**决策程序**：
1. 评估债务人继续营业的必要性和可行性
2. 分析继续营业的成本与收益
3. 报请人民法院许可

**法律依据**：《企业破产法》第26条

📋 营业继续/停止报告模板参见 `templates/营业继续报告.md` 和 `templates/营业停止报告.md`

## 步骤5：合同处理（破产法第18条）

### 5.1 合同处理期限

**关键期限**：
- **决定期限**：受理后2个月内决定解除或继续履行
- **催告答复期限**：对方催告后30日内答复
- **逾期后果**：逾期未通知视为解除合同

### 5.2 合同处理决策

**决策因素**：
1. 合同对破产财产的价值影响
2. 继续履行的成本与收益
3. 对方当事人的履约能力
4. 对债权人整体利益的影响

### 5.3 合同处理方式

**方式一：继续履行**
- 向对方发送继续履行通知
- 提供担保（对方要求时）
- 监督合同履行

**方式二：解除合同**
- 向对方发送解除通知
- 处理解除后的法律后果
- 对方损害赔偿请求权作为破产债权申报

📋 合同处理决定书模板参见 `templates/合同处理决定书_继续履行.md` 和 `templates/合同处理决定书_解除.md`

## 步骤6：聘用工作人员

### 6.1 聘用原则

**需经法院许可**：管理人聘用工作人员应当经人民法院许可

**适用情形**：
- 管理人团队人员不足
- 需要专业财务人员
- 需要专业评估人员
- 需要其他专业人员

### 6.2 聘用程序

1. 确定聘用需求和岗位
2. 拟定人员信息和费用预算
3. 向法院提交聘用工作人员报告
4. 获许可后签订聘用合同

### 6.3 可聘请的专业人员

- 律师
- 会计师
- 评估师
- 拍卖师
- 其他必要专业人员

📋 聘用人员报告模板参见 `templates/聘用人员报告.md`

## 步骤7：管理人团队管理

### 7.1 组长负责制

**组织架构**：
- 管理人负责人（组长）
- 各专业小组（财务组、法律组、资产组等）
- 各小组负责人

### 7.2 工作小组设置及职责

**（一）综合协调组**
- 负责与法院、债权人、债务人沟通协调
- 负责管理人内部会议组织
- 负责文件收发、档案管理
- 负责对外公告、通知发布

**（二）财务审计组**
- 负责接管债务人财务账簿
- 负责委托审计机构进行审计
- 负责债权审查、编制债权表
- 负责管理人账户管理

**（三）资产管理组**
- 负责接管债务人财产
- 负责财产清查、评估
- 负责财产保管、维护
- 负责财产处置

**（四）法律事务组**
- 负责诉讼、仲裁案件处理
- 负责合同审查、处理
- 负责法律意见书出具
- 负责债权人会议法律事务

### 7.3 执业责任保险

**参考厦门指引**：管理人被指定后应当投保执业责任保险

**保险要求**：
- 保险金额应与案件复杂程度、财产规模相匹配
- 保险期间应覆盖管理人履职期间
- 保险范围应包括因管理人过错造成的损失

📋 管理人团队分工方案模板参见 `templates/管理人团队分工方案.md`

## 步骤8：管理人报酬方案

### 8.1 报酬计算依据

**法律依据**：《企业破产法》第28条、最高人民法院《关于审理企业破产案件确定管理人报酬的规定》

**计算基数**：债务人最终清偿的无担保财产价值总额

**计算标准**：
| 清偿财产价值范围 | 比例 |
|----------------|------|
| 不超过100万元（含） | 12% |
| 超过100万元至500万元（含） | 10% |
| 超过500万元至1000万元（含） | 8% |
| 超过1000万元至5000万元（含） | 6% |
| 超过5000万元至1亿元（含） | 3% |
| 超过1亿元至5亿元（含） | 1% |
| 超过5亿元 | 0.5% |

### 8.2 报酬方案制定

**制定步骤**：
1. 预估债务人可清偿无担保财产价值
2. 根据工作量预测报酬
3. 向法院提交报酬方案
4. 法院审查确定

### 8.3 报酬调整

**调整情形**：
- 案件复杂程度超出预期
- 工作量显著增加
- 其他应当调整的情形

**调整程序**：向法院申请调整，说明理由和依据

📋 管理人报酬方案模板参见 `templates/管理人报酬方案.md`
