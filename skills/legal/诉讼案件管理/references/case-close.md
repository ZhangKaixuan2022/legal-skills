# 案件关闭与结案归档

案件关闭不等于删除案件。关闭后仍保留业务文件和系统记录。

## 关闭触发

- 判决、裁定、调解、和解、撤诉、执行或履行已经处理完。
- 委托终止。
- 客户明确不继续。
- 刑事某阶段结束且无后续委托。
- 长期无动作并经用户确认暂停或关闭。

## 关闭前检查

```markdown
### 结案/关闭检查
- 最终文书：
- 是否已告知客户：
- 上诉/申诉/执行/履行期限：
- 费用是否处理：
- 原件是否处理：
- 案件总结：
- 交付清单：
- 后续提醒：
- 业务文件归档：
- 系统记录更新：
- 关闭状态：
```

## 同步记录

关闭时必须更新：

- `当前状态.md`
- `事件记录.md` 或 `阶段记录.md`
- `期限台账.md`
- `案件简报.md`
- `组合状态.md`
- `结案归档记录.md`
- `<LEGAL_WORKSPACE>/_系统记录/诉讼案件总览.md`

如存在未到期的上诉、申诉、执行、履行或回访事项，应保留飞书提醒，不因案件关闭自动删除。
