# 传票与通知处理

适用于传票、开庭通知、举证通知、应诉通知、缴费通知、判决书、裁定书、调解书、执行通知、公安/检察通知等诉讼程序文书。

## 处理流程

1. 按法律工作总控文件读取规范读取文书并输出/记录读取复查摘要。
2. 抽取程序字段、期限字段和行动事项。
3. 写入 `传票与通知台账.md`。
4. 写入或更新 `期限台账.md`。
5. 更新 `程序时间线.md` 和 `当前状态.md`。
6. 调用飞书日历创建提醒；材料准备事项可创建飞书任务。
7. 将飞书链接写入 `飞书同步记录.md`。

## 必须抽取字段

- 文书名称。
- 案号。
- 法院、仲裁委、公安、检察院或其他机关。
- 当事人。
- 文书日期。
- 送达日期、签收日期、签收人。
- 开庭/到场时间。
- 地点、法庭、联系方式。
- 举证期限、答辩期限、上诉期限、缴费期限、提交材料期限。
- 需提交材料。
- 未明确、冲突或 OCR 存疑内容。

## 输出模板

```markdown

- 文书名称：
- 案号：
- 机关/法院：
- 送达/签收日期：
- 开庭/到场时间：
- 地点：
- 触发期限：
- 需准备事项：
- 已写入期限台账：
- 已创建飞书提醒：
- 需要客户确认：

```

## 禁止事项

- 未读取文书就创建期限或日历。
- 日期、时间、地点、法庭、案号存疑时静默创建提醒。
- 把 OCR 存疑字段当作确定期限。
