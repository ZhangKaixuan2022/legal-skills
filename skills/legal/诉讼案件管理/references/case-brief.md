# 案件简报

案件简报分为内部简报、客户简报、庭前简报。

## 内部简报

用于律师快速恢复上下文，可包含内部策略和风险判断。

```markdown
# 案件简报

## 一、当前状态
## 二、关键事实
## 三、程序进展
## 四、核心争议/辩点
## 五、已完成工作
## 六、关键期限
## 七、材料缺口
## 八、风险提示
## 九、下一步动作
## 十、需要客户配合
```

## 客户简报

用于形成可外发内容的系统草稿，不暴露内部推演和不适合外发的策略。正式发给客户的版本应另存业务文件区。

```markdown
# 案件进展简报

## 当前进展
## 已完成工作
## 下一步安排
## 需要您配合的事项
## 重要时间节点
## 风险提示
```

## 庭前简报

围绕开庭、谈判、听证、会见或提交材料。

```markdown
# 庭前简报

## 开庭/会议信息
## 争议焦点/辩护重点
## 已准备材料
## 待补材料
## 发问/质证/陈述要点
## 关键期限
```

## 来源要求

每项关键事实、期限、争点或风险判断必须标注来源，不能把工作笔记中的推演写成客户确认事实。
