# 劳动争议仲裁程序管理Skill

## 功能概述

本Skill覆盖劳动争议仲裁程序全流程管理，是劳动争议诉讼Agent在L5-L9阶段的核心执行Skill。主要功能包括：

1. **仲裁前置强制检测** - 判断案件是否必须经过仲裁前置程序
2. **仲裁时效计算器** - 精确计算1年仲裁时效，处理中断/中止/持续侵害等特殊规则
3. **一裁终局自动判断** - 根据请求金额和争议类型自动判断是否属于终局裁决
4. **仲裁申请书生成（W角色）** - 劳动者代理的仲裁申请书起草
5. **仲裁答辩书生成（E角色）** - 用人单位代理的仲裁答辩书起草
6. **仲裁请求清单生成** - 按争议类型列出标准请求项
7. **L9分支路由** - 裁决后5种走向的判断和引导
8. **仲裁策略报告** - 综合评估输出，指导后续程序
9. **用人单位风险敞口评估** - E角色在答辩、调解或补证前评估败诉点、金额暴露、终局裁决和撤销路径

## 核心功能模块

### 1. 仲裁前置强制检测

根据《劳动争议调解仲裁法》第5条，劳动争议处理实行"四级递进"制度：

```
协商（自愿）→ 调解（企业、行业、社会）→ 【仲裁（必经）】→ 诉讼（对仲裁不服）
```

**必须仲裁前置的争议类型**（第2条）：
1. 确认劳动关系纠纷
2. 订立、履行、变更、解除和终止劳动合同纠纷
3. 除名、辞退和辞职、离职纠纷
4. 工作时间、休息休假、社会保险、福利、培训、劳动保护纠纷
5. 劳动报酬、工伤医疗费、经济补偿或者赔偿金纠纷

**不需要仲裁前置的例外情形**：

| 情形 | 法律依据 | 处理路径 |
|-----|---------|---------|
| 劳动者凭工资欠条直接起诉 | 司法解释（二）第3条 | 可直接诉讼 |
| 解除/终止协议不履行 | 司法解释（一）第35条 | 可直接诉讼 |
| 非劳动争议的民事纠纷 | 司法解释（一）第1条 | 按普通民事纠纷 |

### 2. 仲裁时效计算器

**基本规则**：1年时效（《劳动争议调解仲裁法》第27条）

**起算点**：从当事人知道或应当知道权利被侵害之日起

**特殊规则**：

| 特殊规则 | 适用情形 | 计算方式 |
|---------|---------|---------|
| 持续侵害 | 拖欠工资等持续性侵害 | 从最后一次侵害起算 |
| 时效中断 | 主张权利/请求救济/对方同意 | 从中断时起重新计算 |
| 时效中止 | 不可抗力/正当理由 | 障碍消除后继续计算 |

📖 完整时效计算规则与模板 → [references/仲裁时效计算规则.md](仲裁时效计算规则.md)

### 3. 一裁终局自动判断

根据《劳动争议调解仲裁法》第47条，以下两类争议实行一裁终局：

**第一类：金额限制型**（不超过当地月最低工资标准×12个月）
- 追索劳动报酬
- 工伤医疗费
- 经济补偿
- 赔偿金

**第二类：标准执行型**
- 因执行国家劳动标准在工时方面发生的争议
- 因执行国家劳动标准在休假方面发生的争议
- 因执行国家劳动标准在社会保险方面发生的争议

**判断流程**：
```
输入：仲裁请求列表
    ↓
遍历每项请求 → 判断争议类型 → 判断金额是否超标准
    ↓
├─ 全部请求均符合终局条件 → 一裁终局
└─ 任一请求不符合 → 非终局裁决
```

**诉讼权差异**：

| 裁决类型 | 劳动者权利 | 用人单位权利 |
|---------|-----------|-------------|
| 一裁终局 | 15日内起诉（基层法院） | 30日内申请撤销（中级法院） |
| 非终局 | 15日内起诉（基层法院） | 15日内起诉（基层法院） |

### 5. 仲裁请求清单生成

| 争议类型 | 典型请求要点 |
|---------|-------------|

| 追索劳动报酬 | 拖欠工资+加班费+年休假工资+逾期利息 |

| 未签合同双倍工资 | 双倍工资差额（最多11个月） |
| 工伤保险待遇 | 伤残补助金+医疗补助金+就业补助金+停工留薪期工资等 |

## 特殊保护检测

当案件涉及以下群体时，自动触发特殊保护检测：

| 保护类型 | 检测条件 | 保护内容 |
|---------|---------|---------|
| 三期女职工 | 孕期/产期/哺乳期 | 不得依第40/41条解除 |
| 工伤职工 | 丧失或部分丧失劳动能力 | 停工留薪期内不得解除 |
| 医疗期职工 | 患病/非因工负伤医疗期内 | 不得依第40/41条解除 |
| 距退休不足5年 | 本单位满15年+距退休不足5年 | 不得依第40/41条解除 |

### 三期女职工保护检测

**检测项**：
- 是否处于孕期（根据怀孕时间、预产期判断）
- 是否处于产期（根据分娩时间判断）
- 是否处于哺乳期（婴儿出生1年内）
- 解除行为是否违法

**保护规则**：
- 用人单位不得依据第40条（非过失性解除）、第41条（经济性裁员）解除
- 劳动合同期满，应当续延至相应情形消失时终止
- 违法解除的，劳动者可要求继续履行或2N赔偿

### 工伤职工保护检测

**检测项**：
- 工伤认定是否完成
- 伤残等级
- 停工留薪期是否到期
- 解除行为是否违法

**保护规则**：
- 患职业病或因工负伤并被确认丧失或部分丧失劳动能力的，不得依第40/41条解除
- 停工留薪期内，原工资福利待遇不变
- 1-4级伤残：保留劳动关系，退出工作岗位
- 5-6级伤残：保留劳动关系，安排适当工作
- 7-10级伤残：劳动合同期满可终止或劳动者提出可解除

### 医疗期职工保护检测

**医疗期长度计算**：

| 实际工作年限 | 本单位工作年限 | 医疗期 |
|-------------|---------------|-------|
| 10年以下 | 5年以下 | 3个月 |
| 10年以下 | 5年以上 | 6个月 |
| 10年以上 | 5年以下 | 6个月 |
| 10年以上 | 5-10年 | 9个月 |
| 10年以上 | 10-15年 | 12个月 |
| 10年以上 | 15-20年 | 18个月 |
| 10年以上 | 20年以上 | 24个月 |

**保护规则**：
- 医疗期内，不得依据第40/41条解除
- 劳动合同期满，应当续延至医疗期满时终止
- 医疗期满解除需满足：不能从事原工作 + 不能从事另行安排的工作 + 提前30日通知或N+1

### 距退休不足5年职工保护检测

**检测条件**：
- 法定退休年龄：男60岁/女干部55岁/女工人50岁
- 在本单位连续工作满15年
- 距法定退休年龄不足5年

**保护规则**：
- 满足上述条件的，用人单位不得依据第40/41条解除
- 可以依据第39条（过失性解除）解除

## 工作流程

### 主流程

```
用户输入客户编号 + 仲裁相关指令
    ↓
步骤1：读取系统记录/复盘台账
    - 获取客户基本信息、案件状态、前期文档路径
    ↓
步骤2：前置文档检查
    - 检查法律服务建议书（必需）
    - 检查经济补偿测算表（W角色）/风险敞口评估（E角色）
    - 检查举证责任分配表（必需）
    ↓
步骤3：仲裁前置检测
    - 判断是否需要仲裁前置
    - 如不需要，提示可直接诉讼
    ↓
步骤4：仲裁时效评估
    - 计算时效起算点、截止日
    - 评估中断/中止机会
    - 输出时效评估报告
    ↓
步骤5：一裁终局预判
    - 根据请求金额和类型判断
    - 输出判断结果和诉讼权提示
    ↓
步骤6：特殊保护检测（如适用）
    ↓
步骤7：交互收集信息
    - 确认/补充当事人信息
    - 确认仲裁请求事项
    ↓
步骤8：生成仲裁策略报告
    ↓
步骤9：生成仲裁文书
    - W角色：仲裁申请书 + 证据清单 + 请求清单
    - E角色：仲裁答辩书 + 证据清单 + 质证意见
    ↓
步骤10：保存文件与系统记录同步
    ↓
完成
```

## 注意事项

1. **仲裁前置强制性**：劳动争议必须先仲裁，未经仲裁直接起诉将被驳回（例外情形除外）

2. **时效问题优先**：仲裁时效1年，持续侵害从最后一次起算，务必在时效评估中重点关注

3. **一裁终局注意**：终局裁决对用人单位限制更大（30日撤销期vs劳动者15日起诉期），策略运用时注意双方权利差异

4. **特殊保护优先**：涉及"三期"女职工、工伤职工等特殊群体时，优先适用保护性规定

5. **举证责任倾斜**：充分利用劳动争议举证责任倒置规则，减轻劳动者举证负担

6. **调解优先**：劳动争议调解成功率较高，任何阶段都应重视调解机会

7. **文档格式规范**：仲裁申请书/答辩书需符合仲裁委格式要求，确保受理

8. **时间节点管理**：关键期限（时效、起诉期、撤销期）务必准确计算并提醒用户

## Wiki 知识库查阅规则

当处理的案件涉及具体罪名或法律领域时，必须查阅 Wiki 知识库：

1. **检查知识库**：检查 `~/wiki/wiki/` 下是否有对应知识页（刑事：`books/criminal/{罪名}.md`，民事：`books/civil/{领域}/{具体问题}.md`）
2. **读取知识卡片**：若有对应页面，读取页面顶部 `<!-- CARD_START -->` 到 `<!-- CARD_END -->` 之间的知识卡片（约800字浓缩摘要）
3. **按需深入**：卡片信息足够则直接引用；需要深入了解具体辩护策略、案例细节、司法解释原文时，读取全文对应板块
4. **融入输出**：将 Wiki 知识自然融入当前工作输出，引用时标注来源页面路径
5. **案例知识页**：同时检查 `~/wiki/wiki/cases/` 下是否有相关判例知识页可供参考
