# 仲裁申请书生成规范（W角色）

## 文书结构

```
劳动争议仲裁申请书

一、申请人信息（7字段）
二、被申请人信息
三、仲裁请求事项
四、事实与理由
五、法律依据
六、证据清单
七、附件说明
```

## 申请人信息字段（7字段）

| 字段 | 内容要求 | 示例 |
|-----|---------|------|
| 姓名 | 申请人全名 | 张三 |
| 性别 | 男/女 | 男 |
| 民族 | 民族名称 | 汉族 |
| 出生日期 | XXXX年XX月XX日 | 1985年3月15日 |
| 身份证号码 | 18位身份证号 | 440XXXXXXXXXXXXXXX |
| 住址 | 详细居住地址 | 深圳市福田区XX路XX号 |
| 联系电话 | 手机号码 | 138XXXXXXXX |

## 被申请人信息字段

| 字段 | 内容要求 | 示例 |
|-----|---------|------|
| 单位名称 | 用人单位全称 | 深圳市XX科技有限公司 |
| 统一社会信用代码 | 18位代码 | 9144XXXXXXXXXXXXXX |
| 住所地 | 注册地址 | 深圳市南山区XX路XX号 |
| 法定代表人 | 姓名+职务 | 李四（总经理） |
| 联系电话 | 单位电话 | 0755-XXXXXXXX |

## 事实与理由撰写规范

**结构要求**：

```
事实与理由：

一、劳动关系建立情况
（一）入职经过
    申请人于[日期]入职被申请人处，担任[岗位名称]，双方[签订/未签订]书面劳动合同。

（二）劳动合同主要内容
    合同约定：工作地点为[地点]，工作时间为[工时制度]，月工资为[金额]元。

二、争议事实经过
（一）[具体争议事件1]
    [时间线清晰的事实描述]

（二）[具体争议事件2]
    [时间线清晰的事实描述]

三、被申请人的违法行为
（一）[违法行为1]
    [具体违法事实及证据]

（二）[违法行为2]
    [具体违法事实及证据]

四、损害后果
    [申请人遭受的损失描述]

五、法律依据
    根据《[法律名称]》第[X]条规定："[法条原文]"
    被申请人的上述行为违反了上述法律规定，应当承担[法律责任]。

综上所述，被申请人的[行为]违反了劳动法律法规，侵害了申请人的合法权益。恳请贵委依法支持申请人的全部仲裁请求。
```

## 证据清单格式

```
证据清单

| 序号 | 证据名称 | 证明目的 | 页数 |
|-----|---------|---------|------|
| 1 | 劳动合同 | 证明劳动关系及约定内容 | X页 |
| 2 | 工资银行流水 | 证明工资标准及拖欠事实 | X页 |
| 3 | 考勤记录 | 证明加班事实 | X页 |
| 4 | 解除通知书 | 证明解除事实及时间 | X页 |
| 5 | 微信聊天记录 | 证明催告事实 | X页 |
| ... | ... | ... | ... |

证据提交人：[申请人姓名]
日期：[日期]
```

## 排版格式

严格执行法院/仲裁委要求的排版标准：

| 元素 | 格式要求 |
|-----|---------|
| 纸张 | A4纵向（210mm × 297mm） |
| 页边距 | 上下2.54cm、左右3.18cm |
| 标题 | 华文中宋小二号加黑，居中 |
| 一级标题（一、二、三） | 黑体三号 |
| 二级标题（（一）（二）） | 黑体小三号 |
| 三级标题（1. 2.） | 黑体小四号 |
| 正文 | 宋体小四，1.5倍行距 |
| 段落 | 首行缩进2个字符 |
| 法条引用 | 楷体小四 |

# 仲裁答辩书生成规范（E角色）

## 文书结构

```
劳动争议仲裁答辩书

一、答辩人信息
二、被答辩人（申请人）信息
三、答辩请求
四、答辩理由
    （一）仲裁时效论证
    （二）事实认定论证
    （三）法律适用论证
五、综合结论
```

## 答辩请求

```
答辩请求：

1. 请求驳回被答辩人的全部仲裁请求；
2. 请求确认答辩人与被答辩人之间的[解除/终止]行为合法有效；
3. 请求裁决被答辩人承担本案仲裁费用。
```

## 常见答辩要点

### 违法解除抗辩

```
四、关于解除合法性的答辩

（一）解除依据
答辩人解除与被答辩人的劳动合同，依据为《劳动合同法》第39条第[X]项：
"[法条原文]"

（二）事实依据
被答辩人存在以下[严重违纪/严重失职]行为：
1. [具体违纪事实1]，证据：[证据名称]
2. [具体违纪事实2]，证据：[证据名称]

（三）程序合法性
1. 规章制度制定程序合法：[说明民主程序]
2. 已向被答辩人公示告知：[说明告知程序]
3. 已通知工会：[说明工会程序]

综上所述，答辩人解除劳动合同事实清楚、依据充分、程序合法，不属于违法解除。
```

### 经济补偿抗辩

```
五、关于经济补偿的抗辩

（一）无需支付经济补偿的情形
根据《劳动合同法》第46条规定，[分析本案不属于应当支付经济补偿的情形]。

（二）已支付经济补偿
如已支付，提供支付凭证。

（三）计算基数异议
如对经济补偿计算基数有异议，说明理由并提供证据。
```

### 加班费抗辩

```
六、关于加班费的抗辩

（一）不存在加班事实
被答辩人主张的加班缺乏事实依据。

（二）已足额支付加班费
如已支付，提供工资条、考勤记录等证据。

（三）超过时效部分
被答辩人主张的加班费中，[具体时间段]部分已超过仲裁时效。
```

## 综合结论模板

```
综上所述：

1. 被答辩人的仲裁申请已超过法定时效，依法应予驳回；
2. 被答辩人在申请书中陈述的事实与客观情况不符；
3. 答辩人的行为符合劳动法律法规规定，不存在违法情形；
4. 被答辩人的仲裁请求缺乏事实和法律依据。

恳请贵委依法驳回被答辩人的全部仲裁请求。

此致
[仲裁委员会名称]

答辩人：[单位名称]（盖章）
法定代表人：[姓名]（签字）

[日期]

附：
1. 本答辩书副本[X]份
2. 证据目录及证据材料[X]份
```
