# 仲裁时效计算规则

## 基本时效规则

**时效期间**：1年（《劳动争议调解仲裁法》第27条）

**起算点**：从当事人知道或者应当知道其权利被侵害之日起计算

## 特殊规则

### （一）持续侵害规则

```
适用情形：拖欠工资等持续性侵害行为

起算规则：
- 劳动关系存续期间：从最后一次侵害行为发生之日起算
- 示例：2023年1月开始欠薪，2024年3月又欠薪 → 从2024年3月起算

劳动关系终止后：
- 应当在1年内提出（从终止之日起算）
- 超过1年则丧失仲裁权利
```

### （二）时效中断规则

根据《劳动争议调解仲裁法》第27条第2款，以下情形导致时效中断，从中断时起重新计算：

| 中断情形 | 认定标准 | 证据要求 |
|---------|---------|---------|
| 向对方主张权利 | 劳动者向用人单位提出支付/补偿要求 | 催告函、聊天记录、录音等 |
| 向有关部门请求权利救济 | 向劳动监察、工会、调解组织等投诉 | 投诉记录、回执、受理通知书 |
| 对方同意履行义务 | 用人单位承诺支付/补偿 | 承诺书、还款计划、部分履行凭证 |

### （三）时效中止规则

根据《劳动争议调解仲裁法》第27条第3款，因不可抗力或其他正当理由无法申请仲裁的，时效中止：

| 中止情形 | 适用条件 | 恢复计算 |
|---------|---------|---------|
| 不可抗力 | 地震、疫情等无法预见、避免的情况 | 不可抗力消除后继续计算 |
| 其他正当理由 | 劳动者重病、被限制人身自由等 | 障碍消除后继续计算 |

## 双倍工资时效特别说明

**逐月计算规则**：
- 从应当支付双倍工资的每个月起分别计算1年时效
- 超过1年的月份丧失请求权

**时效起算示例**：
```
入职：2023年1月1日
未签合同期间：2023年2月-2023年12月（11个月）
申请仲裁：2024年6月1日

时效检查：
2023年2月双倍工资 → 时效至2024年2月 → 已过期
2023年3月双倍工资 → 时效至2024年3月 → 已过期
...
2023年6月双倍工资 → 时效至2024年6月 → 未过期
2023年7月双倍工资 → 时效至2024年7月 → 未过期
...

可主张：2023年6月-2023年12月（7个月）
```
