# L9分支路由详解（裁决后走向判断）

## 5种分支的判断条件

根据《劳动争议调解仲裁法》第47-50条，仲裁裁决后存在5种可能的走向：

```
【L9分支路由图】

仲裁裁决（L9）
    ↓
├─ 一裁终局？
│   ├─ 是 → 双方是否起诉/申请撤销？
│   │       ├─ 均未在期限内提出 → L9a（终局+双方接受）→ L15结案归档
│   │       ├─ 劳动者15日内起诉 → L9b（终局+劳动者不服）→ L10一审
│   │       └─ 用人单位30日内申请撤销 → L9c（终局+单位申请撤销）→ L10一审（撤销之诉）
│   │
│   └─ 否 → 任何一方15日内起诉？
│           ├─ 是 → L9d（非终局+不服）→ L10一审
│           └─ 否 → L9e（裁决生效+执行）→ L15结案归档
```

## 各分支详细说明

### L9a：一裁终局+双方接受

| 项目 | 内容 |
|-----|------|
| 判断条件 | 裁决为一裁终局，双方均未在法定期限内起诉或申请撤销 |
| 时间节点 | 劳动者15日期限届满 + 用人单位30日期限届满 |
| 法律效力 | 裁决发生法律效力 |
| 后续路径 | → L15 结案归档 |
| 执行启动 | 可申请法院强制执行 |

### L9b：一裁终局+劳动者不服

| 项目 | 内容 |
|-----|------|
| 判断条件 | 裁决为一裁终局，劳动者在15日内起诉 |
| 起诉期限 | 收到裁决书之日起15日内 |
| 管辖法院 | 用人单位所在地或劳动合同履行地基层法院 |
| 后续路径 | → L10 一审（劳动者起诉） |
| 注意事项 | 起诉后仲裁裁决不发生效力 |

### L9c：一裁终局+用人单位申请撤销

| 项目 | 内容 |
|-----|------|
| 判断条件 | 裁决为一裁终局，用人单位在30日内申请撤销 |
| 申请期限 | 收到裁决书之日起30日内 |
| 管辖法院 | 中级人民法院（仲裁委所在地） |
| 撤销事由 | 《劳动争议调解仲裁法》第49条规定的6种情形 |
| 后续路径 | → L10 一审（撤销之诉） |

**撤销裁决的6种法定情形**（第49条）：
1. 适用法律、法规确有错误的
2. 劳动争议仲裁委员会无管辖权的
3. 违反法定程序的
4. 裁决所根据的证据是伪造的
5. 对方当事人隐瞒了足以影响公正裁决的证据的
6. 仲裁员在仲裁该案时有索贿受贿、徇私舞弊、枉法裁决行为的

### L9d：非终局裁决+不服

| 项目 | 内容 |
|-----|------|
| 判断条件 | 裁决为非终局，任何一方在15日内起诉 |
| 起诉期限 | 收到裁决书之日起15日内 |
| 管辖法院 | 用人单位所在地或劳动合同履行地基层法院 |
| 后续路径 | → L10 一审 |
| 注意事项 | 双方均可起诉，先起诉方为原告 |

### L9e：裁决生效+执行

| 项目 | 内容 |
|-----|------|
| 判断条件 | 非终局裁决，双方均未在15日内起诉 |
| 时间节点 | 收到裁决书之日起15日期限届满 |
| 法律效力 | 裁决发生法律效力 |
| 后续路径 | → L15 结案归档 |
| 执行启动 | 可申请法院强制执行 |
