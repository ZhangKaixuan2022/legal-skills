# 12类争议的标准仲裁请求项

## 确认劳动关系

```
仲裁请求：
1. 确认申请人与被申请人自[日期]至[日期]存在劳动关系
```

## 追索劳动报酬纠纷

```
仲裁请求：
1. 请求裁决被申请人支付拖欠的工资人民币[X]元（[起始日期]至[截止日期]）；
2. 请求裁决被申请人支付加班工资人民币[X]元（工作日加班[X]小时、休息日加班[X]小时、法定节假日加班[X]小时）；
3. 请求裁决被申请人支付未休年休假工资人民币[X]元；
4. 请求裁决被申请人支付上述款项的逾期利息（按LPR计算，自[日期]起至实际支付之日止）。
```

## 经济补偿金纠纷

```
仲裁请求：
1. 请求裁决被申请人支付解除劳动合同经济补偿金人民币[X]元（[工作年限]年×[月平均工资]元）；
2. 请求裁决被申请人支付代通知金人民币[X]元（未提前30日通知）。
```

## 违法解除赔偿纠纷

```
仲裁请求：
1. 请求裁决被申请人支付违法解除劳动合同赔偿金人民币[X]元（[工作年限]年×[月平均工资]元×2）；
2. 请求裁决被申请人出具解除劳动合同证明；
3. 请求裁决被申请人办理社会保险关系转移手续。
```

## 未签劳动合同双倍工资纠纷

```
仲裁请求：
1. 请求裁决被申请人支付未签订书面劳动合同的双倍工资差额人民币[X]元（[起始月份]至[截止月份]，共[X]个月）；
2. 请求裁决确认申请人与被申请人之间存在劳动关系（[起始日期]至[截止日期]）。
```

## 工伤保险待遇纠纷

```
仲裁请求：
1. 请求裁决被申请人支付一次性伤残补助金人民币[X]元（[月平均工资]元×[X]个月）；
2. 请求裁决被申请人支付一次性工伤医疗补助金人民币[X]元；
3. 请求裁决被申请人支付一次性伤残就业补助金人民币[X]元；
4. 请求裁决被申请人支付停工留薪期工资人民币[X]元（[起始日期]至[截止日期]）；
5. 请求裁决被申请人支付医疗费人民币[X]元、住院伙食补助费人民币[X]元、护理费人民币[X]元；
6. 请求裁决被申请人支付劳动能力鉴定费人民币[X]元。
```

## 社会保险纠纷

```
仲裁请求：
1. 补缴社会保险；
2. 赔偿社保待遇损失[X]元
```

## 年休假纠纷

```
仲裁请求：
1. 支付未休年休假工资[X]元（300%标准）
```

## 竞业限制纠纷

```
仲裁请求：
1. 支付竞业限制补偿金[X]元；
2. 继续履行竞业限制协议
```

## 培训费纠纷

```
仲裁请求：
1. 返还违法收取的培训费[X]元
```

## 押金/保证金纠纷

```
仲裁请求：
1. 返还违法收取的押金/保证金[X]元
```

## 其他福利待遇

```
仲裁请求：
1. 支付[具体福利项目]差额[X]元
```
