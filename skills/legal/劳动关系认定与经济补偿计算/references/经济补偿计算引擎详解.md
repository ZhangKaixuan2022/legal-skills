# 经济补偿计算引擎详解

## N/N+1/2N判断逻辑

### N（经济补偿）- 适用7类法定情形（劳动合同法第46条）

| 情形 | 法条依据 | 适用条件 | 支付主体 |
|-----|---------|---------|---------|
| 劳动者依法解除 | 第38条 | 用人单位未及时足额支付劳动报酬、未缴社保、违章指挥等 | 用人单位 |
| 协商一致解除 | 第36条 | 双方协商一致解除 | 用人单位 |
| 非过失性解除 | 第40条 | 患病/非因工负伤医疗期满不能工作、不能胜任、客观情况变化 | 用人单位 |
| 经济性裁员 | 第41条 | 破产重整、经营困难、转产等 | 用人单位 |
| 固定期限合同到期不续签 | 第44条① | 用人单位不同意续签或降低条件 | 用人单位 |
| 用人单位破产 | 第44条④ | 用人单位被依法宣告破产 | 用人单位 |
| 用人单位解散/吊销 | 第44条⑤ | 被吊销营业执照、责令关闭、撤销或提前解散 | 用人单位 |

### N+1（经济补偿+代通知金）- 适用情形

| 情形 | 法条依据 | 适用条件 | 计算方式 |
|-----|---------|---------|---------|
| 非过失性解除未提前通知 | 第40条 | 依据第40条解除但未提前30日书面通知 | N + 1个月工资 |

### 2N（违法解除赔偿金）- 适用情形

| 情形 | 法条依据 | 适用条件 | 计算方式 |
|-----|---------|---------|---------|
| 违法解除/终止 | 第87条 | 无合法理由解除、程序违法解除、"三期"女职工违法解除等 | N × 2 |

### 无需补偿情形

| 情形 | 法条依据 | 说明 |
|-----|---------|------|
| 劳动者主动辞职（非第38条情形） | 第37条 | 提前30日通知解除 |
| 过失性解除 | 第39条 | 试用期不符合、严重违纪、严重失职等 |
| 劳动者开始享受养老保险待遇 | 第44条② | 劳动合同终止 |

## 月工资基数确定

### 计算公式

```
月工资基数 = 解除/终止前12个月工资总额 ÷ 12
```

### 包含项目

| 项目类型 | 是否计入 | 说明 |
|---------|---------|------|
| 基本工资 | ✓ | 固定发放的基本工资 |
| 岗位工资 | ✓ | 与岗位相关的工资 |
| 绩效工资 | ✓ | 考核发放的绩效工资 |
| 奖金 | ✓ | 年终奖、季度奖等 |
| 津贴补贴 | ✓ | 交通补贴、餐补、通讯补贴等 |
| 加班费 | ✓ | 加班工资应计入（最高法院观点） |
| 提成 | ✓ | 销售提成等 |

### 不包含项目

| 项目类型 | 是否计入 | 说明 |
|---------|---------|------|
| 社保代扣代缴部分 | ✗ | 个人承担社保费用 |
| 公积金代扣部分 | ✗ | 个人承担公积金 |
| 个人所得税 | ✗ | 代扣个税 |
| 一次性补偿 | ✗ | 解除时一次性支付的补偿 |
| 福利待遇 | ✗ | 节日福利、礼品等 |

### 工作不满12个月的情况

```
月工资基数 = 实际工作月数工资总额 ÷ 实际工作月数
```

## 三层封顶规则

### 第一层：月工资上限封顶

```
若：实际月工资 > 当地上年度职工月平均工资 × 3
则：按 当地上年度职工月平均工资 × 3 计算
```

### 第二层：工作年限上限封顶

```
若：月工资超过当地平均工资3倍（即触发第一层封顶）
则：工作年限最高不超过12年
```

### 第三层：总额封顶

```
最高补偿 = min(实际月工资, 社平×3) × min(工作年限, 12)

示例：
- 实际月工资：30,000元
- 当地社平工资：8,000元
- 工作年限：15年

计算：
第一层封顶：8,000 × 3 = 24,000元
第二层封顶：min(15, 12) = 12年
第三层总额：24,000 × 12 = 288,000元
```

### 非高收入不受12年限制

```
若：实际月工资 ≤ 当地上年度职工月平均工资 × 3
则：工作年限按实际计算，不受12年限制

示例：
- 实际月工资：10,000元
- 当地社平工资：8,000元
- 工作年限：20年

计算：
第一层：10,000 < 24,000，不触发封顶
第二层：不触发12年限制
第三层：10,000 × 20 = 200,000元
```

## 工作年限计算规则

### 基本规则

```
- 每满1年支付1个月工资
- 6个月以上不满1年 → 按1年计算（支付1个月）
- 不满6个月 → 支付半个月工资
```

### 2008年前后的分段计算

| 时间段 | 计算规则 | 封顶规则 |
|-------|---------|---------|
| 2008年1月1日前 | 按当时规定（各地可能不同） | 按当时规定 |
| 2008年1月1日后 | 按劳动合同法第47条 | 三层封顶规则 |

### 合并计算规则

- 非因本人原因从原单位被安排到新单位工作，原单位工作年限合并计算
- 原单位已支付经济补偿的，该部分年限不再重复计算

### 计算示例

```
入职日期：2015年3月1日
离职日期：2026年1月15日

计算：
完整年份：2015.3.1-2025.3.1 = 10年
剩余期间：2025.3.1-2026.1.15 = 10个月15天

10个月15天 > 6个月 → 按1年计算

总工作年限 = 10 + 1 = 11年
```
