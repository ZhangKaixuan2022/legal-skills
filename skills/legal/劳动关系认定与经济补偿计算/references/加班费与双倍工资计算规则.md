# 加班费与双倍工资计算规则

## 一、加班费三档计算

### 1.1 加班费计算基数确定

**基数确定规则（各地有差异，以当地规定为准）：**

| 确定方式 | 适用情况 | 计算公式 |
|---------|---------|---------|
| 劳动合同约定 | 合同有明确约定 | 按约定工资计算 |
| 实际工资 | 合同未约定或约定不明 | 前12个月平均工资（剔除加班费） |
| 基本工资 | 地方规定以基本工资为基数 | 基本工资 |
| 最低工资 | 基数低于最低工资时 | 按最低工资计算 |

**常见基数构成：**
```
加班费基数 = 月工资 ÷ 21.75天

或

加班费基数 = 月工资 ÷ 21.75天 ÷ 8小时（按小时计算时）
```

### 1.2 三档加班费计算公式

**工作日加班（150%）：**
```
工作日加班费 = 月工资 ÷ 21.75 ÷ 8 × 加班小时 × 150%

或按天计算：
工作日加班费 = 月工资 ÷ 21.75 × 加班天数 × 150%
```

**休息日加班（200%，不能补休时）：**
```
休息日加班费 = 月工资 ÷ 21.75 ÷ 8 × 加班小时 × 200%

或按天计算：
休息日加班费 = 月工资 ÷ 21.75 × 加班天数 × 200%
```

**法定节假日加班（300%）：**
```
法定节假日加班费 = 月工资 ÷ 21.75 ÷ 8 × 加班小时 × 300%

或按天计算：
法定节假日加班费 = 月工资 ÷ 21.75 × 加班天数 × 300%
```

### 1.3 加班事实举证

**劳动者举证责任：**
- 证明存在加班事实（初步证据）

**用人单位举证责任：**
- 提供考勤记录（保存义务）
- 提供加班审批记录（如实行审批制）
- 提供工资发放记录（证明已付加班费）

**证据类型：**

| 证据类型 | 证明力 | 获取方式 |
|---------|-------|---------|
| 考勤记录 | 强 | 用人单位提供/申请调取 |
| 加班审批单 | 强 | 用人单位提供 |
| 工资条 | 中 | 劳动者保存 |
| 工作邮件/聊天记录 | 中 | 截图保存 |
| 同事证言 | 弱 | 证人出庭 |
| 工作成果时间戳 | 中 | 系统记录 |

### 1.4 加班费计算明细表输出格式

| 日期 | 加班类型 | 加班时长 | 计算基数 | 倍数 | 当日加班费 | 备注 |
|-----|---------|---------|---------|-----|-----------|-----|
| | 工作日/休息日/法定节假日 | X小时 | | 150%/200%/300% | | |
| **合计** | | | | | **总金额** | |

## 二、双倍工资计算

### 2.1 适用情形

**未签书面劳动合同：**
- 用工之日起超过1个月不满1年未签书面劳动合同
- 从第2个月起至第12个月，每月支付双倍工资
- 最多11个月

**应签无固定期限合同未签：**
- 符合订立无固定期限劳动合同条件（连续工作满10年等）
- 用人单位未与劳动者订立无固定期限劳动合同
- 自应当订立之日起每月支付双倍工资

### 2.2 计算公式

```
双倍工资 = 月工资 × 未签合同月数（最多11个月）

起算：入职第2个月
截止：入职满1年（视为已订立无固定期限合同）
```

**计算示例：**
```
入职日期：2023年1月1日
首次签合同日期：2023年8月1日
月工资：8,000元

未签合同期间：2023年2月1日 - 2023年7月31日（6个月）

双倍工资 = 8,000 × 6 = 48,000元
```

### 2.3 仲裁时效

**逐月计算规则：**
- 从应当支付双倍工资的每个月起分别计算1年时效
- 超过1年的月份丧失请求权

**时效起算示例：**
```
入职：2023年1月1日
未签合同期间：2023年2月-2023年12月（11个月）
申请仲裁：2024年6月1日

时效检查：
2023年2月双倍工资 → 时效至2024年2月 → 已过期
2023年3月双倍工资 → 时效至2024年3月 → 已过期
...
2024年2月双倍工资 → 时效至2025年2月 → 未过期

可主张：2024年2月-2024年6月（5个月）
```

### 2.4 双倍工资计算表输出格式

| 月份 | 应签日期 | 双倍工资基数 | 金额 | 时效状态 | 备注 |
|-----|---------|-------------|-----|---------|-----|
| | | | | 未过期/已过期 | |
| **合计（可主张）** | | | | | |

## 三、竞业限制补偿计算

### 3.1 补偿标准

**法定最低标准：**
```
竞业限制补偿 ≥ 离职前12个月平均工资 × 30%
```

**各地标准差异：**

| 地区 | 标准 | 备注 |
|-----|------|------|
| 全国（司法解释） | 30% | 低于此标准可申请调整 |
| 深圳 | 50% | 地方性规定 |
| 北京 | 30% | 但低于最低工资按最低工资 |
| 上海 | 20%-50% | 根据竞业限制范围确定 |

### 3.2 计算公式

```
月竞业限制补偿 = 离职前12个月平均工资 × 约定比例（不低于30%）

总补偿 = 月竞业限制补偿 × 竞业限制月数（最长24个月）
```

### 3.3 违约金认定规则

**违约金调整：**
- 约定违约金过高的，可请求法院/仲裁委适当减少
- 一般以实际损失为基础，兼顾合同履行情况、当事人过错程度等

**常见违约金计算：**

| 约定方式 | 效力 | 调整可能 |
|---------|------|---------|
| 固定金额 | 有效 | 过高可调整 |
| 月补偿的倍数 | 有效 | 过高可调整 |
| 年薪的倍数 | 有效 | 过高可调整 |
