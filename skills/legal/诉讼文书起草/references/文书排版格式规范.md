# 文书排版格式规范

> **来源**：诉讼文书起草Skill - 排版格式规范（方案五）
> **用途**：严格执行法院要求格式的表格化排版标准

---

## 一、通用格式要求

严格执行法院要求格式：

| 元素 | 格式要求 |
|-----|---------|
| 纸张 | A4纵向（210mm × 297mm） |
| 页边距 | 上下2.54cm、左右3.18cm |
| 标题 | 华文中宋小二号加黑，居中 |
| 一级标题（一、二、三） | 黑体三号 |
| 二级标题（（一）（二）） | 黑体小三号 |
| 三级标题（1. 2.） | 黑体小四号 |
| 四级标题（（1）（2）） | 宋体小四号 |
| 正文 | 宋体小四，1.5倍行距 |
| 段落 | 首行缩进2个字符 |
| 法条引用 | 楷体小四 |
| 表格内容 | 宋体五号 |
| 页码 | 居中，宋体小四 |

---

## 二、编号层级规范（《法律文书汇编》标准）

**层级顺序**：
- 第一层：一、二、三
- 第二层：（一）（二）（三）
- 第三层：1. 2. 3.
- 第四层：（1）（2）（3）

**重要规则**：编号层级不超过四级，超过时应调整文书结构。
