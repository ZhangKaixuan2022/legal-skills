# 起诉状格式标准

> **来源**：诉讼文书起草Skill - 起诉状/答辩状格式规范
> **用途**：起诉状特有格式要求及与答辩状的区别

---

## 一、起诉状完整结构（7部分）

### 1. 标题
- **内容**："民事起诉状"
- **格式**：方正小标宋简体，居中，sz=44

### 2. 当事人信息
- **顺序**：原告 → 被告（无诉讼代理人；如有法定代理人紧跟其后）
- **自然人格式**：
  ```
  原告：[姓名]，[性别]，[出生日期]出生，[民族]，住[住址]，身份证号[号码]，联系电话[电话]。
  ```
- **法人格式**：
  ```
  原告：[名称]，住所地[地址]，法定代表人[姓名]，职务[职务]，联系电话[电话]。
  ```
- **重要规则**：起诉状中**不列示诉讼代理人信息**，如有法定代理人则列法定代理人

### 3. 诉讼请求
- **标题**："诉讼请求"
- **格式**：标题加粗，下用"1、2、3"数字编号
- **要求**：具体明确，违约金/利息等必须计算到具体金额

### 4. 事实与理由
- **标题**："事实与理由"
- **结构**：叙述事实 → 引入法律依据 → 损害计算
- **法条援引**：只写法条号，不写括号中的法条名称/内容说明

### 5. 结尾语
- **格式**："综上所述，..."总结段落
- **示例**："综上所述，原告的诉讼请求具有事实和法律依据，恳请贵院依法支持原告全部诉讼请求。"

### 6. 落款
```
此致
[法院名称]

具状人：[原告签名/盖章]

[年  月  日]
```

### 7. 附注
```
附：
一、本起诉状副本X份
二、证据目录及证据材料
```

## 二、起诉状格式规范

### 字体排版要求

| 元素 | 格式要求 |
|-----|---------|
| 标题字体 | 方正小标宋简体，居中，sz=44 |
| 正文字体 | 方正仿宋_GB2312，首行缩进，sz=32 |
| 章节标题 | 加粗，sz=32 |
| 当事人信息 | 左对齐，sz=32 |

## 三、答辩状格式规范

参考已有案例文档或律师提供的模板，如无可用模板则使用以下标准格式：

**答辩状文风格式特点**：

| 要素 | 规范要求 |
|-----|---------|
| 标题 | "答辩书"（非"民事答辩状"），方正小标宋简体，居中，sz=44 |
| 当事人信息 | 答辩人（我方）在前，列明名称、住所、法定代表人（法人时） |
| 案由引入 | "答辩人与XXX[案由]一案，现进行如下答辩：" |
| 结构层次 | 用"1. 2. 3."数字编号分点论述，非"一、二、三" |
| 事实陈述 | 先完整陈述本案事实经过（时间线清晰） |

**答辩论点结构**：
- 小标题（如"郭雪驰不服从公司工作调动..."）
- "依据上述事实："引出法律依据
- 法条原文完整引用（如《劳动合同法》第三十九条）
- 结合事实分析法条适用

**结论与结尾**：
- 结论段："综上，答辩人认为...，应当予以驳回。"
- 结尾："此致 [法院/仲裁委名称]"，答辩人盖章处，年月日
- 字体同起诉状：方正小标宋简体标题 + 方正仿宋_GB2312正文

---

## 四、起诉状与答辩状的主要区别

| 对比项 | 起诉状 | 答辩状 |
|-------|-------|-------|
| 标题 | 民事起诉状 | 答辩书 |
| 主体在前 | 原告 | 答辩人 |
| 编号方式 | 一、二、三/（一）（二） | 1. 2. 3. 数字编号 |
| 核心内容 | 诉讼请求、事实与理由 | 针对对方主张逐项反驳 |
| 结论段 | 恳请贵院依法支持原告全部诉讼请求 | 应当予以驳回 |
