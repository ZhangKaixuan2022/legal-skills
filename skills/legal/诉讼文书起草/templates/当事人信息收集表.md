# 当事人信息收集表

> **来源**：诉讼文书起草Skill - 步骤2
> **用途**：起诉状、答辩状等诉讼文书中当事人信息的收集模板，确保法定必要内容完整

## 收集原则

**当事人信息是起诉状的法定必要内容，必须逐项核实，信息不全时必须追问补全，不得留空或填"/"。**

---

## 一、原告信息

### 1.1 原告为自然人

| 字段 | 说明 |
|------|------|
| 姓名 | 必填 |
| 性别 | 必填 |
| 年龄 | 必填（或出生日期，二选一） |
| 民族 | 必填 |
| 职业 | 必填，不知填"无业"或"自由职业" |
| 工作单位 | 必填，无工作单位填"无" |
| 住所（户籍地） | 必填 |
| 经常居住地 | 与户籍地不同时必填 |
| 联系方式 | 必填，包括电话 |

### 1.2 原告为法人或其他组织

| 字段 | 说明 |
|------|------|
| 名称 | 必填 |
| 住所（注册地） | 必填 |
| 法定代表人/主要负责人姓名 | 必填 |
| 法定代表人/主要负责人职务 | 必填 |
| 联系方式 | 必填 |

---

## 二、诉讼代理人信息（如有）

| 字段 | 说明 |
|------|------|
| 姓名 | 必填 |
| 单位名称 | 必填（律师事务所名称） |
| 职务/身份 | 必填（如：律师） |
| 联系方式 | 必填 |

如无诉讼代理人，明确标注"原告自行诉讼"，不在正文中列出代理人信息。

---

## 三、被告信息

### 3.1 被告为自然人

| 字段 | 说明 |
|------|------|
| 姓名 | 必填 |
| 性别 | 必填 |
| 工作单位 | 尽量收集，不知填"不详" |
| 住所 | 必填，用于确定管辖 |
| 联系方式 | 尽量收集，不知填"不详" |

### 3.2 被告为法人或其他组织

| 字段 | 说明 |
|------|------|
| 名称 | 必填 |
| 住所 | 必填 |
| 法定代表人/主要负责人姓名 | 尽量收集，不知填"不详" |
| 联系方式 | 尽量收集，不知填"不详" |

---

## 四、交互规则（关键）

**每次交互必须一次性列出所有缺失字段，用户确认后方可继续，不得分多轮追问单个字段。**

### 交互格式示例

```
请确认/补充以下当事人信息：

【原告】
姓名：___
性别：___
年龄：___（或出生日期）
民族：___
职业：___
工作单位：___
住所：___
联系方式：___

【被告】
姓名：___
性别：___
工作单位：___（不知填"不详"）
住所：___
联系方式：___（不知填"不详"）

【诉讼代理人】（如无请注明）
姓名：___
单位：___
联系方式：___

【管辖法院】：___
【起诉日期】：___年___月___日
```
