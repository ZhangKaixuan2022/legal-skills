# 答辩状框架

> **来源**：诉讼文书起草Skill - 步骤4（答辩状分支）
> **用途**：提供否认、阻却、抵消、反诉四类策略框架及答辩状文档结构

---

## 一、四类答辩策略框架

### 策略一：否认策略（否定对方事实）

```
一、原告主张的[事实]不成立
（一）原告的证据不足以证明[事实]
    1. [证据名称]存在[真实性/合法性/关联性]问题
       [具体分析]
    2. [反驳证据]表明事实并非如此
       [具体说明]
（二）实际情况为[反驳事实]
    1. [事实依据]
    2. [证据支持] → 证据第X号
```

### 策略二：阻却策略（承认事实但主张法律抗辩）

```
一、即使原告主张的事实成立，其请求权也不应得到支持
（一）[抗辩事由]（如诉讼时效已过）
    1. 法律依据
       根据《民法典》第[X]条规定："[法条原文]"
    2. 事实依据
       [具体说明抗辩事由的成立条件]
（二）[第二项抗辩事由]（如有）
    ...
```

### 策略三：抵消策略（提出反请求）

```
一、原告存在[违约/过错]事实
（一）原告应承担[相应责任]
    1. 事实依据
       [原告违约/过错的具体事实]
    2. 法律依据
       根据《[法律名称]》第[X]条规定："[法条原文]"
二、被告有权要求[抵消/减免]
（一）抵消的法律依据
    根据《民法典》第五百六十八条规定，当事人互负债务，该债务的标的物种类、
    品质相同的，任何一方可以将自己的债务与对方的到期债务抵销。
（二）抵消金额计算
    [具体计算]
```

### 策略四：反诉策略

```
反诉请求：
一、判令反诉被告[具体诉求]；
二、判令反诉被告承担反诉费用。

反诉事实与理由：
一、反诉的请求权基础
（一）法律依据
    ...
（二）构成要件分析
    ...
二、要件事实与证据对应
（一）[要件一]
    ...
```

## 二、答辩状文档结构

```
答辩状

答辩人：[名称/姓名]
住所地：[详细地址]
法定代表人：[姓名]  职务：[职务]（法人时）

被答辩人：[对方当事人信息]

答辩人与[被答辩人][案由]一案（案号：[如有]），现提出如下答辩意见：

【根据选定策略填充对应内容】

一、案件基本事实
（一）[时间线事实陈述]
    ...
（二）[关键事实强调]
    ...

二、答辩意见
（一）[针对对方第一项主张的答辩]
    1. 对方主张
       [概述对方主张]
    2. 答辩意见
       [运用上述策略之一进行答辩]
    3. 法律依据
       根据《[法律名称]》第[X]条规定："[法条原文]"

（二）[针对对方第二项主张的答辩]
    ...

三、结论
综上所述，答辩人认为：
1. [核心结论一]；
2. [核心结论二]；
3. [核心结论三]。
[被答辩人]的诉讼请求缺乏事实和法律依据，恳请贵院依法驳回其全部诉讼请求。

此致
[法院/仲裁委员会名称]

答辩人：[答辩人名称/签名]
[XXXX年XX月XX日]

附：
一、本答辩状副本X份
二、证据目录及证据材料
```

## 三、答辩状导出链路

最终本地 `.docx` 必须调用 `法律文书模板与导出` Skill。本文模板只保留答辩状正文结构、论点组织和特殊格式要求；排版与导出由统一导出 Skill 接收语义 HTML 后完成。

**答辩状特有格式**：
- 标题："答辩书"（非"民事答辩状"）
- 结构编号："1. 2. 3."数字编号（非"一、二、三"）
- 论点结构：小标题 → "依据上述事实：" → 法条引用 → 事实分析 → 结论
- 法条引用：完整引用原文，不省略
- 字体同起诉状
