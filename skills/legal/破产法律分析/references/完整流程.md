# 破产法律分析Skill

## 功能概述

运用《中华人民共和国企业破产法》及相关司法解释，对破产案件进行系统性法律分析，包括破产原因认定、适用程序判断、角色定位分析、管辖法院确定、债权清偿评估、可撤销行为识别等，最终生成标准化的破产法律分析报告。按总控当前事项记录和复盘台账衔接。

## 触发条件

当用户发送以下格式时自动触发：
- 【客户编号】+ "破产法律分析"
- 【客户编号】+ "破产分析"
- 破产案件流程中自动衔接（BP1→BP2）

## 第三部分：分析工作流

### 步骤1：案件基本情况梳理

**操作指令：**
1. 从案件沟通记录中提取：
   - 客户基本信息（债务人/债权人/管理人身份）
   - 企业基本情况（名称、类型、注册资本、经营范围、股东结构）
   - 债务概况（债务总额、债务构成、到期债务情况）
   - 资产概况（资产总额、主要资产类型、资产变现能力）
   - 案件背景（经营困境原因、已采取的措施、相关诉讼情况）

2. **初步角色识别：**
   - R1（管理人）：是否接受法院指定、案件复杂度评估
   - R2（债权人代理）：债权清偿可能性评估、债权优先级分析
   - R3（债务人代理）：申请破产/重整利弊分析、时机选择

**输出要求：**
- 案件基本情况表
- 初步角色定位建议

**智能追问触发条件：**
> 角色定位不明确时："根据案件情况，您在本案中的角色定位是：A. 管理人（接受法院指定）；B. 债权人代理（代表债权人主张权利）；C. 债务人代理（代表债务人申请破产或重整）？"

---

### 步骤2：破产原因分析

**操作指令：**

依据《企业破产法》第2条及司法解释（一）第1-4条，分析是否符合破产原因：

**2.1 破产原因法定要件**

| 破产原因类型 | 构成要件 | 法律依据 |
|-------------|---------|---------|
| 类型一 | 不能清偿到期债务 + 资产不足以清偿全部债务 | 《企业破产法》第2条第1款 |
| 类型二 | 不能清偿到期债务 + 明显缺乏清偿能力 | 《企业破产法》第2条第1款 |

**2.2 "不能清偿到期债务"认定标准**

依据《司法解释（一）》第2条，需同时满足：
1. 债权债务关系依法成立
2. 债务履行期限已经届满
3. 债务人未完全清偿债务

**2.3 "资产不足以清偿全部债务"认定标准**

依据《司法解释（一）》第3条：
- 债务人的资产负债表、审计报告、资产评估报告等显示其全部资产不足以偿付全部负债
- 有相反证据足以证明债务人资产能够偿付全部负债的除外

**2.4 "明显缺乏清偿能力"认定情形**

依据《司法解释（一）》第4条，有下列情形之一的，应当认定：
1. 因资金严重不足或者财产不能变现等原因，无法清偿债务
2. 法定代表人下落不明且无其他人员负责管理财产，无法清偿债务
3. 经人民法院强制执行，无法清偿债务
4. 长期亏损且经营扭亏困难，无法清偿债务
5. 导致债务人丧失清偿能力的其他情形

**2.5 重整原因特殊规定**

依据《企业破产法》第2条第2款：
- 企业法人有明显丧失清偿能力可能的，可以依法进行重整

**输出要求：**
- 破产原因分析表（是否符合、符合类型、证据支撑）
- 重整原因分析（如适用）

---

### 步骤3：适用程序判断

**操作指令：**

根据案件具体情况，分析适用何种破产程序：

**3.1 破产清算程序**

| 项目 | 内容 |
|-----|------|
| 适用条件 | 企业无救治可能，通过变价分配清偿债权 |
| 程序特点 | 不可逆程序，企业法人资格终止 |
| 适用情形 | 企业已停止经营、无重整价值、债权人一致同意清算 |
| 法律依据 | 《企业破产法》第107-124条 |

**3.2 重整程序**

| 项目 | 内容 |
|-----|------|
| 适用条件 | 企业有救治可能，通过重整计划调整债务 |
| 程序特点 | 可逆程序，保留企业法人资格，减免债务、延期清偿 |
| 适用情形 | 企业有持续经营价值、有重整投资人、债务人有重整意愿 |
| 法律依据 | 《企业破产法》第70-94条 |

**3.3 和解程序**

| 项目 | 内容 |
|-----|------|
| 适用条件 | 债务人与债权人达成和解协议 |
| 程序特点 | 债务人主动申请，程序相对简便 |
| 适用情形 | 债务规模不大、债权人人数较少、和解可能性较高 |
| 法律依据 | 《企业破产法》第95-106条 |

**3.4 程序选择对比分析**

| 对比维度 | 破产清算 | 重整 | 和解 |
|---------|---------|------|------|
| 企业存续 | 终止 | 保留 | 保留 |
| 债务减免 | 按清偿比例 | 可大幅减免 | 协商确定 |
| 程序周期 | 相对较短 | 较长（6+9个月） | 较短 |
| 控制权 | 管理人 | 债务人/管理人 | 债务人 |
| 适用难度 | 较低 | 较高 | 中等 |

**输出要求：**
- 适用程序建议表
- 程序选择理由分析

**关键节点确认（必须执行）：**
> "根据案件分析，建议适用【破产清算/重整/和解】程序。理由如下：【具体理由】。请确认该程序选择是否符合您的预期？如不符合，请说明您的倾向性意见。"

---

### 步骤4：管辖法院分析

**操作指令：**

依据《企业破产法》第3条确定管辖法院：

**4.1 地域管辖**

- **一般规则**：破产案件由债务人住所地人民法院管辖
- **住所地认定**：债务人主要办事机构所在地
- **主要办事机构不明**：由债务人注册地人民法院管辖

**4.2 级别管辖**

| 企业类型 | 管辖法院 |
|---------|---------|
| 县、县级市或区的工商行政管理机关核准登记 | 基层人民法院 |
| 地区、地级市（含本级）以上工商行政管理机关核准登记 | 中级人民法院 |
| 纳入国家计划调整的企业破产案件 | 中级人民法院 |

**4.3 特殊情形**

- 金融机构破产：由中级人民法院管辖
- 上市公司破产：由中级人民法院管辖
- 关联企业实质合并破产：由核心企业住所地或主要财产所在地法院管辖

**输出要求：**
- 管辖法院确定表（地域管辖+级别管辖）

---

### 步骤5：角色定位分析

**操作指令：**

根据确认的角色（R1/R2/R3），进行针对性分析：

**5.1 R1（管理人视角）**

| 分析项目 | 分析内容 |
|---------|---------|
| 接受指定评估 | 案件复杂度、工作量预估、团队配置建议 |
| 案件难度评估 | 资产规模、债务结构、债权人人数、法律关系复杂度 |
| 工作量预估 | 资产清查、债权审查、诉讼处理、分配执行等 |
| 团队配置建议 | 律师人数、会计师/评估师需求、其他专业人员 |
| 报酬预估 | 按《最高人民法院关于审理企业破产案件确定管理人报酬的规定》估算 |
| 风险识别 | 潜在诉讼风险、债权人冲突、资产流失风险 |

**5.2 R2（债权人代理视角）**

| 分析项目 | 分析内容 |
|---------|---------|
| 债权清偿可能性 | 清偿比例预估（基于资产/债务比例） |
| 债权优先级分析 | 担保债权→职工债权→税款→普通债权的顺位 |
| 担保债权分析 | 担保物价值、担保范围、优先受偿金额 |
| 可撤销行为识别 | 破产申请前1年内的无偿转让、明显不合理价格交易 |
| 个别清偿撤销 | 破产申请前6个月内的个别清偿行为 |
| 股东出资追缴 | 未缴出资、抽逃出资的追缴可能性 |
| 债权申报策略 | 申报期限、申报材料、债权确认异议 |

**5.3 R3（债务人代理视角）**

| 分析项目 | 分析内容 |
|---------|---------|
| 申请利弊分析 | 破产保护的利弊、对经营的影响 |
| 时机选择 | 申请时机的合理性、紧迫性评估 |
| 股东后果分析 | 股权清零风险、出资义务加速到期 |
| 管理层后果分析 | 任职限制、配合义务、法律责任 |
| 重整可行性分析 | 重整价值评估、重整投资人寻找、重整方案设计 |
| 和解可能性分析 | 与主要债权人协商和解的可行性 |
| 替代方案评估 | 庭外重组、债务重组等其他途径 |

**输出要求：**
- 角色定位分析报告（根据确认角色输出对应内容）

**关键节点确认（必须执行）：**
> "角色定位确认为【R1管理人/R2债权人代理/R3债务人代理】。如角色定位有误，请及时告知更正。"

### 步骤6：关键风险提示

**操作指令：**

根据案件具体情况，识别并提示以下风险：

**6.1 程序性风险**

| 风险类型 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 管辖异议风险 | 对方可能提出管辖权异议 | 提前准备管辖依据 |
| 程序转换风险 | 重整失败转为清算 | 制定应急预案 |
| 期限风险 | 各类期限（申报、异议、起诉） | 建立期限管理台账 |

**6.2 实体性风险**

| 风险类型 | 风险描述 | 应对措施 |
|---------|---------|---------|
| 债权确认风险 | 债权金额或性质存在争议 | 充分准备证据材料 |
| 资产流失风险 | 破产前资产被转移或隐匿 | 申请财产保全、追查可撤销行为 |
| 清偿比例风险 | 实际清偿比例低于预期 | 合理设定预期、制定备选方案 |

**6.3 可撤销行为重点提示**

依据《企业破产法》第31-33条：

| 行为类型 | 时间范围 | 法律依据 |
|---------|---------|---------|
| 无偿转让财产 | 受理前1年内 | 第31条第（一）项 |
| 明显不合理价格交易 | 受理前1年内 | 第31条第（二）项 |
| 对无担保债务提供担保 | 受理前1年内 | 第31条第（三）项 |
| 提前清偿未到期债务 | 受理前1年内 | 第31条第（四）项 |
| 放弃债权 | 受理前1年内 | 第31条第（五）项 |
| 个别清偿 | 受理前6个月内 | 第32条 |
| 欺诈性转让 | 任何时候 | 第33条 |

**输出要求：**
- 关键风险提示清单
- 风险应对措施建议

---

### 步骤7：时间节点预估

**操作指令：**

根据适用程序，预估关键时间节点：

**7.1 破产清算程序时间节点**

| 阶段 | 法定时限 | 备注 |
|-----|---------|------|
| 债权申报 | 公告之日起30日-3个月 | 最短不得少于30日 |
| 第一次债权人会议 | 债权申报期限届满后15日内 | 法院召集 |
| 破产财产变价 | 无明确时限 | 依案件复杂程度 |
| 破产财产分配 | 变价完成后及时分配 | 多次分配情形 |
| 程序终结 | 分配完结后 | 法院裁定 |

**7.2 重整程序时间节点**

| 阶段 | 法定时限 | 备注 |
|-----|---------|------|

| 重整程序终止 | 计划执行完毕或失败 | 法院裁定 |

| 阶段 | 法定时限 | 备注 |
|-----|---------|------|

## 第八部分：注意事项

1. **检索局限说明**：必须在分析报告中注明检索基于公开渠道，部分资源需登录账号，建议后续补充检索

2. **分析局限性**：破产法律分析基于现有信息和公开渠道检索，随着案件进展和资产清查、债权申报等工作推进，分析结论可能需要调整

3. **不构成承诺**：破产法律分析报告仅供决策参考，不构成对案件结果的承诺或保证

4. **保密义务**：所有案件信息严格保密，仅用于本案办理

5. **时效性**：法律法规和裁判规则可能更新，建议定期复核

6. **不包含费用预算**：本Skill生成的破产法律分析报告不包含费用预算内容，费用相关事宜由委托合同Skill处理

7. **记录同步异常处理**：
   - 如当前事项系统记录不存在，提示建立或切换当前事项
   - 如客户编号未能定位当前事项，新建记录
   - 回写失败时保存备份并提示用户

8. **关键节点确认**：适用程序选择和角色定位必须在分析过程中与用户确认，未经确认不得擅自推进
