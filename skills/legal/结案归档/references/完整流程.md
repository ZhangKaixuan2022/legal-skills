# 结案归档Skill

## 功能概述

本Skill定位在民事一审诉讼体系的最终环节（判决/调解生效后），系统化完成案卷归档、案件总结、客户案卷移交、费用结算、客户回访等结案工作，实现案件全流程闭环管理。

### 核心功能

1. **案卷归档目录生成**：按标准自动生成完整的案卷归档目录，扫描核对已有文档
2. **案件结案报告**：基于全部案件材料，自动生成结案报告/办案小结
3. **判决书简单记录**：提取判决书关键信息，生成判决结果与诉讼请求对比表
4. **客户案卷移交**：生成移交清单和确认书，指引五步移交流程
5. **费用结算提示**：核查律师费结算情况，生成结算提醒清单
6. **客户回访**：生成回访要点和记录模板
7. **系统记录最终更新**：回写终态字段，标记案件为"已结案"

## 触发方式

### 被动触发

用户发送客户编号 + 提及以下关键词时自动触发：

## 第二部分：核心工作流程

### 步骤1：读取客户档案与案件材料

**操作指令**：

1. **读取当前事项系统记录**：根据客户编号读取客户/案件信息，获取客户信息、收费信息、案件状态、各文档路径

2. **扫描案件文件夹**：
   - 遍历案件文件夹全部子目录
   - 识别并记录所有文档的名称、路径、类型
   - 统计文档数量

3. **读取判决书**（如已生效）：
   - 提取案号、判决日期、判决结果
   - 识别支持/驳回的诉请

4. **信息整合提示**：
   > "已读取【客户称呼】的案件档案。案件状态：【状态】，已识别文档【X】份。开始结案归档流程。"

#### 5.1 移交清单生成

**需移交给客户的材料清单**：

| 序号 | 材料名称 | 原件/复印件 | 份数 | 备注 |
|-----|---------|-----------|------|------|
| 1 | 判决书/调解书/裁定书 | 原件 | 1 | |

**注意事项**：
- 只移交成品，不移交半成品
- 律师办案过程中的文字记录、内部讨论材料不移交
- 原件核对后归还客户

#### 5.2 移交五步流程提示

**向客户移交案卷材料的标准流程**：

1. **整理案卷材料**
   - 将所有案卷材料按时间顺序和重要性分类排序
   - 核对材料完整性

2. **制作移交清单**
   - 列明所有移交材料的名称、原件/复印件、份数
   - 编制页码索引

3. **与客户约定移交时间**
   - 提前与客户沟通，约定合适的时间和地点
   - 建议选择在律所进行移交

4. **当面移交并签署确认**
   - 双方当面清点材料
   - 签署《案卷移交确认书》

5. **留存移交记录**
   - 移交确认书双方各执一份
   - 扫描存档

#### 5.3 案卷移交确认书

**模板路径**：`templates/案卷移交确认书.md`

**主要内容**：
- 移交双方信息
- 移交材料清单
- 材料性质（原件/复印件）
- 移交时间、地点
- 双方签字确认

### 步骤7：客户回访

#### 7.1 回访时机提示

**建议回访时间**：结案后1-2周

**回访目的**：
- 了解客户对服务的满意度
- 解答客户后续疑问
- 维护客户关系
- 获取转介绍机会

#### 7.2 回访要点

**回访内容要点**：
1. **案件结果确认**
   - 对判决结果是否满意
   - 是否考虑上诉（如一审结案）
   - 是否需要协助执行

2. **服务评价**
   - 对律师专业能力的评价
   - 对服务过程的评价
   - 对沟通效率的评价

3. **后续需求**
   - 是否有其他法律需求
   - 是否需要常年法律顾问服务

4. **转介绍**
   - 是否有朋友需要法律服务

#### 7.3 回访记录模板

**模板路径**：`templates/客户回访记录.md`

**记录内容**：
- 回访时间
- 回访方式（电话/面谈/微信）
- 客户反馈
- 后续跟进事项

## 第三部分：文档生成规范

- 标题：华文中宋小二号加黑居中
- 正文：宋体小四，1.5倍行距

## 第四部分：交互规范

### 4.1 智能追问机制

**追问原则**：
- 每次只追问一个关键问题
- 追问语言简洁专业
- 说明追问的原因
- 等待用户回复后再继续

**常见追问情形**：

| 触发情形 | 追问示例 |
|---------|---------|
| 需要确认归档范围 | "本案为一审/二审/再审案件，请确认以生成对应归档目录。" |
| 需要确认费用结算 | "委托合同约定为固定收费/风险代理，请确认以核算应收费用。" |
| 需要确认移交方式 | "案卷移交建议当面移交并签署确认书，是否确认？" |
| 需要确认回访时间 | "建议结案后1-2周进行客户回访，是否现在生成回访要点？" |

## 第六部分：注意事项

### 6.1 案卷归档注意事项

1. **完整性原则**：确保所有应归档材料齐全
2. **规范性原则**：按标准目录顺序排列
3. **保密性原则**：涉及客户隐私的材料妥善保管
4. **时效性原则**：及时完成归档，避免遗漏

### 6.2 客户移交注意事项

1. **只移交成品**：不移交半成品和内部工作记录
2. **当面清点**：双方当面核对材料
3. **签署确认**：务必签署移交确认书
4. **留存备份**：移交前扫描备份

### 6.3 费用结算注意事项

1. **按约结算**：严格按照委托合同约定结算
2. **及时催收**：对未收款项及时催收
3. **开具发票**：及时开具律师费发票
4. **退费合规**：如需退费，按律所规定办理

### 6.4 客户回访注意事项

1. **把握时机**：结案后1-2周为宜
2. **真诚沟通**：了解真实反馈，不回避问题
3. **维护关系**：为后续合作和转介绍打基础
4. **记录存档**：回访记录及时存档
