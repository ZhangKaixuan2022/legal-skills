# 证据提取清单模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 辩护人调取证据材料笔录 + 接收材料清单 |
| 适用阶段 | 审查起诉阶段 |
| 使用对象 | 调取物证、书证等非言词类证据时使用 |
| 参考来源 | 《刑事辩护规范化电子版》 |
| 核心规范 | 调取笔录+接收清单配套使用 |

## 物证提取规范

**可提取的物证**：
- 当场拍照
- 制作照片作为提取清单附件

**无法提取的物证**：
- 对提取过程进行录像
- 物证拍照
- 证据持有人在照片上签字捺印确认

## 书证提取规范

- 在原件上注明"本件由本人于____年____月____日提供给____律师"
- 证据持有人签名捺印
- 如持有人要求留存复印件，当场复印交其保存

## 视听资料提取规范

- 当场复制两份
- 原件当场封存
- 证据持有人在封口处签名捺印
- 复制件由持有人签名确认
- 一份提交检察机关，一份留档

## 电子证据调查规范

### 收集方式

| 收集方式 | 适用情形 | 操作要点 |
|---------|---------|---------|
| 复制 | 电子文件、数据库 | 使用专业工具，保证完整性 |
| 打印 | 网页、聊天记录 | 显示完整URL和时间 |
| 截屏 | 即时通讯、社交媒体 | 显示完整界面和时间 |
| 拍照 | 无法操作的设备 | 清晰显示屏幕内容 |
| 录像 | 动态内容、操作过程 | 完整记录操作流程 |

### 记录要素

| 要素 | 说明 |
|-----|-----|
| 时间 | 收集的时间 |
| 地点 | 收集的地点 |
| 原始储存介质 | 存放地点、设备信息 |
| 电子数据来源 | 数据产生的平台、系统 |
| 持有人信息 | 持有人身份及联系方式 |

### 特殊情况处理

- 必要时全程录像记录提取过程
- 可委托公证机构对取证过程进行公证
- 涉及专业技术的，可聘请技术人员协助

## 使用说明

1. 本清单用于调取物证、书证等非言词类证据时使用。
2. 《辩护人调取证据材料笔录》与《接收材料清单》应配合使用，同时制作。
3. 要详细记录调取证据过程，确保调取证据过程合法。
4. 如实记录证据名称、数量、是否原件、介质、形态等特征，并做好接收材料的封存、保管工作。
5. 做到边调查、边记录，一次完成，由证据持有人等签字确认，不可事后加工整理。
6. 强烈建议在条件许可的情况下对调取证据过程进行全程录音录像。
7. 本清单一式三份，一份统一保管，一份附卷，一份交提供材料的单位或个人。
