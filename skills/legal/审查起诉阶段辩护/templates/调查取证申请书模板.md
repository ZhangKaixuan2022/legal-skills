# 调查取证申请书模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 调查取证申请书 |
| 适用阶段 | 审查起诉阶段 |
| 递交对象 | 人民检察院 |
| 参考来源 | 《刑事辩护规范化电子版》+《刑事辩护法律文书标准化文本》 |
| 核心规范 | 证据清单表格、申请理由三要素 |

## 文书结构

```markdown
                调查取证申请书

[检察院名称]：

    [首部段]

    申请事项：
    请求贵院依职权调取以下证据：[证据名称概述]

    事实与理由：
    一、申请调取的证据
    | 序号 | 证据名称 | 证据类型 | 证据来源/持有人 | 证明目的 |
    |-----|---------|---------|---------------|---------|
    | 1 | | | | |
    | 2 | | | | |

    二、申请调取的理由
    （一）[该证据与案件的关联性]
    （二）[辩护人无法自行调取的原因]
    （三）[该证据对案件事实认定的重要性]

    三、法律依据
    根据《中华人民共和国刑事诉讼法》第四十三条的规定...

此致
[检察院名称]

                                    申请人（辩护人）：【律师事务所名称】
                                                      [律师签名]
                                                    年    月    日
```

## 申请理由三要素

| 要素 | 说明 |
|-----|-----|
| 关联性 | 说明该证据与案件的关联性，能够证明什么事实 |
| 必要性 | 说明辩护人无法自行调取的原因（如涉及被害方、需要司法权保障等） |
| 重要性 | 说明该证据对案件事实认定的重要性 |

## 法律依据

| 条款 | 内容 |
|-----|-----|
| 《刑事诉讼法》第43条 | 辩护律师经证人或者其他有关单位和个人同意，可以向他们收集与本案有关的材料 |
| 《刑事诉讼法》第43条 | 辩护律师可以申请人民检察院收集、调取证据 |
| 《刑事诉讼法》第43条 | 辩护律师经人民检察院许可，可以向被害人或其近亲属、被害人提供的证人收集材料 |
| 《刑事诉讼法》第41条 | 辩护人认为侦查机关未提交的无罪罪轻证据，有权申请检察院调取 |

## 许可判断

**必须经检察院许可的调查**：
- 向被害人收集材料
- 向被害人近亲属收集材料
- 向被害人提供的证人收集材料

**可自行进行的调查**：
- 经证人本人同意的调查
- 向有关单位收集书证
- 向有关个人收集书证

## 证据提交时机策略

### 必须立即提交的三种证据

**法律依据**：《刑事诉讼法》第40条

| 证据类型 | 法定要求 | 提交对象 |
|---------|---------|---------|
| 不在场证明 | 应当及时告知 | 人民检察院 |
| 未达刑事责任年龄证明 | 应当及时告知 | 人民检察院 |
| 属于不负刑事责任的精神病人证明 | 应当及时告知 | 人民检察院 |

### 策略性选择提交时机的规则

**可策略性提交的证据**：
- 证明从犯的证据
- 证明自首立功的证据
- 证明犯罪情节轻微的证据
- 质证用证据

| 情形 | 建议时机 | 理由 |
|-----|---------|-----|
| 证据确凿有力 | 审查起诉阶段提交 | 争取不起诉或从轻起诉 |
| 证据存在瑕疵 | 视情况决定 | 避免被针对性反驳 |
| 证据涉及程序违法 | 书面申请时一并提交 | 便于检察院审查 |
