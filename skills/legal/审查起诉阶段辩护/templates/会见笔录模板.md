# 会见笔录模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 会见笔录 |
| 适用阶段 | 审查起诉阶段 |
| 使用对象 | 辩护律师内部使用 |
| 参考来源 | 《刑事辩护规范化电子版》 |
| 核心规范 | 告知事项+问答格式+签字确认 |

## 文书结构

```markdown
                    会 见 笔 录

时间：____年____月____日____时____分至____时____分
地点：[看守所名称]
会见律师：[姓名]、[姓名]
被会见人：[犯罪嫌疑人姓名]

一、告知事项
律师告知被会见人享有的诉讼权利：[列明告知内容]
被会见人表示：[理解/知悉]

二、会见内容

问：[问题]
答：[回答]

问：[问题]
答：[回答]

...

三、被会见人签字确认
以上笔录我已阅读（向我宣读过），内容属实。

                            被会见人签名：        
                                        ____年____月____日

会见律师签名：
```

## 审查起诉阶段分次会见提纲

### 第一次会见：听取意见+核实笔录

**会见目的**：
- 听取对起诉意见书的意见
- 宣读侦查阶段笔录，听取辩解

**会见内容**：

```
一、告知事项
□ 告知委托情况
□ 告知律师工作性质和职责
□ 告知审查起诉阶段程序和期限
□ 告知犯罪嫌疑人享有的诉讼权利

二、起诉意见书核实
□ 宣读起诉意见书主要内容
□ 询问对指控罪名的意见
□ 询问对指控事实的意见
□ 询问对指控证据的意见

三、侦查阶段笔录核实
□ 逐份宣读侦查阶段笔录
□ 核实笔录内容是否与陈述一致
□ 记录不一致之处及原因
□ 询问是否存在刑讯逼供等非法取证

四、基本情况了解
□ 身体健康状况
□ 羁押期间待遇
□ 对家人的嘱托
```

### 第二次会见：背对背核证+辩护方案

**会见目的**：
- 背对背核证
- 探讨辩护意见
- 制订取证计划

**会见内容**：

```
一、背对背核证
□ 对同案犯供述中涉及当事人的事实进行核实
□ 对证人证言中涉及当事人的事实进行核实
□ 对被害人陈述中涉及当事人的事实进行核实
□ 记录核实结果（一致/矛盾/新增）

二、辩护意见探讨
□ 通报阅卷发现的有利情况
□ 探讨可能的辩护方向
□ 听取当事人意见
□ 初步确定辩护策略

三、取证计划制订
□ 了解可能的无罪、罪轻证据线索
□ 确定需要调查的证人
□ 确定需要收集的物证、书证
□ 告知取证的注意事项
```

### 第三次会见：通报取证及辩护意见起草情况

**会见目的**：
- 通报取证进展
- 商讨辩护意见
- 确定基本辩护思路

**会见内容**：

```
一、工作通报
□ 通报调查取证情况
□ 通报辩护意见起草进展
□ 通报与检察机关沟通情况

二、辩护意见商讨
□ 宣读辩护意见主要内容
□ 听取当事人意见
□ 修改完善辩护意见
□ 确定最终辩护思路

三、下一步工作安排
□ 告知后续工作计划
□ 告知可能的案件走向
□ 解答当事人疑问
```

### 第四次会见：告知程序规定+做好起诉/退查准备

**会见目的**：
- 告知程序规定
- 做好起诉或退查的心理准备

**会见内容**：

```
一、程序告知
□ 告知审查起诉期限
□ 告知可能的处理结果（起诉/不起诉/退查）
□ 告知后续诉讼程序

二、结果准备
□ 起诉情形：告知一审程序、庭审注意事项
□ 不起诉情形：告知权利救济途径
□ 退查情形：告知退查期间注意事项

三、心理辅导
□ 稳定当事人情绪
□ 增强信心
□ 传达家人关心
```

### 第五次会见（退查期间）：了解新讯问+核实新笔录

**适用情形**：案件退回补充侦查期间

**会见内容**：

```
一、新讯问情况了解
□ 是否被再次讯问
□ 讯问时间、地点、人员
□ 讯问内容概要
□ 是否存在非法取证

二、新笔录核实
□ 核实新笔录与陈述是否一致
□ 记录不一致之处
□ 分析新笔录对案件的影响

三、法律咨询
□ 解答当事人疑问
□ 提供法律建议
```

### 第六次会见（退查重报后）：核实新证据+商讨新辩护意见

**适用情形**：退查后案件重新移送审查起诉

**会见内容**：

```
一、补充侦查情况了解
□ 了解补充侦查期间发生的情况
□ 了解是否有新的证据补充

二、新证据核实
□ 核实新补充的证据
□ 分析新证据对案件的影响
□ 调整辩护策略

三、新辩护意见商讨
□ 通报补充的辩护意见
□ 听取当事人意见
□ 确定调整后的辩护方向
```

### 第七次会见：同第四次会见内容

**适用情形**：二次退查后重新移送审查起诉

## 翻供风险防控

**核心原则**：

1. **不轻易出示全部案卷**
   - 不将案卷材料全部展示给犯罪嫌疑人
   - 防止产生不实辩解

2. **采用背对背核证**
   - 通过口头询问核实证据
   - 不直接宣读他人证言

3. **进行法律教育**
   - 告知翻供的法律后果
   - 告知如实供述的从宽政策

4. **如实记录**
   - 完整记录当事人陈述
   - 记录翻供原因和依据

**翻供风险评估**：

| 风险等级 | 风险表现 | 应对措施 |
|---------|---------|---------|
| 高风险 | 当事人明确表示要翻供且无合理依据 | 充分告知法律后果，建议如实陈述 |
| 中风险 | 当事人对部分事实有异议 | 核实异议是否有证据支持 |
| 低风险 | 当事人陈述基本稳定 | 继续保持，做好记录 |
