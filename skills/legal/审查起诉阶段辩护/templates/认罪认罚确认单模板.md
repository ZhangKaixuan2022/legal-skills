# 认罪认罚确认单模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 认罪认罚具结书签署情况确认单 |
| 适用阶段 | 审查起诉阶段 |
| 使用对象 | 辩护律师/值班律师见证具结书签署时使用 |
| 参考来源 | 《刑事辩护规范化电子版》 |
| 核心规范 | 告知情况确认+具结书内容确认 |

## 认罪认罚适用前提审查

**认罪认罚的前提条件**：

| 条件 | 审查内容 |
|-----|---------|
| 自愿性 | 犯罪嫌疑人是否自愿认罪认罚，无强迫、诱骗 |
| 明知性 | 犯罪嫌疑人是否了解认罪认罚的法律后果 |
| 事实基础 | 认罪认罚是否有充分的事实和证据支持 |
| 程序合法 | 是否依法保障了犯罪嫌疑人的诉讼权利 |

**不宜适用认罪认罚的情形**：

| 情形 | 说明 |
|-----|-----|
| 犯罪嫌疑人可能不构成犯罪 | 应争取无罪 |
| 事实不清证据不足 | 应争取存疑不起诉 |
| 当事人对定性有重大异议 | 应进行定性辩护 |
| 量刑建议明显过重 | 应协商或不同意 |
| 当事人不理解法律后果 | 应充分告知后再决定 |

## 律师告知义务

**律师应当告知犯罪嫌疑人的内容**：

| 序号 | 告知内容 |
|-----|---------|
| 1 | 认罪认罚从宽制度的法律规定 |
| 2 | 认罪认罚的从宽幅度 |
| 3 | 具结书的法律效力 |
| 4 | 签署具结书后翻供的法律后果 |
| 5 | 犯罪嫌疑人享有的诉讼权利 |
| 6 | 量刑建议的具体内容 |
| 7 | 律师对案件的分析意见 |

## 量刑建议协商

**协商原则**：

| 原则 | 内容 |
|-----|-----|
| 充分协商 | 与检察机关就量刑建议进行充分沟通 |
| 争取有利 | 在认罪基础上争取最大程度的从宽 |
| 尊重意愿 | 最终是否接受由犯罪嫌疑人决定 |
| 如实告知 | 将协商过程和结果如实告知当事人 |

**协商要点**：

| 协商事项 | 协商策略 |
|---------|---------|
| 量刑建议幅度 | 争取较低的量刑区间 |
| 是否建议缓刑 | 符合条件的积极争取 |
| 从宽情节认定 | 确保有利情节被认定 |
| 罚金数额 | 在法定幅度内争取较低数额 |

## 具结书签署流程

**签署前准备**：

| 步骤 | 内容 |
|-----|-----|
| 1 | 会见当事人，详细告知认罪认罚制度 |
| 2 | 分析案件情况，提出是否认罪认罚的建议 |
| 3 | 如决定认罪认罚，与检察机关协商量刑建议 |
| 4 | 将协商结果告知当事人，确认其同意 |

**签署时注意事项**：

| 注意事项 |
|---------|
| 确认当事人理解具结书全部内容 |
| 确认当事人签署系自愿 |
| 在场见证签署全过程 |
| 在具结书上签字确认 |
| 保留签署情况记录 |

## 认罪态度表述指导

**认罪态度表述的重要性**：

认罪态度直接影响量刑，表述方式需要专业指导。

**常见问题及指导**：

| 问题类型 | 错误表述示例 | 正确表述指导 |
|---------|-------------|-------------|
| 全面否认 | "我没有犯罪" | 区分事实认定与法律评价 |
| 避重就轻 | "我只是帮忙而已" | 如实陈述参与程度 |
| 推卸责任 | "都是别人让我做的" | 承认自身责任，同时说明被动因素 |
| 狡辩态度 | 反复翻供、前后矛盾 | 保持陈述稳定，有异议通过律师反映 |

**认罪认罚态度指导**：

| 态度类型 | 表现 | 从宽幅度参考 |
|---------|-----|-------------|
| 如实供述+认罪认罚 | 全面如实交代，接受处罚 | 从宽幅度最大 |
| 如实供述+不认罪 | 交代事实但对定性有异议 | 坦白从宽 |
| 部分供述+部分认罪 | 承认部分事实 | 视情节酌情从宽 |
| 拒不供述+不认罪 | 否认犯罪事实 | 无从宽情节 |

## 法律依据

根据《中华人民共和国刑事诉讼法》第一百七十四条规定：

> 犯罪嫌疑人自愿认罪，同意量刑建议和程序适用的，应当在辩护人或者值班律师在场的情况下签署认罪认罚具结书。
