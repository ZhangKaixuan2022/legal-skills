# 取保候审申请书模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 取保候审申请书 |
| 适用阶段 | 审查起诉阶段 |
| 递交对象 | 人民检察院 |
| 参考来源 | 《刑事辩护规范化电子版》+《刑事辩护法律文书标准化文本》 |
| 核心规范 | 申请事项+事实与理由、附件清单 |

```markdown

[检察院名称]：

    [首部段]

    请求对犯罪嫌疑人[姓名]变更强制措施为取保候审。

    事实与理由：
    一、犯罪嫌疑人基本情况
    [姓名、年龄、住址、羁押情况]

    二、申请取保候审的理由
    （一）[理由1：如可能判处管制、拘役或独立适用附加刑]
    （二）[理由2：如患有严重疾病、生活不能自理等]

    三、保证方式
    [保证人担保/保证金担保，具体说明]
```

## 法定条件说明

根据《中华人民共和国刑事诉讼法》第六十七条，人民检察院对有下列情形之一的犯罪嫌疑人，可以取保候审：

| 序号 | 法定情形 |
|-----|---------|
| 1 | 可能判处管制、拘役或者独立适用附加刑的 |
| 2 | 可能判处有期徒刑以上刑罚，采取取保候审不致发生社会危险性的 |
| 3 | 患有严重疾病、生活不能自理，怀孕或者正在哺乳自己婴儿的妇女，采取取保候审不致发生社会危险性的 |
| 4 | 羁押期限届满，案件尚未办结，需要采取取保候审的 |
