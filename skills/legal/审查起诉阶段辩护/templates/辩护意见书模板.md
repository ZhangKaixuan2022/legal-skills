# 辩护意见书模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 辩护意见书 |
| 适用阶段 | 审查起诉阶段 |
| 递交对象 | 人民检察院 |
| 参考来源 | 《刑事辩护规范化电子版》+《刑事辩护法律文书标准化文本》 |
| 核心规范 | 五段论结构、首部段范式、落款规范 |

## 文书结构

```markdown
                    辩 护 意 见

[检察院名称]：

    [首部段——说明委托情况、工作开展情况]

    辩护人总体认为：[概括性的辩护观点]

    具体辩护意见如下：

    一、关于案件事实
    （详细陈述辩护人认定的案件事实）

    二、关于证据问题
    （一）[证据问题1]
    （二）[证据问题2]
    ...

    三、关于法律适用
    （详细阐述法律分析意见）

    四、关于量刑情节
    （一）法定从轻、减轻情节
    （二）酌定从轻情节

    综上所述，辩护人认为[具体结论]，建议贵院[具体请求]。

    以上辩护意见，请予审查采纳。

                                        辩护人：广东广和（长春）律师事务所
                                                潘睿律师
                                              年    月    日
附：
1. 证据材料清单    1份
2. 相关法律规定    1份
```

## 首部段标准范式

```
[律师事务所名称]依法接受[委托人姓名]的委托，指派潘睿律师担任
涉嫌[罪名]一案犯罪嫌疑人[当事人姓名]审查起诉阶段的辩护人。辩护人接受
委托后，依法向贵院递交了委托手续，查阅、复制了本案全部案卷材料，多次
会见了犯罪嫌疑人，对本案事实和证据进行了认真研究分析。现根据事实和
法律，发表如下辩护意见，供贵院审查起诉时参考：
```

## 五段论结构说明

| 段落 | 名称 | 内容 |
|-----|-----|-----|
| 第一段 | 首部段 | 说明委托来源、工作情况 |
| 第二段 | 事实段 | 陈述案件事实及辩护人认定的事实 |
| 第三段 | 理由段 | 详细阐述辩护理由 |
| 第四段 | 总结段 | 概括总体意见（"辩护人总体认为"） |
| 第五段 | 结论段 | 提出具体请求（"综上所述"） |

## 关键表述格式

- "辩护人总体认为"——宋体四号加粗下划线
- "综上所述"——宋体四号加粗下划线
