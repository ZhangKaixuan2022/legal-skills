# 质证意见表模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 质证意见表 |
| 适用阶段 | 审查起诉阶段 |
| 使用对象 | 辩护律师内部使用，可选择性提交检察机关 |
| 参考来源 | 《刑事辩护规范化电子版》 |
| 核心规范 | 证据三性分析（真实性/合法性/关联性） |

## 文书说明

本表用于审查起诉阶段阅卷后对全案证据进行系统质证分析，可有选择性地将部分质证意见作为辩护意见书的附件提交检察机关。

## 证据种类说明

根据《中华人民共和国刑事诉讼法》第五十条规定的八种证据类型：

| 序号 | 证据类型 | 说明 |
|-----|---------|-----|
| 1 | 物证 | 能够证明案件真实情况的物品 |
| 2 | 书证 | 以文字、符号、图画等记载的内容证明案件事实 |
| 3 | 证人证言 | 证人就其所知道的案件情况向司法机关所作的陈述 |
| 4 | 被害人陈述 | 被害人就其受害情况和案件情况向司法机关所作的陈述 |
| 5 | 犯罪嫌疑人、被告人供述和辩解 | 犯罪嫌疑人、被告人就案件事实所作的陈述 |
| 6 | 鉴定意见 | 鉴定人对案件中的专门性问题所出具的意见 |
| 7 | 勘验、检查、辨认、侦查实验等笔录 | 办案人员对现场、物品、人身等进行勘验检查等活动的记录 |
| 8 | 视听资料、电子数据 | 以录音、录像、电子数据等形式证明案件事实的证据 |

## 质证三性说明

| 维度 | 审查要点 | 常见质证意见 |
|-----|---------|------------|
| 真实性 | 证据内容是否客观真实 | "对真实性有异议/无异议" |
| 合法性 | 取证程序是否合法合规 | "取证程序违法，应予排除" |
| 关联性 | 是否与待证事实相关联 | "与本案无关联性" |

### 真实性审查要点

| 审查项目 | 审查内容 |
|---------|---------|
| 证据来源 | 证据来源是否清楚、合法 |
| 证据形式 | 证据形式是否符合法定要求 |
| 证据内容 | 证据内容是否客观、完整 |
| 证据印证 | 证据之间是否相互印证 |
| 证据矛盾 | 是否存在与其他证据矛盾之处 |

### 合法性审查要点

| 审查项目 | 审查内容 |
|---------|---------|
| 取证主体 | 取证人员是否适格 |
| 取证程序 | 是否符合法定程序 |
| 取证方式 | 是否存在非法方式 |
| 形式要件 | 是否具备法定形式要件 |
| 保管链条 | 证据保管是否连续完整 |

### 关联性审查要点

| 审查项目 | 审查内容 |
|---------|---------|
| 待证事实 | 该证据拟证明什么事实 |
| 关联程度 | 与待证事实的关联程度 |
| 证明方向 | 是有利证据还是不利证据 |
| 证明力 | 对案件事实认定的影响程度 |

## 常见质证意见表述

### 对真实性的异议

| 情形 | 质证意见表述 |
|-----|------------|
| 证据来源不明 | "该证据来源不清，无法确认其真实性" |
| 证据内容虚假 | "该证据内容与客观事实不符，真实性存疑" |
| 证据相互矛盾 | "该证据与[其他证据]存在矛盾，真实性存疑" |
| 复印件问题 | "该证据系复印件，未能核对原件，真实性无法确认" |

### 对合法性的异议

| 情形 | 质证意见表述 |
|-----|------------|
| 非法取证 | "该证据系以非法方法取得，应当依法排除" |
| 程序瑕疵 | "该证据取证程序存在瑕疵，[具体瑕疵]" |
| 形式缺陷 | "该证据缺少[必备要素]，不符合法定形式要件" |
| 主体不适格 | "该证据取证主体不适格，不能作为定案依据" |

### 对关联性的异议

| 情形 | 质证意见表述 |
|-----|------------|
| 无关联性 | "该证据与本案待证事实无关联，不能作为定案依据" |
| 关联性弱 | "该证据与本案关联性较弱，证明力有限" |
| 证明方向相反 | "该证据恰恰证明了[有利于当事人的事实]" |

## 使用说明

1. 证据名称应对应审查起诉阶段案卷材料中所列证据名称。
2. 案卷页码应对应所复印卷宗的页码，便于核对。
3. 质证意见应针对所列证据材料，从证据的真实性、合法性、关联性三方面分别提出法律意见及理由和法律依据。
4. 本表可根据案件复杂程度增减栏目，如难以认定为法定八种证据类型之一，可标注为"其他"。
5. 建议结合阅卷笔录同步制作本表，确保对全案证据进行系统性审查。

## 提交策略

| 情形 | 建议 |
|-----|-----|
| 证据问题明显 | 选择性提交部分质证意见作为辩护意见附件 |
| 证据问题复杂 | 口头向检察官说明，视反馈决定是否书面提交 |
| 关键质证意见 | 可保留至审判阶段当庭提出 |
