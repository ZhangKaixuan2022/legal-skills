# 调查取证笔录模板

> 本文件为审查起诉阶段辩护Skill的法律文书模板子文件

## 文书说明

| 项目 | 内容 |
|-----|-----|
| 文书名称 | 辩护人询问笔录（调查取证笔录） |
| 适用阶段 | 审查起诉阶段 |
| 使用对象 | 向证人调查取证时使用 |
| 参考来源 | 《刑事辩护规范化电子版》 |
| 核心规范 | 权利义务告知+一问一答+每页签名捺印 |

## 调查人员要求

**必须两人以上**：调查人证应当由二名以上人员进行，可以是一名执业律师和一名律师助理。

## 笔录必备要素

| 要素 | 内容 |
|-----|-----|
| 时间 | 调查开始和结束时间 |
| 地点 | 调查进行的地点 |
| 调查人 | 调查律师姓名、执业证号 |
| 记录人 | 记录人员姓名 |
| 被调查人基本信息 | 姓名、出生日期、民族、职业、住址、身份证号 |
| 权利义务告知 | 如实作证义务、作伪证法律责任 |
| 调查内容 | 问答形式记录 |
| 确认语 | "以上笔录看过，和我说的一样，属实" |
| 签名捺印 | 被调查人在每页签名捺印 |

## 签名捺印规范

- 被调查人应在笔录最后签署确认语并签名捺印
- 被调查人应在每页末端签名捺印
- 涂改处应捺指印确认
- 无书写能力的被调查人可仅捺指印

## 无阅读能力证人的处理

- 进行同步录音录像
- 向其宣读笔录
- 注明"以上笔录向你宣读过，是否与你所说一致？""听清了，与我说的一致"
- 由被调查人捺指印确认

## 使用说明

1. 询问证人建议由两名以上律师进行，一人询问一人记录。
2. 向被害人或被害人提供的证人取证，须经人民检察院许可并取得《准许律师调查书》，同时须经被害人或其近亲属、证人本人同意。
3. 做到边调查、边记录，一次完成，由证人签字确认，不可事后加工整理。
4. 笔录开头要告知自己的工作单位，并出示本人的有关证件。此外，还要告知其权利义务，强调须如实回答，不得隐瞒事实和证据。
5. 辩护人在询问过程中切记勿以诱导、欺骗的方式引诱证人作证、指证，否则可能涉嫌辩护人妨害作证罪。
6. 确认笔录无误后，被询问人应在每页底部签名捺手印确认。
7. 强烈建议在条件许可的情况下对询问过程进行全程录音录像，以降低辩护人执业风险。
8. 本笔录一式三份，一份附卷，一份交被询问人，一份留存。

## 禁止行为清单（七条红线）

| 序号 | 禁止行为 | 法律后果 |
|-----|---------|---------|
| 1 | 伪造、毁灭证据 | 《刑法》第306条辩护人伪造证据罪 |
| 2 | 威胁、引诱证人违背事实改变证言 | 《刑法》第306条辩护人妨害作证罪 |
| 3 | 帮助当事人隐匿、毁灭、伪造证据 | 《刑法》第306条 |
| 4 | 帮助当事人串供 | 《刑法》第306条 |
| 5 | 未经许可向被害方调查取证 | 行政处罚，影响证据效力 |
| 6 | 引导证人按预设内容作证 | 证据无效，涉嫌违法 |
| 7 | 让当事人家属参与调查 | 证据可能被污染，涉嫌违法 |

## 人证分离原则

调查取证离不开家属配合寻找证人，但必须做到"人证分离"：
- "人"指家属：只告诉家属要找什么人
- "证"指证人：不告诉家属调查什么内容
- 调查后要求证人保密，不得与家属谈论被调查事项

## 问题设计原则

**设计原则**：
1. 转化为开放式提问
2. 按时间线排列
3. 每个问题对应一项待核实证据
4. 避免诱导性提问

**问题设计示例**：

```
❌ 错误示范：张三说你当时在现场，对不对？
✅ 正确示范：请描述一下案发当天你的行程。

❌ 错误示范：证人说你拿了那把刀，是这样吗？
✅ 正确示范：你当时手里有没有东西？如果有，是什么？
```
