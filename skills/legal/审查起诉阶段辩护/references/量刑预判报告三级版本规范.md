# 量刑预判报告三级版本规范

## 一、核心原则

审查起诉阶段量刑预判**不沿用**侦查阶段报告，需基于完整案卷**重新制作**。

## 二、三级文档架构

| 级别 | 文档名称 | 保密等级 | 使用对象 | 内容边界 |
|-----|---------|---------|---------|---------|
| 第一级 | 内部底稿 | ★★★最高保密 | 仅限承办律师 | 全部内容 |
| 第二级 | 会见材料 | ★★☆高保密 | 当事人会见使用 | 部分限制 |
| 第三级 | 反馈函 | ★☆☆中保密 | 委托人/家属 | 严格限制 |

---

## 三、第一级：内部底稿

### 可包含内容
- 完整证据分析
- 详细辩护策略
- 量刑计算底稿
- 风险评估
- 预判多种可能结果

### 制作要点
- 详尽记录、充分论证
- 作为辩护工作的核心参考
- 严格保密，不得外传

---

## 四、第二级：会见材料

### 可包含内容
- 案件概况
- 指控事实分析
- 法定刑幅度说明
- 量刑情节分析
- 量刑预判结论
- 辩护方向建议

### 禁止内容
- 同案犯供述具体内容
- 证人证言原文
- 可能导致串供的信息

### 使用规范
- 会见时口头说明，不留存于看守所
- 可让当事人阅读，但不复印给当事人

---

## 五、第三级：家属反馈函

### 可包含内容
1. 案件程序进展（目前阶段、下一步程序）
2. 法定刑幅度一般性说明
3. 可能的程序走向（起诉/不起诉）
4. 律师工作概述

### 严格禁止内容
- 任何证据内容
- 辩护策略细节
- 具体量刑预判数字
- 同案犯情况
- 证人信息
- 检察官态度或倾向

### 表述规范

| 不当表述 | 正确表述 |
|---------|---------|
| "可能判3-5年" | "案件正在审查起诉阶段" |
| "证据对我们有利" | "律师正在积极开展辩护工作" |
| "检察官可能不起诉" | "案件结果有待检察机关依法作出决定" |

---

## 六、量刑预判五步递进法

| 步骤 | 内容 | 具体操作 |
|-----|-----|---------|
| 第一步 | 确定指控罪名与法定刑幅度 | 查阅刑法条文，确定量刑区间 |
| 第二步 | 分析犯罪事实与情节 | 审查证据，认定事实 |
| 第三步 | 识别量刑情节 | 参照量刑情节体系，逐项核对 |
| 第四步 | 计算量刑基准与调整 | 依据量刑规范化文件计算 |
| 第五步 | 形成量刑预判结论 | 综合分析，得出预判区间 |

---

## 七、证据链分析表

| 指控事实 | 控方证据 | 证明力评级 | 矛盾/漏洞 | 辩方证据 | 质证意见 |
|---------|---------|-----------|----------|---------|---------|
| | | | | | |

**证明力评级标准**：
- ★★★ 足以认定
- ★★☆ 有瑕疵
- ★☆☆ 证据不足
- ☆☆☆ 难以认定

---

## 八、保密合规检查

### 六类敏感内容检测

| 类型 | 检测关键词 | 风险等级 |
|-----|-----------|---------|
| 同案犯供述 | "XX（同案犯名字）供述"、"同案人称" | 高 |
| 证人证言 | "证人XX称"、"XX证实" | 高 |
| 辩护策略 | "辩护重点"、"质证意见" | 中 |
| 量刑预判 | "可能判处"、"预计刑期" | 中 |
| 检察官态度 | "检察官表示"、"检察院倾向" | 中 |
| 案件证据 | "根据案卷"、"证据显示" | 高 |

### 检查流程
1. 确定文档级别
2. 对照禁止内容清单检查
3. 删除或修改敏感内容
4. 律师审核确认
5. 存档备查
