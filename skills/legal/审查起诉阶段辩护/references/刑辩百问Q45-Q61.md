# 刑辩百问Q45-Q61

> 本文件为审查起诉阶段辩护Skill的参考材料子文件，涵盖阅卷、会见、核证、取证、非法证据排除、与检察机关沟通、认罪认罚等17个核心问题。

---

## Q45. 审查起诉阶段哪些证据需要打印？

**答**：审查起诉阶段律师阅卷后，应当根据案件需要打印以下证据：

1. **犯罪嫌疑人供述与辩解**——这是会见核实的基础材料，需要让当事人确认笔录内容是否与其陈述一致。

2. **关键物证照片**——涉及辨认的物证照片，需要在会见时让当事人辨认确认。

3. **涉及签名的书证**——如合同、收据等，需要当事人确认签名真伪。

4. **鉴定意见**——特别是对案件定性有重要影响的鉴定意见，需要向当事人解释并确认。

5. **辨认笔录照片**——需要核实辨认程序是否合法、辨认结果是否准确。

**特别提醒**：同案犯供述、证人证言等言词证据不宜直接打印给当事人看，应当采用背对背核证的方式口头核实。

## Q46. 审查起诉阶段律师如何进行阅卷？

**答**：审查起诉阶段阅卷是辩护工作的基础，应当注意以下几点：

1. **全面复制**：原则上应当复制全部案卷材料，至少要复制所有言词证据和关键书证、物证照片。

2. **系统整理**：按照证据类型或犯罪事实分类整理，建立证据目录索引。

3. **重点标注**：对有利证据、不利证据、存疑证据分别标注，便于后续分析。

4. **制作阅卷笔录**：按照十项核心内容制作详细的阅卷笔录。

5. **发现问题**：重点关注证据之间的矛盾、证据链的漏洞、取证程序的瑕疵。

## Q47. 审查起诉阶段如何向犯罪嫌疑人核实证据？

**答**：向犯罪嫌疑人核实证据应当遵循以下原则：

1. **本人笔录可直接核实**：犯罪嫌疑人本人的供述与辩解，可以直接让其阅读核对。

2. **客观性证据可展示**：书证、物证照片、鉴定意见等客观性证据可以直接展示给犯罪嫌疑人。

3. **言词证据背对背核实**：同案犯供述、证人证言、被害人陈述等言词证据，不宜直接展示，应当通过口头询问的方式进行背对背核证。

4. **不得引导供述**：核实过程中不得引导犯罪嫌疑人按照他人证言作出陈述。

## Q48. 审查起诉阶段会见应当注意什么？

**答**：审查起诉阶段会见应当注意：

1. **会见目的明确**：每次会见都要有明确的目的和工作内容。

2. **证据核实为重点**：重点核实案卷证据与当事人陈述是否一致。

3. **防范翻供风险**：不轻易出示全部案卷，采用背对背核证方式。

4. **记录完整规范**：制作详细的会见笔录，由当事人签字确认。

5. **保密义务**：不得向当事人透露不应透露的信息。

## Q49. 如何指导当事人正确表达认罪态度？

**答**：认罪态度对量刑有重要影响，律师应当：

1. **解释认罪含义**：向当事人解释认罪是对犯罪事实的承认，与对法律定性的看法可以分开。

2. **分析利弊**：向当事人分析认罪与不认罪的利弊得失。

3. **尊重意愿**：最终是否认罪由当事人自己决定，律师不得强迫。

4. **指导表述**：如当事人决定认罪，指导其如何恰当表述认罪态度。

5. **防止虚假认罪**：如当事人确实无罪，不得建议其认罪。

## Q50. 辩护律师调查取证的证据何时提交？

**答**：辩护律师取得的证据提交时机应当区分情况：

**必须立即提交的证据**（《刑事诉讼法》第40条）：
- 不在场证明
- 未达刑事责任年龄证明
- 属于不负刑事责任的精神病人证明

**可策略性选择提交时机的证据**：
- 证明从轻、减轻情节的证据
- 质证用证据

一般建议在审查起诉阶段提交有力证据，争取不起诉或从轻起诉；部分质证证据可保留至审判阶段使用。

## Q51. 审查起诉阶段律师调查取证应当注意什么？

**答**：审查起诉阶段调查取证应当注意：

1. **两人以上调查**：调查人证应当由两人以上进行。

2. **许可要求**：向被害人或其近亲属、被害人提供的证人取证，必须经检察院许可。

3. **规范制作笔录**：调查笔录要素齐全，签名捺印规范。

4. **人证分离**：不让家属知道调查内容，防止污染证据。

5. **禁止引导**：不得引导证人按预设内容作证。

## Q52. 如何发现和证明指供、诱供？

**答**：发现指供、诱供的线索：

1. **笔录比对**：多份笔录用语高度一致，特别是使用相同的特殊用语。

2. **超出认知**：笔录包含超出当事人认知范围的细节。

3. **表达异常**：使用与当事人文化程度不符的表达方式。

4. **突然变化**：某次讯问后供述发生重大改变。

**收集线索方法**：
- 详细询问当事人被讯问过程
- 比对不同时间的笔录内容
- 申请调取讯问录音录像

## Q53. 如何处理刑讯逼供后的重复自白？

**答**：重复自白的处理应当从两个角度分析：

**合法性角度**：
- 同一侦查人员后续讯问取得的供述，原则上应当排除
- 更换侦查人员或检察机关讯问的，需审查是否消除了刑讯影响

**客观性角度**：
- 审查后续供述与刑讯供述的内容是否完全一致
- 审查后续供述是否有新增细节或合理变化
- 论证后续供述系刑讯影响的延续

## Q54. 申请非法证据排除需要准备什么材料？

**答**：申请非法证据排除应当准备以下材料：

1. **书面申请书**：载明申请排除的证据名称、理由、法律依据。

2. **相关线索和材料**：
   - 涉嫌非法取证的人员（姓名或体貌特征）
   - 涉嫌非法取证的时间
   - 涉嫌非法取证的地点
   - 涉嫌非法取证的方式
   - 非法取证造成的后果

3. **证据材料**（如有）：
   - 当事人关于刑讯逼供的陈述笔录
   - 伤情照片、医疗记录
   - 同监室人员证言线索

## Q55. 审查起诉阶段如何与检察机关沟通？

**答**：与检察机关沟通应当遵循以下原则：

1. **不搞证据突袭**：不将辩护意见藏到庭审时才提出，审查起诉阶段就应当充分沟通。

2. **充分利用检察监督权**：发现侦查违法的，请求检察院监督。

3. **一路审查一路提**：发现问题及时提出，不要等到最后一刻。

4. **常态化沟通**：保持与检察官的经常性沟通，不限于提交意见时。

**沟通方式**：口头沟通与书面意见相结合，成熟意见书面提交，探索性意见口头探讨。

---

## Q56. 辩护意见应当如何提交？

**答**：辩护意见的提交应当注意：

1. **分类提交**：
   - 成熟意见：书面提交+当面说明
   - 探索性意见：口头探讨
   - 敏感意见：慎重提交，确保有充分依据

2. **防止成为补充侦查提纲**：
   - 指出问题但不提供解决方案
   - 强调无法补正的客观性问题
   - 部分关键意见可保留至审判阶段

3. **把握时机**：
   - 无罪意见尽早提出
   - 量刑意见可在退查后集中提出

## Q57. 如何申请检察机关调查取证？

**答**：申请检察机关调查取证的步骤：

1. **提交书面申请**：载明申请调取的证据、理由、与案件的关联性。

2. **说明无法自行调取的原因**：如涉及被害方证人、需要检察权保障等。

3. **跟进处理结果**：检察院应当在一定期限内决定是否调取。

4. **申请在场**：检察院决定调取的，律师可以申请在场。

**特别注意**：向被害人或其近亲属、被害人提供的证人取证，必须先申请检察院许可。

---

## Q58. 申请调取讯问录音录像被拒绝怎么办？

**答**：申请调取录音录像被拒绝时的应对：

1. **书面申请**：首先提交书面申请，要求调取特定时段的录音录像。

2. **要求书面答复**：要求检察院对是否调取作出书面答复。

3. **申请检察院审查**：如不提供复制件，申请检察院审查录像内容。

4. **记录作为证据**：将申请和答复情况记录在案，作为非法证据排除的线索。

5. **庭审申请**：在审判阶段继续申请法庭调取。

## Q59. 如何区分威胁、引诱、欺骗与侦查谋略？

**答**：区分的核心标准是"双重标准"：

**意志自由标准**：是否迫使犯罪嫌疑人违背意愿供述
**虚假供述标准**：是否可能导致虚假供述

**具体分析**：
- 告知坦白从宽政策——合法（据实告知）
- 威胁抓捕家属——非法（以不实威胁迫使供述）
- 许诺不追究刑责——非法（超越权限的虚假承诺）
- 虚构同案犯已供述——存疑（需具体分析是否导致虚假供述）

## Q60. 审查起诉阶段如何争取不起诉？

**答**：争取不起诉应当区分类型制定策略：

1. **法定不起诉**：论证不构成犯罪，如行为不符合犯罪构成、存在违法阻却事由等。

2. **存疑不起诉**：指出证据不足之处，论证证据之间的矛盾，提出合理怀疑。

3. **相对不起诉**：
   - 论证犯罪情节轻微
   - 促成刑事和解、取得被害人谅解
   - 积极退赃退赔
   - 强调初犯偶犯、认罪认罚等情节

## Q61. 认罪认罚从宽制度的适用条件是什么？

**答**：认罪认罚从宽制度的适用需要满足以下条件：

1. **认罪**：犯罪嫌疑人自愿如实供述自己的罪行，对指控的犯罪事实没有异议。

2. **认罚**：犯罪嫌疑人真诚悔罪，愿意接受处罚。

3. **自愿性**：认罪认罚必须出于犯罪嫌疑人本人真实意愿，不是被迫或受欺骗。

4. **程序保障**：犯罪嫌疑人获得有效法律帮助，了解认罪认罚的法律后果。

**不宜适用的情形**：
- 犯罪嫌疑人可能不构成犯罪
- 事实不清证据不足
- 犯罪嫌疑人对定性有重大异议
- 犯罪嫌疑人不理解法律后果
