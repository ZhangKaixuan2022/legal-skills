# 调查取证操作规范

## 一、调查取证权利概述

| 条款 | 内容 |
|-----|-----|
| 《刑诉法》第43条 | 辩护律师经证人同意可向其收集材料 |
| 《刑诉法》第43条 | 辩护律师可申请检察院收集、调取证据 |
| 《刑诉法》第43条 | 经检察院许可可向被害方收集材料 |

**两种途径**：
- **申请检察院调取（优先）**：风险较低，证据效力强
- **辩护律师自行调取（谨慎）**：风险较高，需严格规范

## 二、五步子流程详解

### 第一步：取证方案制订

| 工作要点 | 具体内容 |
|---------|---------|
| 确定取证目标 | 明确需要收集什么证据、证明什么事实 |
| 判断证据类型 | 人证/物证/书证/电子数据/视听资料 |
| 许可判断 | 被害方取证需检察院许可，证人证言需证人同意 |
| 风险评估 | 评估调查取证的执业风险 |
| 制订调查提纲 | 调查问题清单、调查顺序安排 |

### 第二步：人证调查规范

| 规范要点 | 具体要求 |
|---------|---------|
| 人员配置 | **两名以上律师同时在场** |
| 告知义务 | 告知证人权利义务、作伪证的法律后果 |
| 笔录制作 | 一问一答格式、如实记录 |
| 签名捺印 | 证人逐页签名、修改处捺印 |
| 录音录像 | 条件允许时全程录音录像 |

**人证分离原则**：家属只负责找人，不告诉调查内容

### 第三步：物证书证调查规范

| 证据类型 | 调查要点 |
|---------|---------|
| 物证 | 制作提取清单、拍照固定、说明来源 |
| 书证 | 复制原件、注明出处、加盖原件存放处印章 |
| 视听资料 | 复制原件、记录存储介质信息 |

### 第四步：电子证据调查规范

| 取证方式 | 记录要素 |
|---------|---------|
| 复制 | 原始载体信息、复制时间、复制人 |
| 打印 | 打印时间、网址/路径、打印人 |
| 截屏 | 截屏时间、完整界面、URL地址 |
| 拍照 | 拍照时间、设备信息、显示内容 |
| 录像 | 录制时间、操作过程、完整画面 |

### 第五步：证据提交策略

| 证据类型 | 提交策略 | 说明 |
|---------|---------|-----|
| 不在场证明 | **必须立即提交** | 参见刑辩百问Q50 |
| 未达刑事责任年龄证明 | **必须立即提交** | 发现后必须及时告知检察院 |
| 不负刑事责任的精神病人证明 | **必须立即提交** | 发现后必须及时告知检察院 |
| 其他有利证据 | 根据辩护策略决定提交时机 | 可在庭审中适时提出 |

## 三、禁止行为清单（七条红线）

| 序号 | 禁止行为 | 法律风险 |
|-----|---------|---------|
| 1 | 伪造证据 | 刑法306条 |
| 2 | 威胁引诱证人 | 刑法306条 |
| 3 | 帮助当事人隐匿毁灭伪造证据 | 刑法306条 |
| 4 | 帮助串供 | 刑法306条 |
| 5 | 未经许可向被害方取证 | 程序违法 |
| 6 | 引导证人按预设内容作证 | 涉嫌教唆伪证 |
| 7 | 让家属参与调查 | 程序不规范 |

## 四、申请检察院调查取证

### 申请书要素

```
一、申请事项
- 申请调取的证据名称和内容

二、申请理由
- 该证据对案件的证明作用
- 辩护人无法自行调取的原因
- 检察机关调取的必要性

三、证据线索
- 证据存放地点
- 保管人员或单位
- 调取方式建议
```

### 常见申请情形
- 申请调取监控录像
- 申请调取通话记录
- 申请调取银行流水
- 申请重新鉴定
- 申请调取证人证言

## 五、取证风险防控

### 高风险情形识别

| 风险类型 | 具体表现 | 防控措施 |
|---------|---------|---------|
| 证人变卦 | 证人翻证、否认笔录 | 全程录音录像、两人在场 |
| 被诬陷 | 被指控威胁引诱 | 规范取证、留存证据 |
| 证据瑕疵 | 取证程序不规范 | 严格按规范操作 |

### 防控措施
1. 严格遵守两人同时在场原则
2. 条件允许时全程录音录像
3. 笔录制作规范、签字完整
4. 保留取证过程全部证据
5. 不单独与证人接触
