# 庭后沟通与法官交流规范

## 一、与当事人沟通要点

### 1.1 沟通要点结构

```
与当事人沟通要点

一、庭审整体情况概述
   - 开庭时间、地点
   - 庭审流程完成情况
   - 整体氛围

二、对方的主要观点和态度
   - 对方核心主张
   - 对方提交的新证据（如有）
   - 对方态度分析

三、法官关注的重点问题
   - 法官发问的主要方向
   - 法官对证据的关注点
   - 争议焦点归纳情况

四、初步判断（审慎表达）
   - 有利因素
   - 不利因素
   - 风险提示：不作结果承诺

五、后续需要当事人配合的事项
   - 需补充的材料
   - 需核实的问题
   - 其他配合事项

六、下一步工作安排
   - 提交代理词
   - 补充材料提交
   - 案件跟进计划
```

### 1.2 沟通方式建议

| 案件复杂程度 | 建议方式 | 说明 |
|-------------|---------|------|
| 复杂案件 | 约见面谈 | 结合书面材料详细说明 |
| 简单案件 | 电话沟通 | 简要汇报庭审情况 |
| 客户要求 | 书面汇报 | 采用复盘报告形式 |

### 1.3 注意事项

- **不作结果承诺**：明确告知案件结果存在不确定性
- **客观分析**：既不过分乐观也不过分悲观
- **及时回应**：回应当事人的关切和疑问
- **保持专业**：保持良好的沟通态度

## 二、与主审法官沟通

### 2.1 沟通方式提示

| 沟通事项 | 建议方式 | 说明 |
|---------|---------|------|
| 询问案件进度 | 电话 | 简单快捷 |
| 补充材料提交 | 当面 | 重要材料当面递交 |
| 重要观点补充 | 当面 | 争取面谈机会 |
| 程序问题确认 | 电话/当面 | 根据复杂程度选择 |

### 2.2 沟通时机建议

| 阶段 | 建议时间 | 目的 |
|-----|---------|------|
| 庭后初期 | 庭审后1-2周 | 提交代理词、补充材料 |
| 审理中期 | 根据进展 | 了解进度、回复询问 |
| 判决前 | 适当跟进 | 了解裁判进度（注意分寸） |

### 2.3 沟通内容记录模板

**模板路径**：`templates/法官沟通记录模板.md`

**记录要素**：
- 沟通时间、方式
- 沟通事由
- 沟通内容
- 法官反馈
- 后续事项

### 2.4 沟通技巧提示

**基本原则**：
1. **简明扼要**：提前准备沟通要点
2. **尊重时间**：避开法官繁忙时段
3. **有理有据**：内容有事实和法律依据支撑
4. **不卑不亢**：保持专业态度
5. **适当善意**：表达对法官工作的理解

**禁忌事项**：
- 不得向法官明示或暗示案件结束后有好处
- 不得通过不当途径打听案件内部讨论情况
- 不得对法官表示抱怨或不满情绪
- 不得频繁打扰法官

### 2.5 法官沟通注意事项

1. **分寸把握**：有理有据有节，避免施压
2. **记录完整**：每次沟通后及时记录
3. **频率控制**：合理控制沟通频率，避免频繁打扰
4. **合规底线**：严守职业道德，不越红线

---

## 三、补充材料注意事项

1. **期限意识**：严格遵守法官指定的期限
2. **回执留存**：提交时取得书面回执
3. **逾期风险**：充分评估逾期举证的法律风险
4. **当事人确认**：涉及事实的问题由当事人签字确认
