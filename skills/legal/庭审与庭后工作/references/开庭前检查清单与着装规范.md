# 开庭前检查清单与着装规范

## 一、开庭材料清单核查提醒

**必备材料清单**（请逐项核对）：

| 序号 | 材料名称 | 检查要点 | 状态 |
|-----|---------|---------|------|
| 1 | 起诉状/答辩状副本 | 份数是否充足（按当事人人数+法院1份） | □ |
| 2 | 证据原件 | 逐一核对证据目录，确保原件齐全 | □ |
| 3 | 授权委托书原件 | 委托权限是否明确 | □ |
| 4 | 律师执业证 | 原件 | □ |
| 5 | 律所出庭函/介绍信 | 日期、案号是否正确 | □ |
| 6 | 当事人身份证/营业执照副本 | 原件或经核对的复印件 | □ |
| 7 | 法院传票 | 确认开庭时间、地点 | □ |
| 8 | 庭审提纲/代理意见 | 纸质版备用 | □ |
| 9 | 证据目录 | 与提交法院的一致 | □ |
| 10 | 法律法规汇编 | 关键法条打印备查 | □ |
| 11 | 案例检索报告 | 重要案例打印备查 | □ |
| 12 | 可视化图表 | 时间轴、法律关系图等 | □ |
| 13 | 备用文具 | 笔、笔记本、荧光笔 | □ |

**重要提示**：
- 建议提前一天将材料装入公文包
- 出发前再次核对传票上的开庭时间和地点
- 预留充足路途时间，考虑交通拥堵因素

## 二、着装/仪态提醒

### 2.1 着装要求

- [ ] 穿着律师袍或正装（深色西装套装）
- [ ] 内着浅色衬衣
- [ ] 佩带领带（男律师）
- [ ] 携带律师徽章
- [ ] 下着深色皮鞋
- [ ] 女律师可着深色西装套裙

### 2.2 仪态要求

- [ ] 提前30分钟到达法庭
- [ ] 手机调至静音或关机
- [ ] 保持专业、庄重的仪态

### 2.3 特别提示

- 大部分法院对民事案件没有强制要求穿律师袍，建议开庭前与书记员核实
- 如未要求律师袍，应着正装、系领带
- 不穿律师袍可能被当庭警告或训诫

---

## 三、庭审注意事项提示

### 3.1 称呼规范

- 独任审理："审判长"
- 合议庭审理："审判长、审判员"
- 避免使用"法官"等不规范称呼

### 3.2 发言规范

- [ ] 站立发言（如法庭要求）
- [ ] 语速适中，兼顾书记员记录速度
- [ ] 条理清晰，层次分明
- [ ] 使用法言法语，简洁生动
- [ ] 与法官保持眼神交流

### 3.3 禁止事项

- [ ] 不打断对方发言
- [ ] 不与对方当事人直接对话
- [ ] 不进行人身攻击
- [ ] 不情绪激动大声喧哗
- [ ] 法官阻止发言时不与法官争执

### 3.4 庭审纪律提醒

- 遵从审判长指挥，不得擅自发言
- 珍惜庭上每个发言机会
- 确保法官认真倾听时再表达重要观点
- 当事人回答问题时不要擅自阻止

---

## 四、开庭前检查注意事项

1. **材料完整性**：务必逐项核对，避免遗漏
2. **时间预留**：预留充足路途时间，避免迟到
3. **着装规范**：提前确认是否需要穿律师袍
4. **技术准备**：如需播放视听资料，提前测试设备
