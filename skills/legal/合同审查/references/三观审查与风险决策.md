# 三观审查与风险决策

> 本文件为合同审查Skill提供宏观、中观分析与风险决策的方法论指引，基于张海燕律师《合同审查思维体系与实务技能》方法论整理。

---

## 一、宏观分析：交易结构审查

### 1.1 审查目标

宏观层面审查是合同审查的第一步，目标是理解交易的商业逻辑和法律本质，判断交易结构是否合理、合法。

### 1.2 八大基本交易类型

| 交易类型 | 典型特征 | 审查关注点 |
|---------|---------|-----------|
| **转让所有权** | 一方将财产所有权永久转移给另一方 | 权属清晰、转让限制、对价合理 |
| **让渡使用权** | 保留所有权，仅转移使用权 | 使用范围、期限、返还条件 |
| **提供劳务成果** | 以完成特定工作并交付成果为标的 | 成果标准、验收程序、知识产权归属 |
| **提供服务过程** | 以提供劳务过程为标的 | 服务标准、过程监督、责任承担 |
| **委托代办** | 以处理事务为标的 | 授权范围、转委托、费用承担 |
| **合作经营** | 共同投入、共享收益、共担风险 | 出资方式、利润分配、退出机制 |
| **融资担保** | 提供资金或信用支持 | 利率合规、担保效力、风险控制 |
| **身份关系** | 涉及人身关系的安排 | 主体资格、法定条件、强制性规定 |

### 1.3 合同名实一致性检查

**审查要点**：合同名称与实际法律关系是否一致。

| 常见名实不符情形 | 实际法律关系 | 法律后果 |
|----------------|-------------|---------|
| 名为"借款"实为"投资" | 股权投资 | 可能被认定为股东，承担经营风险 |
| 名为"租赁"实为"买卖" | 分期付款买卖 | 适用买卖合同规则，所有权转移时点不同 |
| 名为"合作"实为"借贷" | 民间借贷 | 适用借贷利率限制 |
| 名为"顾问费"实为"回扣" | 商业贿赂 | 合同无效，可能承担刑事责任 |
| 名为"劳务"实为"劳动" | 劳动关系 | 适用《劳动合同法》，用人单位承担法定责任 |
| 名为"委托代理"实为"居间" | 居间合同 | 报酬支付条件不同 |

### 1.4 交易结构优化评估

审查现有交易结构是否存在更优方案：

1. **税务角度**：是否存在税负更低的交易结构
2. **风险角度**：责任承担范围是否可优化
3. **效率角度**：审批流程、办理成本是否可简化
4. **合规角度**：是否符合监管要求和行业规范

### 2.2 语义场景识别

当关键词不明确时，通过语义场景判断合同类型：

| 模糊场景描述 | 类型映射 | 判断依据 |
|-------------|---------|---------|
| "找人帮忙做个网站" | 承揽/委托 | 是否有具体成果物 |
| "长期合作供货" | 买卖（框架协议） | 所有权转移 |
| "使用对方场地经营" | 租赁/承包经营 | 是否使用对方名义经营 |
| "共同出资做项目" | 合伙/合作开发 | 是否共担风险 |
| "帮忙找客户" | 中介/代理 | 是否以对方名义签约 |

### 2.3 易混淆类型辨析表

#### 承揽合同 vs 委托合同

| 辨析要点 | 承揽合同 | 委托合同 |
|---------|---------|---------|
| **标的** | 完成特定工作并交付成果 | 处理事务 |
| **风险承担** | 承揽人自担工作风险 | 委托人承担后果 |
| **任意解除** | 定作人享有（《民法典》第787条） | 双方均享有（《民法典》第933条） |
| **典型情形** | 加工、定作、修理、复制 | 代理签约、代办手续 |

#### 买卖合同 vs 承揽合同

| 辨析要点 | 买卖合同 | 承揽合同 |
|---------|---------|---------|
| **材料提供** | 卖方提供标准产品 | 定作方提供材料或特定要求 |
| **标的特定性** | 种类物为主 | 特定物/定制品 |
| **风险转移** | 交付时转移 | 验收合格时转移 |
| **典型情形** | 采购标准设备 | 定制非标设备 |

#### 劳动关系 vs 劳务关系

| 辨析要点 | 劳动关系 | 劳务关系 |
|---------|---------|---------|
| **法律适用** | 《劳动合同法》 | 《民法典》合同编 |
| **管理关系** | 存在管理与被管理关系 | 相对独立 |
| **薪酬方式** | 按月固定发薪 | 按约定结算 |
| **社保责任** | 雇主承担法定责任 | 风险分配更灵活 |
| **工伤责任** | 工伤保险基金赔付 | 提供劳务方自担或按过错分担 |

#### 租赁合同 vs 承包经营合同

| 辨析要点 | 租赁 | 承包经营 |
|---------|-----|---------|
| **核心标的** | 使用他人财产 | 经营权的转移 |
| **对外名义** | 以承租人自己名义 | 以被承包方名义 |
| **对外责任** | 承租人自担 | 可能涉及被承包方连带责任 |
| **经营风险** | 承租人自担 | 承包人自担 |

#### 特许经营 vs 代理经销

| 辨析要点 | 特许经营 | 代理经销 |
|---------|---------|---------|
| **品牌使用** | 使用特许人品牌 | 仅销售产品 |
| **经营模式** | 使用完整商业模式 | 不使用商业模式 |
| **监管要求** | 需商务部备案 | 无备案要求 |
| **法律依据** | 《商业特许经营管理条例》 | 一般合同规则 |

#### 融资租赁 vs 分期付款买卖

| 辨析要点 | 融资租赁 | 分期付款买卖 |
|---------|---------|-------------|
| **三方关系** | 出租人、承租人、供应商 | 买卖双方 |
| **所有权** | 租赁期间归出租人 | 视约定（《民法典》第641条） |
| **监管要求** | 融资租赁公司需持牌 | 无特殊要求 |
| **税务处理** | 进项抵扣方式不同 | 视具体约定 |

## 三、中观分析：合同结构完整性审查

### 3.1 十项必备条款清单

| 序号 | 条款类型 | 检查标准 | 重要程度 |
|-----|---------|---------|---------|
| 1 | **主体条款** | 当事人信息完整准确、签约资格确认 | 🔴高 |
| 2 | **标的条款** | 交易对象具体描述、品名规格数量明确 | 🔴高 |
| 3 | **质量条款** | 质量标准明确、检验方式可操作 | 🟡中 |
| 4 | **价款条款** | 金额明确、计算方式清楚、税费承担；涉及收付款的合同必须明确收款信息 | 🔴高 |
| 5 | **履行条款** | 时间、地点、方式、费用承担明确 | 🟡中 |
| 6 | **验收条款** | 验收标准、验收程序、异议期限 | 🟡中 |
| 7 | **违约条款** | 违约情形、违约责任、救济措施 | 🟡中 |
| 8 | **解除终止** | 解除条件、终止后果、清算安排 | 🟡中 |
| 9 | **争议解决** | 管辖法院或仲裁机构明确 | 🟡中 |
| 10 | **签署生效** | 生效条件、份数、附件效力 | 🟢低 |

### 3.2 结构逻辑审查

**审查维度**：

1. **条款排列顺序**：是否符合交易逻辑或重要性顺序
2. **前后矛盾检查**：不同条款之间是否存在冲突
3. **重复约定检查**：同一事项是否在多处重复（可能矛盾）
4. **附件一致性**：正文与附件表述是否一致

## 四、风险决策框架

### 4.1 决策导向原则

律师在提供合同审查服务时，需要帮助客户进行决策，而非简单罗列风险。

**核心理念**：
- 不能只告诉客户"这个事情有风险，请你自己判断"
- 需要在不确定性下进行分析、决策
- 提供具有可操作性的法律意见

### 4.2 成本收益分析法

| 分析维度 | 审查问题 | 决策支持 |
|---------|---------|---------|
| **风险损失** | 最坏情况下的损失是什么？ | 量化最大可能损失 |
| **发生概率** | 风险发生的概率有多高？ | 结合交易背景判断 |
| **交易收益** | 本次交易的预期收益是什么？ | 对比风险与收益 |
| **替代方案** | 是否有风险更低的替代方案？ | 提供替代建议 |

### 4.3 可操作性平衡原则

**便利性 vs 风险控制的权衡**：

1. **高风险低收益**：建议放弃或重大修改
2. **高风险高收益**：建议增加担保措施、设置风险隔离
3. **低风险高收益**：可适度接受
4. **低风险低收益**：按常规处理

### 4.4 审查意见表达原则

| 意见类型 | 适用情形 | 表达方式 |
|---------|---------|---------|
| **必须修改** | 存在重大法律风险 | 明确指出风险、给出修改方案 |
| **建议修改** | 存在一般风险或可争取利益 | 说明利弊、提供建议 |
| **提示注意** | 存在不确定性或潜在风险 | 提示关注、不代为决策 |
| **商务事项** | 非法律问题 | 提请客户自行决定 |
