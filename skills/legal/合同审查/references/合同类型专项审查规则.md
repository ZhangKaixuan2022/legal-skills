# 合同类型专项审查规则

本文件是 `合同审查` Skill 的类型化审查层。结构为“原 22 类合同审查卡 + 新增交叉专项卡”。使用顺序：先完成通用读取复查和合同概览，再识别主合同类型，最后加载对应审查卡；混合合同时同时适用多个审查卡。引用具体法条、司法解释或监管规则前，必须按总控规则检索核实现行有效状态。

## 一、通用使用规则

每份合同至少完成四步：

1. 识别交易结构：谁付款、谁交付、交付什么、如何验收、违约后如何救济。
2. 识别合同类型：先匹配原 22 类主类型，再判断是否叠加新增交叉专项卡。
3. 检查必读材料：正文、附件、订单、报价单、补充协议、授权文件、往来确认文件。
4. 输出问题时使用 `contract-workflow-protocol.md` 的问题格式，必须写明法律风险、业务风险、建议修改文本、对方不同意时的谈判落点。

重大风险通常包括：

- 合同主体不存在、无权签约或资质明显不适格
- 标的、价款、履行期限、验收标准缺失导致合同难以履行
- 管辖、仲裁、解除、违约责任等核心条款无效或不可执行
- 付款、交付、权利转移、风险转移安排明显不利于我方且无法补救
- 自动续约、排他、责任限制、赔偿、权利放弃等条款可能锁死客户

## 二、原 22 类合同审查卡

### 1. 公司及其他主体类

适用场景：公司设立、股东协议、股权转让、增资、减资、收购并购、章程配套、合伙企业或其他组织主体安排。

必读材料：公司章程、股东名册、工商档案、出资证明、股东会/董事会决议、财务报表、债务清单、权属证明、审批备案材料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 主体资格 | 股东、投资人、转让方、目标公司主体是否适格 |
| 出资与权属 | 出资是否实缴，股权是否质押、冻结、代持、共有或存在争议 |
| 决策程序 | 股东会/董事会决议、优先购买权、章程限制、外部审批是否完备 |
| 交易对价 | 价款、估值依据、付款节点、税费承担是否清晰 |
| 交割控制 | 工商变更、印章证照、账册资料、控制权交接 |
| 陈述保证 | 债务、税务、劳动、诉讼、知识产权、行政处罚等历史风险 |
| 过渡期 | 经营限制、损益归属、新增债务和重大处分限制 |
| 退出与违约 | 回购、赔偿、竞业限制、违约金、加速到期 |

红线风险：未处理优先购买权；股权存在权利负担但未披露；付款早于控制权交接且无担保；历史债务由受让方兜底。

对方不同意时：可用分期付款、尾款、保证金、披露清单、赔偿承诺替代全面保证；重大权属瑕疵不建议让步。

期限提醒：交割期限、工商变更期限、回购触发期限、业绩承诺期需要进入合同概览；有固定通知期时进入提醒表。

### 2. 买卖转让类

适用场景：买卖、采购、销售、供货、购销、设备采购、资产转让、货物供应。

必读材料：采购订单、报价单、技术规格书、质量标准、验收标准、售后文件、物流保险附件。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 标的 | 名称、型号、规格、数量、品牌、产地是否足以特定化 |
| 价格 | 单价、总价、币种、税率、含税/不含税、发票类型 |
| 交付 | 时间、地点、方式、运输费用、装卸费用、交付凭证 |
| 风险转移 | 毁损灭失风险转移时点是否明确，是否与所有权转移区分 |
| 验收 | 检验期限、检验标准、异议方式、抽检/全检、沉默视为合格规则 |
| 质量与售后 | 质保期、维修、换货、退货、备件供应、响应时间 |
| 所有权保留 | 未付清价款前所有权是否保留，是否影响转售或使用 |
| 违约 | 逾期交付、逾期付款、质量不合格、拒收、解除权 |

红线风险：标的或质量标准不明确；付款节点早于关键交付或验收节点且无担保；验收期过短；违约责任只约束我方。

对方不同意时：可将一次性验收改为初验加终验；可接受较短异议期但从具备检验条件起算；可用尾款、质保金、保函替代高额违约金。

期限提醒：框架采购、年度供货、长期价格锁定协议必须抽取到期日、自动续约和不续约通知期限。

### 3. 租赁类

适用场景：房屋租赁、商铺租赁、场地租赁、设备租赁、车辆租赁。

必读材料：权属证明、授权出租文件、租赁物清单、平面图、设备清单、物业规则、消防/用途限制、交接清单。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 权属与授权 | 出租人是否有出租权，是否抵押、查封、共有或转租受限 |
| 用途 | 约定用途与规划、消防、物业、营业许可是否匹配 |
| 租金押金 | 租金、押金、递增、免租期、支付日、发票、退还条件 |
| 交付 | 交付状态、附属设施、钥匙、交接清单、水电表读数 |
| 装修 | 是否允许装修，审批、恢复原状、装修归属 |
| 维修 | 主体结构、自然损耗、人为损坏、设备维修责任 |
| 转租续租 | 是否可转租，续租优先权，续租通知期限 |
| 退租 | 返还状态、押金退还、逾期占用费 |

红线风险：出租权不清；用途可能违法或无法办照；押金可被单方没收；自动涨租、解除、续租条款明显不利。

对方不同意时：可不要求全部权属材料公开，但至少要求出租权承诺和违约赔偿；可接受恢复原状但排除正常损耗和经同意装修。

期限提醒：必须抽取到期日、续租优先权、提前不续租通知期限、通知方式。

### 4. 承揽、劳务、服务类

适用场景：承揽、加工、定作、修理、咨询、顾问、策划、服务外包、劳务服务。

必读材料：服务清单、工作说明书、报价单、项目计划、成果样式、验收标准、人员名单、变更单。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 服务范围 | 服务内容、边界、排除事项是否清楚 |
| 成果标准 | 交付物格式、数量、质量、验收口径 |
| 人员安排 | 是否指定人员，能否替换，替换是否需同意 |
| 项目变更 | 超范围需求、工期顺延、费用调整机制 |
| 费用支付 | 固定价、工时价、里程碑付款、报销范围 |
| 验收 | 验收期限、修改次数、不合格处理、视为验收 |
| 成果权属 | 知识产权归属、使用范围、源文件交付 |
| 客户配合 | 资料提供、确认期限、延误责任 |

红线风险：服务范围无边界；成果标准无法验收；无限修改、无限返工或结果保证；客户配合义务缺失却由服务方承担全部延期责任。

对方不同意时：可用服务清单加排除事项替代过细条款；可接受结果导向但增加客户配合、资料真实性和变更收费。

期限提醒：顾问、年度服务、运维服务必须抽取服务期限和续约条款。

### 5. 中介、居间、推广、行纪类

适用场景：中介、居间、撮合、推广、渠道介绍、佣金、行纪交易。

必读材料：中介事项说明、客户或项目清单、佣金政策、成交证明、排他期限、费用报销规则。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 服务内容 | 是报告机会、提供媒介、推广服务还是代为交易 |
| 成功标准 | 签约、付款、交付、回款哪个节点触发报酬 |
| 佣金计算 | 基数、比例、税费、分期、退款或退单处理 |
| 排他范围 | 区域、客户、产品、期限是否明确 |
| 不绕过义务 | 委托人绕开中介直接交易的后果 |
| 费用承担 | 前期费用、差旅费、推广费、实报实销标准 |
| 合规边界 | 是否涉及商业贿赂、回扣、利益输送 |

红线风险：成功标准不明；佣金触发过早；不绕过条款过宽；顾问费实质为回扣。

对方不同意时：可将报酬拆成签约部分和回款部分；可限定排他客户清单和保护期；合规风险不建议让步。

期限提醒：排他期、客户保护期、佣金结算期需要进入概览；有固定到期和续约时进入提醒表。

### 6. 知识产权类

适用场景：商标、专利、著作权、软件著作权、技术秘密许可或转让。

必读材料：权属证书、授权链条、登记/备案材料、许可样式、使用清单、侵权争议资料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 权属 | 权利是否有效、是否共有、是否质押或被限制 |
| 许可性质 | 独占、排他、普通许可是否明确 |
| 使用范围 | 地域、期限、产品、渠道、用途 |
| 再许可 | 是否允许转授权、关联方使用 |
| 费用 | 固定费、提成、最低保证金、审计权 |
| 维权 | 侵权发现、维权主体、费用和赔偿归属 |
| 终止后处理 | 库存、宣传物、账号、资料、标识停止使用 |

红线风险：授权人权利来源不清；独占/排他/普通许可混乱；许可范围无法覆盖实际业务；第三方侵权索赔责任缺失。

对方不同意时：可接受普通许可但保障实际使用范围；可用最低销售额换取稳定授权期；权利瑕疵赔偿不建议完全放弃。

期限提醒：许可期限、续展条件、不续约通知期、商标/专利有效期需要记录。

### 7. 劳动用工类

适用场景：劳动合同、竞业限制、保密协议、培训服务期、劳务派遣、员工安置协议。

必读材料：岗位说明、薪酬制度、规章制度、公示签收记录、社保缴纳记录、派遣协议、竞业范围资料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 关系定性 | 劳动、劳务、承揽、派遣是否混淆 |
| 合同期限 | 固定期限、无固定期限、试用期是否合法 |
| 岗位薪酬 | 岗位、地点、工时、工资、绩效、奖金是否清晰 |
| 社保福利 | 社保、公积金、补贴、休假约定是否合规 |
| 规章衔接 | 解除、纪律、考核是否与制度衔接 |
| 竞业限制 | 人员范围、期限、地域、补偿、违约金 |
| 培训服务期 | 专项培训、费用构成、服务期和违约金 |

红线风险：试用期违法；放弃社保；竞业限制无补偿；违约金超出劳动法允许范围；名为劳务实为劳动。

对方不同意时：劳动法强制规则不让步；商业性条款可通过岗位、绩效、保密、交接义务优化。

期限提醒：劳动合同到期、竞业限制期限、服务期届满可进入提醒表。

### 8. 担保类

适用场景：保证、抵押、质押、留置、定金、反担保、差额补足。

必读材料：主合同、担保合同、公司担保决议、抵质押物权属证明、登记材料、保证人资信资料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 主债权 | 主合同编号、金额、期限、债务范围是否明确 |
| 担保方式 | 一般保证/连带保证、抵押、质押、反担保是否清楚 |
| 担保范围 | 本金、利息、违约金、损害赔偿、实现债权费用 |
| 保证期间 | 起算点、届满日、主张方式 |
| 公司担保 | 董事会/股东会决议、章程限制、关联担保 |
| 登记交付 | 抵押登记、质押交付或权利质押登记 |
| 主合同变更 | 债务展期、金额增加、利率变化对担保影响 |

红线风险：保证方式不明；保证期间缺失；公司担保无有效决议；抵押未登记或质押未交付。

对方不同意时：可接受担保范围缩小，但主债权和实现债权费用应覆盖；公司担保决议不建议放弃。

期限提醒：保证期间届满、抵押登记到期、主债务到期均可进入提醒表。

### 9. 民间借贷类

适用场景：自然人借款、企业间借款、借条、还款协议、债务确认、展期协议。

必读材料：借款合同、借条、转账凭证、收据、还款记录、催收记录、担保资料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 本金 | 实际出借金额、大小写、币种、是否预扣利息 |
| 利息 | 利率、逾期利率、复利、违约金叠加是否过高 |
| 期限 | 放款日、到期日、提前到期条件 |
| 用途 | 借款用途、禁止用途、挪用后果 |
| 支付凭证 | 现金交付证明、银行流水、第三方代付说明 |
| 担保 | 保证、抵押、质押、共同债务加入 |
| 实现债权费用 | 律师费、保全费、诉讼费、公告费等承担 |

红线风险：砍头息；利息和违约责任叠加明显过高；无付款凭证；夫妻共同债务或保证责任不清。

对方不同意时：可降低名义利率，保留合理逾期利息和实现债权费用；可用担保替代高利率。

期限提醒：还款日、分期付款日、保证期间、展期届满日需要记录。

### 10. 特许经营、许可加盟类

适用场景：商业特许经营、加盟、连锁、品牌许可、经营模式输出。

必读材料：特许人备案信息、两店一年证明、信息披露文件、品牌权属、运营手册、培训支持文件。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 资质备案 | 特许人是否具备特许经营条件和备案情况 |
| 授权内容 | 品牌、系统、技术、经营模式、区域、期限 |
| 费用 | 加盟费、保证金、管理费、物料费、返利 |
| 支持义务 | 培训、选址、装修、运营、供应链支持 |
| 经营控制 | 价格、采购、装修、人员、营销限制是否合理 |
| 退出处理 | 保证金退还、库存、标识拆除、客户资源 |
| 违约解除 | 窜货、侵权、质量事故、违规经营后果 |

红线风险：无备案或信息披露缺失；保证金扣罚过宽；单方随意调整政策；终止后库存和投入无法处理。

对方不同意时：可接受统一管理，但费用扣罚、政策调整、区域保护必须有边界。

期限提醒：加盟期限、续约条件、提前不续约通知期、保证金退还期限必须记录。

### 11. 承包经营类

适用场景：门店承包、食堂承包、项目承包、经营权托管、内部承包。

必读材料：权属或经营许可文件、场地/设备清单、经营资质、安全责任资料、历史债务和员工安置资料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 经营权来源 | 发包方是否有权发包，是否需审批或备案 |
| 对外名义 | 以谁名义经营，对外债务和投诉由谁承担 |
| 承包费用 | 固定承包费、提成、保底、保证金 |
| 资产交接 | 场地、设备、证照、库存、客户资源 |
| 合规责任 | 食品安全、安全生产、环保、消防、用工 |
| 经营限制 | 经营范围、采购渠道、价格政策、品牌使用 |
| 退出清算 | 资产返还、债务清理、员工和客户交接 |

红线风险：无权发包；对外责任不清；安全和用工责任全部压给我方但缺控制权；历史债务未切割。

对方不同意时：可接受部分经营限制，但必须匹配控制权和责任边界。

期限提醒：承包期限、考核期、续包通知期、保证金退还期需要记录。

### 12. 合作类

适用场景：联合经营、项目合作、渠道合作、联合开发、资源互换、战略合作。

必读材料：项目方案、出资或资源清单、利润分配规则、成本预算、审批文件、客户或渠道清单。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 合作性质 | 合伙、联营、服务、代理、投资还是买卖 |
| 投入义务 | 资金、资源、人员、技术、渠道投入是否量化 |
| 收益分配 | 收入口径、成本扣除、结算周期、亏损承担 |
| 决策机制 | 重大事项、预算、合同签署、资金支出 |
| 财务透明 | 账目、审计、票据、收款账户 |
| 排他限制 | 竞业、独家、客户归属、区域限制 |
| 退出机制 | 期限届满、违约退出、清算、客户和资产处理 |

红线风险：合作名义掩盖借贷或投资；收益分配不清；亏损和债务承担不清；退出机制缺失。

对方不同意时：可先签框架合作加项目确认单；可用预算审批和结算报表替代复杂治理结构。

期限提醒：合作期限、排他期、结算期、退出通知期需要记录。

### 13. 赠与类

适用场景：财产赠与、附义务赠与、公益捐赠、亲属间赠与。

必读材料：赠与财产权属证明、交付或过户资料、附义务说明、家庭关系或公益资质材料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 权属 | 赠与人是否有处分权，是否共有或存在负担 |
| 交付过户 | 动产交付、不动产登记、股权变更等手续 |
| 附义务 | 义务内容、履行期限、违约后果 |
| 撤销权 | 任意撤销、法定撤销、公益赠与限制 |
| 税费 | 契税、个税、印花税等承担 |
| 家庭风险 | 夫妻共同财产、继承争议、债权人撤销权 |

红线风险：处分共有财产无同意；用赠与逃避债务；附义务不清；过户前权利不稳定。

对方不同意时：可用附条件交付、分期赠与或公证增强稳定性。

期限提醒：过户期限、附义务履行期限可记录；一般不做续约提醒。

### 14. 婚姻家庭类

适用场景：婚内财产协议、离婚协议、抚养协议、赡养协议、家庭财产分割。

必读材料：身份关系材料、财产权属证明、贷款信息、子女情况、债务清单、社保和收入材料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 主体关系 | 婚姻、亲子、赡养关系是否清楚 |
| 财产范围 | 房产、车辆、股权、存款、债务是否列明 |
| 分割安排 | 过户、付款、贷款承担、协助义务 |
| 子女抚养 | 抚养权、抚养费、探望、教育医疗费用 |
| 债务承担 | 共同债务、个人债务、对外债权人效力 |
| 履行保障 | 违约责任、协助过户、证件交付 |

红线风险：处分未列明或权属不清财产；债务承担对外效力被误解；抚养费和探望安排不可执行。

对方不同意时：可拆分为即时履行和后续协助义务；重要财产尽量增加公证或过户节点。

期限提醒：抚养费支付日、过户期限、补偿款分期可记录。

### 15. 通用配套文本类

适用场景：授权委托书、确认函、承诺函、通知函、保函、收据、交接单、补充说明等配套文件。

必读材料：主合同、交易背景、授权依据、履行凭证、对应附件。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 与主合同关系 | 是否改变主合同权利义务，效力顺位如何 |
| 授权范围 | 授权事项、期限、转授权、撤销方式 |
| 承诺边界 | 承诺主体、承诺对象、条件、期限 |
| 事实确认 | 是否形成债务确认、验收确认或权利放弃 |
| 送达生效 | 通知方式、地址、送达时间 |
| 证据价值 | 签章、日期、附件、经办人权限 |

红线风险：承诺函无意中放弃重大权利；确认函形成不利债务确认；授权范围过宽。

对方不同意时：可加入“不视为放弃其他权利”“仅为本事项使用”等限制。

期限提醒：授权期限、通知期限、承诺履行期限需要记录。

### 16. 房地产开发、运营类

适用场景：房地产合作开发、招商运营、物业运营、商业综合体运营、销售代理。

必读材料：土地/房产权属、规划许可、施工许可、预售许可、招商政策、物业规则、运营方案。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 权属许可 | 土地、房产、规划、预售、消防等手续 |
| 合作模式 | 开发合作、招商运营、委托管理、租售代理 |
| 收益分配 | 租金、销售佣金、运营费、成本扣除 |
| 招商承诺 | 品牌、客流、开业率、保底收益是否可实现 |
| 工程交付 | 交付标准、整改、延期、验收 |
| 运营责任 | 物业、安全、消防、广告、公共区域维护 |
| 退出清算 | 租户、保证金、装修、未结费用处理 |

红线风险：权属或许可不全；保底收益无实现路径；招商承诺过度；运营责任和控制权不匹配。

对方不同意时：可将结果承诺改为过程义务和阶段目标；收益分配需绑定真实回款。

期限提醒：开业日、交付日、招商考核日、运营期限、续约通知期需要记录。

### 17. 建设工程类

适用场景：建设工程施工、装饰装修、维修工程、设计施工一体化、工程分包。

必读材料：图纸、工程量清单、报价单、预算书、施工计划、变更签证规则、验收标准、资质资料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 工程范围 | 工程地点、内容、边界、图纸和清单是否一致 |
| 资质 | 承包资质、分包限制、违法转包风险 |
| 工期 | 开工、竣工、顺延条件、停工责任 |
| 价款 | 固定总价、固定单价、据实结算、暂估价 |
| 变更签证 | 指令形式、确认期限、价款调整 |
| 验收结算 | 隐蔽工程、阶段验收、竣工验收、结算期限 |
| 质保 | 质保期、质保金比例、返还条件 |
| 安全 | 安全生产、第三人损害、保险 |

红线风险：工程范围和价款模式不匹配；变更签证无确认机制；逾期验收/结算无后果；质保金比例或返还期限不合理。

对方不同意时：可不细化全部工程量，但必须固定图纸、清单和排除项；固定总价应保留变更、停工和不可归责顺延例外。

期限提醒：工期、验收期、结算期、质保期、质保金返还日需要记录。

### 18. 金融、保险、证券、期货类

适用场景：金融服务、保险合同、证券期货服务、理财产品、融资服务、资产管理相关协议。

必读材料：产品说明书、风险揭示书、监管备案或资质文件、收费标准、交易规则、投保单/保单。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 主体资质 | 金融牌照、销售资质、代理权限 |
| 产品信息 | 标的、期限、收益、费用、风险等级 |
| 风险揭示 | 是否充分揭示本金损失、流动性、市场风险 |
| 适当性 | 投资者/投保人适当性匹配 |
| 责任限制 | 免责条款、不可抗力、系统故障责任 |
| 赎回理赔 | 赎回、退保、理赔、除外责任、等待期 |
| 争议解决 | 管辖、仲裁、监管投诉路径 |

红线风险：无资质销售；保本承诺与产品性质不符；风险揭示不足；除外责任未提示。

对方不同意时：强监管和适当性要求不让步；商业收费可谈判。

期限提醒：保险期限、缴费日、犹豫期、赎回期、通知期需要记录。

### 19. 私募基金类

适用场景：私募基金募集、认购、合伙协议、基金管理、投资顾问、份额转让。

必读材料：基金合同/合伙协议、募集说明书、风险揭示书、管理人登记信息、托管协议、投资者适当性材料。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 管理人资质 | 登记状态、实际控制人、异常信息 |
| 募集合规 | 合格投资者、风险测评、冷静期、回访 |
| 投资范围 | 投资标的、限制、关联交易、杠杆 |
| 费用分配 | 管理费、业绩报酬、托管费、费用列支 |
| 决策机制 | 投委会、管理人权限、重大事项表决 |
| 退出机制 | 封闭期、赎回、份额转让、清算 |
| 信息披露 | 定期报告、重大事项披露、审计 |

红线风险：面向非合格投资者募集；保本保收益；管理人异常；投资范围和费用不清。

对方不同意时：监管红线不让步；费用和披露频次可谈判。

期限提醒：缴款日、封闭期届满、赎回窗口、基金到期日、清算期需要记录。

### 20. 软件、IT、游戏开发运营类

适用场景：软件开发、系统集成、SaaS、平台服务、运维、游戏开发运营、技术服务、数据处理。

必读材料：主协议、订单、SOW、功能清单、接口文档、SLA、数据处理说明、安全要求、验收标准。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 服务范围 | 标准产品、定制开发、运维支持是否区分 |
| 开发交付 | 里程碑、版本、源代码、部署环境、验收用例 |
| SLA | 可用性、响应时间、恢复时间、服务补偿 |
| 数据 | 数据归属、处理目的、保存期限、导出、删除 |
| 合规 | 个人信息保护、数据安全、网络安全义务分配 |
| 知识产权 | 既有软件、定制成果、开源组件、第三方授权 |
| 账号权限 | 管理员权限、停服条件、账号交接 |
| 费用续费 | 订阅费、实施费、扩容费、续费涨价规则 |

红线风险：数据泄露、停服、服务中断完全免责；客户数据无法导出或删除；定制成果权属不清；自动续费、涨价、停服权过宽。

对方不同意时：可接受责任上限，但数据泄露、故意/重大过失、知识产权侵权应设例外；可不交付源代码但需退出协助。

期限提醒：订阅期限、自动续费、涨价通知、取消方式、SLA统计周期需要记录。

### 21. 物流运输类

适用场景：物流、运输、仓储、货运代理、快递配送、冷链运输。

必读材料：运单、货物清单、报价单、保险或保价规则、交接单、签收规则、仓储规则、赔付标准。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 货物信息 | 名称、数量、重量、体积、价值、特殊属性 |
| 交接 | 提货、装卸、签收、异常记录 |
| 时效 | 运输期限、延误责任、不可归责情形 |
| 风险 | 灭失、短少、损坏、污染、温控失败 |
| 保价保险 | 是否保价、未保价赔付上限、保险责任 |
| 费用 | 运费、仓储费、装卸费、滞留费 |
| 索赔 | 异议期限、证明材料、赔付计算 |

红线风险：未保价赔付上限过低且未提示；货损、短少、延误责任完全免除；签收规则导致无法验货。

对方不同意时：可接受赔付上限，但高价值货物应保价或另购保险；特殊货物要求必须写入合同。

期限提醒：年度物流服务、仓储服务需抽取服务期限、续约和价格调整通知期。

### 22. 影视剧融资、制作与发行类

适用场景：影视剧投资、联合制作、承制、发行、宣传、演员/导演聘用、版权授权。

必读材料：项目备案或许可、剧本权属、主创合同、预算、发行计划、收益分配表、版权链文件。

重点审查：

| 模块 | 审查要点 |
|---|---|
| 权属链条 | 剧本、音乐、素材、改编权、肖像权、商标使用 |
| 投资制作 | 投资额、付款节点、预算控制、超支承担 |
| 审批合规 | 备案、许可证、审查、播出平台要求 |
| 主创履约 | 档期、替换、违约、不可抗力 |
| 收益分配 | 票房、平台收入、广告、衍生品、海外发行 |
| 宣发发行 | 发行区域、期限、渠道、最低发行承诺 |
| 署名宣传 | 署名位置、海报物料、宣传义务 |
| 退出清算 | 项目停拍、审查未过、发行失败后的处理 |

红线风险：版权链不完整；审批风险由我方单方承担；收益口径不清；投资款支付早于关键权利或备案节点。

对方不同意时：可接受收益后置，但应获得查账权和收入定义；权属链和审批材料不建议让步。

期限提醒：拍摄期、交付期、发行期、收益结算期、授权期限需要记录。

## 三、新增交叉专项卡

以下专项卡不是替代原 22 类，而是在命中相应场景时叠加使用。

### 23. 保密协议 / NDA 专项

适用场景：NDA、保密协议、合作前信息披露、技术资料、客户名单、源代码、商业秘密。

重点审查：保密信息范围、例外情形、使用目的、接触人员、保密期限、返还销毁、违约责任、行为保全。

红线风险：单方保密；保密范围覆盖我方既有技术或公开信息；违约金过高；限制我方正常业务。

对方不同意时：可接受较宽保密范围，但必须补足例外情形；普通保密信息和商业秘密分层保护。

期限提醒：保密期和终止后义务期限进入概览；有固定到期和续签机制时进入提醒表。

### 24. SaaS / 数据处理 / 个人信息专项

适用场景：SaaS、云服务、平台服务、数据处理、个人信息委托处理、系统运维、账号服务。

重点审查：客户数据归属、处理目的、最小必要、跨境传输、数据导出删除、备份、日志、安全事件通知、子处理方、账号停用、SLA。

红线风险：服务方可任意使用客户数据；终止后无法导出或删除；数据泄露完全免责；子处理方不受约束。

对方不同意时：可接受标准安全附件，但安全事件通知、数据导出删除、违法违规使用禁止条款不建议放弃。

期限提醒：订阅期、自动续费、涨价通知、数据保留/删除期限必须记录。

### 25. 和解 / 补充 / 解除 / 债务确认专项

适用场景：和解协议、补充协议、解除协议、债务确认、还款安排、撤诉协议。

重点审查：原合同关系、债权债务基数、付款节点、权利放弃范围、履行失败后果、担保、撤诉/解除节点、效力顺位。

红线风险：“互不追究”覆盖过宽；分期付款无加速到期；和解失败后不能恢复原债权；补充协议与原合同顺位不清。

对方不同意时：可接受部分权利放弃，但必须以对方按期履行为条件；分期付款需加速到期和费用承担。

期限提醒：付款节点、宽限期、提前到期日、撤诉节点可进入提醒表。

### 26. 续约 / 自动续约 / 到期提醒专项

适用场景：任何含固定期限、自动续约、不续约通知、提前解除通知、价格续期调整的合同。

必须抽取：

```text
| 合同编号 | 到期日 | 自动续约方式 | 提前通知天数 | 最晚发通知日 | 通知方式 | 通知地址/邮箱 | 是否需要工作日顺延 | 状态 | 来源条款 |
```

红线风险：自动续约但未提醒；不续约通知期过早或过短；通知方式缺失；通知地址错误；到期后价格自动上涨。

对方不同意时：可接受自动续约，但需提前提醒、明确通知方式、限制涨价幅度或保留不续约权。

执行规则：字段完整时，询问用户是否创建飞书日历提醒；用户确认后调用 `lark-cli calendar +create --as user`。
