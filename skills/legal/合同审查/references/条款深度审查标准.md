# 条款深度审查标准

> 本文件为合同审查Skill提供条款级审查的方法论指引，包含通用审查维度、深度审查框架、SMART原则检验、三点一线完备性检验及审查方法四法。

---

## 一、通用审查维度表

### 1.1 八维度审查框架

| 审查维度 | 审查优先级 | 审查要点 | 常见问题 |
|---------|---------|---------|---------|
| **主体条款** | 🔴高 | 真实性、资质、履约能力、涉诉/失信 | 主体不适格、无签约权限、履约能力不足 |
| **标的条款** | 🔴高 | 特定性、合法性、可转让性、权属清晰 | 标的不特定、权属不清、存在转让限制 |
| **质量条款** | 🟡中 | 标准明确、检验方式、异议期限 | 标准模糊、无检验程序、异议期过短 |
| **价款/报酬** | 🟡中 | 金额明确、支付方式、收款信息、税费承担 | 计价不清、付款节点不明、收款账户缺失、税负约定缺失 |
| **履行条款** | 🟡中 | 时间、地点、方式、费用承担 | 期限不明、地点模糊、方式不可操作 |
| **违约条款** | 🟡中 | 违约情形、责任明确、可执行性 | 情形不周延、责任过重或过轻、难以执行 |
| **争议解决** | 🟡中 | 方式明确、管辖合法 | 管辖约定无效、仲裁条款不规范 |
| **程序条款** | 🟢低 | 生效条件、份数、附件、送达 | 生效条件不明、附件缺失、送达地址不清 |

### 1.2 维度权重说明

- **🔴高优先级维度**：可能导致合同无效、交易失败或重大损失，必须重点审查
- **🟡中优先级维度**：可能导致履行争议或部分损失，应当仔细审查
- **🟢低优先级维度**：一般程序性问题，按常规审查

## 二、五维度深度审查框架

### 2.1 标的维度审查

| 审查要素 | 审查问题 | 风险后果 |
|---------|---------|---------|
| **权属清晰** | 出让方是否有完整处分权？是否存在共有人、权利负担？ | 无权处分导致合同效力瑕疵 |
| **瑕疵担保** | 标的物是否存在质量瑕疵、权利瑕疵？担保责任如何约定？ | 取得存在瑕疵的标的物 |
| **特定性** | 标的物描述是否足够特定？能否区分于其他物？ | 履行标的不明确 |
| **资质要求** | 交易是否需要特定资质、许可、备案？ | 因缺乏资质导致合同无效 |

**法律依据**：
- 《民法典》第597条：无权处分的效力
- 《民法典》第612-617条：瑕疵担保责任

### 2.2 交易流程维度审查

| 审查要素 | 审查问题 | 风险后果 |
|---------|---------|---------|
| **交付程序** | 交付方式、时间、地点是否明确？交付凭证如何确认？ | 交付争议 |
| **验收程序** | 验收标准、验收期限、验收主体、不合格处理？ | 验收纠纷 |
| **时间节点** | 各关键节点是否有明确的起止时间？ | 履行期限不明 |
| **风险转移** | 标的物风险何时转移？运输风险由谁承担？ | 灭失损坏责任不清 |
| **文件交接** | 需要移交的文件、资料是否列明？ | 交接不完整 |

**法律依据**：
- 《民法典》第604-606条：标的物风险转移
- 《民法典》第620-624条：检验期间与异议

### 2.3 权利义务维度审查

| 审查要素 | 审查问题 | 风险后果 |
|---------|---------|---------|
| **对等性** | 双方权利义务是否大体对等？是否存在显失公平？ | 合同可撤销或履行困难 |
| **可执行性** | 约定的义务是否实际可履行？是否过于严苛？ | 约定形同虚设 |
| **隐性义务** | 是否存在附随义务、默示义务未明确？ | 义务范围争议 |
| **执行人员风险** | 经办人员是否有越权、收受利益等风险？ | 代理风险、商业贿赂 |

**审查技巧**：
- 逐条检查每项义务是否有对应的权利保障
- 检查每项权利是否有对应的义务支撑
- 关注"应当"、"必须"、"不得"等义务性表述

### 2.4 违约救济维度审查

| 审查要素 | 审查问题 | 审查标准 |
|---------|---------|---------|
| **违约金合理性** | 违约金是否过高？是否超过实际损失的130%？ | 《民法典》第585条：可请求调整 |
| **解除权** | 解除条件是否明确？是否有约定解除权？ | 避免被动锁定 |
| **损害赔偿** | 赔偿范围是否明确？是否包含可得利益？ | 避免赔偿争议 |
| **继续履行** | 违约后能否要求继续履行？排除情形是否约定？ | 《民法典》第580条 |
| **责任上限** | 是否设置了责任上限？上限是否合理？ | 风险可控性 |

**违约金调整规则**（《民法典》第585条）：
- 约定违约金低于损失：可请求增加
- 约定违约金过分高于损失：可请求适当减少
- 司法实践：一般以不超过实际损失的130%为限

### 2.5 担保措施维度审查

| 审查要素 | 审查问题 | 注意事项 |
|---------|---------|---------|
| **类型适当性** | 担保方式是否与交易风险匹配？ | 高风险交易应选择抵押/质押 |
| **主从关系** | 担保合同与主合同的关系是否明确？ | 《民法典》第682条：主债务消灭担保消灭 |
| **登记要求** | 是否需要办理登记？未登记的法律后果？ | 不动产抵押需登记生效 |
| **担保期间** | 担保期间是否约定？是否合理？ | 未约定推定6个月 |
| **代偿能力** | 保证人是否有代偿能力？ | 需核查保证人资信 |

**担保效力规则**：

| 担保类型 | 设立要件 | 效力产生 |
|---------|---------|---------|
| 保证 | 书面合同 | 合同成立即生效 |
| 不动产抵押 | 抵押合同+登记 | 登记时设立 |
| 动产抵押 | 抵押合同 | 合同生效时设立，登记对抗 |
| 动产质押 | 质押合同+交付 | 交付时设立 |
| 权利质押 | 质押合同+登记/交付权利凭证 | 登记/交付时设立 |

## 三、SMART原则检验

### 3.1 五维度检验标准

| 原则 | 含义 | 检验问题 | 常见缺陷 |
|-----|-----|---------|---------|
| **S - Specific** | 具体的 | 权利义务是否具体描述？主体、对象、行为是否明确？ | "应妥善保管"→未明确保管标准 |
| **M - Measurable** | 可衡量的 | 质量、验收标准能否量化？能否客观判断是否达标？ | "质量良好"→无法判断 |
| **A - Attainable** | 可达到的 | 条款是否可执行？要求是否过于严苛？对方能否实际履行？ | 要求"24小时响应"→实际无法做到 |
| **R - Relevant** | 有关联性 | 条款是否围绕交易目标？是否与核心权利义务相关？ | 大量无关条款分散重点 |
| **T - Time-based** | 有期限的 | 是否有明确起止期限？时间节点是否清楚？ | "尽快交货"→无明确期限 |

### 3.2 SMART检验示例

**不合格条款**：
> "乙方应尽快将货物送至买方指定地点，质量按照国家标准执行。"

**缺陷分析**：
- ❌ 不具体：未明确具体货物、数量
- ❌ 不可衡量：未指明具体国标号
- ❌ 无期限："尽快"无法判断

**合格条款**：
> "乙方应于2024年6月30日前，将本合同附件一所列货物（型号XX，数量100件）送达甲方指定仓库（地址：XX市XX区XX路XX号）。货物质量应符合GB/T XXXX-XXXX标准，甲方应在收货后7日内完成检验。"

## 四、三点一线条款完备性检验

### 4.1 "三点"检验

每条核心条款是否设置了**权利、义务、救济**三要素。

| 要素 | 含义 | 检验问题 |
|-----|-----|---------|
| **权利** | 当事人可以要求的内容 | 约定了义务，权利人有权要求什么？ |
| **义务** | 当事人必须履行的内容 | 权利对应的义务是什么？由谁履行？ |
| **救济** | 义务不履行时的后果 | 违反义务时，权利人如何救济？ |

### 4.2 "一线"检验

**条件→时间→程序**的逻辑是否通顺完整。

| 逻辑环节 | 检验问题 |
|---------|---------|
| **条件** | 触发权利/义务的前提条件是否明确？ |
| **时间** | 行使权利/履行义务的时间节点是否清楚？ |
| **程序** | 具体操作步骤是否可执行？ |

### 4.3 检验示例

**不完备条款**：
> "甲方应支付货款。"

**缺陷分析**：
- 缺条件：何时支付？收货后？验收后？
- 缺时间：多少天内支付？
- 缺程序：支付到哪个账户？
- 缺救济：不支付怎么办？

**完备条款**：
> "乙方交货并经甲方验收合格后（条件），甲方应于10个工作日内（时间）将货款支付至乙方指定账户（程序：账户信息见本合同附件）。逾期支付的，甲方应按日利率万分之五支付逾期利息（救济）。"

## 五、审查方法四法

### 5.1 补增法

**适用场景**：
- 缺少必备条款（如质量标准、验收程序、违约责任）
- 缺少保护委托人的条款（如解除权、优先权）

**审查意见格式**：
```
【需补充条款】
位置：建议在第X条后补充
内容：
  "第X+1条 检验
  买方应在收到货物后___日内进行检验。检验合格的，视为货物符合约定；
  检验不合格的，买方应在检验期满后___日内书面通知卖方，逾期未通知的，
  视为货物符合约定。检验费用由___方承担。"
理由：原合同未约定检验程序，可能导致质量争议时举证困难
```

### 5.2 删改法

**适用场景**：
- **违法条款**：违反法律强制性规定
- **重复条款**：同一事项多处约定
- **矛盾条款**：前后条款相互冲突
- **不当表述**：表述不准确或有歧义
- **文字错误**：错别字、数字错误

**审查意见格式**：
```
【重大风险】利率超过法定上限
【修改位置】第X条第Y款
【风险原因】年利率36%超过合同成立时一年期LPR的4倍，超过部分的利息约定无效，出借人无法主张超过部分的利息（《最高人民法院关于审理民间借贷案件适用法律若干问题的规定》第25条）
【修订建议】
原文："第X条第Y款 借款利率
借款年利率为36%，按月计息。"

建议修改为：
"第X条第Y款 借款利率
借款年利率为合同成立时一年期贷款市场报价利率（LPR）的4倍，即____%（大写：百分之____）。利息自借款实际发放之日起计算，按月结息。"
```

### 5.3 建议法

**适用场景**：
- 需明确的商务事项（付款比例、交货时间等需双方协商确定）
- 可争取的有利条款（解除权、优先权、限制责任条款）
- 需整体调整的情形（条款顺序、结构优化）

**审查意见格式**：
```
【建议修改】付款方式风险过高
【修改位置】第X条
【风险原因】合同签订后一次性支付全部款项，对受让方风险较大。如股权变更登记无法完成或转让方存在隐瞒债务等情形，受让方已支付款项难以追回
【修订建议】
原文："第X条 付款方式
受让方应于本合同签订后5个工作日内一次性向转让方支付全部股权转让款人民币XXX万元。"

建议修改为：
"第X条 付款方式
股权转让款分三期支付：
X.1 第一期：本合同签订后5个工作日内，受让方向转让方支付股权转让款的30%，即人民币XXX万元（大写：XXX万元整），作为定金。
X.2 第二期：目标公司完成股权变更登记并取得新的营业执照后5个工作日内，受让方向转让方支付股权转让款的40%，即人民币XXX万元（大写：XXX万元整）。
X.3 第三期：交割完成且转让方按照本合同约定履行完毕全部义务后___日内，受让方向转让方支付剩余股权转让款30%，即人民币XXX万元（大写：XXX万元整）。"
```

### 5.4 提示法

**适用场景**：
- 存疑事项（主体身份、资质真实性存疑）
- 潜在法律风险（条款效力不确定、存在争议）
- 前瞻性商业事项（市场变化、政策风险）

**审查意见格式**：
```
【提示】第X条
合同相对方为"中国某某网"，但"中国某某网"并非法人单位，
而是北京某某公司所属的网站平台。建议核实并修改为正确的法人主体名称，
以免影响合同效力和权利行使。
```

### 5.5 四法适用原则

| 方法 | 语气强度 | 适用场景 | 决策权 |
|-----|---------|---------|-------|
| 删改 | 强 | 违法、重大风险 | 律师主导 |
| 补增 | 强 | 缺失必备条款 | 律师主导 |
| 建议 | 中 | 商务协商事项 | 客户决定 |
| 提示 | 弱 | 不确定事项 | 客户自行判断 |
