# 条款规范与语言标准

> 本文件为合同起草Skill提供条款表达与语言规范的方法论指引，基于《合同起草审查指南：三观四步法》"合同语言篇"整理。

---

## 一、合同语言三原则

### 1.1 准确性原则

每个词语都应准确表达法律含义，不使用模糊词汇。

**要点**：
- 准确、清晰、全面，避免模糊、歧义、疏漏
- 合同语言无须过于追求简练，只要有利于准确全面界定权利义务即可
- 尽可能量化，以利于争议解决

**示例对比**：

| 模糊表达 | 准确表达 |
|---------|---------|
| 尽快交货 | 于2024年3月15日前交货 |
| 合理费用 | 不超过人民币5000元的费用 |
| 适当补偿 | 按实际损失的120%补偿 |
| 一定比例 | 按合同总金额的5% |

### 1.2 无歧义性原则

条款不得存在两种以上解释的可能。

**常见歧义类型**：
1. **指代不明**：代词指向不清
2. **修饰关系不明**：限定词作用范围不清
3. **并列关系不明**：是"和"还是"或"
4. **数量关系不明**：包含边界还是不包含

**歧义示例与修正**：

❌ "未经甲方允许，不得复制资料内容，严禁在资料上勾、划、改。"
- 歧义："未经甲方允许"是仅限定"不得复制"，还是同时限定"严禁勾划改"？

✅ 修正后：
```
1. 未经甲方书面允许，乙方不得复制资料内容；
2. 乙方严禁在资料上勾、划、改。
```

### 1.3 法言法语原则

使用规范的法律术语，避免口语化。

**要求**：
- 合同语言接近于法律法规条款的语言
- 应适当追求典雅，不适宜口语化
- 语言风格客观、理性、中立
- 避免带有情绪或道德批判意味的表述

## 二、金字塔式写作方法

### 2.1 基本原则

借鉴"金字塔式写作"原则，将合同内容及条款适当整理：

1. **一段话、一句话尽量只表达一个意思**
2. **把所有的意思按层次和逻辑组织好**
3. **复杂的权利义务安排通过层次结构表达，而非长句子**

### 2.2 应用示例

**修改前**（问题条款）：
> 乙方借阅甲方资料期限在两周之内，如有特殊情况，最长不超过一个月。借阅资料要妥善保管，不得转借，未经甲方允许，不得复制资料内容，严禁在资料上勾、划、改。若资料归还时，资料有缺失，甲方有权拒绝接收。

**修改后**（应用金字塔式写作）：
```
第3条 资料借阅

3.1 乙方借阅甲方资料期限在两周之内，如有特殊情况，最长不超过一个月。

3.2 借阅资料要妥善保管，包括但不限于：
    3.2.1 不得转借；
    3.2.2 未经甲方书面允许，不得复制资料内容；
    3.2.3 严禁在资料上勾、划、改。

3.3 若资料归还时资料有缺失，甲方有权拒绝接收。
```

### 2.3 金字塔式写作的好处

1. **避免语言粘连**：减少歧义和争议
2. **结构清晰**：便于阅读和理解
3. **便于修订**：单独删改某项不影响其他项
4. **便于思考**：有助于发现遗漏

## 三、标准句式模板

### 3.1 基本句型

| 句型类型 | 句式结构 | 示例 |
|---------|---------|------|
| **义务句** | 甲方应/须/应当 + 动作 + 对象 + 时间/方式限制 | 甲方应于每月15日前支付当月租金。 |
| **权利句** | 乙方有权 + 动作 + 条件限制 | 乙方有权在甲方逾期付款超过30日时解除本合同。 |
| **禁止句** | 任何一方不得 + 动作 | 任何一方不得将本合同项下的权利义务转让给第三方。 |

### 3.2 条件句型

| 句型 | 示例 |
|-----|------|
| 如/若/在...情况下，...有权/应当... | 如乙方未按期交货，甲方有权解除合同。 |
| 除...外，...不得... | 除不可抗力外，任何一方不得单方解除本合同。 |
| 非经...同意，...不得... | 非经甲方书面同意，乙方不得转包。 |
| 出现下列情形之一的，...有权... | 出现下列情形之一的，甲方有权立即终止合同：... |

### 3.3 概括+列举句型

**包括但不限于**：
```
乙方的保密义务包括但不限于以下情形：
（一）不得向任何第三方披露保密信息；
（二）不得将保密信息用于本合同约定目的以外的任何用途；
（三）不得复制保密信息。
```

**下列情形之一**：
```
出现下列情形之一的，甲方有权解除本合同：
（一）乙方逾期交货超过30日；
（二）乙方交付的货物存在重大质量缺陷且无法修复；
（三）乙方被宣告破产或进入清算程序。
```

### 3.4 正说+反说句型

```
甲方应按本合同约定按时足额支付价款；未按时支付的，应按日利率万分之五支付逾期利息。

乙方有权使用甲方提供的商标进行产品宣传；但不得对商标进行任何形式的修改或变形使用。
```

## 四、条款编号与层级规范

### 4.1 推荐编号样式

**样式一：简单中式**（适合较简单的合同）
```
一、第一级
    1. 第二级
       (1) 第三级
```

**样式二：复杂中式**（适合一般及复杂合同，推荐使用）
```
第1条 第一级
    1.1 第二级
        1.1.1 第三级
            1.1.1.1 第四级
```

**样式三：英式**（适合较复杂的合同）
```
1. 第一级
    1.1 第二级
        1.1.1 第三级
            1.1.1.1 第四级
```

### 4.2 编号规则

1. 全文统一使用一种编号样式
2. 第一级应尽可能设置条标题
3. 复杂合同可在多级编号之外另设"章"作为更高一级
4. 选择项、并列项可单独使用(1)(2)编号

## 五、格式排版标准

### 5.1 字体与字号

| 位置 | 字体 | 字号 |
|-----|-----|-----|
| 合同标题 | 黑体 | 二号或小二号 |
| 章节标题 | 黑体 | 小三号或四号 |
| 条款标题 | 黑体或宋体加粗 | 小四号 |
| 正文 | 宋体或仿宋 | 小四号或四号 |

### 5.2 版式设置

| 项目 | 标准设置 |
|-----|---------|
| 纸张 | A4 |
| 页边距 | 上下2.54cm，左右3.17cm |
| 行距 | 1.5倍行距或固定值24-28磅 |
| 段前段后 | 0.5行或不设 |
| 缩进 | 统一缩进两个汉字字符，或统一不缩进 |
| 页码 | 底部居中，格式"第X页/共Y页" |

### 5.3 特殊格式

| 场合 | 格式要求 |
|-----|---------|
| 合同各方名称 | 加粗 |
| 条款标题 | 可加粗（全文统一） |
| 格式条款/重要提示 | 加粗 |
| 金额 | 大写数字+小写数字 |
| 填充处 | 可使用下划线 |

## 六、风险提示标注规范

### 6.1 格式要求

- **格式**：【风险提示：...】
- **样式**：红色加粗（或黑色加粗+下划线）
- **位置**：紧跟在相关条款之后

### 6.2 内容要求

风险提示应包含：
1. 具体风险说明
2. 可能的法律后果
3. 应对建议（如有）

### 6.3 示例

```
第5条 付款方式

甲方应于货物验收合格后30日内支付全部货款。

【风险提示：如约定的付款周期过长，卖方应考虑要求买方提供担保或分期付款。根据《民法典》第628条，买方未按约定支付价款的，卖方可以要求支付价款或者解除合同。】
```

## 七、常用法律术语对照表

| 口语表达 | 法律术语 | 备注 |
|---------|---------|-----|
| 租房子 | 房屋租赁 | |
| 买东西 | 买卖合同/货物购销 | |
| 借钱 | 借款合同/民间借贷 | |
| 找人干活 | 承揽/劳务/雇佣 | 需辨析适用类型 |
| 入股 | 股权投资/增资入股 | |
| 合伙做生意 | 合伙协议/合作经营 | |
| 押金 | 保证金/定金 | **法律效果不同** |
| 违约金 | 违约金/损害赔偿金 | |
| 欠条 | 借据/欠条 | **法律性质不同** |
| 打官司 | 诉讼/起诉 | |
| 仲裁 | 仲裁 | |
| 担保人 | 保证人/担保人 | |
| 抵押 | 抵押/质押 | **区分动产与不动产** |
| 分红 | 利润分配/股息红利 | |
| 转让 | 转让/让与 | |
| 解除合同 | 解除/终止 | **法律效果不同** |

### 7.1 特别注意

**定金 vs 保证金**：
- 定金：适用定金罚则，交付方违约则定金不予返还，收取方违约则双倍返还
- 保证金：性质取决于约定，不当然适用定金罚则

**借据 vs 欠条**：
- 借据：证明借款关系成立
- 欠条：证明债务存在，但不证明债务原因

**解除 vs 终止**：
- 解除：合同关系向前消灭，可能涉及恢复原状
- 终止：合同关系向后消灭，不涉及已履行部分

## 八、语言质量检查清单

### 8.1 内容检查

- [ ] 主体、对象、标的、时间、具体行为是否准确？
- [ ] 权利义务性质是否准确？
- [ ] 权利义务范围是否明确？

### 8.2 表述检查

- [ ] 层次是否清晰？
- [ ] 语言是否规范，有无歧义？
- [ ] 是否使用了法律术语？
- [ ] 数量、金额、期限是否具体？

### 8.3 格式检查

- [ ] 编号是否连续、统一？
- [ ] 字体字号是否规范？
- [ ] 行距段距是否适当？
- [ ] 页码是否正确？
