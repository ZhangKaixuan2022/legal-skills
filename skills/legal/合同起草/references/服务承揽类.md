# 服务承揽类合同

## 一、适用范围

### 覆盖范围
- **合同库大类**：
  - 四、承揽劳务服务（10个子类，445+模板）
  - 五、中介居间推广行纪（4个子类，89+模板）
  - 十一、承包经营（19模板）
- **典型适用场景**：
  - 加工定制、修理检验
  - 各类专业服务（法律、财务、咨询、设计）
  - 劳务外包、委托代办
  - 中介居间、代理推广
  - 培训教育、会展活动
  - 保管仓储
  - 承包经营

### 法律依据
- 承揽合同：《民法典》合同编第17章（第770-787条）
- 委托合同：《民法典》合同编第23章（第919-936条）
- 中介合同：《民法典》合同编第26章（第961-966条）
- 行纪合同：《民法典》合同编第25章（第951-960条）

## 三、类型特有风险

### 1. 定作人/委托人任意解除权
**承揽合同**：《民法典》第787条规定，定作人在承揽人完成工作前可以随时解除合同。
**委托合同**：《民法典》第933条规定，委托人或者受托人可以随时解除委托合同。
**风险提示**：
- 定作人/委托人享有法定的任意解除权，但应当赔偿损失
- 《民法典》第933条明确：有偿委托合同的解除方应当赔偿对方的**直接损失和可得利益**
**应对建议**（承揽人/受托人角度）：
- 合同中明确约定任意解除需支付的违约金数额或计算方式
- 约定定金作为解约定金，一方解除需承担定金罚则
- 尽量将合同定性为非委托类合同，排除任意解除权

### 2. 承揽与服务与委托的定性争议
**法律影响**：
- **承揽合同**：定作人有任意解除权（第787条），承揽人对工作成果有留置权（第783条），承揽人造成第三人损害时定作人一般不担责（第1193条）
- **委托合同**：双方均有任意解除权（第933条），受托人行为可能对委托人产生代理效力（第925-926条）
**定性参考**：
- 提供有形实物成果→优先定性为承揽
- 处理事务、代理行为→优先定性为委托
- 仅提供劳务服务→可能为无名服务合同
**应对建议**：合同中明确约定合同性质，避免争议时由法院认定。

### 3. 中介报酬请求权
**法律规则**：《民法典》第963条规定，中介人促成合同成立的，委托人应当按照约定支付报酬。中介人未促成合同成立的，不得请求支付报酬。
**风险提示**：
- 中介仅提供信息或机会，未实际促成交易的，无权请求报酬
- 委托人绕过中介直接与第三人交易的（跳单），中介仍可主张报酬
**应对建议**：
- 中介方：明确约定"提供信息即应支付服务费"或约定跳单违约金
- 委托方：明确约定以"促成交易"为付费条件

### 4. 承包经营的法律风险
**法律规则**：企业承包经营涉及《公司法》等规定，承包不改变企业法人资格，发包方仍对外承担责任。
**风险提示**：
- 违法承包（如挂靠经营、出借资质）可能影响合同效力
- 承包人以被承包企业名义从事经营，法律后果由企业承担
- 与劳动关系的区分：承包人一般不与发包企业建立劳动关系
**应对建议**：
- 明确承包人的独立经营主体地位
- 约定承包人违法经营的责任承担

### 5. 承揽人/受托人的留置权
**法律规则**：《民法典》第783条规定，定作人未向承揽人支付报酬或者材料费等价款的，承揽人对完成的工作成果享有留置权或者有权拒绝交付。
**风险提示**：定作人拖欠款项时，承揽人可留置工作成果。
**应对建议**：
- 定作人：约定分阶段付款，避免一次性支付全部款项
- 承揽人：在合同中确认留置权的行使条件和程序

### 6. 服务方造成第三人损害的责任
**法律规则**：《民法典》第1193条规定，承揽人在完成工作过程中造成第三人损害或者自己损害的，定作人不承担侵权责任。但定作人对定作、指示或者选任有过错的，应当承担相应的责任。
**风险提示**：
- 承揽关系中，定作人一般不承担雇主责任
- 但劳务关系中，接受劳务方可能承担雇主责任
**应对建议**：将合同定性为承揽合同，而非劳务合同，可降低定作人的责任风险。

## 五、风险审查要点表

### 1. 标的维度

| 审查项目 | 风险描述 | 应对建议 |
|---------|---------|---------|
| 服务内容界定 | 服务范围不明确导致争议 | 详细列明服务内容清单，附件形式细化 |
| 验收标准 | 成果验收标准不清晰 | 明确约定验收标准、验收程序、验收期限 |
| 质量要求 | 服务或成果质量标准不明 | 约定具体的质量标准或参照行业标准 |
| 资质要求 | 服务方不具备必要资质 | 核实资质证书，合同中列明资质要求 |
| 合同定性 | 承揽/委托/服务界定不清 | 合同中明确约定合同性质 |

### 2. 交易流程维度

| 审查项目 | 风险描述 | 应对建议 |
|---------|---------|---------|
| 交付节点 | 阶段性成果交付不明确 | 分阶段约定交付内容和时间节点 |
| 验收程序 | 验收程序不清导致拖延 | 约定验收期限、验收方式、视为验收通过的情形 |
| 变更程序 | 服务内容变更无程序 | 约定书面变更程序及费用调整机制 |
| 付款与验收挂钩 | 付款不与验收/进度挂钩 | 约定分期付款与阶段验收对应 |

### 3. 权利义务维度

| 审查项目 | 风险描述 | 应对建议 |
|---------|---------|---------|
| 任意解除权 | 委托/承揽合同有法定任意解除权 | 约定解除违约金或转化为非委托类合同 |
| 转委托/分包 | 受托人擅自转委托 | 明确禁止或限制转委托，约定违约责任 |
| 知识产权 | 成果知识产权归属不明 | 明确约定知识产权归属及使用许可 |
| 保密义务 | 商业秘密泄露风险 | 签订保密协议或在合同中约定保密条款 |
| 代理风险 | 受托人以委托人名义行为 | 明确授权范围，控制代理权限 |

### 4. 违约救济维度

| 审查项目 | 风险描述 | 应对建议 |
|---------|---------|---------|
| 违约金约定 | 违约金过高或过低 | 根据可能损失合理约定违约金 |
| 任意解除赔偿 | 解除后损失计算不明 | 明确约定解除时的赔偿计算方式 |
| 迟延履行 | 服务方逾期完成 | 约定逾期违约金及催告后解除权 |
| 质量不符 | 成果质量不达标 | 约定整改、返工、减价、解除等救济措施 |
| 定金条款 | 定金能否抵扣解除权 | 明确定金性质（立约/解约/违约定金） |

### 5. 特殊风险维度

| 审查项目 | 风险描述 | 应对建议 |
|---------|---------|---------|
| 中介跳单 | 委托人绕过中介直接交易 | 约定跳单违约金，明确跳单认定标准 |
| 留置权 | 承揽人行使留置权 | 定作人按约付款，避免触发留置条件 |
| 承包经营责任 | 承包人以企业名义违法经营 | 约定承包人违法经营的责任承担和追偿 |
| 雇主责任 | 服务方造成第三人损害 | 尽量定性为承揽，明确排除雇主责任 |
| 安全责任 | 服务过程中发生安全事故 | 约定安全责任归属，要求投保相应保险 |
