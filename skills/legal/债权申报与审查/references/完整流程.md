# 债权申报与审查Skill

## 功能概述

本Skill适用于破产程序中的债权管理环节，支持三种角色：
- **R1（管理人）**：债权审查与管理，包括发布申报公告、接收登记、实质审查、编制债权表、核查异议处理
- **R2（债权人代理）**：债权申报，包括准备材料、金额计算、提交跟踪、异议应对
- **R3（债务人代理）**：协助核实，包括提供债务信息、发表意见、提供相关材料

### 步骤2：R1管理人流程 - 债权审查与管理

#### 2.1 发布债权申报公告

**触发条件**：【客户编号】+ "债权申报公告"

**法律依据**：
- 《企业破产法》第14条：人民法院应当自裁定受理破产申请之日起二十五日内通知已知债权人，并予以公告
- 《深圳市中级人民法院破产案件审理规程》第28条：管理人应当自收到指定管理人决定书之日起十日内通知已知债权人

**关键时间节点**：
| 事项 | 期限 | 法律依据 |
|-----|------|---------|
| 通知已知债权人 | 受理后10日内 | 操作指引第29条 |
| 发布公告 | 受理后25日内 | 破产法第14条 |
| 债权申报期限 | 30日-90日 | 破产法第45条 |

**公告内容要素**：
1. 案件基本信息（案号、受理法院、受理日期）
2. 债务人基本信息
3. 债权申报期限（明确起止日期）
4. 申报地点及联系方式
5. 申报所需材料清单
6. 第一次债权人会议召开时间、地点
7. 管理人联系方式

**生成文档**：`债权申报公告_【客户姓名】_【YYYYMMDD】.docx`

---

#### 2.2 接收与登记债权申报

**触发条件**：【客户编号】+ "债权登记表"

**登记内容**（操作指引第30条）：
| 登记项目 | 具体内容 |
|---------|---------|
| 债权人基本信息 | 名称/姓名、住所、联系方式 |
| 债权发生原因 | 合同、侵权、无因管理、不当得利等 |
| 债权数额 | 本金、利息、违约金等 |
| 债权性质 | 普通债权/担保债权/职工债权/税款债权 |
| 担保情况 | 是否有担保、担保物、担保权顺序 |
| 证据材料 | 合同、发票、转账凭证、判决书等 |
| 申报日期 | 实际收到申报材料的日期 |
| 申报方式 | 网上申报/书面申报 |

**网上申报效力**：
> 根据《厦门市中级人民法院破产案件债权申报指引》，网上申报与书面申报具有同等法律效力。

**生成文档**：`债权登记表_【客户姓名】_【YYYYMMDD】.xlsx`

---

#### 2.3 实质审查

**触发条件**：【客户编号】+ "债权审查"

**审查内容**（操作指引第31条）：

**（一）真实性审查**
- 债权是否真实发生
- 债权凭证的真实性
- 交易背景的合理性

**（二）合法性审查**
- 债权是否违反法律法规
- 是否存在虚假诉讼
- 是否涉及非法债务

**（三）时效性审查**
- 是否超过诉讼时效
- 是否存在时效中断、中止情形

**（四）数额准确性审查**
- 本金数额确认
- 利息计算（截至受理日，注意24%上限）
- 违约金计算
- 其他费用核实

**审查标准**（参考深圳指引）：

| 债权类型 | 审查标准 |
|---------|---------|
| 有生效法律文书的债权 | 以生效法律文书确认的金额为准 |
| 诉讼/仲裁未决的债权 | 暂缓认定，待裁判生效后确认 |
| 未经诉讼仲裁的债权 | 进行实质审查，核实证据材料 |
| 附利息债权 | 自破产申请受理时起停止计息（破产法第46条） |
| 未到期债权 | 视为到期（破产法第46条） |
| 利息标准 | 超过年利率24%的部分不予认定 |
| 建设工程价款 | 优先受偿权认定（民法典第807条） |
| 有财产担保债权 | 审查担保权设立时间、顺序 |

**利息计算规则**：
```
利息计算截止日期：破产申请受理日
利息计算上限：年利率24%
计算公式：利息 = 本金 × 利率 × 计息天数/365
```

**生成文档**：`债权审查意见书_【债权人名称】_【YYYYMMDD】.docx`

---

#### 2.4 编制债权表

**触发条件**：【客户编号】+ "编制债权表"

**债权分类**（操作指引第32条）：

| 债权类别 | 定义 | 备注 |
|---------|------|------|
| 担保债权 | 以债务人财产设定担保的债权 | 需列明担保财产及担保权顺序 |
| 职工债权 | 债务人所欠职工的工资、医疗、伤残补助、抚恤费用等 | 无需申报，管理人调查公示 |
| 税款债权 | 债务人所欠税款 | 包括滞纳金 |
| 普通债权 | 除上述债权外的其他债权 | 按申报顺序列明 |

**债权表格式**：
| 序号 | 债权人名称 | 申报数额 | 审查认定数额 | 担保财产 | 优先级 | 备注 |
|-----|-----------|---------|-------------|---------|-------|------|
| 1 | | | | | | |
| 2 | | | | | | |

**职工债权公示内容**：
- 职工姓名、身份证号码（部分隐藏）
- 工资、医疗、伤残补助、抚恤费用等具体金额
- 计算期间
- 公示期限（不少于15日）
- 异议处理方式

**生成文档**：
- `债权表_【客户姓名】_【YYYYMMDD】.docx`
- `职工债权公示_【客户姓名】_【YYYYMMDD】.docx`

---

#### 2.5 核查与异议处理

**核查程序**：
1. 将债权表提交第一次债权人会议核查
2. 债务人、债权人对债权表记载的债权有异议的，可提出异议
3. 异议处理：向人民法院提起债权确认之诉

**异议处理流程**：
```
债权人/债务人提出异议
        ↓
管理人核实异议内容
        ↓
确有错误的→更正债权表
        ↓
异议人不服的→向受理法院提起债权确认之诉
```

**诉讼期限**：
- 异议人应在规定期限内（通常为15日）向受理破产案件的法院提起诉讼
- 逾期未起诉的，视为无异议

---

#### 2.6 补充申报

**补充申报规则**：
- 在破产财产最后分配前，债权人可以补充申报债权
- 此前已进行的分配，不再对其补充分配
- 为审查和确认补充申报债权的费用，由补充申报人承担

### 步骤3：R2债权人代理流程 - 债权申报

#### 3.1 准备申报材料

**触发条件**：【客户编号】+ "债权申报书"

**申报材料清单**：

| 材料名称 | 份数 | 要求 |
|---------|------|------|
| 债权申报书 | 原件1份 | 载明债权发生原因、金额、是否有担保 |
| 身份证明 | 复印件1份 | 自然人：身份证；法人：营业执照+法定代表人身份证明 |
| 授权委托书 | 原件1份 | 委托代理人申报时提供 |
| 合同/协议 | 复印件 | 证明债权发生的基础法律关系 |
| 发票/收据 | 复印件 | 证明交易金额 |
| 转账凭证 | 复印件 | 证明款项支付 |
| 判决书/裁定书 | 复印件 | 生效法律文书确认的债权 |
| 担保合同 | 复印件 | 证明担保债权 |
| 其他证据材料 | 复印件 | 与债权相关的其他证明材料 |

**债权申报书内容**：
1. 债权人基本信息
2. 债务人名称
3. 债权发生原因及经过
4. 债权金额（本金、利息、违约金等分项列明）
5. 债权担保情况
6. 证据材料清单
7. 申报日期

**生成文档**：`债权申报书_【客户姓名】_【YYYYMMDD】.docx`

---

#### 3.2 债权金额计算

**计算项目**：

**（一）本金确认**
- 根据合同、发票、转账凭证确认本金数额
- 多笔交易的，分别列明并汇总

**（二）利息计算**
- 计算期间：债权发生日至破产受理日
- 利率标准：按合同约定，但不超过年利率24%
- 计算公式：利息 = 本金 × 年利率 × 计息天数 ÷ 365

**（三）违约金计算**
- 按合同约定的违约金标准计算
- 违约金过高的，可请求调整

**（四）担保物价值评估**
- 担保物为不动产的，提供评估报告或市场价值说明
- 担保物为动产的，提供购置发票或评估报告

**生成文档**：`债权证据清单_【客户姓名】_【YYYYMMDD】.docx`

---

#### 3.3 提交与跟踪

**提交要求**：
- 在债权申报期限内提交（受理后30-90日）
- 可网上申报或书面申报
- 保留申报回执或提交证明

**跟踪事项**：
1. 确认管理人收到申报材料
2. 跟踪管理人审查进度
3. 准备参加第一次债权人会议
4. 核查债权表记载内容

---

#### 3.4 异议应对

**触发条件**：【客户编号】+ "债权异议"/"债权确认"

**异议类型**：

**（一）对管理人不予确认的债权**
- 向管理人提出书面异议
- 异议不成立的，向人民法院提起债权确认之诉

**（二）对其他债权人债权有异议**
- 在债权人会议核查债权表时提出异议
- 向人民法院提起债权确认之诉

**债权确认诉讼材料**：
1. 起诉状
2. 证据材料（申报债权时的全部证据）
3. 管理人审查意见
4. 债权表记载内容
5. 异议书及管理人回复

**生成文档**：
- `债权异议申请书_【客户姓名】_【YYYYMMDD】.docx`（如需）
- `债权确认起诉状_【客户姓名】_【YYYYMMDD】.docx`（如需）

---

### 步骤4：R3债务人代理流程 - 协助核实

#### 4.1 提供债务信息

**协助管理人核实内容**：
- 提供债务人账簿、凭证
- 提供与债权相关的合同、协议
- 提供付款记录、对账单
- 说明债务发生原因及履行情况

#### 4.2 对债权申报发表意见

**发表意见内容**：
- 债权是否真实存在
- 债权金额是否准确
- 债权是否已清偿或部分清偿
- 债权是否存在抵销情形
- 债权是否超过诉讼时效

#### 4.3 提供相关材料

**提供材料清单**：
- 债务人的财务账簿
- 与债权相关的合同、协议
- 付款凭证、银行流水
- 往来函件、对账单
- 其他与债权相关的材料

## 注意事项

1. **期限把控**：严格把控债权申报期限、公告期限、异议期限，避免程序瑕疵
2. **利息计算**：附利息债权自破产申请受理时起停止计息，注意计算截止日期
3. **利率上限**：利息计算注意年利率24%的上限规定
4. **职工债权**：职工债权无需申报，由管理人调查公示
5. **担保债权**：审查担保权设立时间、顺序，确认担保物价值
6. **证据完整**：债权申报材料应完整、真实，证据链应闭合
7. **异议期限**：对债权表有异议的，应在规定期限内提起债权确认之诉
8. **补充申报**：最后分配前可补充申报，但不再享受已进行的补充分配

## Wiki 知识库查阅规则

当处理的案件涉及具体罪名或法律领域时，必须查阅 Wiki 知识库：

1. **检查知识库**：检查 `~/wiki/wiki/` 下是否有对应知识页（刑事：`books/criminal/{罪名}.md`，民事：`books/civil/{领域}/{具体问题}.md`）
2. **读取知识卡片**：若有对应页面，读取页面顶部 `<!-- CARD_START -->` 到 `<!-- CARD_END -->` 之间的知识卡片（约800字浓缩摘要）
3. **按需深入**：卡片信息足够则直接引用；需要深入了解具体辩护策略、案例细节、司法解释原文时，读取全文对应板块
4. **融入输出**：将 Wiki 知识自然融入当前工作输出，引用时标注来源页面路径
5. **案例知识页**：同时检查 `~/wiki/wiki/cases/` 下是否有相关判例知识页可供参考
