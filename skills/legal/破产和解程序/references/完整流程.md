# 破产和解程序Skill

## 功能概述

本Skill定位在破产程序体系中的和解环节，为管理人、债权人代理人、债务人代理人提供破产和解程序的全流程支持。破产和解是指在法院受理破产申请后、宣告破产前，债务人与债权人会议达成和解协议，中止破产程序的法律制度。

### 核心功能

| 功能模块 | 具体内容 |
|---------|---------|
| **和解申请指导** | 协助债务人起草和解申请书，明确申请条件与时机 |
| **协议草案编制** | 生成符合法律要求的和解协议草案，包含清偿方案、债务减免、延期安排 |
| **表决支持** | 提供表决规则说明、协助统计表决结果 |
| **效力确认** | 明确协议对各方当事人的约束力范围 |
| **执行监督** | 监督协议执行，处理协议失败后的转清算程序 |
| **系统记录同步** | 记录和解程序状态、表决结果、执行进度等信息 |

### 重要区分

> **⚠️ 与民事诉讼调解的区别**
> 
> | 对比项 | 破产和解 | 民事诉讼调解 |
> |-------|---------|-------------|
> | **法律依据** | 《企业破产法》第95-99条 | 《民事诉讼法》第48条、第96-102条 |
> | **适用程序** | 破产程序中 | 普通民事诉讼 |
> | **申请主体** | 仅债务人可申请 | 双方当事人均可 |
> | **表决机制** | 债权人会议表决 | 双方协商一致 |
> | **担保影响** | 不影响担保权人权利 | 可约定放弃担保 |
> | **失败后果** | 转破产清算 | 继续诉讼 |

## 第二部分：核心工作流程

### 模块1：和解申请（《企业破产法》第95条）

**申请主体**：仅债务人可以申请和解

**申请时机**：法院受理破产申请后、宣告破产前

**申请要件**：
1. 债务人提出和解申请
2. 提交和解协议草案
3. 说明债务人财产状况
4. 说明和解可行性分析

**法院审查与裁定**：
- 审查内容：申请主体、申请时机、草案内容、可行性
- 裁定结果：裁定和解（中止破产程序）或驳回申请

📋 和解申请书模板参见 [references/破产和解程序文书模板.md](破产和解程序文书模板.md)

### 模块2：和解协议草案编制

#### 2.1 草案内容要求

| 条款类别 | 具体内容 |
|---------|---------|
| **债权清偿方案** | 清偿比例、清偿方式（现金/实物/股权）、清偿期限 |
| **债务减免方案** | 减免金额、减免条件、减免范围 |
| **延期清偿安排** | 分期清偿计划、每期金额、时间节点 |
| **继续经营方案** | 经营计划、资金来源、盈利预测 |
| **担保安排** | 如有新担保，明确担保方式、范围、期限 |
| **违约条款** | 违约情形、违约责任、救济措施 |

#### 2.2 担保债权人权利保护（《企业破产法》第96条）

> **⚠️ 重要提示：和解协议不影响担保债权人权利**
> 
> 根据《企业破产法》第96条规定：
> "对债务人的特定财产享有担保权的权利人，自人民法院裁定和解之日起可以行使权利。"
> 
> **实务要点**：
> 1. 担保债权人对特定财产的担保权不受和解协议约束
> 2. 担保债权人可以就担保财产优先受偿
> 3. 和解协议中的清偿方案仅适用于无担保债权
> 4. 担保债权人可以选择参加和解程序，也可以单独行使担保权

#### 2.3 清偿比例测算

| 项目 | 破产清算 | 和解方案 |
|-----|---------|---------|
| 资产变现总额 | 【金额】 | 【金额】（持续经营价值） |
| 清算费用/管理费用 | 【金额】 | 【金额】 |
| 职工债权 | 【金额】 | 【金额】 |
| 税款债权 | 【金额】 | 【金额】 |
| 有担保债权 | 【金额】（优先受偿） | 【金额】（优先受偿） |
| 可供普通债权分配 | 【金额】 | 【金额】 |
| **清偿比例** | **【比例】** | **【比例】** |

### 模块3：债权人会议表决（《企业破产法》第97条）

#### 3.1 表决规则

**法定要求**（双重标准须同时满足）：
- 由出席会议的有表决权的债权人**过半数同意**
- 且其所代表的债权额占无担保债权总额的**2/3以上**

**表决权计算**：
- 有担保债权人就担保财产享有优先受偿权，其担保债权额不计入表决基数
- 无担保债权总额 = 债权总额 - 有担保债权额
- 弃权票不计入同意或反对票数

#### 3.2 表决程序

1. **会议召集**：管理人提前通知全体债权人（时间、地点、议程、草案）
2. **管理人报告**：报告债务人财产状况
3. **草案说明**：债务人说明和解协议草案
4. **债权人质询**
5. **进行表决**
6. **统计结果、形成决议**

#### 3.3 法院裁定认可

**提交材料**：
1. 债权人会议决议
2. 表决结果统计表
3. 和解协议草案
4. 管理人审查意见

**裁定结果**：
- 裁定认可：和解协议生效
- 裁定不予认可：说明理由，可重新协商或转入清算

### 模块4：和解协议的效力（《企业破产法》第98-99条）

#### 4.1 协议生效时间

和解协议经人民法院裁定认可后生效。

#### 4.2 协议约束力范围

| 主体 | 效力内容 |
|-----|---------|
| **债务人** | 按约履行清偿义务；恢复财产和经营管理权；继续经营并执行方案 |
| **无担保债权人** | 受和解协议约束；按约接受清偿；放弃超出比例部分债权 |
| **担保债权人** | 不受和解协议约束，可单独行使担保权 |
| **管理人** | 终止执行职务；向债务人移交财产和营业事务；提交执行职务报告 |

#### 4.3 程序效力

- **中止破产程序**：解除保全措施，中止相关民事诉讼或仲裁
- **恢复债务人管理权**：债务人恢复对财产和营业事务的管理权

---

### 模块5：和解协议的执行监督

#### 5.1 债务人义务

| 义务类型 | 具体内容 |
|---------|---------|
| **清偿义务** | 按约定比例、期限、方式履行；确保清偿资金来源 |
| **报告义务** | 定期向债权人会议报告经营情况、财务状况和清偿进度；重大事项及时报告 |

#### 5.2 债权人监督权

- 监督债务人是否按约履行清偿义务
- 监督债务人财产状况是否发生重大变化
- 检查债务人是否存在转移财产、欺诈等行为
- 可要求提供财务报告、检查账簿、委托审计

📋 执行监督报告模板参见 [references/破产和解程序文书模板.md](破产和解程序文书模板.md)

---

### 模块6：和解协议的失败与转清算（《企业破产法》第99条）

#### 6.1 终止执行的法定情形

| 情形 | 说明 |
|-----|------|
| **债务人不执行** | 未按期履行、未按约定方式履行、拒绝履行 |
| **债务人不能执行** | 财产不足、经营状况严重恶化、丧失履行能力 |
| **财产状况继续恶化** | 资产大幅减值、负债持续增加、影响协议执行 |
| **欺诈行为** | 虚假陈述、转移财产、隐匿资产 |

#### 6.2 转清算程序

- **申请主体**：债权人、管理人、利害关系人
- **法院裁定**：裁定终止和解协议执行，宣告债务人破产，转入破产清算程序
- **法律后果**：和解协议终止；减免的债权恢复效力；已清偿部分不予返还

## 第三部分：各角色工作重点

### R1（管理人）

| 工作阶段 | 核心职责 |
|---------|---------|
| **和解申请阶段** | 审查和解申请及协议草案的合法性和可行性；向法院提交审查意见 |
| **草案编制阶段** | 协助债务人完善和解协议草案；核实债权债务情况；测算清偿比例 |
| **表决阶段** | 召集债权人会议；组织表决程序；统计表决结果；向法院报告 |
| **协议生效后** | 向法院提交终止职务申请；向债务人移交财产和营业事务；提交执行职务报告 |
| **执行监督阶段** | 协助债权人监督协议执行；发现违约情形及时报告 |
| **转清算阶段** | 申请法院终止和解协议执行；恢复管理人职务；转入清算程序 |

### R2（债权人代理）

| 工作阶段 | 核心职责 |
|---------|---------|

| **转清算阶段** | 申请终止和解协议执行；主张债权恢复；参与清算程序 |

### R3（债务人代理）

| 工作阶段 | 核心职责 |

## 第六部分：关键法条

### 《中华人民共和国企业破产法》

**第九十五条（和解申请）**
> 债务人可以依照本法规定，直接向人民法院申请和解；也可以在人民法院受理破产申请后、宣告债务人破产前，向人民法院申请和解。
> 债务人申请和解，应当提出和解协议草案。

**第九十六条（担保权不受影响）**
> 人民法院经审查认为和解申请符合本法规定的，应当裁定和解，予以公告，并召集债权人会议讨论和解协议草案。
> 对债务人的特定财产享有担保权的权利人，自人民法院裁定和解之日起可以行使权利。

**第九十七条（表决规则）**
> 债权人会议通过和解协议的决议，由出席会议的有表决权的债权人过半数同意，并且其所代表的债权额占无担保债权总额的三分之二以上。

**第九十八条（协议效力）**
> 债权人会议通过和解协议的，由人民法院裁定认可，终止和解程序，并予以公告。管理人应当向债务人移交财产和营业事务，并向人民法院提交执行职务的报告。

**第九十九条（转清算）**
> 和解协议草案经债权人会议表决未获得通过，或者已经债权人会议通过的和解协议未获得人民法院认可的，人民法院应当裁定终止和解程序，并宣告债务人破产。
> 债务人不能执行或者不执行和解协议的，人民法院经和解债权人请求，应当裁定终止和解协议的执行，并宣告债务人破产。

## 第八部分：注意事项

### 8.1 和解申请注意事项

1. **申请时机**：必须在宣告破产前提出，一旦宣告破产，不得再申请和解
2. **申请主体**：仅债务人可申请，债权人、管理人均无权申请
3. **草案完整性**：和解协议草案应具备基本内容，否则法院可能驳回申请

### 8.2 和解协议草案注意事项

1. **清偿比例合理性**：清偿比例应高于预估清算清偿比例，否则难以获得通过
2. **担保债权人权利**：不得损害担保债权人的优先受偿权
3. **履行能力评估**：债务人应具备履行和解协议的能力
4. **经营方案可行性**：继续经营方案应具有可操作性

### 8.3 表决与执行注意事项

1. **表决基数**：以无担保债权总额为表决基数，有担保债权不计入
2. **双重多数**：需同时满足人数过半和债权额2/3以上
3. **严格履行**：债务人应严格按照协议约定履行
4. **违约救济**：发现违约情形应及时申请终止和解协议执行

### 8.4 执业风险提示

1. 注意和解方案是否存在利益冲突
2. 确保债务人财产状况等信息真实完整
3. 和解过程中的信息应严格保密
4. 妥善保存和解过程中的重要文件
