# 破产和解表决程序细则与角色工作重点

## 一、表决规则详解（《企业破产法》第97条）

### 1.1 法定要求

**通过标准**：
- 由出席会议的有表决权的债权人过半数同意
- 且其所代表的债权额占无担保债权总额的2/3以上

**双重标准须同时满足**。

### 1.2 表决权计算

- 有担保债权人就担保财产享有优先受偿权，其担保债权额不计入表决基数
- 无担保债权总额 = 债权总额 - 有担保债权额
- 弃权票不计入同意或反对票数

### 1.3 表决程序

**会议召集**：
1. 管理人提前通知全体债权人
2. 通知内容包括：会议时间、地点、议程、和解协议草案
3. 债权人在会议前审阅和解协议草案

**会议表决**：
1. 管理人报告债务人财产状况
2. 债务人说明和解协议草案
3. 债权人质询
4. 进行表决
5. 统计表决结果
6. 形成会议决议

### 1.4 表决结果统计表

| 序号 | 债权人名称 | 债权性质 | 债权金额（元） | 表决意见 | 代表债权额（元） |
|-----|-----------|---------|--------------|---------|----------------|
| 1 | 【名称】 | 无担保 | 【金额】 | 同意/反对/弃权 | 【金额】 |
| 2 | 【名称】 | 无担保 | 【金额】 | 同意/反对/弃权 | 【金额】 |
| ... | ... | ... | ... | ... | ... |
| **合计** | - | - | **【金额】** | - | **【金额】** |

**表决结果判定**：
- 同意票数：【票数】票，占出席会议有表决权债权人的【比例】%
- 同意债权额：【金额】元，占无担保债权总额的【比例】%
- 表决结果：【通过/未通过】

## 二、法院裁定认可

### 2.1 提交材料

1. 债权人会议决议
2. 表决结果统计表
3. 和解协议草案
4. 管理人审查意见

### 2.2 法院审查内容

- 表决程序是否合法
- 表决结果是否符合法定比例
- 和解协议内容是否损害债权人利益
- 和解协议是否违反法律强制性规定

### 2.3 裁定结果

- **裁定认可和解协议**：和解协议生效
- **裁定不予认可**：说明理由，可重新协商或转入清算

---

## 三、和解协议效力（《企业破产法》第98-99条）

### 3.1 协议生效时间

和解协议经人民法院裁定认可后生效。

### 3.2 协议约束力范围

**对债务人的效力**：
- 按照和解协议约定履行清偿义务
- 恢复对财产和经营的管理权
- 继续经营并执行经营方案

**对债权人的效力**：
- 无担保债权人受和解协议约束
- 按照和解协议约定接受清偿
- 放弃超出清偿比例部分的债权
- 担保债权人不受和解协议约束，可单独行使担保权

**对管理人的效力**：
- 管理人终止执行职务
- 向债务人移交财产和营业事务
- 提交执行职务报告

### 3.3 程序效力

**中止破产程序**：
- 法院裁定和解后，破产程序中止
- 解除对债务人财产的保全措施
- 中止有关债务人的民事诉讼或仲裁（与和解协议履行相关的除外）

**恢复债务人管理权**：
- 债务人恢复对财产的管理权
- 债务人恢复对营业事务的管理权
- 管理人不再执行职务

---

## 四、和解失败与转清算（《企业破产法》第99条）

### 4.1 终止执行的法定情形

1. **债务人不执行和解协议**：
   - 未按期履行清偿义务
   - 未按约定方式履行
   - 拒绝履行

2. **债务人不能执行和解协议**：
   - 财产不足以履行清偿义务
   - 经营状况严重恶化
   - 丧失履行能力

3. **财产状况继续恶化**：
   - 资产大幅减值
   - 负债持续增加
   - 影响和解协议执行

4. **欺诈行为**：
   - 虚假陈述
   - 转移财产
   - 隐匿资产

### 4.2 转清算程序

**申请主体**：债权人、管理人、利害关系人

**法院裁定**：
- 裁定终止和解协议执行
- 宣告债务人破产
- 转入破产清算程序

**法律后果**：
- 和解协议终止
- 减免的债权恢复效力
- 已清偿部分不予返还
- 按照破产清算程序分配剩余财产

## 五、三角色工作重点

### R1（管理人）工作重点

| 工作阶段 | 具体职责 |
|---------|---------|
| **和解申请阶段** | 审查和解申请及协议草案的合法性和可行性；向法院提交审查意见 |
| **草案编制阶段** | 协助债务人完善和解协议草案；核实债权债务情况；测算清偿比例 |
| **表决阶段** | 召集债权人会议；组织表决程序；统计表决结果；向法院报告 |
| **协议生效后** | 向法院提交终止职务申请；向债务人移交财产和营业事务；提交执行职务报告 |
| **执行监督阶段** | 协助债权人监督协议执行；发现违约情形及时报告 |
| **转清算阶段** | 申请法院终止和解协议执行；恢复管理人职务；转入清算程序 |

**注意事项**：
1. 严格审查和解协议草案，确保不损害债权人利益
2. 确保表决程序合法合规
3. 妥善保管破产档案和资料
4. 移交时制作详细的移交清单

### R2（债权人代理）工作重点

| 工作阶段 | 具体职责 |
|---------|---------|
| **草案审查阶段** | 分析和解方案对己方权益影响；比较和解清偿比例vs清算清偿比例；评估和解可行性 |
| **表决阶段** | 向委托人解释表决事项和后果；根据委托人指示参与表决；建议投赞成/反对票 |
| **协议执行阶段** | 监督协议执行情况；定期向委托人报告；发现违约及时采取措施 |
| **转清算阶段** | 申请终止和解协议执行；主张债权恢复；参与清算程序 |

**代理策略建议**：
- 重点关注清偿比例是否优于清算
- 评估债务人履约能力
- 关注担保安排是否充分
- 监督执行进度，及时预警

### R3（债务人代理）工作重点

| 工作阶段 | 具体职责 |
|---------|---------|
| **申请阶段** | 起草和解申请书；准备申请材料；论证和解可行性 |
| **草案编制阶段** | 起草和解协议草案；设计清偿方案；制定继续经营方案 |
| **表决阶段** | 向债权人会议说明和解方案；回答债权人质询；推动协议获得通过 |
| **协议执行阶段** | 协助债务人执行协议；处理履行中的法律问题；防范违约风险 |
| **经营管理阶段** | 协助债务人恢复经营管理；完善公司治理；防范法律风险 |

**注意事项**：
1. 确保和解协议内容合法可行
2. 清偿方案要切合实际，避免过度承诺
3. 制定切实可行的经营改善方案
4. 严格按约履行，避免违约

## 六、关键时间节点

| 节点 | 时间要求 | 法律依据 |
|-----|---------|---------|
| **和解申请** | 法院受理破产申请后、宣告破产前提出 | 《企业破产法》第95条 |
| **草案提交** | 与和解申请同时提交 | 《企业破产法》第95条 |
| **债权人会议** | 法院裁定和解后及时召集 | 实务操作 |
| **表决期限** | 会议召开后当场表决 | 实务操作 |
| **法院裁定认可** | 表决通过后及时申请 | 实务操作 |
| **清偿期限** | 按和解协议约定 | 和解协议 |
| **转清算申请** | 发现违约情形后及时申请 | 《企业破产法》第99条 |

## 七、与民事诉讼调解的重要区分

| 对比项 | 破产和解 | 民事诉讼调解 |
|-------|---------|-------------|
| **法律依据** | 《企业破产法》第95-99条 | 《民事诉讼法》第48条、第96-102条 |
| **适用程序** | 破产程序中 | 普通民事诉讼 |
| **申请主体** | 仅债务人可申请 | 双方当事人均可 |
| **表决机制** | 债权人会议表决（双重多数） | 双方协商一致 |
| **担保影响** | 不影响担保权人权利（第96条） | 可约定放弃担保 |
| **失败后果** | 转破产清算 | 继续诉讼 |
| **约束范围** | 对全体无担保债权人有约束力 | 仅约束协议双方 |
