# 破产和解程序文书模板

## 一、和解申请书

```
和解申请书

申请人（债务人）：【债务人名称】
统一社会信用代码：【代码】
住所地：【地址】
法定代表人：【姓名】，职务：【职务】

申请事项：
请求贵院裁定债务人和解，中止破产程序。

事实与理由：
一、破产程序基本情况
【债务人名称】因【破产原因】，经【申请人名称】申请，贵院于【日期】裁定受理破产申请，案号为【案号】，并指定【管理人名称】担任管理人。

二、债务人财产状况
根据管理人调查，债务人现有资产情况如下：
1. 固定资产：【金额】元
2. 流动资产：【金额】元
3. 无形资产：【金额】元
4. 资产总计：【金额】元

负债情况：
1. 有担保债权：【金额】元，债权人【名称】
2. 职工债权：【金额】元
3. 税款债权：【金额】元
4. 普通债权：【金额】元
5. 负债总计：【金额】元

三、和解可行性分析
1. 债务人具有继续经营的价值和条件：【说明】
2. 通过和解可提高债权清偿比例：
   - 预估清算清偿比例：【比例】
   - 拟议和解清偿比例：【比例】
3. 债务人已制定切实可行的经营改善方案

四、和解协议草案要点
1. 债权清偿方案：【清偿比例、方式、期限】
2. 债务减免方案：【减免范围】
3. 延期清偿安排：【延期期限】
4. 继续经营方案：【经营计划】

五、法律依据
根据《中华人民共和国企业破产法》第95条规定，债务人可以依照本法规定，直接向人民法院申请和解；也可以在人民法院受理破产申请后、宣告债务人破产前，向人民法院申请和解。

综上所述，债务人具备和解条件，和解有利于提高债权清偿比例，维护债权人合法权益，请求贵院裁定债务人和解。

此致
【法院名称】

申请人：【债务人名称】（盖章）
法定代表人：【签名】
日期：    年  月  日

附件：
1. 和解协议草案
2. 债务人财产状况说明
3. 继续经营方案
4. 债权清册
5. 其他相关材料
```

## 二、和解协议草案

```
和解协议草案

协议编号：【编号】
签订日期：【日期】
签订地点：【地点】

甲方（债务人）：【债务人名称】
统一社会信用代码：【代码】
住所地：【地址】
法定代表人：【姓名】

乙方（债权人会议代表）：【代表名称】
代表债权额：【金额】元
代表债权比例：【比例】

鉴于：
1. 甲方因【破产原因】，经【法院名称】于【日期】裁定受理破产申请；
2. 甲方提出和解申请，并提交本和解协议草案；
3. 双方就和解事宜达成一致意见。

根据《中华人民共和国企业破产法》及相关法律规定，经友好协商，达成如下协议：

第一条 债权确认
1.1 经管理人审查确认，乙方代表的无担保债权总额为人民币【金额】元（大写：【大写金额】）。
1.2 有担保债权人的权利不受本协议影响，担保债权人可依法就担保财产优先受偿。

第二条 债权清偿方案
2.1 清偿比例：甲方按照无担保债权总额的【比例】%进行清偿。
2.2 清偿金额：乙方代表债权可获得清偿金额为人民币【金额】元。
2.3 清偿方式：【现金/实物/股权/组合方式】。

第三条 清偿期限与方式
3.1 清偿期限：本协议生效后【期限】内完成全部清偿。
3.2 分期安排：
    第一期：于【日期】前支付【比例】%，计人民币【金额】元；
    第二期：于【日期】前支付【比例】%，计人民币【金额】元；
    第三期：于【日期】前支付剩余【比例】%，计人民币【金额】元。
3.3 收款账户：
    账户名称：【名称】
    开户银行：【银行】
    银行账号：【账号】

第四条 债务减免
4.1 对于本协议约定清偿比例之外的部分债权，乙方同意予以免除。
4.2 减免金额计算：以债权总额减去实际清偿金额为准。

第五条 继续经营方案
5.1 甲方承诺按照以下方案继续经营：
    （一）经营计划：【具体计划】
    （二）资金来源：【资金来源说明】
    （三）盈利预测：【预测说明】
5.2 甲方应定期向债权人会议报告经营情况和财务状况。

第六条 担保安排
【如有新担保安排，详细说明担保方式、担保人、担保范围、担保期限等】

第七条 违约责任
7.1 甲方未按本协议约定履行清偿义务的，构成违约。
7.2 违约情形包括：
    （一）未按期支付任何一期款项；
    （二）未按约定方式履行清偿义务；
    （三）转移、隐匿财产，影响协议履行；
    （四）其他严重影响债权人利益的行为。
7.3 违约责任：
    （一）加速到期：未到期债权立即到期；
    （二）违约金：按未清偿金额的【比例】%支付违约金；
    （三）恢复原债权：减免的债权恢复效力；
    （四）其他救济措施：【具体措施】。

第八条 协议执行监督
8.1 甲方应按照本协议约定履行清偿义务。
8.2 乙方有权监督协议执行情况。
8.3 甲方应定期向乙方报告执行进度。

第九条 和解失败的处理
9.1 有下列情形之一的，视为和解失败：
    （一）甲方不能执行或者不执行和解协议；
    （二）甲方财产状况继续恶化，影响和解协议执行；
    （三）甲方有欺诈行为，损害债权人利益。
9.2 和解失败的后果：
    （一）法院裁定终止和解协议执行；
    （二）宣告甲方破产；
    （三）转入破产清算程序。

第十条 争议解决
10.1 本协议履行过程中发生争议的，双方应协商解决。
10.2 协商不成的，任何一方均可向【法院名称】提起诉讼。

第十一条 其他约定
11.1 本协议经债权人会议表决通过并经法院裁定认可后生效。
11.2 本协议对全体无担保债权人均有约束力。
11.3 本协议一式【份数】份，各方各执一份，具有同等法律效力。

第十二条 生效条件
本协议自以下条件全部满足之日起生效：
1. 债权人会议表决通过；
2. 【法院名称】裁定认可和解协议。

甲方（盖章）：【债务人名称】
法定代表人或授权代表（签字）：
日期：    年  月  日

乙方（债权人会议代表签字）：
日期：    年  月  日
```
