# 各角色工作重点详解

## 一、R1（管理人）工作重点

### 1.1 核心职责总览

| 工作阶段 | 核心职责 |
|---------|---------|
| **编制或审查重整计划草案** | 调查债务人财产状况；分析债权结构；制定债权调整方案；评估经营方案可行性 |
| **组织分组表决** | 召集债权人会议；说明重整计划草案；组织分组表决；统计表决结果 |
| **向法院申请批准** | 准备批准申请材料；说明表决情况；论证重整计划合法性 |
| **监督计划执行** | 定期监督执行情况；编制监督报告；发现异常及时报告 |
| **编制执行情况报告** | 按期提交执行监督报告；申请延长监督期限（如需）；申请终止重整程序 |

### 1.2 详细工作清单

#### 阶段一：重整计划编制

1. 全面调查债务人财产状况
2. 审核债权申报和确认情况
3. 分析债权人结构和分组情况
4. 评估债务人经营能力和重整可行性
5. 测算清偿比例和清偿方案
6. 协调各方利益，制定可行方案
7. 起草或审查重整计划草案
8. 与法院沟通重整计划内容

#### 阶段二：债权人会议表决

1. 确定会议时间、地点
2. 发送会议通知及重整计划草案
3. 准备会议材料
4. 主持债权人会议
5. 说明重整计划草案内容
6. 组织各组分组表决
7. 统计并宣布表决结果
8. 形成会议纪要

#### 阶段三：批准程序

1. 准备批准申请材料
2. 撰写重整计划合法性说明
3. 如适用强制批准，论证符合条件
4. 向法院提交批准申请
5. 配合法院审查

#### 阶段四：执行监督

1. 监督债务人执行情况
2. 核查债权清偿进度
3. 审查资金使用情况
4. 定期编制监督报告
5. 发现异常及时报告法院
6. 必要时申请延长监督期限
7. 执行完毕后申请终止程序

## 二、R2（债权人代理）工作重点

### 2.1 核心职责总览

| 工作阶段 | 核心职责 |
|---------|---------|
| **审查重整计划对己方权益影响** | 分析债权调整方案；测算清偿比例；评估受偿可行性；对比清算与重整收益 |
| **参与分组表决** | 分析投赞成/反对票的利弊；准备表决意见；参与债权人会议表决 |
| **监督执行情况** | 关注重整计划执行进展；核查债权清偿情况；监督债务人履约 |
| **对计划不公平条款提出异议** | 识别歧视性条款；提出修改建议；必要时申请法院审查 |

### 2.3 债权人代理策略要点

#### 赞成票策略

适用情形：
- 重整清偿比例高于清算清偿比例
- 债务人具有重整价值和可行性
- 清偿方案切实可行，履行有保障
- 经营方案合理，有持续盈利可能

#### 反对票策略

适用情形：
- 重整清偿比例低于或等于清算清偿比例
- 债务人不具备重整条件
- 清偿方案缺乏可行性
- 存在损害本债权人利益的条款

#### 附条件同意策略

- 要求修改清偿方案
- 增加担保措施
- 调整清偿期限
- 增加违约责任条款

## 三、R3（债务人代理）工作重点

### 3.1 核心职责总览

| 工作阶段 | 核心职责 |
|---------|---------|
| **参与编制重整计划** | 提供企业经营信息；协助制定经营方案；论证重整可行性 |
| **申请自行管理** | 准备自行管理申请材料；论证自行管理条件；建立监督配合机制 |
| **推动重整计划获批** | 与债权人沟通协商；争取债权人支持；配合法院审查 |
| **配合执行重整计划** | 按计划清偿债务；实施经营方案；接受管理人监督 |

### 3.2 详细工作清单

#### 阶段一：重整申请

1. 评估债务人是否符合重整条件
2. 准备重整申请材料
3. 论证重整原因和重整价值
4. 代理债务人向法院提交申请
5. 配合法院审查和听证

#### 阶段二：自行管理申请

1. 评估债务人是否具备自行管理条件
2. 准备自行管理申请材料
3. 论证自行管理的必要性和可行性
4. 设计管理人监督配合机制
5. 代理债务人提交申请

#### 阶段三：重整计划编制

1. 提供债务人经营和财务信息
2. 参与制定经营方案
3. 协助设计清偿方案
4. 论证重整可行性
5. 审核重整计划草案内容
6. 提出修改完善建议

#### 阶段四：推动表决通过

1. 与主要债权人沟通协商
2. 解释重整计划内容和优势
3. 争取债权人支持
4. 处理债权人异议
5. 必要时调整方案内容

#### 阶段五：计划执行配合

1. 协助债务人按计划执行
2. 监督清偿进度
3. 处理执行中的法律问题
4. 配合管理人监督
5. 定期向委托人报告

### 3.3 债务人代理注意事项

1. **信息披露**：确保向管理人和法院提供真实、完整的信息
2. **利益平衡**：兼顾债务人利益和债权人合理诉求
3. **方案可行性**：确保经营方案和清偿方案切实可行
4. **时间节点**：严格把控各项时间节点，避免逾期
5. **合规经营**：指导债务人在重整期间合规经营
6. **配合监督**：积极配合管理人监督，避免对抗

## 四、三角色协作机制

### 4.1 信息沟通机制

| 沟通事项 | R1（管理人） | R2（债权人代理） | R3（债务人代理） |
|---------|-------------|-----------------|-----------------|
| 重整计划草案 | 主动提供 | 及时审阅 | 参与编制 |
| 债权审查结果 | 主动通报 | 核对确认 | 配合确认 |
| 财产状况报告 | 定期编制 | 审阅监督 | 提供信息 |
| 经营情况报告 | 审核发布 | 关注监督 | 如实报告 |
| 执行进度报告 | 编制报告 | 监督核查 | 配合执行 |

### 4.2 利益协调原则

1. **债权人利益优先**：重整方案应保障债权人获得不低于清算的清偿比例
2. **公平对待**：同一组别债权人应获得平等待遇
3. **兼顾效率**：在保障各方权益前提下，提高重整效率
4. **透明公开**：重大事项及时披露，接受各方监督

### 4.3 争议解决机制

1. **协商优先**：各方就争议事项优先协商解决
2. **管理人调解**：管理人可协调各方立场，促进达成一致
3. **法院裁定**：协商不成的，提请法院裁定
4. **权利救济**：对裁定不服的，依法行使申诉权
