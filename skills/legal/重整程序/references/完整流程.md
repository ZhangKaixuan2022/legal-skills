## 第二部分：工作流程

### 步骤1：读取全量前置文档

读取并解析以下文档（如存在）：
1. **债权表** - 获取债权人清单、债权金额、债权性质、担保情况
2. **财产状况报告** - 获取债务人资产、负债、经营状况
3. **债权人会议纪要** - 获取债权人会议决议、分组意向
4. **审计报告** - 获取财务审计结论、偿债能力分析
5. **评估报告** - 获取资产评估价值、清算价值测算

### 步骤2：确定重整申请主体与时机

#### 2.1 申请主体（《企业破产法》第70条）

| 申请主体 | 申请条件 | 申请时机 |
|---------|---------|---------|
| 债务人 | 具备重整原因 | 可直接申请 |
| 债权人 | 具备重整原因 | 可直接申请 |
| 出资人 | 出资额占注册资本1/10以上 | 法院受理破产清算申请后、宣告破产前 |
| 管理人 | 认为债务人具备重整价值 | 破产清算程序中 |

#### 2.2 重整原因认定

**法律依据**：《企业破产法》第2条

- 企业法人不能清偿到期债务，并且资产不足以清偿全部债务
- 或者明显缺乏清偿能力
- 或者有明显丧失清偿能力可能

#### 2.3 法院裁定重整后的效力

1. **程序效力**：重整程序开始，清算程序中止
2. **保全措施**：有关债务人财产的保全措施应当解除
3. **执行程序**：执行程序应当中止
4. **债务清偿**：债权人不得个别行使权利

### 步骤3：重整期间管理

#### 3.1 管理模式选择

| 模式 | 适用条件 | 说明 |
|-----|---------|------|
| **债务人自行管理** | 经人民法院批准 | 债务人在管理人监督下自行管理财产和营业事务（第73条） |
| **管理人管理** | 默认模式 | 未获自行管理批准或法院认为不宜由债务人管理 |

#### 3.2 重整期间特殊规则

| 规则事项 | 法律规定 | 适用说明 |
|---------|---------|---------|
| 担保权暂停行使 | 第75条 | 重整期间担保权暂停行使，但担保物有毁损或价值明显减少可能除外 |
| 新融资作为共益债务 | 第75条第2款 | 为继续营业而借款，可作为共益债务优先清偿 |
| 股权限制 | 第77条 | 董监高转让股权须经人民法院同意 |
| 取回权限制 | 第76条 | 债务人合法占有的他人财产，权利人要求取回须符合约定条件 |

### 步骤4：重整计划草案编制

#### 4.1 编制主体与期限

| 事项 | 法律规定 | 说明 |
|-----|---------|------|
| 编制主体 | 债务人或管理人 | 债务人自行管理时由债务人编制；管理人管理时由管理人编制 |
| 提交期限 | 6个月 | 自法院裁定重整之日起6个月内提交 |
| 延长期限 | 3个月 | 有正当理由可申请延长3个月 |
| 逾期后果 | 宣告破产 | 逾期未提交或草案未获通过，法院应裁定终止重整程序并宣告破产 |

**法律依据**：《企业破产法》第79条

#### 4.2 重整计划内容（《企业破产法》第81条）

**法定必备内容**：

1. **债务人的经营方案**：主营业务调整、资产处置/重组、引入战略投资人、未来发展规划
2. **债权分类**：担保债权组、职工债权组、税款债权组、普通债权组、出资人权益调整组
3. **债权调整方案**：各类债权调整比例、调整方式（现金清偿/债转股/延期清偿等）
4. **债权受偿方案**：清偿资金来源、清偿时间安排、清偿方式及比例
5. **重整计划执行期限**：明确的执行起止时间、关键节点
6. **监督期限**：管理人监督期限、监督报告提交安排
7. **其他方案**：职工安置方案、业务整合方案等

#### 4.3 债权分类与分组（《企业破产法》第82条）

| 组别 | 债权类型 | 说明 |
|-----|---------|------|
| 第一组 | 担保债权组 | 对债务人特定财产享有担保权的债权 |
| 第二组 | 职工债权组 | 债务人所欠职工工资、医疗伤残补助、抚恤费用、社保费用、补偿金 |
| 第三组 | 税款债权组 | 债务人所欠税款 |
| 第四组 | 普通债权组 | 普通破产债权 |
| 第五组 | 出资人组 | 债务人的出资人权益调整事项 |

**可增设分组**：小额债权组（参考《文书样式》）

### 步骤5：债权分组表决（《企业破产法》第84条）

#### 5.1 表决规则

**通过标准**（双重标准须同时满足）：
- **人数标准**：出席会议的同一表决组的债权人过半数同意
- **金额标准**：同意的债权人所代表的债权额占该组债权总额的2/3以上

#### 5.2 表决程序

1. **会议召集**：管理人提前通知债权人会议时间、地点、议题
2. **草案说明**：管理人或债务人向债权人会议说明重整计划草案
3. **分组讨论**：各表决组分别讨论重整计划草案
4. **分组表决**：各组分别进行表决
5. **结果统计**：统计各组表决结果

### 步骤6：法院批准程序

#### 6.1 正常批准

**适用情形**：各表决组均通过重整计划草案

**法律依据**：《企业破产法》第86条

**程序**：
1. 管理人/债务人向法院提出批准申请
2. 法院审查重整计划内容合法性
3. 法院自收到申请之日起30日内裁定批准
4. 终止重整程序并公告

#### 6.2 强制批准（《企业破产法》第87条）

**适用情形**：部分表决组未通过重整计划草案，但符合法定条件

| 组别 | 强制批准条件 |
|-----|-------------|
| 担保债权组 | 就该特定财产将获得全额清偿，或延期清偿所受损失得到公平补偿，且担保权未受到实质性损害 |
| 职工债权组 | 所获得的清偿比例不低于破产清算程序中可获得的清偿比例 |
| 税款债权组 | 所获得的清偿比例不低于破产清算程序中可获得的清偿比例 |
| 普通债权组 | 所获得的清偿比例不低于破产清算程序中可获得的清偿比例 |
| 出资人组 | 权益调整方案公平、公正，或出资人权益已被调整为零 |

**其他必备条件**：
1. 重整计划公平对待同一表决组的成员
2. 债务人的经营方案具有可行性
3. 债权人会议决议或法院裁定符合法律规定

### 步骤7：重整计划执行与监督

#### 7.1 执行主体与监督

- **执行人**：债务人（《企业破产法》第89条）
- **监督人**：管理人
- **监督期限**：重整计划规定的监督期限

#### 7.2 监督内容

1. 重整计划执行情况
2. 债权清偿情况
3. 经营方案实施情况
4. 资金使用情况

#### 7.3 执行完毕与程序终止

**执行完毕标准**：
- 重整计划规定的清偿义务已履行完毕
- 经营方案已实施完毕
- 其他重整事项已完成

**终止重整程序**（《企业破产法》第94条）：
- 法院裁定重整计划执行完毕
- 重整程序终止
- 按照重整计划减免的债务，债务人不再承担清偿责任

#### 7.4 执行不能的处理（《企业破产法》第93条）

**法律后果**：
1. 经管理人或者利害关系人请求，法院裁定终止重整计划执行
2. 宣告债务人破产
3. 债权人在重整计划中作出的债权调整承诺失去效力
4. 为重整计划执行提供的担保继续有效

## 第三部分：各角色工作重点

### R1（管理人）

| 工作阶段 | 核心职责 |
|---------|---------|
| **编制/审查重整计划** | 调查财产状况；分析债权结构；制定调整方案；评估经营可行性 |
| **组织分组表决** | 召集债权人会议；说明草案；组织表决；统计结果 |
| **向法院申请批准** | 准备材料；说明表决情况；论证合法性 |
| **监督计划执行** | 定期监督；编制报告；发现异常及时报告 |
| **编制执行报告** | 按期提交监督报告；申请延长（如需）；申请终止程序 |

### R2（债权人代理）

| 工作阶段 | 核心职责 |
|---------|---------|
| **审查权益影响** | 分析调整方案；测算清偿比例；对比清算与重整收益 |
| **参与表决** | 分析投票利弊；准备表决意见；参与会议表决 |
| **监督执行** | 关注执行进展；核查清偿情况；监督债务人履约 |
| **提出异议** | 识别不公平条款；提出修改建议；申请法院审查 |

### R3（债务人代理）

| 工作阶段 | 核心职责 |
|---------|---------|
| **参与编制计划** | 提供经营信息；协助制定方案；论证可行性 |
| **申请自行管理** | 准备申请材料；论证条件；建立监督配合机制 |
| **推动获批** | 与债权人沟通；争取支持；配合法院审查 |
| **配合执行** | 按计划清偿；实施经营方案；接受监督 |

### 分段提醒机制

- **3个月提醒**：重整计划编制进度检查
- **5个月提醒**：重整计划提交期限临近预警
- **6个月提醒**：最后期限，必须提交或申请延期
- **执行期提醒**：按重整计划节点提醒执行进度

## 第七部分：注意事项

1. **重整计划合法性**：内容必须符合《企业破产法》第81条规定，不得违反法律强制性规定
2. **债权分类准确性**：严格按法定标准分类，不得损害特定组别利益
3. **清偿比例公平性**：普通债权清偿比例不得低于清算清偿比例（强制批准条件）
4. **经营方案可行性**：经营方案必须具有现实可操作性
5. **表决程序合规性**：严格按法定程序召集债权人会议、组织分组表决
6. **时间节点把控**：严格把控重整计划提交期限，避免逾期宣告破产
7. **记录同步异常处理**：
   - 如当前事项系统记录不存在，提示建立或切换当前事项
   - 如客户编号未能定位当前事项，提示用户核实
   - 回写失败时保存备份并提示用户
   - 如字段不存在则自动新增
