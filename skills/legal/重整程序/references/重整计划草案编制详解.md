# 重整计划草案编制详解

## 一、编制主体与期限

| 事项 | 法律规定 | 说明 |
|-----|---------|------|
| 编制主体 | 债务人或管理人 | 债务人自行管理时由债务人编制；管理人管理时由管理人编制 |
| 提交期限 | 6个月 | 自法院裁定重整之日起6个月内提交 |
| 延长期限 | 3个月 | 有正当理由可申请延长3个月 |
| 逾期后果 | 宣告破产 | 逾期未提交或草案未获通过，法院应裁定终止重整程序并宣告破产 |

**法律依据**：《企业破产法》第79条

---

## 二、重整计划内容（《企业破产法》第81条）

### 2.1 法定必备内容

**1. 债务人的经营方案**
   - 主营业务调整方案
   - 资产处置或重组方案
   - 引入战略投资人方案
   - 未来经营发展规划

**2. 债权分类**
   - 担保债权组
   - 职工债权组
   - 税款债权组
   - 普通债权组
   - 出资人权益调整组（如涉及）

**3. 债权调整方案**
   - 各类债权的调整比例
   - 调整方式（现金清偿、债转股、延期清偿等）
   - 担保债权的特定安排

**4. 债权受偿方案**
   - 清偿资金来源
   - 清偿时间安排
   - 清偿方式及比例

**5. 重整计划的执行期限**
   - 明确的执行起止时间
   - 关键节点安排

**6. 重整计划执行的监督期限**
   - 管理人监督期限
   - 监督报告提交安排

**7. 有利于债务人重整的其他方案**
   - 职工安置方案
   - 业务整合方案
   - 其他必要安排

---

## 三、债权分类与分组

### 3.1 法定分组（《企业破产法》第82条）

| 组别 | 债权类型 | 说明 |
|-----|---------|------|
| 第一组 | 担保债权组 | 对债务人特定财产享有担保权的债权 |
| 第二组 | 职工债权组 | 债务人所欠职工工资、医疗伤残补助、抚恤费用、社保费用、补偿金 |
| 第三组 | 税款债权组 | 债务人所欠税款 |
| 第四组 | 普通债权组 | 普通破产债权 |
| 第五组 | 出资人组 | 债务人的出资人权益调整事项 |

### 3.2 可增设分组

- **小额债权组**：参考《文书样式》，可将小额普通债权单独设组

## 四、重整计划草案模板

```
【债务人名称】重整计划草案

第一部分 债务人基本情况
  一、设立及历史沿革
  二、股权结构
  三、组织架构及人员情况
  四、主营业务及经营状况
  五、资产及负债情况

第二部分 重整原因分析
  一、债务危机形成原因
  二、经营困境分析
  三、重整价值分析

第三部分 债权申报及审查情况
  一、债权申报情况
  二、债权审查情况
  三、债权表

第四部分 债权分类及调整方案
  一、债权分类
  二、债权调整方案
     （一）担保债权组调整方案
     （二）职工债权组调整方案
     （三）税款债权组调整方案
     （四）普通债权组调整方案
     （五）出资人权益调整方案

第五部分 债权受偿方案
  一、清偿资金来源
  二、清偿期限及方式
  三、担保债权清偿安排

第六部分 经营方案
  一、经营维持方案
  二、资产重组方案
  三、引入投资人方案
  四、未来发展规划

第七部分 重整计划执行
  一、执行主体
  二、执行期限
  三、执行监督
  四、执行完毕标准

第八部分 其他事项
  一、重整计划生效条件
  二、重整计划效力
  三、重整计划变更
  四、其他必要安排

附件：
  1. 债权表
  2. 资产评估报告
  3. 审计报告
  4. 其他相关材料
```

## 五、延期提交申请模板

**适用情形**：无法在6个月法定期限内提交重整计划草案

```
延期提交重整计划草案申请

申请人：【债务人名称/管理人名称】

申请事项：
请求贵院准予延期【X】个月提交重整计划草案。

事实与理由：
一、案件基本情况
   【简述重整案件受理时间、当前进展】

二、延期事由
   1. 【事由一：如资产情况复杂需进一步调查】
   2. 【事由二：如债权人人数众多需充分沟通】
   3. 【事由三：如战略投资人尚未确定】

三、延期期间工作安排
   1. 【具体工作计划及时间节点】
   2. 【预计完成时间】

综上所述，因【具体原因】，无法在法定期限内提交重整计划草案，恳请贵院批准延期。

此致
【法院名称】

申请人：【盖章】
XXXX年XX月XX日
```

## 六、债务人自行管理申请模板

**适用情形**：债务人具备自行管理能力，申请法院批准自行管理

```
债务人自行管理申请

申请人：【债务人名称】
住所地：【地址】
法定代表人：【姓名】

申请事项：
请求贵院批准申请人在管理人监督下自行管理财产和营业事务。

事实与理由：
一、债务人具备自行管理的条件
   1. 公司治理结构完整，经营管理团队稳定
   2. 具备继续营业的必要条件
   3. 不存在损害债权人利益的情形

二、自行管理有利于重整成功
   1. 债务人熟悉业务，有利于维持经营价值
   2. 有利于制定切实可行的重整计划
   3. 有利于获得债权人支持

三、管理人监督机制
   1. 定期向管理人报告经营情况
   2. 重大决策须经管理人同意
   3. 接受债权人会议和债权人委员会监督

综上所述，申请人具备自行管理条件，自行管理有利于重整程序顺利进行，恳请贵院批准。

此致
【法院名称】

申请人：【债务人名称】（盖章）
法定代表人：【签字】
XXXX年XX月XX日
```

## 七、重整计划批准申请模板

```
重整计划批准申请

申请人：【债务人名称/管理人名称】

申请事项：
请求贵院裁定批准【债务人名称】重整计划草案。

事实与理由：
一、重整程序基本情况
   【简述重整受理时间、重整计划编制及提交情况】

二、债权人会议表决情况
   【列明各组表决结果】

三、重整计划合法性说明
   1. 债权分类符合法律规定
   2. 债权调整方案公平合理
   3. 经营方案具有可行性
   4. 【如适用强制批准，逐项说明符合第87条各项条件】

四、重整计划批准的必要性
   【说明批准重整计划对各方利益的积极影响】

综上所述，重整计划草案【已获得全部表决组通过/符合强制批准条件】，内容合法可行，恳请贵院裁定批准。

此致
【法院名称】

申请人：【盖章】
XXXX年XX月XX日

附件：
1. 重整计划草案
2. 债权人会议决议
3. 表决结果统计表
4. 其他相关材料
```

```
三、申请事项
   请求贵院裁定确认重整计划执行完毕，并裁定终止重整程序。

XXXX年XX月XX日
```
