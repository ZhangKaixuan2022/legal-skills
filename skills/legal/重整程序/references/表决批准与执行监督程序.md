# 表决批准与执行监督程序详解

## 一、债权分组表决（《企业破产法》第84条）

### 1.1 表决规则

**通过标准**（双重标准须同时满足）：
- **人数标准**：出席会议的同一表决组的债权人过半数同意
- **金额标准**：同意的债权人所代表的债权额占该组债权总额的2/3以上

### 1.2 表决程序

1. **会议召集**：管理人提前通知债权人会议时间、地点、议题
2. **草案说明**：管理人或债务人向债权人会议说明重整计划草案
3. **分组讨论**：各表决组分别讨论重整计划草案
4. **分组表决**：各组分别进行表决
5. **结果统计**：统计各组表决结果

### 1.3 表决结果报告模板

```
重整计划草案表决结果报告

   会议形式：【现场/网络/书面】
   出席情况：应到【X】人，实到【X】人
```

## 二、法院批准程序

### 2.1 正常批准

**适用情形**：各表决组均通过重整计划草案

**法律依据**：《企业破产法》第86条

**程序**：
1. 管理人/债务人向法院提出批准申请
2. 法院审查重整计划内容合法性
3. 法院自收到申请之日起30日内裁定批准
4. 终止重整程序并公告

### 2.2 强制批准（《企业破产法》第87条）

**适用情形**：部分表决组未通过重整计划草案，但符合法定条件

**法定条件**：

| 组别 | 强制批准条件 |
|-----|-------------|
| 担保债权组 | 就该特定财产将获得全额清偿，或延期清偿所受损失得到公平补偿，且担保权未受到实质性损害 |
| 职工债权组 | 所获得的清偿比例不低于破产清算程序中可获得的清偿比例 |
| 税款债权组 | 所获得的清偿比例不低于破产清算程序中可获得的清偿比例 |
| 普通债权组 | 所获得的清偿比例不低于破产清算程序中可获得的清偿比例 |
| 出资人组 | 权益调整方案公平、公正，或出资人权益已被调整为零 |

**其他必备条件**：
1. 重整计划公平对待同一表决组的成员
2. 债务人的经营方案具有可行性
3. 债权人会议决议或法院裁定符合法律规定

## 三、重整计划执行与监督

### 3.1 执行主体与监督

- **执行人**：债务人（《企业破产法》第89条）
- **监督人**：管理人

### 3.2 监督内容

1. 重整计划执行情况
2. 债权清偿情况
3. 经营方案实施情况
4. 资金使用情况

### 3.3 监督期限

- **基本期限**：重整计划规定的监督期限
- **延长监督**：管理人在监督期限届满前可申请延长（《企业破产法》第91条）

### 3.4 执行监督报告模板

```

   （一）债权清偿情况
       1. 担保债权清偿：【金额/比例】
       2. 职工债权清偿：【金额/比例】
       3. 税款债权清偿：【金额/比例】
       4. 普通债权清偿：【金额/比例】

   （二）经营方案执行情况
       1. 【具体执行事项及进展】

   （三）资产重组情况
       【资产处置、投资人引入等进展】

```

## 四、执行完毕与程序终止

### 4.1 执行完毕标准

- 重整计划规定的清偿义务已履行完毕
- 经营方案已实施完毕
- 其他重整事项已完成

### 4.2 终止重整程序（《企业破产法》第94条）

- 法院裁定重整计划执行完毕
- 重整程序终止
- 按照重整计划减免的债务，债务人不再承担清偿责任

### 4.3 执行不能的处理

**适用情形**：债务人不能执行或不执行重整计划（《企业破产法》第93条）

**法律后果**：
1. 经管理人或者利害关系人请求，法院裁定终止重整计划执行
2. 宣告债务人破产
3. 债权人在重整计划中作出的债权调整承诺失去效力
4. 为重整计划执行提供的担保继续有效

## 五、重整程序时间轴

| 时间节点 | 期限 | 法律依据 | 后果 |
|---------|------|---------|------|
| 重整计划提交 | 6个月（可延3个月） | 第79条 | 逾期宣告破产 |
| 法院批准裁定 | 收到申请后30日内 | 第86条 | - |
| 执行期限 | 按重整计划规定 | 第89条 | - |
| 监督期限 | 按重整计划规定 | 第91条 | 可申请延长 |

### 分段提醒机制

- **3个月提醒**：重整计划编制进度检查
- **5个月提醒**：重整计划提交期限临近预警
- **6个月提醒**：最后期限，必须提交或申请延期
- **执行期提醒**：按重整计划节点提醒执行进度
