# 破产清算Skill

## 功能概述

本Skill定位在破产程序中的清算阶段，覆盖从宣告破产到程序终结的全流程工作。主要为R1（管理人）提供工作指引，R2（债权人代理）和R3（债务人代理）参与监督。

### 核心功能

1. **宣告破产处理**：依据破产法第107条，处理宣告破产情形及效力
2. **破产财产范围确认**：界定破产财产范围，识别取回权财产
3. **财产变价方案**：编制变价方案，组织拍卖或整体转让
4. **破产财产分配**：按法定顺序编制和执行分配方案
5. **追加分配处理**：程序终结后2年内发现财产的追加分配
6. **程序终结与注销**：办理程序终结、注销登记、管理人终止职务
7. **各角色工作指引**：区分R1/R2/R3的工作重点和职责

## 第二部分：核心工作流程

### 步骤1：宣告破产（破产法第107条）

#### 1.1 宣告情形

依据《企业破产法》第107条，法院裁定宣告破产的情形：

| 情形 | 法律依据 | 适用条件 |
|-----|---------|---------|
| 情形一 | 第107条第1款 | 债务人符合破产条件且不能重整/和解 |
| 情形二 | 第107条第2款 | 重整计划草案未获通过或批准，重整终止 |
| 情形三 | 第107条第3款 | 和解协议执行失败 |

#### 1.2 宣告效力

宣告破产裁定生效后产生以下法律效力：

1. **债务人身份变更**
   - 债务人成为"破产人"
   - 丧失对财产的管理处分权

2. **财产性质变更**
   - 债务人财产成为"破产财产"
   - 用于清偿破产债权

3. **管理人职责延续**
   - 管理人继续管理破产财产
   - 开始变价和分配工作

#### 1.3 宣告后工作清单

**R1（管理人）**：
- [ ] 接收宣告破产裁定书
- [ ] 通知债权人宣告破产事项
- [ ] 确定破产财产范围
- [ ] 启动财产变价准备工作

**R2（债权人代理）**：
- [ ] 确认宣告破产裁定内容
- [ ] 关注债权确认情况
- [ ] 准备参与变价方案和分配方案表决

**R3（债务人代理）**：
- [ ] 配合管理人移交财产
- [ ] 协助提供财产相关资料

### 步骤2：破产财产范围确认（破产法第30、107条）

#### 2.1 破产财产构成

依据《企业破产法》第30条、第107条，破产财产包括：

| 财产类型 | 范围说明 | 法律依据 |
|---------|---------|---------|
| 现有财产 | 宣告破产时债务人的全部财产 | 第30条 |
| 新增财产 | 破产程序终结前取得的财产 | 第107条 |
| 追回财产 | 管理人依第31-33条追回的财产 | 第31-33条 |

#### 2.2 不属于破产财产的情形

**取回权财产**（权利人可取回）：
- 债务人占有的不属于债务人的财产
- 基于所有权保留的财产
- 基于融资租赁的财产
- 基于借用、保管、仓储等关系的财产

**处理流程**：
1. 权利人向管理人主张取回权
2. 管理人审查取回权主张
3. 确认属实的，返还财产
4. 有争议的，通过诉讼解决

#### 2.3 财产清单核对

**核对要点**：
```
破产财产范围核对清单

一、固定资产
□ 土地使用权及房屋建筑物
□ 机器设备
□ 运输工具
□ 办公设备

二、流动资产
□ 货币资金
□ 应收账款
□ 存货
□ 预付账款

三、无形资产
□ 专利权
□ 商标权
□ 著作权
□ 特许经营权

四、对外投资
□ 股权投资
□ 债权投资

五、其他财产
□ 知识产权
□ 域名
□ 其他可变现资产

六、需排除财产
□ 取回权财产（权利人：____）
□ 担保财产（担保权人：____）
```

### 步骤3：财产变价（破产法第111-112条）

#### 3.1 变价方式（优先顺序）

依据《企业破产法》第111条、第112条，变价方式按以下优先顺序：

| 优先顺序 | 变价方式 | 适用条件 |
|---------|---------|---------|
| 第一 | 拍卖 | 原则上采用拍卖方式 |
| 第二 | 债权人会议决议的其他方式 | 债权人会议另有决议 |
| 第三 | 整体转让 | 对破产企业整体或主要资产 |

#### 3.2 变价方案编制

**方案内容**：
```
破产财产变价方案

一、变价财产范围
（列明拟变价的财产清单）

二、变价方式
1. 拍卖方式：网络拍卖/现场拍卖
2. 拍卖平台：____
3. 整体转让方案（如适用）

三、变价时间安排
1. 评估完成时间：____
2. 拍卖公告时间：____
3. 拍卖时间：____

四、保留价确定
1. 评估价值：____
2. 保留价：____
3. 降价幅度：____

五、变价费用预算
1. 评估费：____
2. 拍卖佣金：____
3. 其他费用：____
```

#### 3.3 变价方案表决程序

**表决流程**：
1. 管理人编制变价方案
2. 提交债权人会议表决
3. 债权人会议通过→执行变价
4. 债权人会议不通过→申请法院裁定

**表决规则**：
- 出席会议的有表决权债权人过半数同意
- 所代表债权额占无财产担保债权总额的二分之一以上

#### 3.4 不动产出租注意事项

参考厦门中院破产审判指引：

| 事项 | 要求 |
|-----|------|
| 租赁形式 | 采用不定期租赁 |
| 租金标准 | 不低于市场价格 |
| 交付时间 | 承租人须在财产变价成交后交付 |
| 合同备案 | 向管理人备案租赁合同 |

### 步骤4：破产财产分配（破产法第113-116条）

#### 4.1 清偿顺序（法定优先级）

依据《企业破产法》第113条，破产财产按以下顺序清偿：

**第一顺位：破产费用和共益债务**
- 随时清偿，不受顺序限制
- 破产费用不足时，按比例清偿

| 类别 | 具体项目 |
|-----|---------|
| 破产费用 | 诉讼费、管理人报酬、管理人执行职务费用、聘用工作人员费用 |
| 共益债务 | 继续营业产生的债务、管理人借款、无因管理、不当得利 |

**第二顺位：职工债权**
- 工资、医疗费、伤残补助、抚恤费
- 基本养老保险、基本医疗保险
- 经济补偿金

**第三顺位：社保费用和税款**
- 欠缴的社会保险费用
- 欠缴的税款

**第四顺位：普通破产债权**
- 无担保的普通债权
- 担保债权未受偿部分

#### 4.2 担保债权处理

**优先受偿规则**：
- 对特定财产享有担保权的债权人→就该特定财产优先受偿
- 超出担保物价值部分→作为普通债权参与分配

**处理流程**：
1. 确认担保权及担保财产价值
2. 担保权人就担保财产优先受偿
3. 计算优先受偿金额
4. 未受偿部分转入普通债权

#### 4.3 分配方案编制

**方案内容**：
```
破产财产分配方案

一、可供分配财产总额
1. 货币资金：____
2. 变价所得：____
3. 其他收入：____
4. 合计：____

二、优先拨付费用
1. 破产费用：____
2. 共益债务：____
3. 小计：____

三、第一顺位：职工债权
1. 债权总额：____
2. 清偿金额：____
3. 清偿比例：____

四、第二顺位：社保费用和税款
1. 社保费用：____
2. 税款：____
3. 小计：____

五、第三顺位：普通债权
1. 债权总额：____
2. 可供分配金额：____
3. 清偿比例：____

六、分配实施安排
1. 中间分配：____
2. 最后分配：____

七、分配公告
1. 公告时间：____
2. 公告方式：____
```

#### 4.4 分配方案表决与实施

**表决程序**：
1. 管理人编制分配方案
2. 提交债权人会议表决
3. 债权人会议通过→执行分配
4. 债权人会议不通过→申请法院裁定

**分配实施**：
- **中间分配**：大额财产变价后可先行分配
- **最后分配**：全部财产变价完毕后进行
- **分配公告**：管理人发布分配公告

### 步骤5：追加分配（破产法第123条）

#### 5.1 追加分配条件

依据《企业破产法》第123条：

| 条件 | 说明 |
|-----|------|
| 时间条件 | 程序终结后2年内 |
| 财产条件 | 发现可供分配的财产 |
| 主体条件 | 债权人可请求法院追加分配 |

#### 5.2 追加分配流程

1. **发现财产**
   - 管理人发现或债权人报告
   - 确认财产属于破产财产

2. **申请追加分配**
   - 债权人向法院提出申请
   - 或管理人主动提请

3. **法院裁定**
   - 法院审查后裁定追加分配
   - 指定管理人执行

4. **实施追加分配**
   - 按原分配方案比例分配
   - 发布追加分配公告

#### 5.3 追加分配注意事项

- 追加分配适用原分配方案的清偿比例
- 追加分配费用从追加财产中优先支付
- 超过2年发现的财产不再追加分配

---

### 步骤6：终结破产程序（破产法第120条）

#### 6.1 终结情形

依据《企业破产法》第120条：

| 情形 | 适用条件 |
|-----|---------|
| 情形一 | 最后分配完结 |
| 情形二 | 无财产可供分配 |

#### 6.2 终结程序流程

**最后分配完结终结**：
1. 管理人完成最后分配
2. 向法院提交终结申请
3. 法院裁定终结破产程序
4. 管理人办理注销登记

**无财产可供分配终结**：
1. 确认无财产可供分配
2. 向法院提交终结申请
3. 法院裁定终结破产程序
4. 管理人办理注销登记

#### 6.3 终结后续工作

**注销登记**：
- 管理人向破产人原登记机关办理注销登记
- 提交注销登记申请及相关材料

**管理人终止职务**：
- 管理人向法院提交终止执行职务报告
- 法院确认后管理人终止职务

**债权处理**：
- 未受清偿的债权不再清偿
- 法人主体资格消灭

## 第三部分：文档生成规范

## 第七部分：注意事项

### 7.1 财产变价注意事项

1. **优先采用拍卖**：原则上应采用拍卖方式变价
2. **保留价合理**：保留价不得低于评估价值的70%
3. **公告充分**：拍卖公告应充分，保障竞买人知情权
4. **整体转让优先**：对破产企业整体或主要资产，优先考虑整体转让

### 7.2 财产分配注意事项

1. **清偿顺序严格**：必须严格按照法定顺序清偿
2. **同顺序按比例**：同一顺序不足清偿的，按比例分配
3. **担保债权优先**：担保债权人就担保财产优先受偿
4. **分配公告及时**：分配方案应公告，保障债权人知情权

### 7.3 程序终结注意事项

1. **注销登记及时**：程序终结后应及时办理注销登记
2. **管理人职责终止**：注销登记后管理人终止执行职务
3. **未受偿债权**：未受清偿的债权不再清偿
4. **追加分配时效**：注意2年追加分配时效

## Wiki 知识库查阅规则

当处理的案件涉及具体罪名或法律领域时，必须查阅 Wiki 知识库：

1. **检查知识库**：检查 `~/wiki/wiki/` 下是否有对应知识页（刑事：`books/criminal/{罪名}.md`，民事：`books/civil/{领域}/{具体问题}.md`）
2. **读取知识卡片**：若有对应页面，读取页面顶部 `<!-- CARD_START -->` 到 `<!-- CARD_END -->` 之间的知识卡片（约800字浓缩摘要）
3. **按需深入**：卡片信息足够则直接引用；需要深入了解具体辩护策略、案例细节、司法解释原文时，读取全文对应板块
4. **融入输出**：将 Wiki 知识自然融入当前工作输出，引用时标注来源页面路径
5. **案例知识页**：同时检查 `~/wiki/wiki/cases/` 下是否有相关判例知识页可供参考
