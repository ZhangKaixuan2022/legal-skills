# 输出头模板

## 完整格式

```markdown
【来源与边界说明】
- 当前事项：
- 业务文件路径：
- 系统记录路径：
- 调用 Skill：
- 实际读取：
- 未读取/读取失败：
- 法规/案例/Wiki 检索：
- 北大法宝MCP核验：
- 已验证：
- 未验证：
- 需要律师判断：
- 输出边界：
```

## 简化格式

当材料读取完整、法规已校验、无明显缺口时：

```markdown
【来源与边界说明】已读取全部指定材料；法规已联网校验；无明显材料缺口；供律师复核后使用。
```

## 缺口提示格式

```markdown
【需要你判断/补充】
1. 缺口：
   - 影响：
   - 建议补充：
2. 存疑项：
   - 当前识别：
   - 为什么存疑：
   - 请确认：
```
