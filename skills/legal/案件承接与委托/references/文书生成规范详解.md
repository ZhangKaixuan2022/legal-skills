# 刑事案件承接文书生成规范详解

---

## 一、版面格式规范（所有文书必须遵循）

| 项目 | 规格 |
|------|------|
| 页边距（上下） | 25.4mm |
| 页边距（左右） | 31.8mm |
| 标题 | 黑体四号加粗 |
| 一级标题 | 黑体四号加粗，缩进2字符 |
| 二级标题 | 宋体四号加粗，缩进2字符 |
| 正文 | 宋体四号，首行缩进2字符 |
| 行距 | 固定27磅 |

---

## 二、标题命名规范

**标题构成**：当事人 + 案件名称 + 诉讼阶段 + 事项

**侦查阶段表述**："涉嫌某罪一案"

**示例**：
- 张三涉嫌故意伤害罪一案刑事委托代理合同
- 张三涉嫌故意伤害罪一案授权委托书
- 张三涉嫌故意伤害罪一案律师事务所函

---

## 三、法律依据引用规范

文书中涉及法律依据时，必须准确引用：
- 《中华人民共和国刑事诉讼法》
- 《中华人民共和国律师法》
- 《律师执业管理办法》
- 相关司法解释

---

## 四、刑事法律服务方案格式说明

**文档标题格式**：《关于XXX涉嫌XXX罪一案刑事法律服务方案》

**示例**：
- 《关于张三涉嫌故意伤害罪一案刑事法律服务方案》
- 《关于李四涉嫌诈骗罪一案刑事法律服务方案》

**文档格式规范**：
| 项目 | 规格 |
|------|------|
| 文档标题 | 黑体二号加粗，居中 |
| 一级标题（一、二、三…） | 黑体四号加粗 |
| 二级标题（（一）（二）…） | 宋体四号加粗 |
| 表格标题行 | 宋体四号加粗 |
| 正文及表格内容 | 宋体四号 |
| 行距 | 固定27磅 |
| 页边距 | 上下25.4mm，左右31.8mm |

**输出路径**：保存至法律工作总控确认的当前事项业务文件路径下的案件承接目录

**文件命名规范**：【当事人姓名】涉嫌【罪名】罪一案刑事法律服务方案.docx

**正式 Word 导出规范**：
```
业务文书先生成语义 HTML，并经 `法律文书出稿前审查` 生成 `draft_checked.html`；审查状态为 `PASS` 或 `FIXED_PASS` 后，调用 `法律文书模板与导出` 统一导出 `.docx`。

导出元数据：
- 文档类型：刑事法律服务方案
- 模板结构：九部分（案件基本信息、案情概述、案件分析、案件结果预判、法律服务内容、工作计划、收费方案、律师团队介绍、特别说明）
- 数据来源：步骤1采集的案件信息 + 步骤2评估结果 + 律师确认的收费金额
- 输出路径：01-案件承接/1-接待/
```

## 五、各类文书核心内容要求

### 5.1 受案登记审批表

**核心内容**：
- 案件名称、案件类型
- 委托人姓名、当事人姓名
- 案件所处阶段、办案机关、承办人
- 基本案情简介
- 是否被羁押、羁押地点
- 利益冲突审查结果
- 接待律师、跟进律师
- 审批意见

### 5.2 刑事委托代理合同

**核心结构要素**：
- 委托人信息（姓名、身份证号、联系方式、住址）
- 受托律师事务所及律师信息
- 委托事项（为犯罪嫌疑人/被告人XXX涉嫌XXX罪一案提供辩护）
- 委托事项——辩护阶段范围：侦查/审查起诉/一审/二审
- 代理权限（一般授权/特别授权）
- 律师费用及支付方式
- 双方权利义务
- 合同期限及解除条件
- 争议解决方式
- 签订日期及地点

**签订规范**：
- 委托协议可以由委托人以外的人签署
- 可一次性签订委托协议，但应分阶段签署授权委托书
- 一式二份，一份交委托人，一份律所存档

### 5.3 授权委托书

**核心结构要素**：
- 委托人基本信息
- 受委托律师姓名、执业证号、律师事务所名称
- 委托权限范围
- 委托期限
- 委托人签名、日期

**签署规范**：
1. 每个阶段至少一式三份
2. 三个阶段一般需要签署九份授权委托书
3. 委托人应在授权委托书上签字捺指印
4. 犯罪嫌疑人被羁押的，首次会见时由其签名捺指印确认

### 5.4 律师事务所函

**核心结构要素**：
- 致函对象（办案机关名称）
- 律师事务所名称及介绍
- 指派律师姓名及执业证号
- 案件信息（当事人姓名、涉嫌罪名）
- 请求事项（安排会见、提供案件相关信息等）
- 律师事务所通信地址和律师联系电话（务必注明）
- 律师事务所盖章、日期

**递交规范**：
- 接受委托后及时递交
- 原则上当面向承办人递交
- 特殊情况可电话确认后寄交
- 先提交委托手续后，再开展辩护工作
