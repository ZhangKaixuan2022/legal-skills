# 刑事辩护案件承接与委托Skill

## 功能概述

依据《刑事辩护实务操作技能与执业风险防范》（徐宗新著）第一章"基本原则"及第二章"收案"、《刑事辩护规范化：文书卷宗示范》第一编"收案"的规范要求，协助律师完成刑事案件的**初次接待、案件评估、风险告知、委托手续办理及案件信息采集**，生成标准化委托文书并建立案件档案。

**核心价值**：规范化、标准化完成刑事案件承接全流程，确保委托关系合法有效，防范执业风险。

**输入**：客户编号、委托人/当事人信息、案件基本情况
**输出**：刑事法律服务方案、刑事委托代理合同、授权委托书、委托人权利义务告知书、律师事务所函、案件档案文件夹

---

## 刑事辩护十大基本原则（精要版）

**本原则贯穿刑事辩护全流程所有阶段，是刑事辩护律师执业的根本行为准则。**

| 序号 | 原则名称 | 核心要义 |
|------|---------|---------|
| 1 | 以事实为依据 | 观点必须基于事实和证据 |
| 2 | 以法律为准绳 | 每一步工作都应有法律依据 |
| 3 | 恪守职业道德 | 遵守执业纪律，不违规执业 |
| 4 | 维护合法权益 | 用正当合法手段维护当事人合法权益 |
| 5 | 独立辩护 | 依法独立进行，不受委托人意志不当限制 |
| 6 | 保守秘密 | 保守国家秘密、商业秘密和当事人隐私 |
| 7 | 有理有利有节 | 不请客送礼、不贿赂司法人员 |
| 8 | 不打包票 | 只承诺专业能力和敬业态度 |
| 9 | 不贬损同行 | 不贬损其他律师和律师事务所 |
| 10 | 不诋毁司法 | 不诋毁司法机关形象 |

### 步骤1：案件信息初步采集（摘要）

**咨询四步要诀**：
- **认真听**：不要打断当事人的发言，让当事人在平和状态下将所知所想都说出来
- **着重记**：将当事人回答及主要案件事实记录下来，必要时画图厘清案情
- **适时问**：针对案件相关要素适时发问（如被告人年龄、刑拘时间、逮捕罪名等）
- **全面说**：全面介绍实体法、程序法、权利义务、律师作用及服务方案

**必采信息**：
- 当事人信息：姓名、性别、出生日期、身份证号、住址、联系电话、职业、是否有前科
- 委托人信息：姓名、与当事人关系、身份证号、联系电话
- 案件信息：涉嫌罪名、案件阶段、办案机关、羁押场所、拘留/逮捕日期、案情简述

**委托人资格**：犯罪嫌疑人本人、法定代理人、近亲属（配偶、父母、子女、同胞兄弟姐妹）、被指定的其他亲友或单位

**交互规则（强制要求）**：每次交互必须一次性列出所有缺失字段，用户确认后方可继续，不得分多轮追问单个字段。

---

### 步骤2：案件初步评估（摘要）

**利益冲突审查（前置必查项）**：
- 同案代理冲突 → 拒绝受理
- 同所代理冲突 → 审查后决定（如存在主从犯之争等利益冲突，即便当事人同意也不得受理）
- 关联案件冲突 → 拒绝受理
- 被害方代理冲突 → 拒绝受理

**案件复杂程度评估**：

| 等级 | 判断标准 | 典型情形 |
|------|---------|---------|
| 简单 | 事实清楚、证据充分、单一罪名 | 轻微盗窃、交通肇事（无争议）、醉驾 |
| 一般 | 事实基本清楚、涉及一至两个争议焦点 | 一般故意伤害、普通诈骗、职务侵占 |
| 复杂 | 事实存在重大争议、多个罪名、证据链不完整 | 共同犯罪、经济犯罪、非法证据排除 |
| 重大 | 可能判处无期徒刑或死刑、社会影响重大 | 故意杀人、贩毒大案、涉黑涉恶 |

**评估内容**：
1. 案件性质判断：罪名构成要件分析、可能刑罚幅度
2. 案件复杂程度：简单/一般/复杂/重大
3. 辩护方向：无罪/罪轻/程序/量刑辩护
4. 认罪认罚适用性评估
5. 预计办案周期和工作量

---

### 步骤3：出具刑事法律服务方案（摘要）

**服务方案九部分内容**：
1. 案件基本信息
2. 案情概述
3. 案件分析（罪名法条、构成要件、量刑幅度、有利/不利因素）
4. 案件结果预判（最严重/最轻/最可能结果）
5. 法律服务内容（根据案件阶段）
6. 工作计划与时间安排
7. 收费方案（金额由律师填写，AI不自动生成）
8. 律师团队介绍
9. 特别说明

**预计办案周期参考**：

| 阶段 | 标准期限 | 延长情形 |
|------|---------|---------|
| 侦查阶段（刑拘） | 一般14日，最长37日 | 流窜、多次、结伙作案 |
| 侦查阶段（逮捕后） | 一般2个月 | 复杂案件可延长1-4个月 |
| 审查起诉阶段 | 一般1个月，可延长0.5个月 | 退回补充侦查各1个月 |
| 一审阶段 | 2-3个月 | 死刑、附带民事可延长3个月 |
| 二审阶段 | 2个月 | 复杂案件可延长2个月 |

📋 文书格式详见 → [references/文书生成规范详解.md](文书生成规范详解.md)

---

### 步骤4：风险告知（摘要）

**告知内容**：
1. 刑事辩护的一般风险（结果风险、证据风险、司法裁量风险、案件变化风险）
2. 律师执业范围和限制
3. 犯罪嫌疑人的合法权益（辩解权、休息权、申诉控告权、委托律师权）
4. 会见、阅卷的权利与时间安排
5. 费用收取标准和方式说明
6. 委托人的权利和义务
7. 拒绝辩护的法定事由

**犯罪嫌疑人四项权利**：
- **辩解权**：如实陈述，要求如实记录，可拒签不符笔录
- **休息权**：遭遇连续审讯可要求休息
- **申诉控告权**：对违法办案可申诉控告
- **委托律师权**：委托辩护律师，获得法律帮助

⚠️ 告知后必须要求用户确认"已知悉"

---

### 步骤5：委托手续办理（摘要）

**生成文书清单**：
1. 受案登记审批表
2. 刑事委托代理合同
3. 授权委托书（每阶段至少三份）
4. 委托人权利义务告知书
5. 律师事务所函
6. 案件流程告知书
7. 重大敏感案件备案文书（如适用）

**关键操作要点**：
- **授权委托书**：每个阶段至少三份，委托人签字捺指印，被羁押者首次会见时由犯罪嫌疑人签名捺指印确认
- **律师事务所函**：须注明律师事务所通信地址和律师联系电话
- **委托手续递交**：先提交委托手续，后开展辩护工作

📋 文书核心内容要求 → [references/文书生成规范详解.md](文书生成规范详解.md)

### 步骤6：建立案件档案（摘要）

**基础路径**：`<当前事项业务文件路径>/`

**档案结构**：
```
【当事人姓名】/
├── 01-案件承接/（接待、受案、律所证明、其他）
├── 02-侦查阶段/
├── 03-审查起诉阶段/
├── 04-一审阶段/
├── 05-二审阶段/
└── 06-其他/
```

> **强制规则**：系统记录同步必须按法律工作总控确认后的当前事项和记录路径执行；涉及复盘台账字段变更时，先向用户说明更新内容，不得静默写入。

**复盘台账更新**：更新 复盘台账（按法律工作总控确认后读取/更新） 中的案件信息和状态

---

### 步骤7：确认与交付

**交付流程**：
1. 展示所有生成文书的摘要供律师逐项确认
2. 保存所有文书至案件档案文件夹
3. 更新复盘台账
4. 提示下一步工作建议（启动侦查阶段辩护Skill）

**费用约定规范**：
- 费用必须书面约定，不得口头承诺
- 明确收费金额、支付时间、支付方式
- 明确费用是否包含差旅费、鉴定费等
- 委托人应将费用直接支付至律师事务所账户

## 执业风险防范要点（精要版）

### 禁止事项

| 禁止行为 | 法律依据 |
|---------|---------|
| 承诺案件结果 | 《律师法》第四十条 |
| 接受非法请求 | 《刑事诉讼法》第四十二条 |
| 泄露案件信息 | 《律师法》第三十八条 |
| 私自收取费用 | 《律师服务收费管理办法》 |
| 帮助串供/伪造证据 | 《刑事诉讼法》第四十二条 |
| 向司法人员行贿 | 《律师法》第四十九条 |
| 以与司法机关有特殊关系招揽业务 | 《律师职业道德和执业纪律规范》第二十一条 |
| 违规会见、传递信息 | 《律师职业道德和执业纪律规范》第二十三条 |

### 必须事项

- 严格核实委托人身份和委托权限
- 律师费用书面约定，明确透明
- 会见笔录和沟通记录必须留存
- 授权委托书须由犯罪嫌疑人本人确认（被羁押者首次会见时）
- 委托手续及时递交办案机关
- 重大敏感案件依规备案

### 拒绝辩护与代理

**拒绝事由**：委托事项违法、利用律师服务从事违法活动、故意隐瞒与案件有关的重要事实、提出不合理要求致使律师无法正常履职

**处理程序**：经律师事务所主任或授权负责人同意 → 告知委托人 → 记录在卷 → 报律师协会备案 → 向办案机关出具解除委托关系的公函

## 材料收集清单（摘要）

### 委托人应提供的必备材料

- 委托人身份证原件及复印件
- 当事人身份证复印件
- 近亲属关系证明（如适用）
- 拘留/逮捕通知书（如已收到）
- 案情说明材料

### 律师应准备的材料

**接待阶段**：接待登记表、接待笔录、服务方案

**受案阶段**：受案登记审批表、委托协议、委托人须知、刑事授权委托书、收费凭证、案件流程告知书

**会见材料**：律师执业证（原件）、律师事务所证明/所函、授权委托书、会见介绍信及存根

**材料收集注意事项**：
- **身份核实**：必须核实委托人身份证原件，留存复印件
- **关系证明**：近亲属代为委托的，必须核实并留存亲属关系证明
- **签字捺印**：授权委托书须委托人签字捺印
- **存根管理**：律师事务所函、会见介绍信等须留存根备查

## 错误处理机制

| 错误类型 | 处理方式 |
|---------|---------|
| 信息采集不完整 | 一次性列出全部缺失项，要求补充 |
| 委托人身份无法核实 | 提示需提供有效身份证明，暂停办理 |
| 存在利益冲突 | 告知委托人，依法拒绝委托 |
| 文书生成失败 | 检查路径权限，询问是否更换路径 |
| 档案创建失败 | 报告错误，手动指定替代路径 |

## Wiki 知识库查阅规则

当处理的案件涉及具体罪名或法律领域时，必须查阅 Wiki 知识库：

1. **检查知识库**：检查 `~/wiki/wiki/` 下是否有对应知识页（刑事：`books/criminal/{罪名}.md`，民事：`books/civil/{领域}/{具体问题}.md`）
