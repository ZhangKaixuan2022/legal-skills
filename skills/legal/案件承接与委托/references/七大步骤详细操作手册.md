# 刑事案件承接七大步骤详细操作手册

依据《刑事辩护实务操作技能与执业风险防范》第二章"收案"及《刑事辩护规范化：文书卷宗示范》第一编"收案"。

---

## 步骤1：案件信息初步采集

**依据**：《刑事辩护实务操作技能与执业风险防范》第二章"咨询四步要诀"

### 咨询四步要诀

1. **认真听**：不要打断当事人的发言，让当事人在平和状态下将所知所想都说出来
2. **着重记**：将当事人回答及主要案件事实记录下来，必要时画图厘清案情
3. **适时问**：针对案件相关要素适时发问（如被告人年龄、刑拘时间、逮捕罪名等）
4. **全面说**：全面介绍实体法、程序法、权利义务、律师作用及服务方案

### 1.1 当事人信息（犯罪嫌疑人/被告人）

| 字段 | 说明 | 必填 |
|------|------|------|
| 姓名 | 犯罪嫌疑人/被告人姓名 | 是 |
| 性别 | 男/女 | 是 |
| 出生日期 | YYYY年MM月DD日 | 是 |
| 身份证号 | 18位身份证号码 | 是 |
| 住址 | 户籍地址或经常居住地 | 是 |
| 联系电话 | 如被羁押，填写家属电话 | 是 |
| 职业 | 工作单位及职务 | 是 |
| 文化程度 | 教育背景 | 否 |
| 是否有前科 | 是否曾受刑事处罚、是否累犯 | 是 |

### 1.2 委托人信息（如非当事人本人）

| 字段 | 说明 | 必填 |
|------|------|------|
| 姓名 | 委托人姓名 | 是 |
| 与当事人关系 | 近亲属需说明具体关系（配偶、父母、子女、同胞兄弟姐妹） | 是 |
| 身份证号 | 18位身份证号码 | 是 |
| 联系电话 | 委托人联系方式 | 是 |
| 住址 | 联系地址 | 是 |

**委托人资格**：
1. **犯罪嫌疑人、被告人本人**
2. **法定代理人**：未成年人、限制行为能力人的法定代理人
3. **近亲属**：配偶、父母、子女、同胞兄弟姐妹
4. **犯罪嫌疑人、被告人指定的人**：包括其他亲友、所在单位

### 1.3 案件基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 涉嫌罪名 | 可能涉及多罪名 | 是 |
| 案件当前所处阶段 | 侦查/审查起诉/一审/二审 | 是 |
| 办案机关 | 公安局/检察院/法院名称 | 是 |
| 承办人及联系方式 | 办案人员姓名、电话 | 否 |
| 案件编号 | 如尚未获知可后补 | 否 |
| 羁押场所 | 看守所名称 | 是 |
| 被拘留/逮捕日期 | YYYY年MM月DD日 | 是 |
| 案情简述 | 委托人描述的基本事实 | 是 |
| 是否有同案犯 | 是/否，如有需说明人数及关系 | 是 |
| 是否有被害人 | 是/否，如有需说明人数及伤亡情况 | 是 |
| 此前是否已有律师介入 | 是/否，如有需说明律师信息 | 是 |
| 案件来源 | 电话/网络/他人介绍/直接上门 | 否 |

### 1.4 咨询时应向委托人讲明的八项内容

| 序号 | 告知事项 | 要点 |
|------|---------|------|
| 1 | 刑事诉讼程序及期限 | 侦查、审查起诉、一审、二审、再审、死刑复核的程序及诉讼期限 |
| 2 | 所涉罪名及刑罚 | 罪名构成要件、量刑档次、法定刑幅度 |
| 3 | 强制措施 | 拘传、取保候审、监视居住、刑事拘留、逮捕的种类和适用条件 |
| 4 | 从轻情节 | 法定和酌定的从轻、减轻、免除处罚情节 |
| 5 | 合法权益 | 犯罪嫌疑人及其亲友的合法权益 |
| 6 | 法律服务方案 | 律师的作用、服务内容、工作安排 |
| 7 | 案件发展趋势 | 可能的结果预判（最严重/最轻/最可能的结果） |
| 8 | 收费标准 | 律师事务所收案程序与收费标准 |

## 步骤2：案件初步评估

### 2.1 利益冲突审查（前置必查项）

| 审查事项 | 冲突情形 | 处理方式 |
|---------|---------|---------|
| 同案代理 | 同一律师不得接受同一案件两名以上犯罪嫌疑人及其亲属的委托 | 拒绝受理 |
| 同所代理 | 同一律师事务所分别为同案共犯指派不同律师，须告知委托人并经其同意；如存在主从犯之争等利益冲突，即便当事人同意也不得受理 | 审查后决定 |
| 关联案件 | 同一律师不得为两名或两名以上未同案处理但犯罪事实存在关联的被告人辩护 | 拒绝受理 |
| 被害方代理 | 同时为同一案件被害人代理与犯罪嫌疑人辩护 | 拒绝受理 |

### 2.2 案件性质判断

**罪名构成要件分析**：
- 明确涉嫌罪名的主体、主观方面、客观方面、客体
- 分析定罪的关键要件
- 识别罪与非罪、此罪与彼罪的区分要点

### 2.3 案件复杂程度评估

| 等级 | 判断标准 | 典型情形 |
|------|---------|---------|
| 简单 | 事实清楚、证据充分、单一罪名 | 轻微盗窃、交通肇事（无争议）、醉驾 |
| 一般 | 事实基本清楚、涉及一至两个争议焦点 | 一般故意伤害、普通诈骗、职务侵占 |
| 复杂 | 事实存在重大争议、多个罪名、证据链不完整 | 共同犯罪、经济犯罪、非法证据排除 |
| 重大 | 可能判处无期徒刑或死刑、社会影响重大 | 故意杀人、贩毒大案、涉黑涉恶 |

### 2.4 辩护方向初步判断

| 类型 | 适用情形 |
|------|---------|
| 无罪辩护 | 事实不清、证据不足，或不构成犯罪 |
| 罪轻辩护 | 罪名较轻、情节较轻、从轻减轻情节 |
| 程序辩护 | 侦查程序违法、非法证据排除 |
| 量刑辩护 | 事实清楚，争取从轻、减轻处罚 |

### 2.5 预计办案周期

| 阶段 | 标准期限 | 延长情形 |
|------|---------|---------|
| 侦查阶段（刑拘） | 一般14日，最长37日 | 流窜、多次、结伙作案 |
| 侦查阶段（逮捕后） | 一般2个月 | 复杂案件可延长1-4个月 |
| 审查起诉阶段 | 一般1个月，可延长0.5个月 | 退回补充侦查各1个月 |
| 一审阶段 | 2-3个月 | 死刑、附带民事可延长3个月 |
| 二审阶段 | 2个月 | 复杂案件可延长2个月 |

## 步骤3：出具刑事法律服务方案

### 服务方案文档内容结构

**一、案件基本信息**：当事人姓名、涉嫌罪名、案件阶段、办案机关等

**二、案情概述**：基于委托人描述的案件基本事实、时间线

**三、案件分析**：
- 涉嫌罪名法律规定及构成要件
- 量刑幅度分析
- 有利因素与不利因素
- 辩护方向初步判断

**四、案件结果预判**：最严重结果、最轻结果、最可能结果

**五、法律服务内容**：根据案件阶段列明服务项目

**六、工作计划与时间安排**

**七、收费方案**（金额由律师填写）

**八、律师团队介绍**

**九、特别说明**

## 步骤4：风险告知

### 4.1 刑事辩护的一般风险

1. **结果风险**：不能保证案件必然达到委托人期望的结果
2. **证据风险**：案件结果取决于事实和证据
3. **司法裁量风险**：最终裁判由人民法院独立作出
4. **案件变化风险**：罪名、情节在诉讼中可能变化

### 4.2 律师执业范围和限制

**律师的作用**：法律帮助、法律咨询、亲情沟通、案件跟踪、申诉控告

**律师的执业限制**：
- 不得帮助串供、伪造证据、妨害作证
- 不得向司法工作人员行贿
- 不得私自收取费用
- 不得违规会见、传递信息

### 4.3 犯罪嫌疑人的合法权益

| 权利名称 | 内容 |
|---------|------|
| 辩解权 | 如实陈述，要求如实记录，可拒签不符笔录 |
| 休息权 | 遭遇连续审讯可要求休息 |
| 申诉控告权 | 对违法办案可申诉控告 |
| 委托律师权 | 委托辩护律师，获得法律帮助 |
| 处理紧急事务权 | 委托律师处理紧急事务 |

### 4.4 委托人的权利和义务

**权利**：知情权、建议权、解除委托权

**义务**：如实陈述、按时支付费用、配合律师工作、不得要求违法行为

## 步骤5：委托手续办理

### 5.1 受案登记审批表

案件名称、委托人、当事人、案件阶段、利益冲突审查结果、审批意见

### 5.2 刑事委托代理合同

- 委托人信息、受托律师事务所及律师信息
- 委托事项及阶段范围
- 代理权限、律师费用及支付方式
- 双方权利义务、合同期限
- 签订日期及地点

### 5.3 授权委托书

- 每个阶段至少三份
- 委托人签字捺指印
- 被羁押者首次会见时由犯罪嫌疑人签名捺指印确认

### 5.4 委托人权利义务告知书

委托人权利（知情权、建议权、解除委托权）及义务

### 5.5 律师事务所函

致函对象、指派律师信息、请求事项、联系方式
