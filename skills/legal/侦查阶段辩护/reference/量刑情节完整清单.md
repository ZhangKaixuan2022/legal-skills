## 二、自首认定要点

### 一般自首（刑法第67条第1款）

**自动投案**：
- 在尚未受到讯问、未被采取强制措施时主动投案
- 向公安机关、检察机关、法院或其他有关机关投案
- 委托他人先代为投案
- 先以信电投案
- 因病、伤或为减轻犯罪后果，委托他人先代为投案，或先以信电投案
- 在投案途中被抓获
- 经亲属规劝、陪同投案
- 公安机关通知犯罪嫌疑人的亲属，或亲属主动报案后，将犯罪嫌疑人送去投案

**如实供述**：
- 供述自己的主要犯罪事实
- 共同犯罪中供述同案犯
- 数罪中供述主要罪行

### 准自首（刑法第67条第2款）

- 被采取强制措施的犯罪嫌疑人、被告人
- 正在服刑的罪犯
- 如实供述司法机关尚未掌握的本人其他罪行
- 该罪行与已掌握罪行属不同种罪行

### 供述变化的处理

- 先供后翻又如实供述：一般不影响自首认定
- 翻供后在一审判决前如实供述：可认定自首
- 翻供后经查证翻供内容属实：不认定自首

## 三、立功认定要点

### 一般立功（刑法第68条）

- 揭发他人犯罪行为，经查证属实
- 提供重要线索，从而得以侦破其他案件
- 阻止他人犯罪活动
- 协助司法机关抓捕其他犯罪嫌疑人
- 具有其他有利于国家和社会的突出表现

### 重大立功

- 犯罪嫌疑人揭发的他人犯罪可能被判处无期徒刑以上刑罚
- 提供的重要线索侦破的案件中被告人可能被判处无期徒刑以上刑罚
- 其他重大立功表现

### 立功线索来源的限制

- 不能是犯罪嫌疑人通过非法手段获取的线索
- 不能是犯罪嫌疑人在羁押期间从同监室人员处获取的线索（有争议）

## 五、酌定从重情节（9项）

| 情节类型 | 说明 |
|---------|-----|
| **犯罪手段特别残忍** | 造成严重后果的残忍手段 |
| **犯罪对象特殊** | 老人、儿童、残疾人等弱势群体 |
| **主犯/首要分子** | 在共同犯罪中起组织、策划、指挥作用 |
| **社会危害特别严重** | 造成重大社会影响或严重后果 |
| **犯罪后态度恶劣** | 拒不认罪、毫无悔意 |

### 酌定从重情节深层评估因素

| 评估维度 | 具体情形 | 从重程度参考 |
|---------|---------|-------------|
| **主观恶性程度** | 预谋周密、蓄意报复、寻衅滋事、恶意明显 | 酌情增加基准刑5%-15% |
| **人身危险性** | 再犯可能性大、缺乏悔改表现、属监管对象期间作案 | 酌情增加基准刑5%-15% |
| **多名主犯区分** | 首要分子（策划组织指挥）→ 主要分子（直接实行）→ 次要分子（参与程度浅） | 首要分子从重幅度最大 |
| **再犯风险** | 曾因同类犯罪受处罚、有在逃经历、前科累累 | 酌情增加基准刑10%-20% |

## 六、缓刑可能性评估

```
【缓刑适用条件核查】
□ 判处三年以下有期徒刑
□ 犯罪情节较轻
□ 有悔罪表现
□ 没有再犯罪的危险
□ 宣告缓刑对所居住社区没有重大不良影响

【禁止适用缓刑情形】
□ 累犯
□ 犯罪集团首要分子
```

## 七、核查表使用说明

1. **核查状态说明**：
   - ✓已确认：有明确证据支持
   - ✗已排除：有明确证据否定
   - ？待核实：需进一步调查或会见确认

2. **使用时机**：
   - 首次会见后进行初步核查
   - 每次会见后更新核查结果
   - 撰写法律文书前最终确认

3. **证据对照**：
   - 每项情节应标注证据来源
   - 区分"当事人陈述"与"客观证据"
