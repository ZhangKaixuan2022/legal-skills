# 不予批准逮捕律师意见书模板

> 本文件为侦查阶段辩护Skill的法律文书模板子文件，来源于主文件 `SKILL.md` 第11.4节。

## 一、文书结构模板

```
XX涉嫌XX罪一案
不予批准逮捕的律师意见书

XX人民检察院
尊敬的承办检察官：

XX因涉嫌XX罪由贵院审查逮捕。受XX近亲属委托并经其本人确认，XX律师事务所指派[律师姓名]律师担任XX的辩护人，为其辩护。接受指派后，辩护人会见了XX，详细听取了其陈述，对本案有了一定的了解。现根据XX的陈述，提出如下辩护意见，供贵院审查。

辩护人总体认为，根据XX的陈述及辩护人从侦查机关了解的基本案情，⋯⋯观点概述⋯⋯XX符合不予批准逮捕的条件。具体意见如下：

一、⋯⋯（证据方面）
二、⋯⋯（定性方面）
三、⋯⋯（社会危险性方面）

综上，辩护人认为，⋯⋯观点概述⋯⋯根据《人民检察院审查逮捕质量标准》第X条、《最高人民检察院、公安部关于逮捕社会危险性条件若干问题的规定（试行）》第X条之规定，恳请贵院对XX不予批准逮捕！

以上意见供贵院参考，望予采纳！

                              辩护人：XX律师事务所
                                      XX律师________
                                      X年X月X日
附：
1. 谅解书；
2. 民事赔（补）偿协议；
3. 其他证明材料。
```

## 二、核心论证要素（三项）

### 第一项：证据方面

| 论证角度 | 具体内容 | 论证目标 |
|---------|---------|---------|
| 证据标准 | 证据是否达到"有证据证明有犯罪事实"的标准 | 证明证据不足 |
| 关键证据瑕疵 | 关键证据是否存在来源不明、程序违法、矛盾等问题 | 证明证据不可采 |
| 证据链完整性 | 证据链是否闭合、是否存在逻辑漏洞 | 证明事实不清 |
| 言词证据 | 是否仅有口供、是否有其他证据印证 | 排除孤证定案 |

### 第二项：定性方面

| 论证角度 | 具体内容 | 论证目标 |
|---------|---------|---------|
| 是否构成犯罪 | 犯罪构成四要件是否齐备 | 证明不构成犯罪 |
| 罪名认定 | 此罪与彼罪的区分、罪名是否准确 | 证明罪名认定错误 |
| 出罪事由 | 是否存在正当防卫、紧急避险等 | 证明行为合法 |
| 情节轻微 | 是否属于情节显著轻微、危害不大 | 证明不认为是犯罪 |

### 第三项：社会危险性方面

| 论证角度 | 具体内容 | 法律依据 |
|---------|---------|---------|
| 五种情形排除 | 逐一排除《刑诉法》第81条规定的五种社会危险性情形 | 《刑诉法》第81条 |
| 取保候审可行性 | 论证取保候审足以防止社会危险性 | 《逮捕社会危险性规定》 |
| 人身危险性评估 | 系初犯、偶犯、主观恶性小、平时表现良好 | 综合评估 |
| 羁押必要性 | 论证不具有逮捕羁押的必要性 | 比例原则 |

## 三、首部段标准表述

**标准首部段**（必须包含以下要素）：
1. 案件来源说明："XX因涉嫌XX罪由贵院审查逮捕"
2. 委托关系："受XX近亲属委托并经其本人确认"
3. 律所指派："XX律师事务所指派[律师姓名]律师担任XX的辩护人"
4. 工作说明："辩护人会见了XX，详细听取了其陈述"
5. 了解程度：**"对本案有了一定的了解"**（非"全面了解"）
6. 意见说明："现根据XX的陈述，提出如下辩护意见，供贵院审查"

## 四、结论与跟进语

**结论措辞**（标准表述）：
```
恳请贵院对XX不予批准逮捕！
```

**跟进语**（紧接结论后）：
```
以上意见供贵院参考，望予采纳！
```

**格式要求**：结论和跟进语均使用宋体四号加粗、下划线，段前段后空1行。

## 五、"辩护人总体认为"开篇格式

**标准格式**：
```
辩护人总体认为，根据XX的陈述及辩护人从侦查机关了解的基本案情，[案件定性意见/证据不足意见/社会危险性意见等核心观点概述]，XX符合不予批准逮捕的条件。具体意见如下：
```

**格式要求**：
- 字体：宋体四号
- 强调："辩护人总体认为"加粗、下划线
- 段落：段前段后间距1行

## 六、法律依据

- 《中华人民共和国刑事诉讼法》第八十一条
- 《人民检察院刑事诉讼规则》
- 《最高人民检察院、公安部关于逮捕社会危险性条件若干问题的规定（试行）》
