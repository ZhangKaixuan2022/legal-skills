# 取保候审申请书模板

> 本文件为侦查阶段辩护Skill的法律文书模板子文件，来源于主文件 `SKILL.md` 第11.3节。

## 一、文书结构模板

```
XX涉嫌XX罪一案
取保候审申请书

申请人：XX律师事务所 XX律师
地址：XXXXXXXXXX
联系电话：XXXXXXXXXX

申请事项：为犯罪嫌疑人XX申请变更强制措施，办理取保候审

事实与理由：

辩护人总体认为，根据XX的陈述及辩护人从侦查机关了解的基本案情，⋯⋯观点概述⋯⋯XX符合取保候审的条件。具体意见如下：

一、⋯⋯（具体理由详述）
（1. 涉案罪刑是否符合取保候审的刑期条件；2. 事实是否已基本查清，证据是否已基本收集固定；3. 嫌疑人社会危险性，是否可能妨碍诉讼以及是否有逃跑可能性；4. 嫌疑人的认罪悔罪态度；5. 赔偿谅解情况；6. 政策及社会效果；7. 其他可以取保候审的情节）

综上，辩护人认为⋯⋯观点概述⋯⋯根据《中华人民共和国刑事诉讼法》第六十七条、第八十一条规定，特申请贵局考虑上述因素，对XX变更强制措施为取保候审。

此致
XX市公安局

                              辩护人：XX律师事务所
                                      XX律师________
                                      X年X月X日
附：
1. 谅解书；
2. 民事赔（补）偿协议；
3. 犯罪嫌疑人悔过书。
```

## 二、核心论证要素

**必备论证要素（七项）**：

| 序号 | 论证要素 | 内容要点 | 重要程度 |
|-----|---------|---------|---------|
| 1 | 犯罪嫌疑人基本情况 | 姓名、年龄、职业、家庭状况、前科情况 | ★★★ |
| 2 | 涉嫌罪名及案件进展 | 涉嫌罪名、拘留/逮捕日期、羁押地点 | ★★★ |
| 3 | 符合取保候审法定条件 | 详见下表 | ★★★★★ |
| 4 | 社会危险性分析 | 不具有《刑诉法》第81条规定的五种情形 | ★★★★★ |
| 5 | 保证方式说明 | 保证人资格/保证金数额及缴纳能力 | ★★★★ |
| 6 | 量刑情节说明 | 法定、酌定从轻减轻情节 | ★★★★ |
| 7 | 结论请求 | 明确请求变更强制措施为取保候审 | ★★★ |

## 三、取保候审法定条件论证框架

| 法定情形 | 论证要点 | 法律依据 |
|---------|---------|---------|
| 可能判处管制、拘役或独立适用附加刑 | 罪行较轻，预判刑期在三年以下 | 《刑诉法》第67条第1款第1项 |
| 可能判处有期徒刑以上刑罚，取保不致发生社会危险 | 有固定住所、稳定职业、家庭牵挂；系初犯、偶犯；认罪悔罪态度好 | 《刑诉法》第67条第1款第2项 |
| 患有严重疾病、生活不能自理 | 提供医疗诊断证明，说明羁押条件不适宜 | 《刑诉法》第67条第1款第3项 |
| 怀孕或哺乳自己婴儿的妇女 | 提供医疗证明或出生证明 | 《刑诉法》第67条第1款第4项 |
| 羁押期限届满，案件尚未办结 | 计算羁押期限，说明已届满情况 | 《刑诉法》第67条第1款第5项 |

## 四、社会危险性否定论证

**必须逐项排除**：
```
【社会危险性分析】
犯罪嫌疑人XX不具有《刑事诉讼法》第八十一条规定的社会危险性情形：
□ 不存在可能实施新的犯罪的情形
□ 不存在危害国家安全、公共安全或社会秩序的现实危险
□ 不存在可能毁灭、伪造证据，干扰证人作证或串供的情形
□ 不存在可能对被害人、举报人、控告人实施打击报复的情形
□ 不存在企图自杀或逃跑的情形
```

## 五、附件清单要求

| 附件类型 | 证明目的 | 是否必须 |
|---------|---------|---------|
| 谅解书 | 证明被害人已谅解 | 有则必附 |
| 民事赔偿协议 | 证明已赔偿损失 | 有则必附 |
| 犯罪嫌疑人悔过书 | 证明认罪悔罪态度 | 建议附上 |
| 保证人身份证明及资产证明 | 证明保证人资格 | 采用保证人方式时必附 |
| 保证金缴纳能力证明 | 证明有能力缴纳保证金 | 采用保证金方式时建议附 |
| 固定住所证明 | 证明不会逃跑 | 建议附上 |
| 疾病诊断证明 | 证明身体不适宜羁押 | 以此为由时必附 |

## 六、法律依据

- 《中华人民共和国刑事诉讼法》第六十七条
- 《中华人民共和国刑事诉讼法》第八十一条
- 《中华人民共和国刑事诉讼法》第九十七条
