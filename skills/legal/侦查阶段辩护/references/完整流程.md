# 刑事辩护侦查阶段工作Skill

## 一、功能概述

> 本Skill亦可由刑事辩护总调度Skill路由进入：skills/legal/刑事辩护总调度/SKILL.md

本Skill为刑事辩护侦查阶段提供完整的工作流程指导，涵盖从案件承接后进入侦查阶段到移送审查起诉的全部辩护工作。核心功能包括：

1. **会见全流程管理**：首次会见七步法、分次会见差异化内容、会见笔录生成、会见情况反馈函
2. **案件法律分析与量刑预判**：五步递进法分析、量刑情节核查、辩护策略建议
3. **批捕时间节点管理**：智能计算批捕截止日期、最佳申请时机提示
4. **法律文书生成**：六类标准化法律文书（取保候审申请书、不予批准逮捕律师意见书等）
5. **执业风险全程提示**：刑事责任风险、会见纪律风险、保密义务风险、程序合规风险
6. **认罪认罚从宽嵌入**：制度告知、利弊评估、从宽幅度分析

4. 委托人信息（姓名、与当事人关系、联系方式）

## 三、刑事辩护十大基本原则（贯穿全程）

侦查阶段辩护工作必须始终遵循以下十大基本原则：

| 原则 | 核心要义 |
|-----|---------|
| **原则一：忠诚原则** | 在法律允许范围内最大限度维护犯罪嫌疑人合法权益 |
| **原则二：保密原则** | 对执业所获悉的委托人信息有权保密，但涉及重大安全犯罪除外 |
| **原则三：独立原则** | 依法独立发表辩护意见，不受非法干涉 |
| **原则四：依法原则** | 在法律规定范围内维护权益，不帮助逃避法律制裁 |
| **原则五：诚信原则** | 不作虚假承诺，不收取"关系费""活动费" |
| **原则六：勤勉原则** | 及时会见、及时沟通、及时提交辩护意见 |
| **原则七：尊重原则** | 对司法机关及工作人员保持理性、平和、专业态度 |
| **原则八：回避原则** | 发现利益冲突或影响公正辩护的情形应主动回避 |
| **原则九：规范原则** | 依照执业规范制作法律文书，建立完整工作档案 |
| **原则十：风险防范原则** | 增强执业风险意识，防范刑法第306条等刑事风险 |

## 五、会见全流程

### 5.1 会见手续准备

**三证材料**（各准备三份）：
1. **律师执业证**（复印件）
2. **律师事务所介绍信**（律师会见在押犯罪嫌疑人专用介绍信）
3. **委托书或法律援助公函**

**提交对象**：看守所

**会见审批规则**：

| 案件类型 | 审批要求 | 法律依据 |
|---------|---------|---------|
| 普通案件 | 无须许可，持三证直接会见，看守所应在48小时内安排 | 《刑事诉讼法》第39条 |
| 危害国家安全犯罪 | 需侦查机关许可 | 《刑事诉讼法》第39条 |
| 恐怖活动犯罪 | 需侦查机关许可 | 《刑事诉讼法》第39条 |
| 特别重大贿赂犯罪 | 需侦查机关许可 | 《刑事诉讼法》第39条 |

### 5.2 首次会见（七步法）

| 步骤 | 内容要点 |
|-----|---------|
| **第一步：说明来意** | 介绍律师身份、委托来源，消除戒备心理，建立信任关系 |
| **第二步：告知监听风险** | 明确告知可能存在监控风险，提醒谨慎发言 |
| **第三步：提供法律咨询** | 实体法（犯罪构成、量刑幅度）、程序法（期限、权利） |
| **第四步：了解讯问笔录情况** | 问"笔录是怎么做的"而非"案件真实情况" |
| **第五步：全面法律分析** | 根据笔录情况结合法律规定进行分析，核对辩解是否已记录 |
| **第六步：提出法律服务方案** | 告知律师、家属、当事人各应做的工作 |
| **第七步：解决其他问题** | 转达问候、了解生活状况、心理疏导 |

**侦查阶段第一次会见提纲**：

| 序号 | 会见事项 | 具体内容 |
|-----|---------|---------|
| 1 | 介绍辩护人情况及委托关系 | 说明律师身份、委托来源 |
| 2 | 确认关系、转达问候 | 确认与委托人关系，转达近亲属问候 |
| 3 | 了解到案经过、羁押情况 | 告知诉讼权利义务及保障措施 |
| 4 | 了解基本情况 | 身份信息、强制措施、涉嫌罪名、案情及讯问情况 |
| 5 | 了解案件证据收集情况 | 讯问次数、笔录内容、有无辩解未被记录 |
| 6 | 了解侦查程序合法性 | 是否存在刑讯逼供等非法取证 |
| 7 | 了解从轻减轻情节 | 有无自首、立功、从犯等情节 |
| 8 | 沟通辩护方案 | 与犯罪嫌疑人商量确定辩护方向 |
| 9 | 签名确认 | 阅读会见笔录，签名捺手印确认 |

### 5.3 分次会见差异化内容

#### 5.3.1 批捕前会见

**会见目的**：
1. 核实拘留原因和案件进展
2. 了解是否有新的讯问及笔录情况
3. 为《不予提请批准逮捕辩护意见》收集材料
4. 辅导检察官提审时的注意事项

**会见提纲要点**：
| 序号 | 会见事项 | 具体内容 |
|-----|---------|---------|
| 1 | 关怀、告知家庭情况 | 转达家属问候，了解当事人近况 |
| 2 | 回顾上次会见情况 | 核实辩护工作进展，了解需要补充的情况 |
| 3 | 告知案件进展 | 说明目前案件所处阶段和程序走向 |
| 4 | 告知批准逮捕的程序 | 解释批捕审查程序、时限、审查标准 |
| 5 | 逐条核对逮捕条件 | 详见下方核对框架 |
| 6 | 预先告知检察官提审注意事项 | 辅导如何面对检察官提审 |

**批准逮捕条件分析框架**（重点核对）：
```
【逮捕条件分析】
一、是否有证据证明有犯罪事实
□ 有证据证明发生了犯罪事实
□ 有证据证明犯罪事实是犯罪嫌疑人实施的
□ 证据需要进一步侦查

二、是否可能判处有期徒刑以上刑罚
□ 涉嫌罪名法定刑包含有期徒刑以上
□ 犯罪情节是否可能判处有期徒刑

三、是否具有社会危险性
□ 可能实施新的犯罪
□ 有危害国家安全、公共安全或者社会秩序的现实危险
□ 可能毁灭、伪造证据，干扰证人作证或者串供
□ 可能对被害人、举报人、控告人实施打击报复
□ 企图自杀或者逃跑
```

#### 5.3.2 批捕后会见

**会见目的**：告知批准逮捕决定、详细了解案情、复盘批捕过程、评估羁押必要性审查、准备后续辩护材料

**复盘分析要点**：分析侦查机关认定的犯罪事实可能存在的误差，评估哪些无罪和罪轻的证据可能被遗漏

#### 5.3.3 侦查终结前会见

**会见目的**：核实全部供述内容、查缺补漏、为审查起诉阶段做准备

## 六、案件法律分析与量刑预判（五步递进法）

### 6.1 第一步：罪名分析

**犯罪构成四要件逐项比对**：

| 要件 | 分析内容 | 核查要点 |
|-----|---------|---------|
| 犯罪主体 | 一般/特殊主体、刑事责任年龄、刑事责任能力 | 年龄、精神状态、特殊身份 |
| 犯罪主观方面 | 故意/过失、目的、动机 | 故意类型、认识错误 |
| 犯罪客体 | 侵犯的社会关系 | 具体侵犯的法益 |
| 犯罪客观方面 | 危害行为、危害结果、因果关系 | 行为方式、结果程度、因果关系成立与否 |

### 6.2 第二步：法律法规检索与适用

**检索范围**：刑法条款、司法解释、地方量刑指导意见、指导案例、立案追诉标准

### 6.3 第三步：量刑档次预判

**基准刑确定方法**：确定法定刑幅度 → 根据犯罪事实确定量刑起点 → 根据数额、情节确定基准刑

### 6.4 第四步：法定量刑情节逐项核查

> **按需参考**：进行量刑情节核查时，请读取 `reference/量刑情节完整清单.md` 获取法定15项+酌定18项+从重9项的完整情节清单及认定要点。

**核查要点概览**：
- 法定从轻减轻情节（15项）：自首、坦白、立功、从犯、未成年人等
- 酌定从轻情节（18项）：被害人过错、认罪悔罪、初犯偶犯、赔偿谅解等
- 酌定从重情节（9项）：手段残忍、犯罪对象特殊、主犯等

### 6.5 第五步：综合预判

**缓刑可能性评估**：判处三年以下、情节较轻、有悔罪表现、无再犯危险、对社区无重大不良影响

**存放路径**：`<当前事项业务文件路径>/02-侦查阶段/工作底稿/`

**内容结构**：案件基本信息、涉嫌罪名及法律依据、犯罪构成分析、证据情况分析、量刑档次预判、法定情节核查结果、酌定情节分析、综合量刑预判（区间）、辩护策略建议、风险提示

**可嵌入内容**：案件方向概括性说明、有利情节类型概述、下一步工作计划

## 八、批捕时间节点智能提示

### 8.1 批捕时间计算规则

| 案件类型 | 刑拘期限 | 提请批捕期限 | 最长刑拘期限 |
|---------|---------|-------------|-------------|
| 普通案件 | 3日内（特殊延长1-4日） | 拘留后14日内 | 7日 |
| 特殊案件（流窜/多次/结伙） | 可延长至30日 | 审查批捕7日 | 37日 |

### 8.2 智能提示机制

**输入信息**：拘留日期、案件类型

**自动计算输出**：
```
【批捕时间节点计算】
案件类型：普通案件
拘留日期：XXXX年XX月XX日
提请批捕截止：XXXX年XX月XX日（拘留后14日）
检察院审查截止：XXXX年XX月XX日（再加7日）

【最佳申请时机提示】
建议在XXXX年XX月XX日前（提请批捕前3-5日）提交《不予批准逮捕律师意见书》
```

### 8.3 提示触发

- 拘留后第7日提示：距离普通案件提请批捕截止还有7日
- 拘留后第25日提示：距离特殊案件提请批捕截止还有5日
- 提请批捕前5日提示：建议提交不予批捕意见书

## 九、不予批捕条件细化

### 9.1 不予批捕的法定情形

**一、无逮捕必要性**：取保候审、监视居住足以防止社会危险性

**二、证据不足**：没有证据证明有犯罪事实或犯罪嫌疑人实施

**三、不构成犯罪**：不符合犯罪构成要件、存在违法阻却事由、情节显著轻微

**四、犯罪情节轻微**：可能判处管制、拘役或独立适用附加刑、可能免予刑事处罚

### 9.2 不予批捕的常见理由

| 类别 | 具体理由 |
|-----|---------|
| 证据方面 | 仅有口供无其他印证、关键证据存在瑕疵、证据链不完整 |
| 主体方面 | 未达刑事责任年龄、精神病人（待鉴定）、主体身份不符 |
| 主观方面 | 故意/过失难以认定、存在认识错误 |
| 客观方面 | 因果关系不成立、行为方式不符合构成要件 |
| 社会危险性方面 | 有固定住所和工作、系初犯偶犯、已取得谅解、已全额退赔 |

## 十、自首/立功认定条件细化

### 10.1 自首的认定

**一般自首**（刑法第67条第1款）：
- **自动投案**：未受讯问、未被强制措施时主动投案
- **如实供述**：供述自己的主要犯罪事实

**准自首**（刑法第67条第2款）：被采取强制措施后，如实供述司法机关尚未掌握的本人其他罪行

**供述变化处理**：翻供后又如实供述一般不影响自首认定；翻供内容经查证属实则不认定自首

### 10.2 立功的认定

**一般立功**：揭发他人犯罪（经查证属实）、提供重要线索、阻止他人犯罪、协助抓捕其他嫌疑人

**重大立功**：揭发或提供线索侦破的案件可能被判处无期徒刑以上

> **按需参考**：自首/立功认定的详细情形和判断标准，请参见 `reference/量刑情节完整清单.md` 第二、三节。

## 十二、执业风险全程提示

### 12.1 四类风险分类概要

| 风险类型 | 核心风险点 |
|---------|-----------|
| **刑事责任风险** | 刑法第306条：毁灭伪造证据、妨害作证 |
| **会见纪律风险** | 传递信件物品、录音录像、让嫌疑人使用通讯工具 |
| **保密义务风险** | 向家属透露案情、向无关人员透露信息、网上公开案件 |
| **程序合规风险** | 会见手续不全、文书格式不规范、档案不完整 |

### 12.2 禁止事项清单（10条红线）

1. **禁止**向犯罪嫌疑人传递信件、物品
2. **禁止**为犯罪嫌疑人与外界传递串供信息
3. **禁止**帮助犯罪嫌疑人毁灭、伪造证据
4. **禁止**威胁、引诱证人改变证言
5. **禁止**诱导、唆使犯罪嫌疑人作虚假陈述
6. **禁止**向无关人员透露案件秘密
7. **禁止**向家属透露可能影响诉讼的案件信息
8. **禁止**收取"关系费""活动费"或承诺案件结果
9. **禁止**向办案人员请客送礼或行贿
10. **禁止**在会见时录音录像（未经允许）或让犯罪嫌疑人使用通讯工具

### 12.3 必须事项清单（10条底线）

1. **必须**携带完整的三证材料会见
2. **必须**制作规范的会见笔录并由犯罪嫌疑人签字确认
3. **必须**告知犯罪嫌疑人诉讼权利义务
4. **必须**如实向委托人反馈工作进展（脱敏处理）
5. **必须**及时向侦查机关提交委托手续
6. **必须**在法定期限内提交相关申请和意见
7. **必须**对辩护工作全程留痕、建立完整档案
8. **必须**独立、客观、依法进行辩护
9. **必须**保守案件秘密和当事人隐私
10. **必须**在发现非法取证时及时申请排除

> **按需参考**：查看详细的风险场景分析和自查表，请读取 `checklists/执业风险禁止清单.md`。

## 十三、当事人和解（侦查阶段工作提示）

### 13.1 和解启动时机

**越早越好**：刑事和解贯穿诉讼全程，侦查阶段和解对当事人最为有利。侦查阶段达成和解，公安机关会直接建议检察院从轻认定，有利于后续争取不起诉或从宽处罚。

### 13.2 和解案件范围

| 案件类型 | 适用条件 | 法律依据 |
|---------|---------|---------|
| 民间纠纷引起的侵犯人身/财产犯罪 | 刑法分则第四、五章，可能判处三年以下 | 《刑事诉讼法》第277条 |
| 过失犯罪 | 七年以下刑罚（渎职犯罪除外） | 《刑事诉讼法》第277条 |
| **排除情形** | 五年内曾故意犯罪的不适用 | 《刑事诉讼法》第277条 |

**重罪案件和解**：故意伤害致人重伤、民间纠纷引起的故意杀人等案件，虽不在法定和解范围内，但和解仍可获得酌定从轻情节。

### 13.3 侦查阶段律师和解职责

1. **评估和解可能性**：了解案件类型是否适合和解、被害人意愿
2. **建议当事人态度**：告知和解利弊、不同阶段和解的法律后果差异
3. **协调沟通**：代表当事人与被害方初步沟通和解意向
4. **法律咨询**：为当事人计算合理赔偿金额

> **重要提示**：侦查阶段和解工作主要是评估和铺垫，正式和解协议通常在审查起诉或审判阶段签署。

## 十四、认罪认罚从宽（侦查阶段职责嵌入）

### 14.1 认罪认罚的认定条件

**认罪**：承认指控的犯罪事实、对行为性质没有异议
**认罚**：愿意接受处罚、同意量刑建议、签署认罪认罚具结书

### 14.2 从宽幅度

| 阶段 | 从宽幅度 |
|-----|---------|
| 侦查阶段认罪认罚 | 可减少基准刑30%以下 |
| 审查起诉阶段认罪认罚 | 可减少基准刑20%以下 |
| 审判阶段认罪认罚 | 可减少基准刑10%以下 |

### 14.3 律师职责

1. **审查证据充分性**：审查在案证据是否确实充分
2. **告知法律后果**：认罪认罚的法律含义、从宽幅度、上诉权利
3. **权衡利益得失**：帮助当事人作出理性决策
4. **争取最大从宽**：与检察机关协商量刑建议
5. **防止被诱导、胁迫**：确保认罪认罚系自愿
